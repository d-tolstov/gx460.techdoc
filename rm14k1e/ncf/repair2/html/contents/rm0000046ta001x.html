GLASS HATCH OPENER SYSTEM > DETAILS



FUNCTION OF MAIN COMPONENTS

  1. Glass Hatch Opener System

    Component
    		Function
    Glass Hatch Opener Switch
    		By pressing this switch, the back door glass is opened.
    Transmitter Glass Hatch Opener Button
    		By pressing this button, the back door glass is opened.
    Main Body ECU
    		Receives a signal from each switch and ECU and sends the glass hatch open request signal to the glass hatch opener relay.
    Door Control Receiver
    		Receives the glass hatch open request signal from the transmitter.


OPERATING CONDITION

  1. The glass hatch opener system activates when all of the following conditions are met:

    1. The back door is closed.
    2. Vehicle speed is less than 5 km/h (3 mph).
    3. A glass hatch open request signal is input into the main body ECU.


FUNCTION
