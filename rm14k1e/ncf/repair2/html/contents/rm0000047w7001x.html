FUNCTION OF MAIN COMPONENTS

Component
Function
Display and Navigation Module Display*1
Receives image signals transmitted from the parking assist ECU and displays them.
Accessory Meter Assembly*2
Receives image signals transmitted from the parking assist ECU and displays them.
Driving Support Switch Control ECU
  1. Transmits signals from the multi-information switch to the parking assist ECU.
  2. Transmits signals from the parking assist ECU to the accessory meter assembly.*2
Television Camera Assemblies
Outputs the images received by television camera assemblies installed on the front grille, outer rear view mirror and back door to the parking assist ECU.
Parking Assist ECU
  1. Turns the multi-terrain monitor system on/off in accordance with the signals input from the ECUs and switches.
  2. Outputs visual signals from the television camera assemblies to the display and navigation module display*1 or the accessory meter assembly*2.
Combination Meter Assembly
Illuminates the multi-terrain indicator light while the multi-terrain select is on.
Multi-information Switch (Steering Pad Switch Assembly)
  1. The multi-terrain select can be operated by operating this switch.
  2. The multi-terrain monitor screen can be switched by operating this switch while multi-terrain select is operating.
Wide View Front and Side Monitor Switch
Switches between the screen of the multi-terrain monitor and wide view front and side monitor system.
ECM
Outputs shift position signals to the parking assist ECU.
Park/Neutral Position Switch Assembly
Transmits shift position signals to the ECM.
Skid Control ECU
Transmits speed sensor signal to the parking assist ECU.
  1. *1: Models with navigation system
  2. *2: Models without navigation system

OPERATING CONDITION

  1. The multi-terrain monitor system will display under the following conditions:

    1. Engine switch is on (IG) or on (ACC).

    2. Multi-terrain select is on.

    3. Outer rear view monitor is opened.

    4. Vehicle speed is 10 km/h or less.

    5. Multi-terrain monitor system is on.

  2. Checking Traveling Direction and Avoiding Obstacles



    1. In order to avoid obstacles while traveling forward, the driver should steer in such a way that the wheel path does not overlap with any obstacles. By doing so, obstacles can be avoided and passed.



      HINT:
      1. As it is difficult to understand the height differences of obstacles using the images received from the cameras, care should be taken of large differences in height.
      2. As it is difficult to perceive distances using the images of the wide view front camera assembly, distances to items displayed should be confirmed directly.


FAIL-SAFE

  1. The table below indicates the conditions of detecting malfunctions in the components of this system:

    Malfunction Parts
    		Detection Item
    		Function
    Steering Angle Sensor
    1. Transmission of sensor malfunction signal
    2. Open circuit in sensor signal
    3. Communication malfunction between the steering angle sensor and parking assist ECU
    		Displays "Guidance Unavailable"
    Television Camera Assemblies
    		Transmission of television camera malfunction signal
    		Stops signal reception and displays a dark screen
    Parking Assist ECU
    		Malfunction of parking assist ECU
    		Stops system operation


DIAGNOSIS

  1. The multi-terrain monitor system is equipped with a diagnosis function which can display the service menus. The method for entering the service menu screen is the same as the method used for the multi display. For details, refer to the corresponding Repair Manual for this model.
