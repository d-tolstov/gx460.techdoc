FEATURES > PERFORMANCE



OUTLINE

  1. 1UR-FE Engine

    1. The 1UR-FE engine is a V-type, 8-cylinder, 4.6-liter, 32-valve DOHC engine.

    2. On this engine, the Dual Variable Valve Timing-intelligent (Dual VVT-i) system, Electronic Throttle Control System-intelligent (ETCS-i), air injection system and Exhaust Gas Recirculation (EGR) control are used. These control functions have been optimized to further improve engine performance and fuel economy, and to reduce exhaust emissions.

      Engine Type
      		1UR-FE
      Item
      		No. of Cylinders and Arrangement
      		8-cylinder, V-type
      Valve Mechanism
      		32-valve DOHC, Chain Drive (with Dual VVT-i)
      Displacement
      		4608 cm3 (281.2 cu. in.)
      Maximum Output [SAE-NET]
      		225 kW @ 5600 rpm
      Maximum Torque [SAE-NET]
      		446 N*m @ 3400 rpm

  2. A760F Automatic Transmission

    1. A760F 6-speed automatic transmissions is used.

  3. Suspension

    1. A double-wishbone independent suspension is used for the front.

    2. A 4-link coil spring with lateral rod type suspension is used for the rear.

    3. A rear air suspension system is used.

    4. An Adaptive Variable Suspension System (AVS) is used.

    5. A Kinetic Dynamic Suspension System (KDSS) is used.

  4. Brake

    1. A ventilated disc brake is provided for the front brake.

    2. A drum in type ventilated disc brake is provided for the rear brake.

    3. A duo-servo drum brake is provided for the parking brake.

    4. A brake control system equipped with an Anti-lock Brake System (ABS), Multi-terrain Anti-lock Brake System (Multi-terrain ABS), Electronic Brake force Distribution (EBD), brake assist, Traction Control (TRC), Active Traction Control (A-TRC), Vehicle Stability Control (VSC), hill-start assist control, downhill assist control and CRAWL is provided.

  5. Steering

    1. A rack and pinion type steering gear with a hydraulic power steering assist is used.

    2. A power tilt and power telescopic system is used.

    3. A heated steering wheel system is used.

  6. Frame Structure

    1. A strong frame structure is used to achieve a high level of reliability as a genuine SUV.
