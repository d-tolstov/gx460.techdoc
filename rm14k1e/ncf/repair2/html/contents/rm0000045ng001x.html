POWER WINDOW CONTROL SYSTEM > DETAILS



SYSTEM CONTROL

  1. The power window control system has the following functions:

    Function
    		Outline
    		Customization
    Manual Up-and-down
    		The manual up-and-down function causes the window to open or close while the multiplex network master switch assembly or power window regulator switch assembly is being pulled halfway up or pushed halfway down. The window stops as soon as the switch is released.
    		Standard (Not Customizable)
    One-touch Auto Up-and-down
    		The one-touch auto up-and-down function enables the window to be fully opened or closed by firmly operating of the multiplex network master switch assembly or power window regulator switch assembly.
    		Default Setting is On*
    Variable Speed Control
    		Up-and-down movement speed of the door windows is controlled using variable control to reduce operation noise.
    		Standard (Not Customizable)
    Jam Protection
    		The jam protection function automatically stops the power window and moves it downward if a foreign object gets jammed in the window during one-touch auto-up operation or manual up operation.
    		Standard (Not Customizable)
    Remote Control
    		The multiplex network master switch assembly can control the up-and-down operation of all door windows.
    		Default Setting is On*
    Window Lock
    		Operation of the passenger windows is disabled when the window lock switch is pressed.
    		Standard (Not Customizable)
    Key Off Operation
    		The key off operation function makes it possible to operate the power windows for approximately 45 seconds after the engine switch is turned off, if either front door is not opened.
    		Standard (Not Customizable)
    Mechanical Key-linked Up-and-down
    1. When the driver door is locked, and the key cylinder in the driver door is turned and maintained in the lock direction for 2 seconds or more, the main body ECU activates the power window motors to raise all door windows while the mechanical key is turned.
    2. When the driver door is unlocked, turning and maintaining the driver door key cylinder in the unlock direction for 2 seconds or more will cause all door windows to be lowered.
    		Default Setting is Off *
    Transmitter-linked Up-and-down
    1. When the door control receives an unlock signal from the transmitter for longer than 3 seconds, the main body ECU controls the power window motors in accordance with the signal to open the windows.
    2. When the door control receives a lock signal from the transmitter for longer than 3 seconds, the main body ECU controls the power window motors in accordance with the signal to close the windows.
    		Default Setting is Off *

    HINT:
    *: Customizable parameters can be configured by means of the customized body electronics system. For details, refer to the corresponding Repair Manual for this model.


    1. The ECU built into each power window motor stores the initial position of its door window. The initial position is not cleared even if battery terminals, fuses or the power window motor connector is disconnected. However, after removing or reinstalling window components, or replacing the power window regulator sub-assembly or power window regulator motor, the stored initial position data must be cleared and initialization must be performed. For details, refer to the corresponding Repair Manual for this model.



  2. Jam Protection Function

    1. The jam protection function automatically stops the power window and moves it downward if a foreign object gets jammed in the door window during one-touch auto up operation or manual up operation.

    2. The operation of the jam protection function is described below:

      Door Window Position from Fully Closed
      		Operation
      200 mm (7.87 in.) or more
      		Down operation of 125 mm (4.93 in.) or 2.5 seconds
      200 mm (7.87 in.) or less
      		Down operation until door window downward movement of 200 mm (7.87 in.) is reached, or until 5 seconds have elapsed



    3. The magnet and 2 Hall ICs in the power window motor assembly are used to enable power window jam protection.




      Text in Illustration 
      *1
      		Hall IC
      		*2
      		Magnet
      *3
      		Worm Gear
      		*4
      		ECU

    4. The Hall ICs convert the changes in the magnetic flux that occur through the rotation of the magnet into pulse signals and output them to the ECU built into the power window motor assembly.



    5. To control the jam protection function, the ECU determines the amount of movement and jamming of the window glass based on the pulse signals from the Hall IC1, and judges the direction of the window glass movement based on the phase difference between the pulses from the Hall IC1 and Hall IC2.






FAIL-SAFE

  1. If a Hall IC in the power window motor malfunctions, some power window functions will be prohibited for the fail-safe mode:

    1. Each power window operates when the corresponding power window switch is fully pushed down or pulled up and held in that position.

    2. When any power window ECU detects an abnormality in a Hall IC, which detects the position, speed and direction of the window, it switches to fail-safe mode.
