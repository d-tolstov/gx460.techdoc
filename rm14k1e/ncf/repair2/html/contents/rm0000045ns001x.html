MIRROR SYSTEM > DETAILS



OPERATING CONDITION

  1. Reverse-linked Function

    1. This function operates when all of the following conditions are met:

      1. Engine switch is on (IG).
      2. Either the R or L position of the mirror select switch has been selected.
      3. Shift lever is moved to R.

  2. Mirror Heater Function

    1. The mirror heaters are activated when the engine switch is turned on (IG), and the defogger switch is pushed. The air conditioning amplifier assembly keeps the mirror heaters on for approximately 15 minutes.

  3. Memory System Function

    1. This function operates when any of the following conditions are met:

      1. When the seat memory switch is operated with the engine switch turned on (IG), the shift lever moved to P and the mirror position stored in the memory system.
      2. When the seat memory switch is operated within approximately 60 seconds after closing the driver side door, with the engine switch turned off, the shift lever moved to P and the mirror position stored in the memory system.
      3. When the seat memory switch is operated within approximately 180 seconds after opening the driver side door with the engine switch turned off, the shift lever moved to P and the mirror position stored in the memory system.
      4. With the engine switch turned off, driver side door closed and the key ID registration of the memory call function complete, the doors are unlocked using the smart access function or wireless door unlock function and then the driver side door is opened.

  4. Automatic Glare-resistant EC Dimming Function

    1. This function operates when both of the following conditions are met:

      1. When the AUTO switch of the inside rear view mirror is pushed to select AUTO mode (When the engine switch is turned from off to on (IG), this system defaults to AUTO mode).
      2. When the inside rear view mirror detects a large difference between brightness of the surroundings and the headlights of a following car.


FUNCTION

  1. The outer rear view mirrors have the following functions:

    Function
    		Outline
    Electric Remote Mirror
    		Remote Control Function
    		The adjustment axis of the mirror surface of the electric remote mirrors can be controlled by using the outer mirror switch assembly.
    Manual Retractable Function
    		The outer mirror switch assembly can be used to retract the outer rear view mirrors.
    Reverse-linked Function
    		This function tilts the mirror surface downward to improve visibility when the shift lever is in R.
    Mirror Heater Function
    1. The mirror heaters use a heater element in both outer rear view mirrors to help defog them.
    2. The mirror heaters are controlled by the air conditioning amplifier assembly.
    Memory System
    		The memory system can store the driver seat, outer rear view mirror positions and tilt and telescopic steering positions. Up to 2 memory positions can be stored in the system memory and recalled.
    Automatic Glare-resistant EC Dimming Function
    		During nighttime driving, if a large difference in intensity exists between the surrounding light and the light reflected off the inside rear view mirror from the headlights behind, the automatic glare-resistant Electrochromic (EC) mirror automatically reduces the reflection rate of the inside or outside rear view mirrors and thus dampens the glare from the mirror.
