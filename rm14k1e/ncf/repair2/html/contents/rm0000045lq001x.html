POWER DOOR LOCK CONTROL SYSTEM > DETAILS



FUNCTION

  1. The power door lock control system has the following functions:

    Function
    		Outline
    Manual Lock and Unlock
    		This function can lock or unlock all doors by operation of the door control switch.
    Manual Unlock Prohibition
    		Performing the door lock operation with a transmitter or a key will prohibit the doors from being unlocked by the door lock control switch.
    Key-linked Lock and Unlock
    		This function, linked with the driver door key cylinder, can lock or unlock all the doors when a lock or unlock operation is effected.
    Key Confinement Prevention
    		When the key is inserted in the interior actuation area and the door lock operation is performed while the driver door is open, all the doors will be unlocked.
    2-step Unlock
    		When the mechanical key is inserted into the door lock cylinder and is turned once, the driver door is unlocked. Then, when the key is turned again, the rest of the doors are unlocked.
    Shift-linked Automatic Door Lock
    		When the conditions listed below are met, this function causes all the doors to be automatically locked:
    1. The engine is running.
    2. All of the doors are closed.
    3. The shift lever is moved to any position other than P.
    4. Any of the doors are unlocked.
    Shift-linked Automatic Door Unlock
    		When the engine switch is on (IG) and the shift lever is moved to P from any position, all of the doors will be automatically unlocked.
    Collision Door Lock Release
    		If the SRS airbag system has been deployed, and the main body ECU has detected the signal that is output by the center airbag sensor assembly, the main body ECU unlocks all the doors.
    Speed Sensitive Automatic Door Lock
    		After the engine switch has been turned to on (IG) with any of the doors unlocked, this function causes all doors to be automatically locked once the vehicle speed reaches approximately 20 km/h (13 mph) or more. After this function has operated, it will not operate again until either of the following conditions is met:
    1. Any of the doors are opened after the vehicle has stopped.
    2. The engine switch is turned off after the vehicle has stopped.
    Opening Driver's Door Linked Automatic Door Unlock
    		All doors are unlocked automatically when the driver's door is opened within 10 seconds after the engine switch is turned from on (IG) to on (ACC) or off.
    Customization
    		The customize setting of some functions can be turned on or off. For details, refer to the corresponding Repair Manual for this model.
