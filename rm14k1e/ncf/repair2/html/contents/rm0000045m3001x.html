ENTRY AND START SYSTEM > GENERAL



OUTLINE

  1. The entry and start system has the following main functions, making it a very user-friendly system:

    Function
    		Outline
    Start Function
    		The start function is operated by simply pressing the engine switch while carrying the key. When the engine switch is pressed, the power management control ECU turns on/off the ACC relay, IG1 relay, and IG2 relay to switch the power source pattern.
    1. The power source mode changes depending on depression of the brake pedal and the shift lever position.
    Start Function
    		Engine Immobiliser
    		The engine immobiliser function compares the ID code registered in the ID code box with the ID code of the transponder chip embedded in the key.
    Steering Lock
    		The steering lock function is operated to lock and unlock the steering wheel by using a lock/unlock motor in steering column assembly.
    Entry Function
    		Entry Unlock:
    1. All doors are unlocked by grabbing the driver or front passenger outside door handle or by pressing the unlock switch of the back door.
    Entry Lock:
    1. All doors are locked by touching the lock sensor of the driver or front passenger outside door handle or pressing the lock switch of the back door.
    Entry Function
    		Glass Hatch Opener
    		The glass hatch is opened by pressing the glass hatch opener switch.
    Wireless Door Lock Control
    		The wireless door lock control function is a convenient feature to lock or unlock all doors, or to open the glass hatch at a distance. The basic operation is the same as that of the wireless door lock control system.
