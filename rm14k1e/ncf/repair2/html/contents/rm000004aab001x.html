BRAKE CONTROL SYSTEM > GENERAL



OUTLINE

  1. The following brake control functions are provided: Anti-lock Brake System (ABS), Multi-terrain Anti-lock Brake System (Multi-terrain ABS), Electronic Brake force Distribution (EBD), brake assist, Traction Control (TRC), Active Traction Control (A-TRC), Vehicle Stability Control (VSC), hill-start assist control, downhill assist control and CRAWL.

  2. Multi-terrain select is used.

  3. A VSC OFF switch is used.


MAIN FEATURES

  1. Multi-terrain Select

    1. The multi-terrain select provides operation assistance and prompts the driver to switch to the transfer drive mode most appropriate for the road mode selected by the driver's switch operation.

    2. When multi-terrain select control is active, A-TRC brake control and the engine output control most appropriate for the selected road mode are actuated automatically. In this way, superior off-road driveability is achieved.


PRECAUTION

  1. When the brake control system is activated, the brake pedal may shudder. This is a normal occurrence of the system in operation and should not be considered a malfunction.

  2. The TRC and VSC functions control brake hydraulic pressure and engine output to ensure vehicle stability. Therefore, operate the VSC OFF switch (stop the operation of the TRC and VSC functions) only when it is necessary.

  3. If the A-TRC, hill-start assist control, downhill assist control and CRAWL functions are used continuously for a long time, the temperature of the actuator in the hydraulic brake booster may rise, possibly causing the operation of each function to halt temporarily. In this event, the indicator light will illuminate, and the driver will be informed by a display shown on the multi-information display and by the sounding of the buzzer in the combination meter assembly.
