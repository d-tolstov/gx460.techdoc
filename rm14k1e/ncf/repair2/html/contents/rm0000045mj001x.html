THEFT DETERRENT > DETAILS



FUNCTION

  1. The theft deterrent system has the following functions:

    Function
    		Outline
    Alarm
    		When the theft deterrent system enters the alarm state, the warning items (hazard warning lights, vehicle horn and security horn) are operated for approx. 27.5 seconds.
    Security Indicator
    1. When the state of the theft deterrent system enters the armed preparation or alarm state, the security indicator light is illuminated to inform the user.
    2. This indicator is also the engine immobiliser indicator, and it blinks when the engine immobiliser is in an engaged state.
    Forced Door Lock
    		The main body ECU transmits a door lock signal to all the doors when all of the following conditions are satisfied:
    1. The theft deterrent system is in the alarm state.
    2. One of the doors has been unlocked.
    The function stops when one of the following conditions is met:
    1. All doors are locked.
    2. The warning ends.


OPERATION


  1. The states of the theft deterrent system are as follows:

    State
    		Description
    		Theft Detection
    Disarmed State A (Engine Hood or Glass Hatch is Closed)
    		The theft deterrent system has not been set by the user.
    		Not possible
    Disarmed State B (Engine Hood or Glass Hatch is Open)
    Armed Preparation State
    		This is the preparation state before the theft deterrent system becomes active (enters the armed state) when the system has been set by the user.
    		Not possible
    Armed State
    		The theft deterrent system is active. (Theft can be detected.)
    		Possible
    Alarm State
    		A theft attempt has been detected and the warning operation activates.
    		Possible



  2. The theft deterrent system activates as shown in the diagram above when one of the items in the chart occurs in order to cause the system to enter the respective state.

    Condition
    		Item
    (1)
    		When the engine switch is off, disarmed state B will be switched to if either of the following conditions are met:
    1. All doors are locked using either the entry and start system while all doors are closed and either the engine hood or glass hatch is open.
    (2)
    		Disarmed state A will be switched to if any of the following conditions are met:
    1. When one of the doors is unlocked.
    2. When one of the doors is opened.
    3. When the engine switch is pushed.
    4. When the engine switch has been turned to on (IG) or on (ACC).
    5. When a terminal is disconnected from the battery and reconnected.
    6. All doors are unlocked using either the entry and start system.
    (3)
    		When the engine switch is off, the armed preparation state can be switched to via the following operation:
    1. All doors are locked using either the entry and start system while all doors, the engine hood and the glass hatch are closed.
    (4)
    		Armed preparation state will be switched to when the glass hatch or the engine hood is closed.
    (5)
    		Disarmed state A will be switched to if any of the following conditions are met:
    1. All doors are unlocked using either the entry and start system.
    2. When one of the doors is unlocked.
    3. When one of the doors is opened.
    4. When the engine switch is pushed.
    5. When the engine switch has been turned to on (IG) or on (ACC).
    6. When a terminal is disconnected from the battery and reconnected.
    (6)
    		Disarmed state B will be switched to if any of the following conditions are met:
    1. Engine hood is opened.
    2. Glass hatch is opened.
    3. The glass hatch is opened using either the entry and start system.
    (7)
    		The armed state will be switched to either 30 seconds or 10 seconds* after switching to the armed preparation state.
    (8)
    		Disarmed state A will be switched to if any of the following conditions are met:
    1. All doors are unlocked using either the entry and start system or the engine switch is on (ACC).
    2. 2 seconds have elapsed since the engine was started.
    3. The engine switch is turned to on (IG) after 5seconds or more have pressed since the presses of the engine switch.
    (9)
    		Disarmed state B will be switched to if the following condition is met:
    1. The glass hatch has been opened using either the entry and start system.
    (10)
    		The armed state will be switched to after approximately 27.5 seconds + 5 seconds* (alarm time + standby time) has passed.
    (11)
    		The alarm state will be switched to if any of the following conditions are met:
    1. Any door, the engine hood, or the glass hatch is opened.
    2. When a terminal is disconnected from the battery and reconnected.
    3. Any door is unlocked through an operation other than the key.
    4. When the power source mode has been turned from off to on (IG) after 5 seconds or more have passed since the pressing of the engine switch.

    1. *: Models for China
