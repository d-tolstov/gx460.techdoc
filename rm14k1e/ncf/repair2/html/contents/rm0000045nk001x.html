SLIDING ROOF SYSTEM > DETAILS



SYSTEM CONTROL

  1. The sliding roof system has the following functions:

    Function
    		Outline
    		Customization
    Manual Open-and-close
    		Opens or closes the sliding roof while the open switch or close switch is pressed for 0.3 seconds or less. The sliding roof stops as soon as the switch is released.
    		Standard (Not Customizable)
    One-touch Auto Open
    1. Opens the sliding roof up to the fully opened position when the open switch is pressed for 0.3 seconds or more.
    2. To suppress wind noise, this function stops opening the roof at the comfort stop position. By pressing the open switch once again, the sliding roof can be fully opened.
    		Standard (Not Customizable)
    One-touch Auto Close
    		Fully closes the sliding roof when the close switch is pressed for 0.3 seconds or more.
    		Standard (Not Customizable)
    Manual Tilt Up-and-down
    		Tilts up or down the sliding roof while the up switch or down switch is pressed for 0.3 seconds or less. The sliding roof stops as soon as the switch is released.
    		Standard (Not Customizable)
    One-touch Auto Tilt Up-and-down
    		Fully tilts up or down the sliding roof when the up switch or down switch is pressed for 0.3 seconds or more.
    		Standard (Not Customizable)
    Jam Protection
    		The jam protection function automatically stops the sliding roof and moves it open halfway (or to the fully tilted up position) if a foreign object gets jammed in the sliding roof during close or tilt down operation.
    		Standard (Not Customizable)
    Key Off Operation
    		The key off operation function makes it possible to operate the sliding roof for approximately 45 seconds after the engine switch is turned off, if the front door is not opened.
    		Standard (Not Customizable)
    Mechanical Key-linked Open-and-close
    1. When the key cylinder in the driver door is turned and held in the unlock position for 2 seconds or more, the sliding roof ECU activates the sliding roof motor to open the sliding roof while the key cylinder is turned.
    2. Similarly, turning and holding the driver door key cylinder in the lock position will cause the sliding roof to close.
    		Default Setting is Off*
    Mechanical Key-linked Tilt Up-and-down
    1. When the key cylinder in the driver door is turned and held in the unlock position for 2 seconds or more, the sliding roof ECU activates the sliding roof motor to tilt up the sliding roof while the key cylinder is turned.
    2. Similarly, turning and holding the driver door key cylinder in the lock position will cause the sliding roof to tilt down.
    		Default Setting is Off*
    Transmitter-linked Open-and-close
    1. When the main body ECU receives an unlock signal from the key continuously for 3 seconds or more via the door control receiver, the sliding roof ECU opens the sliding roof.
    2. Similarly, when the main body ECU receives a lock signal from the key continuously for 3 seconds or more via the door control receiver, the sliding roof ECU closes the sliding roof.
    		Default Setting is Off*
    Transmitter-linked Tilt Up-and-down
    1. When the main body ECU receives an unlock signal from the key continuously for 3 seconds or more via the door control receiver, the sliding roof ECU tilts up the sliding roof.
    2. Similarly, when the main body ECU receives a lock signal from the key continuously for 3 seconds or more via the door control receiver, the sliding roof ECU tilts down the sliding roof.
    		Default Setting is Off*
    Sliding Roof Open Warning
    		When the engine switch is turned from on (IG) to off and the driver door is opened with the sliding roof open, the buzzer in the combination meter assembly sounds once. Then, a warning message appears on the multi-information display and the master warning light blinks.
    		Standard (Not Customizable)

    HINT:
    *: Customizable parameters can be configured by means of the customized body electronics system. For details, refer to the corresponding Repair Manual for this model.


    1. The sliding roof ECU memorizes the initial position of the sliding roof. This memory is not cleared if the battery terminals are disconnected. However, initialization is necessary after the sliding roof drive gear sub-assembly is replaced.

    2. After replacing the sliding roof drive gear sub-assembly, the sliding roof will not operate correctly. In this case, the sliding roof position must be initialized as described below:

      1. Keep pressing the slide roof switch "up or close" until initialization completes. This will enable the sliding roof ECU to start initializing and perform the tilt up and tilt down operations of the sliding roof in sequence.
      2. Keep the switch pressed for 1 second after the tilt-up operation is completed.
      3. The sliding roof ECU performs the tilt down operation.
      4. The initialization process ends when the close operation is completed.

    3. Keep the tilt up or slide close switch pressed during initialization. If the tilt up or slide close switch is released during initialization, the system will not be able to complete initialization. If this occurs, the aforementioned steps must be performed again.

    4. For details, refer to the corresponding Repair Manual for this model.



  2. Jam Protection Function



    1. To control the jam protection function, the sliding roof ECU monitors the amount of movement and judges jamming of the sliding roof based on the pulse signals from Hall IC1, and the movement direction of the sliding roof from the phase difference between the pulses from Hall IC1 and Hall IC2.
