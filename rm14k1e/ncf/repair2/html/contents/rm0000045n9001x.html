WINDSHIELD / WINDOW GLASS SYSTEM > GENERAL



OUTLINE

  1. Windshield Deicer System

    1. The windshield deicer system uses a heater element in the windshield glass to help reduce the formation of ice where the front wipers rest.

  2. Window Defogger System

    1. The window defogger system uses a heater wire on the back door glass.

    2. This system is controlled by the air conditioning amplifier assembly.
