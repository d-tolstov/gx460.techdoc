FEATURES > EQUIPMENT



OUTLINE

  1. Entry and Start System

    1. This system mainly consists of the wireless door lock control function, engine immobiliser function, start function and entry function.

    2. The start function allows the power source mode to be switched between off, on (ACC), and on (IG). The power source can be switched by pushing the engine switch while the key is being carried by the driver. In addition, when the key is carried by the driver, the driver can start the engine by pushing the engine switch with the brake pedal depressed.

    3. For the entry function, the following functions can be operated when the driver is carrying the key:

      1. The doors unlock if the outside door handle is gripped or by pressing the back door lock switch (Entry Unlock Function).
      2. All doors are locked by touching the lock sensor of the front outside door handle or by pressing the back door lock switch (Entry Lock Function).

  2. Pre-crash Safety System

    1. 2 types of systems are used; pre-crash safety system (radar type) and pre-crash safety system (radar type with stereo camera and driver monitor camera).



  3. Combination Meter Assembly

    1. An Optitron display type combination meter assembly or an analog display type combination meter assembly is used.



    2. The Optitron display type combination meter assembly achieves excellent visibility through the use of smoke acrylic in the protective panel, and bright Light Emitting Diodes (LEDs) that have high contrast to illuminate the indicator and the dial.



  4. LEXUS Parking Assist-sensor System

    1. The LEXUS parking assist-sensor system uses ultrasonic sensors and is equipped with a beeping alarm. It can detect obstacles at the rear of the vehicle, and provide information to the driver regarding the distance.

  5. Monitor System

    1. To assist the driver in parking the vehicle by monitoring the area behind it, the rear view monitor system has a rear television camera assembly .

    2. This system consists of the rear television camera assembly, display and navigation module display*1, and accessory meter assembly*2.

      1. *1: Models with navigation system
      2. *2: Models without navigation system

  6. Wide View Front and Side Monitor System

    1. The wide view front and side monitor system uses television camera assemblies located on the front grille and passenger-side outer rear view mirror and displays images on the display and navigation module display*1 or accessory meter assembly*2 to allow the driver to see blind spots in the front and passenger side areas, thus assisting the driver's vision.

    2. The screen display mode can be switched between manual display mode and automatic display mode, in consideration of usability.

    3. A round monitor main switch has been placed on the steering wheel for switching the screen display and switching the wide view front and side monitor system on and off.

    4. The system consists of the front television camera assembly, side television camera assembly, display and navigation module display*1, accessory meter assembly*2 and parking assist ECU.

      1. *1: Models with navigation system
      2. *2: Models without navigation system



  7. Multi-terrain Monitor System

    1. In the multi-terrain select, the multi-terrain monitor system uses television camera assemblies located on the front grille, outer rear view mirror assembly and displays images on the multi display assembly to allow the driver to see blind spots around the vehicle, thus assisting the driver's vision.

    2. A switch to turn the multi-terrain monitor system on and off and switch the screen display is located on the steering wheel.



    3. The system consists of the television camera assemblies, display and navigation module display*1, accessory meter assembly*2, steering pad switch and parking assist ECU.

      1. *1: Models with navigation system
      2. *2: Models without navigation system

  9. Seat

    1. Seat Variation

      1. An 8-way power seats are provided for the front seats.
      2. A 60/40 split slide walk-in seat is provided for the rear No. 1 seat.

        HINT:
        Slide walk-in mechanism is for the 40% side only

      3. A double-flat seat (power) is provided for the rear No. 2 seat.

    2. Climate control seat system

      1. A climate control seat system is provided in accordance with specification.

    3. Front Power Seat

      1. A power seat control system is used for the driver seat and front passenger seat.
      2. A memory function is used in the power seat control system for the driver seat.

    4. Rear No. 2 Power Seat with Folding Function

      1. A power seat control system is used for rear No. 2 seat.
      2. The rear No. 2 seat is stowed automatically by the folding function.

    5. Seat Cover

      1. Smooth leather and semi-aniline leather are provided for the seat cover.



  10. Glass Hatch Opener System

    1. The glass hatch is opened by pressing a button.


  11. HDD Navigation System

    1. A navigation system which uses map data stored on a Hard Disc Drive (HDD) is used.



  12. G-BOOK System (Models for China)



    1. The G-BOOK system is a telematics service that links the vehicle with the G-BOOK network (which provides information from the G-BOOK center or content providers).

  13. Instrument Panel



    1. A surface-wrapped pad is provided on the upper section of the instrument panel for the front passenger side to improve tactility and express a feeling of luxury.





  14. Center Console

    1. The combination of wood-grain and metallic panel material expresses both dynamism and sophistication.

    2. Each function switch is laid out in the panel section on the surface of the console.

    3. For the side section of the console, soft pads are provided in consideration of passenger's knees, thus improving the comfort and tactility for occupants.



    4. USB port, stereo jack adapter assembly, and front power outlet socket assembly (DC 12V) are provided inside the console box, thus improving convenience.

    2. A removable inner tray with a built-in handle has been provided inside the large console box in order to easily organize strage objects and improve accessibility to the bottom. By removing the inner tray from the console box, a large object can be stored inside.
