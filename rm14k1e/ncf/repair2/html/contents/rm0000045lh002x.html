LIN COMMUNICATION SYSTEM > GENERAL



OUTLINE

  1. The Local Interconnect Network (LIN) is used for communication within the body electrical system.

  2. The LIN is used in the following systems:

    1. Steering lock system

    2. Power door lock control system

    3. Entry and start system

    4. Air conditioning system

    5. Sliding roof system

    6. Power window control system

    7. Wiper and washer system

    8. Lighting system

    9. Accessory meter system

  3. See the respective system for details.
