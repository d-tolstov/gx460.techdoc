DESCRIPTION

The distance control switch sets the vehicle-to-vehicle distance setting. The distance control switch is installed in the steering pad switch. The vehicle-to-vehicle distance setting can be changed by operating the distance control switch in the steering pad switch while the dynamic radar cruise control system is in operation.
