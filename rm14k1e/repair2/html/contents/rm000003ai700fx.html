REAR SEAT ENTERTAINMENT ECU > INSTALLATION

for Preparation Click here



1. INSTALL NO. 3 MULTI-DISPLAY CONTROLLER BRACKET



  1. Set the aligning protrusions of the bracket, labeled A, into the holes of the controller.

  1. Install the No. 3 multi-display controller bracket with the 2 screws labeled B.



2. INSTALL NO. 1 MULTI-DISPLAY CONTROLLER BRACKET



  1. Set the aligning protrusions of the bracket, labeled A, into the holes of the controller.

  1. Install the No. 1 multi-display controller bracket with the 2 screws labeled B.



3. INSTALL MULTI-DISPLAY CONTROLLER SUB-ASSEMBLY WITH BRACKET



  1. Install the multi-display controller sub-assembly with bracket with the bolt and screw.

    Torque:
    for bolt A:
    8.0 N*m{ 82 kgf*cm , 71 in.*lbf }

  1. Return the floor carpet to its original position as shown in the illustration.

  1. Attach the clamp and connect the 4 connectors.



4. INSTALL MULTI-MEDIA MODULE COVER

  1. Install the amplifier cover with the 3 clips.




  1. Return the floor carpets to their original positions as shown in the illustration.



5. INSTALL FRONT SEAT ASSEMBLY RH

  1. Install the front seat assembly RH (Click here).



6. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).



7. CHECK SRS WARNING LIGHT

  1. Check the SRS warning light (Click here).
