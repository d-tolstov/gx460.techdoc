FRONT SHOCK ABSORBER > INSTALLATION

for Preparation Click here


HINT:
  1. Use the same procedure for the RH and LH sides.
  2. The procedure listed below is for the LH side.
  3. A bolt without a torque specification is shown in the standard bolt chart (Click here).



1. TEMPORARILY INSTALL FRONT SHOCK ABSORBER WITH COIL SPRING



  1. Install the coil spring to the vehicle body with the lower end of the coil spring facing the rear side of the vehicle.
    Text in Illustration 
    *a
    		Lower End
    *b
    		Outer Side
    *c
    		Rear Side




  1. Install the 3 nuts to the top of the front shock absorber with coil spring.

    Torque:
    64 N*m{ 653 kgf*cm , 47 ft.*lbf }




  1. Temporarily install the bolt, washer and nut as shown in the illustration.




  1. Using SST, install the bracket with the nut.

    SST
    09961-00950  

    Torque:
    15 N*m{ 153 kgf*cm , 11 ft.*lbf }




  1. Using a 4 mm hexagon wrench, install the absorber control actuator with the 2 bolts.

    Torque:
    7.8 N*m{ 80 kgf*cm , 69 in.*lbf }

  1. Attach the 3 clamps and connect the connector.



2. INSTALL FRONT STABILIZER BAR



  1. Install the 2 front stabilizer lower bracket bushes to the front stabilizer bar.

    HINT:
    Align the protrusions on the bushes with the identification marks on the front stabilizer bar with the protrusions facing inward.

    Text in Illustration 
    *a
    		Protrusion
    *b
    		Mark Position




  1. With the identification marks of the front stabilizer bar facing downwards, support the front stabilizer bar with a jack.

    NOTICE:
    Place a wooden block between the jack and front stabilizer bar to prevent damage.

    HINT:
    Place the jack under the center of the vehicle.




  1. Install the front stabilizer bar and 2 front stabilizer lower brackets with the 4 bolts.

    Torque:
    40 N*m{ 408 kgf*cm , 30 ft.*lbf }

    HINT:
    Make sure that the protrusions on the front stabilizer lower bracket bushes are positioned within 180° of the stabilizer cylinder and stabilizer link.

    Text in Illustration 
    *1
    		Stabilizer Link
    *2
    		Stabilizer Cylinder



3. INSTALL FRONT STABILIZER END BRACKET



  1. Install the 2 front stabilizer brackets and 2 front stabilizer link bushes with the 4 bolts.

    Torque:
    75 N*m{ 765 kgf*cm , 55 ft.*lbf }

    HINT:
    There are stamps on the front stabilizer brackets to distinguish between the right and left brackets.



4. INSTALL FRONT SUSPENSION MEMBER BRACE SUB-ASSEMBLY

  1. Install the 2 member braces with the 6 bolts.

    Torque:
    30 N*m{ 306 kgf*cm , 22 ft.*lbf }



5. INSTALL NO. 1 ENGINE UNDER COVER SUB-ASSEMBLY



  1. Hook the No. 1 engine under cover to the vehicle body as shown in the illustration.

  1. Install the 4 bolts.

    Torque:
    29 N*m{ 296 kgf*cm , 21 ft.*lbf }



6. INSTALL LOWER FRONT BUMPER COVER

  1. Install the lower front bumper cover with the 5 bolts and clip.

    Torque:
    8.0 N*m{ 82 kgf*cm , 71 in.*lbf }



7. INSTALL FRONT WHEEL

Torque:
112 N*m{ 1142 kgf*cm , 83 ft.*lbf }



8. STABILIZE SUSPENSION

  1. Lower the vehicle.

  1. Bounce the vehicle up and down several times to stabilize the suspension.



9. TIGHTEN FRONT SHOCK ABSORBER WITH COIL SPRING



  1. Tighten the nut.

    Torque:
    95 N*m{ 969 kgf*cm , 70 ft.*lbf }



10. INSPECT AND ADJUST FRONT WHEEL ALIGNMENT

  1. Inspect and adjust the front wheel alignment (Click here).
