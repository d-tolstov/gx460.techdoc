REAR ACCELERATION SENSOR > INSTALLATION

for Preparation Click here



1. INSTALL REAR ACCELERATION SENSOR



  1. Install the rear acceleration sensor with the bolt as shown in the illustration.

    Torque:
    8.5 N*m{ 87 kgf*cm , 75 in.*lbf }

    NOTICE:
    1. Replace the rear acceleration sensor with a new one if it is dropped.
    2. Install the rear acceleration sensor is oriented correctly.

  1. Connect the connector.

    NOTICE:
    Securely connect the connector.



2. INSTALL DECK TRIM SIDE PANEL ASSEMBLY LH

  1. Install the deck trim side panel assembly LH (Click here).



3. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).
