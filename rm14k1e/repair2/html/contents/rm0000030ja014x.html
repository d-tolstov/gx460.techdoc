1. INSTALL AIR INJECTION CONTROL DRIVER

  1. Install the air injection control driver with the 2 screws.

    Torque:
    2.0 N*m{ 20 kgf*cm , 18 in.*lbf }



2. INSTALL AIR INJECTION CONTROL DRIVER WITH BRACKET

  1. Install the air injection control driver with bracket with the bolt.

    Torque:
    13 N*m{ 127 kgf*cm , 9 ft.*lbf }

  1. Connect the 2 air injection control driver connectors.



3. INSTALL ENGINE ROOM SIDE COVER LH

  1. Install the engine room side cover LH with the 4 clips.
