FRONT WIPER MOTOR > INSTALLATION

for Preparation Click here



1. INSTALL WIPER MOTOR WIRE

  1. Attach the claw to install the wiper motor wire.



2. INSTALL WINDSHIELD WIPER MOTOR ASSEMBLY

  1. Using a T30 "TORX" socket, install the wiper motor with the 2 bolts.

    Torque:
    7.5 N*m{ 76 kgf*cm , 66 in.*lbf }

  1. Connect the connector.




  1. Apply MP grease to the crank arm pivot of the front wiper crank.
    Text in Illustration 
    *1
    		Crank Arm Pivot

    		MP grease




  1. Connect the rod to the crank arm pivot of the front wiper crank.



3. INSTALL WINDSHIELD WIPER MOTOR AND LINK



  1. Attach the grommet as shown in the illustration.

    Torque:
    7.0 N*m{ 71 kgf*cm , 62 in.*lbf }

    Text in Illustration 
    *1
    		Grommet

    NOTICE:
    Be careful not to damage the windshield when installing the windshield wiper motor and link assembly.

  1. Install the windshield wiper motor and link with the 2 bolts.

  1. Connect the connector.

  1. Attach the clamp.



4. INSTALL COWL TOP VENTILATOR LOUVER SUB-ASSEMBLY



  1. Attach the 10 guides (B).

  1. Attach the 6 claws and guide (A) as shown in the illustration.

  1. Install the 2 clips to the cowl top ventilator louver sub-assembly.
    Text in Illustration 
    *1
    		Guide A
    *2
    		Guide B



5. INSTALL CENTER NO. 2 COWL TOP VENTILATOR LOUVER

  1. Attach the 2 claws and 3 guides to install the center No. 2 cowl top ventilator louver.



6. INSTALL HOOD TO COWL TOP SEAL


7. INSTALL FRONT WIPER ARM AND BLADE ASSEMBLY RH

  1. Operate the wiper and stop the windshield wiper motor at the automatic stop position.




  1. When reusing the front wiper arm and blade assembly RH:

    1. Clean the wiper arm serrations.
      Text in Illustration 
      *1
      		Wiper Arm Serration
      *2
      		Wire Brush
      *3
      		Wiper Pivot Serration

  1. When reusing the windshield wiper link assembly:

    1. Clean the wiper pivot serrations with a wire brush.




  1. Install the front wiper arm and blade assembly RH with the nut to the position shown in the illustration.

    Torque:
    25 N*m{ 255 kgf*cm , 18 ft.*lbf }

    Standard:
    Area
    		Standard Condition
    a
    		7.5 mm (0.295 in.)
    b
    		7.5 mm (0.295 in.)

    Text in Illustration 
    *1
    		Ceramic Dot
    *a
    		Upper Limit
    *b
    		Lower Limit

    HINT:
    1. Hold the wiper arm by hand while tightening the nut.
    2. Raise and lower the wiper arm a few times to confirm that it has settled to the specified position.



8. INSTALL FRONT WIPER ARM AND BLADE ASSEMBLY LH



  1. When reusing the front wiper arm and blade assembly LH:

    1. Clean the wiper arm serrations.
      Text in Illustration 
      *1
      		Wiper Arm Serration
      *2
      		Wire Brush
      *3
      		Wiper Pivot Serration

  1. When reusing the windshield wiper link assembly:

    1. Clean the wiper pivot serrations with a wire brush.




  1. Install the front wiper arm and blade assembly LH with the nut to the position shown in the illustration.

    Torque:
    25 N*m{ 255 kgf*cm , 18 ft.*lbf }

    Standard:
    Area
    		Standard Condition
    a
    		7.5 mm (0.295 in.)
    b
    		7.5 mm (0.295 in.)

    Text in Illustration 
    *1
    		Ceramic Dot
    *a
    		Upper Limit
    *b
    		Lower Limit

    HINT:
    1. Hold the wiper arm by hand while tightening the nut.
    2. Raise and lower the wiper arm a few times to confirm that it has settled to the specified position.

  1. Operate the front wipers while spraying washer fluid on the windshield glass. Make sure that the front wipers function properly and there is no interference with the vehicle body.
