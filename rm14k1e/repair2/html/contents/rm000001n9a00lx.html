DTC    C1792/92    Height Control Switch Circuit (Test Mode DTC)

for Preparation Click here


DESCRIPTION

The vehicle height can be adjusted by pressing the height control switch.
By pressing the "up" side, vehicle height is raised.
By pressing the "down" side, vehicle height is lowered.

DTC Code
DTC Detection Condition
Trouble Area
C1792/92
The height control switch signal does not change.
  1. Height control switch (combination switch)
  2. Harness or connector
  3. Suspension control ECU

HINT:
This DTC is stored only during test mode.
