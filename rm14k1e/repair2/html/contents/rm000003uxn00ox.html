DTC    U0100    Lost Communication with ECM

DTC    U0142    Lost Communication with Main Body ECU

DTC    U0155    Lost Communication with Combination Meter

for Preparation Click here


DESCRIPTION

The air conditioning amplifier communicates with the ECM, main body ECU (multiplex network body ECU) and combination meter through the CAN communication system.

DTC Code
DTC Detection Condition
Trouble Area
U0100
Lost communication with the ECM.
  1. CAN communication system
  2. ECM
U0142
Lost communication with the main body ECU.
  1. CAN communication system
  2. Main body ECU (multiplex network body ECU)
U0155
Lost communication with the combination meter.
  1. CAN communication system
  2. Combination meter assembly
