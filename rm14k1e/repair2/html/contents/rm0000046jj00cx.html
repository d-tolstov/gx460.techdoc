DOOR CONTROL RECEIVER > INSTALLATION

for Preparation Click here



1. INSTALL DOOR CONTROL RECEIVER

  1. Install the receiver with the bolt.

    Torque:
    5.5 N*m{ 56 kgf*cm , 49 in.*lbf }

  1. Connect the connector.



2. INSTALL REAR QUARTER TRIM PANEL ASSEMBLY LH

  1. Attach the 4 clips and 2 guides and pass the rear No. 2 seat outer belt floor anchor through the rear quarter trim panel.

  1. Connect the connector.

  1. Attach the 7 clips and 2 guides to install the rear quarter trim panel.



3. INSTALL DECK TRIM SIDE PANEL ASSEMBLY LH

  1. Connect each connector.

  1. Attach the 3 clips and 8 claws to install the deck trim side panel.

  1. Install the 2 bolts and 2 screws.

  1. Install the rear No. 2 seat outer belt floor anchor with the bolt.

    Torque:
    42 N*m{ 428 kgf*cm , 31 ft.*lbf }

  1. Install the rear No. 1 seat outer belt floor anchor with the bolt.

    Torque:
    42 N*m{ 428 kgf*cm , 31 ft.*lbf }



4. INSTALL ASSIST GRIP SUB-ASSEMBLY



  1. Assemble the assist grip, 2 clips and 2 covers as shown in the illustration.

  1. Install the assist grip.



5. INSTALL ASSIST GRIP PLUG

HINT:
Use the same procedure for all assist grip plugs.

  1. Attach the 2 claws to install the assist grip plug.



6. INSTALL FRONT DECK SIDE TRIM COVER (w/ Tonneau Cover)

HINT:
Use the same procedure for the other front deck side trim cover.

  1. Attach the 2 claws to install the front deck side trim cover



7. INSTALL NO. 1 TONNEAU COVER HOLDER CAP (w/o Tonneau Cover)

HINT:
Use the same procedure for the other No. 1 tonneau cover holder cap.

  1. Attach the 2 claws to install the No. 1 tonneau cover holder cap.



8. INSTALL NO. 1 LUGGAGE COMPARTMENT TRIM HOOK

HINT:
Use the same procedure for the other No. 1 luggage compartment trim hook.

  1. Insert the No. 1 luggage compartment trim hook with the hook oriented horizontally to install it.



9. INSTALL REAR NO. 1 SEAT OUTER LAP BELT ANCHOR COVER

HINT:
Use the same procedure for the other rear No. 1 seat outer lap belt anchor cover.

  1. Attach the 3 claws to install the rear No. 1 seat outer lap belt anchor cover.



10. INSTALL TONNEAU COVER ASSEMBLY (w/ Tonneau Cover)

  1. Install the tonneau cover.



11. INSTALL REAR DOOR OPENING TRIM WEATHERSTRIP LH



  1. Align the paint mark on the rear door opening trim weatherstrip with the mark position on the vehicle and install the rear door opening trim weatherstrip as shown in the illustration.
    Text in Illustration 
    *1
    		Paint Mark
    *2
    		Mark Position



12. INSTALL REAR DOOR SCUFF PLATE LH

  1. Attach the 3 clips, 6 claws and 2 guides to install the rear door scuff plate.



13. INSTALL QUARTER SCUFF PLATE LH

  1. Install the quarter scuff plate with the 2 bolts.



14. INSTALL REAR NO. 1 FLOOR STEP COVER

HINT:
Use the same procedure for all rear No. 1 floor step covers.

  1. Attach the 2 claws to install the rear No. 1 floor step cover.



15. INSTALL REAR NO. 1 SEAT ASSEMBLY

  1. for 60/40 Split Slide Walk-in Seat Type LH Side:
    Install the rear No. 1 seat assembly (Click here).

  1. for 60/40 Split Slide Walk-in Seat Type RH Side:
    Install the rear No. 1 seat assembly (Click here).



16. INSTALL REAR NO. 2 SEAT ASSEMBLY

  1. Install the rear No. 2 seat assembly (Click here).



17. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).
