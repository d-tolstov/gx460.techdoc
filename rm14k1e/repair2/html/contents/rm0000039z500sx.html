REAR DOOR WINDOW FRAME MOULDING > INSTALLATION

for Preparation Click here


HINT:
  1. Use the same procedure for the RH and LH sides.
  2. The procedure listed below is for the LH side.



1. INSTALL REAR DOOR FRONT WINDOW FRAME MOULDING LH

HINT:
When installing the window frame moulding, heat the vehicle body and window frame moulding using a heat light.

Standard:
Item
Temperature
Vehicle Body
40 to 60°C (104 to 140°F)
Window Frame Moulding
20 to 30°C (68 to 86°F)

NOTICE:
Do not heat the vehicle body or window frame moulding excessively.

  1. Clean the vehicle body surface.

    1. Using a heat light, heat the vehicle body surface.

    2. Remove the double-sided tape from the vehicle body surface.

    3. Wipe off any tape adhesive residue with cleaner.

  1. Install a new rear door front window frame moulding.

    1. Using a heat light, heat a new rear door front window frame moulding and the vehicle body surface.

    2. Remove the peeling paper from the face of the rear door front window frame moulding.

      HINT:
      After removing the peeling paper, keep the exposed adhesive free from foreign matter.

    3. Attach the clip and double-sided tape to install the rear door front window frame moulding.

      HINT:
      Press the rear door front window frame moulding firmly to install it.

  1. Install a new door window frame moulding clip.



2. INSTALL REAR DOOR WEATHERSTRIP LH

  1. Attach the clip to install the upper part of the rear door weatherstrip.



3. INSTALL REAR DOOR BELT MOULDING LH

  1. Install the rear door belt moulding (Click here).
