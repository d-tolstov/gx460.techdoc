DTC    C1433    Steering Angle Sensor Internal Circuit

for Preparation Click here


DESCRIPTION

Steering angle sensor signals are sent to the skid control ECU via the CAN communication system. When there is a malfunction in the CAN communication system, it is detected by the steering angle sensor zero point malfunction diagnostic function.

DTC Code
DTC Detection Condition
Trouble Area
C1433
Either condition is met:
  1. With the IG1 terminal voltage 9.5 to 17.4 V, the skid control ECU receives an error signal from the steering angle sensor for 0.012 seconds.
  2. An open in the sensor power supply circuit for 1 second or more.
Steering angle sensor (Spiral cable sub-assembly)

WIRING DIAGRAM

Refer to DTC C1432 (Click here).

INSPECTION PROCEDURE

HINT:
  1. When DTC U0073, U0123, U0124 and/or U0126 is output together with DTC C1433, inspect and repair the trouble areas indicated by DTC U0073, U0123, U0124 and/or U0126 first (Click here).
  2. When any of the speed sensors or the yaw rate and acceleration sensor has trouble, DTCs for the steering angle sensor may be stored even when the steering angle sensor is normal. When DTCs for the speed sensors or yaw rate and acceleration sensor are output together with DTCs for the steering angle sensor, inspect and repair the speed sensor and yaw rate and acceleration sensor first, and then inspect and repair the steering angle sensor.
