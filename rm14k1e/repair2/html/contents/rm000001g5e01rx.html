REAR NO. 1 SEAT OUTER BELT ASSEMBLY > INSTALLATION

for Preparation Click here


HINT:
  1. Use the same procedure for the RH and LH sides.
  2. The procedure listed below is for the LH side.



1. INSTALL FRONT SHOULDER BELT ANCHOR ADJUSTER ASSEMBLY



  1. Align the claw with the anchor plate positioning hole, and install the anchor adjuster with the 2 bolts.

    Torque:
    42 N*m{ 428 kgf*cm , 31 ft.*lbf }

    HINT:
    First install the lower bolt, and then install the upper bolt.



2. INSTALL REAR NO. 1 SEAT OUTER BELT ASSEMBLY LH



NOTICE:
When installing the retractor, make sure the claws of the vehicle (labeled "a" in the illustration) only contact the installation areas of the retractor.

  1. Align the seat belt positioning holes with the claws and install the retractor of the seat belt with the 2 bolts as shown in the illustration.

    HINT:
    First install bolt A, and then install bolt B.

    Torque:
    for bolt A:
    8.5 N*m{ 87 kgf*cm , 75 in.*lbf }
    for bolt B:
    42 N*m{ 428 kgf*cm , 31 ft.*lbf }




  1. Connect the pretensioner connector as shown in the illustration.

  1. Install the rear No. 1 seat outer belt shoulder anchor with the nut.

    Torque:
    42 N*m{ 428 kgf*cm , 31 ft.*lbf }



3. INSTALL REAR SIDE NO. 2 AIR DUCT (for RH Side, w/ Rear Air Conditioning System)



  1. Detach the 2 clips.

  1. Detach the clamp and remove the duct.



4. INSTALL FRONT QUARTER TRIM PANEL ASSEMBLY LH

  1. Pass the rear No. 1 seat outer belt floor anchor through the front quarter trim panel.

  1. Attach the 2 clips and 2 guides to install the front quarter trim panel.



5. INSTALL DECK TRIM SIDE PANEL ASSEMBLY LH

  1. Attach the 3 clips and 8 claws to install the deck trim side panel.

  1. Install the 2 bolts and 2 screws.

  1. Install the rear No. 2 seat outer belt floor anchor with the bolt.

    Torque:
    42 N*m{ 428 kgf*cm , 31 ft.*lbf }

  1. Install the rear No. 1 seat outer belt floor anchor with the bolt.

    Torque:
    42 N*m{ 428 kgf*cm , 31 ft.*lbf }



6. INSTALL ASSIST GRIP SUB-ASSEMBLY

HINT:
Use the same procedure for the other assist grip.

  1. Install the assist grip with the 2 bolts.



7. INSTALL ASSIST GRIP PLUG

HINT:
Use the same procedure for all assist grip plugs.

  1. Attach the 2 claws to install the assist grip plug.



8. INSTALL FRONT DECK SIDE TRIM COVER (w/ Tonneau Cover)

HINT:
Use the same procedure for the other front deck side trim cover.

  1. Attach the 2 claws to install the front deck side trim cover



9. INSTALL NO. 1 LUGGAGE COMPARTMENT TRIM HOOK

HINT:
Use the same procedure for the other No. 1 luggage compartment trim hook.

  1. Insert the No. 1 luggage compartment trim hook with the hook oriented horizontally to install it.



10. INSTALL REAR NO. 1 SEAT OUTER LAP BELT ANCHOR COVER

HINT:
Use the same procedure for the other rear No. 1 seat outer lap belt anchor cover.

  1. Attach the 3 claws to install the rear No. 1 seat outer lap belt anchor cover.



11. INSTALL REAR DOOR OPENING TRIM WEATHERSTRIP LH



  1. Align the paint mark on the rear door opening trim weatherstrip with the mark position on the vehicle and install the rear door opening trim weatherstrip as shown in the illustration.
    Text in Illustration 
    *1
    		Paint Mark
    *2
    		Mark Position



12. INSTALL REAR DOOR SCUFF PLATE LH

  1. Attach the 3 clips, 6 claws and 2 guides to install the rear door scuff plate.



13. INSTALL QUARTER SCUFF PLATE LH

  1. Install the quarter scuff plate with the 2 bolts.



14. INSTALL REAR NO. 1 FLOOR STEP COVER

HINT:
Use the same procedure for all rear No. 1 floor step covers.

  1. Attach the 2 claws to install the rear No. 1 floor step cover.



15. INSTALL REAR NO. 2 SEAT ASSEMBLY LH

  1. Install the rear No. 2 seat assembly LH (Click here).



16. INSTALL TONNEAU COVER ASSEMBLY (w/ Tonneau Cover)

  1. Install the tonneau cover.
