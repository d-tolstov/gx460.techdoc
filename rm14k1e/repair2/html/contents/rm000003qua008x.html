HIGH INTENSITY DISCHARGE HEADLIGHT BULB > INSTALLATION

for Preparation Click here


HINT:
  1. Use the same procedure for the RH and LH sides.
  2. The procedure listed below is for the LH side.



1. INSTALL DISCHARGE HEADLIGHT BULB



  1. Lock the set spring as shown in the illustration to install the discharge headlight bulb.

    NOTICE:
    Do not touch the glass part of the bulb.




  1. Turn the headlight control ECU socket in the direction indicated by the arrow to connect the headlight control ECU socket.



2. INSTALL NO. 1 HEADLIGHT BACK COVER



  1. Turn the back cover in the direction indicated by the arrow to install it.



3. INSTALL UPPER RADIATOR SUPPORT SEAL (for RH Side)

  1. Install the upper radiator support seal with the 13 clips.



4. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).
