INSTRUMENT PANEL SPEAKER > INSTALLATION

for Preparation Click here


HINT:
  1. Use the same procedure for the RH and LH sides.
  2. The procedure listed below is for the LH side.
  3. A bolt without a torque specification is shown in the standard bolt chart (Click here).



1. INSTALL FRONT NO. 4 SPEAKER ASSEMBLY

  1. Connect the speaker connector.

  1. Align the speaker with the positioning pins of the instrument panel and temporarily install the speaker.

  1. Install the front No. 4 speaker with the 2 screws.

    NOTICE:
    1. Do not touch the cone of the speaker.
    2. When installing the speaker to the instrument panel, be careful that the wires do not get caught between parts.



2. INSTALL UPPER INSTRUMENT CLUSTER FINISH PANEL

  1. Attach the 11 clips to install the upper instrument cluster finish panel.



3. INSTALL FRONT NO. 2 SPEAKER ASSEMBLY

  1. Connect the connector.

  1. Align the speaker with the positioning pins of the instrument panel and temporarily install the speaker.

  1. Install the front No. 2 speaker with the 2 screws.

    NOTICE:
    1. Do not touch the cone of the speaker.
    2. When installing the speaker to the instrument panel, be careful that the wires do not get caught between parts.



4. INSTALL NO. 1 INSTRUMENT PANEL SPEAKER PANEL SUB-ASSEMBLY

  1. Attach the 2 clips, claw and 2 guides to install the No. 1 instrument panel speaker panel.



5. INSTALL FRONT PILLAR GARNISH LH

  1. Attach the 3 guides to install the front pillar garnish.



6. INSTALL NO. 1 ASSIST GRIP

HINT:
Use the same procedure for the other No. 1 assist grip.

  1. Attach the 2 claws to install the No. 1 assist grip.

  1. Install the 2 bolts.



7. INSTALL FRONT NO. 1 ASSIST GRIP PLUG LH

HINT:
Use the same procedure for the other front No. 1 assist grip plug.

  1. Attach the 2 claws to install the front No. 1 assist grip plug.



8. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).



9. CHECK SRS WARNING LIGHT

  1. Check the SRS warning light (Click here).
