1. INSTALL PLANETARY GEAR



  1. Apply high-temperature grease to the planetary gears and pin parts of the planetary shaft as shown in the illustration.

  1. Install the 3 planetary gears to the starter center bearing clutch.



2. INSTALL STARTER ARMATURE PLATE



  1. Align the protrusion of the starter center bearing clutch sub-assembly with the notch of the starter armature plate.
    Text in Illustration 
    *1
    		Protrusion
    *2
    		Notch

  1. Insert the starter armature plate to the starter center bearing clutch.



3. INSTALL STARTER ARMATURE ASSEMBLY

  1. Install the starter armature to the starter center bearing clutch.



4. INSTALL STARTER YOKE ASSEMBLY



  1. Align the protrusion of the starter center bearing clutch with the notch of the starter yoke and install the starter yoke.
    Text in Illustration 
    *1
    		Protrusion
    *2
    		Notch



5. INSTALL STARTER COMMUTATOR END FRAME ASSEMBLY

  1. Install the starter brush holder.




  1. Connect the 4 brushes to the starter brush holder.

    1. Using a screwdriver, hold back the brush spring.

    2. Connect the brush to the starter brush holder.

      NOTICE:
      Check that the positive (+) lead wires are not grounded.

  1. Install a new dust starter protector.




  1. Align the starter commutator end frame rubber with the groove of the starter yoke.
    Text in Illustration 
    *1
    		Rubber
    *2
    		Groove




  1. Install the starter commutator end frame with the 2 screws.

    Torque:
    3.8 N*m{ 39 kgf*cm , 34 in.*lbf }



6. INSTALL STARTER DRIVE HOUSING ASSEMBLY



  1. Apply high-temperature grease to the starter pinion drive lever as shown in the illustration.




  1. Install the starter pinion drive lever to the starter center bearing clutch.




  1. Align the protrusion of the starter center bearing clutch with the notch inside the starter drive housing.
    Text in Illustration 
    *1
    		Protrusion
    *2
    		Notch




  1. Using a T30 "TORX" socket wrench, install the starter drive housing with the 2 bolts.

    Torque:
    6.0 N*m{ 61 kgf*cm , 53 in.*lbf }



7. INSTALL MAGNET STARTER SWITCH ASSEMBLY



  1. Temporarily install the magnet starter switch by hooking its tip onto the upper side of the starter pinion drive lever.

  1. Install the 2 nuts.

    Torque:
    7.5 N*m{ 76 kgf*cm , 66 in.*lbf }




  1. Connect the lead wire to terminal C with the nut.

    Torque:
    6.5 N*m{ 66 kgf*cm , 58 in.*lbf }
