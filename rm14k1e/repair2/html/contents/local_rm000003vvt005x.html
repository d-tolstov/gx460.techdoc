DTC    B2658    Short in Sensor with Motor Power Supply Circuit


DESCRIPTION

WIRING DIAGRAM

INSPECTION PROCEDURE

CHECK FOR DTC

CHECK HARNESS AND CONNECTOR (FRONT POWER SEAT SWITCH - SLIDE MOTOR)

CHECK FRONT POWER SEAT SWITCH LH (SLIDE MOTOR CIRCUIT)

CHECK HARNESS AND CONNECTOR (FRONT POWER SEAT SWITCH - FRONT VERTICAL MOTOR)

CHECK FRONT POWER SEAT SWITCH LH (FRONT VERTICAL MOTOR CIRCUIT)

CHECK HARNESS AND CONNECTOR (FRONT POWER SEAT SWITCH - LIFTER MOTOR)

CHECK FRONT POWER SEAT SWITCH LH (LIFTER MOTOR CIRCUIT)

CHECK HARNESS AND CONNECTOR (FRONT POWER SEAT SWITCH - RECLINING MOTOR)

CHECK FRONT POWER SEAT SWITCH LH (RECLINING MOTOR CIRCUIT)

CHECK SEPARATE TYPE FRONT SEAT CUSHION SPRING ASSEMBLY LH

CHECK SEPARATE TYPE FRONT SEATBACK SPRING ASSEMBLY LH
