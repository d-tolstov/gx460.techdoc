DTC    C1831/31    Accumulator Solenoid Malfunction / Upside

DTC    C1832/32    Accumulator Solenoid Malfunction / Downside

for Preparation Click here


DESCRIPTION

The stabilizer control ECU receives information from the steering angle sensor, skid control ECU (speed signal) and yaw rate and acceleration sensor via CAN communication. Based on this information, the stabilizer control ECU turns the stabilizer control solenoid valve (built into the stabilizer control with accumulator housing assembly) on or off.

DTC Code
DTC Detection Condition
Trouble Area
C1831/31
C1832/32
Either condition is met:
  1. An open circuit signal is detected for a second continuously when current is not applied to the solenoid valve.
  2. A short circuit signal is detected 8 seconds continuously when current is applied to the solenoid valve.
  1. Harness or connector
  2. Stabilizer control solenoid valve (Stabilizer control with accumulator housing assembly)
  3. Stabilizer control ECU
