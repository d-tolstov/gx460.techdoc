DTC    B156B    Multi-media Interface ECU Disconnected

DTC    B156E    No Response from Multi-media Interface ECU

for Preparation Click here


DESCRIPTION

The display and navigation module display and the multi-media interface ECU are connected by the AVC-LAN communication line.
When an AVC-LAN communication error occurs between the display and navigation module display and the multi-media interface ECU, these DTCs are stored.

DTC Code
DTC Detection Condition
Trouble Area
B156B*
Either condition is met:
  1. The multi-media interface ECU is/was not connected while the engine switch is on (IG) or on (ACC).
  2. Communication between the master unit and multi-media interface ECU is not possible when the engine is started.
  1. AVC-LAN circuit
  2. Multi-media interface ECU
  3. Display and navigation module display
B156E
Within the time-out period, there is no response from the multi-media interface ECU to the following requests from the master unit:
  1. The system inspection request.
  2. The system inspection result request.
  3. The diagnostic memory request.

HINT:
  1. *: Even if no fault is present, this DTC may be stored depending on the battery condition or engine start voltage.
  2. The display and navigation module display is the master unit.
