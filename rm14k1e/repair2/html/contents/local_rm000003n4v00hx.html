DTC    C1AEC    Front Sensor Communication Circuit


DESCRIPTION

WIRING DIAGRAM

INSPECTION PROCEDURE

CHECK FOR DTC (C1AEC)

CHECK HARNESS AND CONNECTOR (CLEARANCE WARNING ECU - FRONT CORNER SENSOR LH)

CHECK HARNESS AND CONNECTOR (FRONT CORNER SENSOR LH - FRONT CENTER SENSOR LH)

CHECK HARNESS AND CONNECTOR (FRONT CENTER SENSOR LH - FRONT CENTER SENSOR RH)

CHECK HARNESS AND CONNECTOR (FRONT CENTER SENSOR RH - FRONT CORNER SENSOR RH)

INSPECT NO. 1 ULTRASONIC SENSOR (FRONT CORNER SENSOR LH)

INSPECT NO. 1 ULTRASONIC SENSOR (FRONT CENTER SENSOR LH)

INSPECT NO. 1 ULTRASONIC SENSOR (FRONT CENTER SENSOR RH)

REPLACE NO. 1 ULTRASONIC SENSOR (FRONT CORNER SENSOR RH)

CHECK FOR DTC (C1AEC)

REPLACE NO. 1 ULTRASONIC SENSOR (FRONT CENTER SENSOR LH)

CHECK FOR DTC (C1AEC)

REPLACE NO. 1 ULTRASONIC SENSOR (FRONT CENTER SENSOR RH)

CHECK FOR DTC (C1AEC)

REPLACE NO. 1 ULTRASONIC SENSOR (FRONT CORNER SENSOR RH)

CHECK FOR DTC (C1AEC)

CHECK HARNESS AND CONNECTOR (PARKING ASSIST ECU - FRONT CORNER SENSOR LH)

CHECK HARNESS AND CONNECTOR (FRONT CORNER SENSOR LH - FRONT CENTER SENSOR LH)

CHECK HARNESS AND CONNECTOR (FRONT CENTER SENSOR LH - FRONT CENTER SENSOR RH)

CHECK HARNESS AND CONNECTOR (FRONT CENTER SENSOR RH - FRONT CORNER SENSOR RH)

INSPECT NO. 1 ULTRASONIC SENSOR (FRONT CORNER SENSOR LH)

INSPECT NO. 1 ULTRASONIC SENSOR (FRONT CENTER SENSOR LH)

INSPECT NO. 1 ULTRASONIC SENSOR (FRONT CENTER SENSOR RH)

REPLACE NO. 1 ULTRASONIC SENSOR (FRONT CORNER SENSOR RH)

CHECK FOR DTC (C1AEC)

REPLACE NO. 1 ULTRASONIC SENSOR (FRONT CENTER SENSOR LH)

CHECK FOR DTC (C1AEC)

REPLACE NO. 1 ULTRASONIC SENSOR (FRONT CENTER SENSOR RH)

CHECK FOR DTC (C1AEC)

REPLACE NO. 1 ULTRASONIC SENSOR (FRONT CORNER SENSOR LH)

CHECK FOR DTC (C1AEC)
