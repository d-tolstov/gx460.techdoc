for Preparation Click here


DESCRIPTION

The multi-media interface ECU sends a sound signal to the display and navigation module display through this circuit.
If there is an open or short in the circuit, sound cannot be heard from the speakers even if there is no malfunction in the stereo component amplifier assembly, radio receiver assembly, display and navigation module display or speakers.
