DTC    U110B    Lost Communication with Parking Assist Control Module "B"

for Preparation Click here


DESCRIPTION


DTC Code
DTC Detection Condition
Trouble Area
U110B
There is no communication from the parking assist ECU.
  1. Power source or inside parking assist ECU
  2. Parking assist ECU branch wire or connector
  3. Parking assist ECU
  1. For vehicles with a LEXUS parking assist-sensor system (with parking assist system and/or side monitor system) only.

WIRING DIAGRAM




INSPECTION PROCEDURE

NOTICE:
Inspect the fuses for circuits related to this system before performing the following inspection procedure.

HINT:
Operating the engine switch, any switches or any doors triggers related ECU and sensor communication with the CAN, which causes resistance variation.

1.DISCONNECT CABLE FROM NEGATIVE BATTERY TERMINAL

  1. Disconnect the cable from the negative (-) battery terminal before measuring the resistances of the main wire and branch wire.

    CAUTION:
    Wait at least 90 seconds after disconnecting the cable from the negative (-) battery terminal to disable the SRS system.

    NOTICE:
    1. w/ Navigation System (for HDD):
      After the engine switch is turned off, the display and navigation module display (HDD navigation system) records various types of memory and settings. As a result, after turning the engine switch off, be sure to wait for the time specified in the following table before disconnecting the cable from the negative (-) battery terminal.

      Waiting Time before Disconnecting Cable from Negative (-) Battery Terminal:

      Condition
      		Waiting Time
      Vehicle enrolled in G-BOOK system
      		6 minutes
      Vehicle not enrolled in G-BOOK system
      		1 minute

    2. When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).


NEXT


2.CHECK FOR OPEN IN CAN BUS WIRE (PARKING ASSIST ECU BRANCH WIRE)



  1. Disconnect the I3 parking assist ECU connector.

  1. Measure the resistance according to the value(s) in the table below.

    Standard Resistance:
    Tester Connection
    		Switch Condition
    		Specified Condition
    I3-22 (CANH) - I3-21 (CANL)
    		Engine switch off
    		54 to 69 Ω

3.CHECK HARNESS AND CONNECTOR (PARKING ASSIST ECU - BATTERY AND BODY GROUND)



  1. Connect the cable to the negative (-) battery terminal.

    NOTICE:
    When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).

  1. Measure the resistance according to the value(s) in the table below.

    Standard Resistance:
    Tester Connection
    		Condition
    		Specified Condition
    I3-3 (GND1) - Body ground
    		Always
    		Below 1 Ω

  1. Measure the voltage according to the value(s) in the table below.

    Standard Voltage:
    Tester Connection
    		Condition
    		Specified Condition
    I3-1 (+B) - Body ground
    		Always
    		11 to 14 V
    I3-7 (IG) - Body ground
    		Engine switch on (IG)
    		11 to 14 V
