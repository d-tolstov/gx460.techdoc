6.INSPECT METER WARNING LIGHT

  1. Check if a warning light on the combination meter assembly comes on or a warning message on the multi-information display is displayed even when there is no problem.

    OK:
    There are no problems with warning lights or warning messages.

    HINT:
    The warning notification function of the G-BOOK system uses the warning notification signals sent and received by the meter and gauge system. Therefore, if the meter and gauge system is not operating normally, the warning notification function will not operate normally.



NG
GO TO METER / GAUGE SYSTEM (Click here)
OK
