1. INSTALL INSTRUMENT PANEL CUP HOLDER ASSEMBLY



  1. Insert the instrument panel cup holder into the upper console panel as shown in the illustration.

  1. Install the instrument panel cup holder with the 4 screws.

  1. Connect the connector and attach the 3 clamps.

  1. Slide the cup holder and install the 2 screws.



2. INSTALL REFRESHING SEAT SWITCH

  1. Attach the 4 claws to install the switch.

  1. Connect the connector.

    HINT:
    Use the same procedure for both switches.



3. INSTALL SHIFT POSITION INDICATOR

  1. Install the shift position indicator to the upper console panel with the screw.

  1. Attach the connector clamp.



4. INSTALL COMBINATION SWITCH ASSEMBLY

  1. Install the combination switch (absorber control switch) with the 4 screws.



5. INSTALL SHIFTING HOLE COVER SUB-ASSEMBLY

  1. Attach the 7 claws and 2 guides to install the shifting hole cover.



6. INSTALL REAR CONSOLE BOX HOLDER CUP

  1. Attach the 4 claws to install the rear console box holder cup.

  1. Install the 2 screws.



7. INSTALL CONSOLE COMPARTMENT DOOR SUB-ASSEMBLY

  1. Install the console compartment door with the 6 screws.
