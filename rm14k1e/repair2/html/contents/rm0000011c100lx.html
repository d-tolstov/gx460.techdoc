PARKING BRAKE PEDAL > INSTALLATION

for Preparation Click here



1. INSTALL PARKING PEDAL PAD

  1. Install the parking pedal pad.



2. INSTALL NO. 1 PARKING BRAKE CABLE ASSEMBLY

  1. Connect the No. 1 parking brake cable No. 3 clamp to the No. 1 parking brake cable.

  1. Temporarily install the No. 1 parking brake cable assembly with the adjusting nut and lock nut.



  1. Install the No. 1 parking brake cable assembly with the 4 nuts and 3 bolts.



    Torque:
    for nut A:
    6.0 N*m{ 61 kgf*cm , 53 in.*lbf }
    for bolt B:
    13 N*m{ 127 kgf*cm , 9 ft.*lbf }




  1. Pass the No. 1 parking brake cable through the parking brake pedal.

  1. Install the clip.

  1. Bend the parking brake pedal claw.

  1. Temporarily install the adjusting nut and lock nut to the No. 1 parking brake cable.



3. INSTALL PARKING BRAKE CONTROL PEDAL ASSEMBLY

  1. Install the parking brake control pedal assembly with the 3 nuts.

    Torque:
    13 N*m{ 127 kgf*cm , 9 ft.*lbf }

  1. Connect the parking brake switch connector.



4. INSTALL TRANSMISSION FLOOR SHIFT ASSEMBLY

  1. Install the transmission floor shift assembly (Click here).



5. INSTALL CENTER AIRBAG SENSOR ASSEMBLY

  1. Turn the engine switch off.

  1. Disconnect the cable from the negative (-) battery terminal.

    CAUTION:
    Wait at least 90 seconds after disconnecting the cable from the negative (-) battery terminal to disable the SRS system.

    NOTICE:
    1. w/ Navigation System (for HDD):
      After the engine switch is turned off, the display and navigation module display (HDD navigation system) records various types of memory and settings. As a result, after turning the engine switch off, be sure to wait for the time specified in the following table before disconnecting the cable from the negative (-) battery terminal.

      Waiting Time before Disconnecting Cable from Negative (-) Battery Terminal:

      Condition
      		Waiting Time
      Vehicle enrolled in G-BOOK system
      		6 minutes
      Vehicle not enrolled in G-BOOK system
      		1 minute

    2. When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).

  1. Install the airbag sensor with the 3 bolts.

    Torque:
    18 N*m{ 184 kgf*cm , 13 ft.*lbf }

    NOTICE:
    1. If the airbag sensor has been dropped, or there are any cracks, dents or other defects in the case, bracket or connector, replace it with a new one.
    2. When installing the airbag sensor, be careful that the SRS wiring does not interfere with other parts and that it is not pinched between other parts.

  1. Check that the airbag sensor is not loose.

  1. Connect the connector.



6. INSTALL YAW RATE AND ACCELERATION SENSOR

  1. Attach the clamp to the hole of the sensor bracket.

  1. Connect the connector.

    NOTICE:
    Securely connect the connector.

  1. Install the yaw rate and acceleration sensor with the 2 bolts.

    Torque:
    15 N*m{ 148 kgf*cm , 11 ft.*lbf }

    NOTICE:
    1. If the sensor is dropped, replace it.
    2. Make sure the bracket rotation stopper touches the installation position.
    3. Make sure that the sensor is oriented correctly.
    4. When installing the sensor, make sure to avoid impacts to the sensor.



7. INSTALL TURN SIGNAL FLASHER ASSEMBLY

  1. Install the turn signal flasher with the nut.

  1. Connect the connector.



8. INSTALL REAR CONSOLE BOX ASSEMBLY

  1. Install the rear console box assembly (Click here).



9. INSTALL LOWER NO. 1 INSTRUMENT PANEL AIRBAG ASSEMBLY

  1. Install the lower No. 1 instrument panel airbag assembly (Click here).



10. ADJUST PARKING BRAKE SHOE CLEARANCE AND PARKING BRAKE PEDAL TRAVEL

  1. Remove the lower instrument panel finish panel sub-assembly (Click here).

  1. Completely release the parking brake pedal.




  1. Loosen the lock nut and adjusting nut to completely release the parking brake cable.

  1. Temporarily install the hub nuts to the hub bolts.

    HINT:
    Securely install the hub nuts to the rear disc.




  1. Turn the shoe adjuster so that it expands until the disc locks.
    Text in Illustration 

    		Shoe Adjuster Contracts

    		Shoe Adjuster Expands

  1. Turn the shoe adjuster so that it contracts until the disc can rotate smoothly.

    Standard:
    8 notches

  1. Check that there is no brake drag against the shoe.




  1. Turn the adjusting nut until the parking brake pedal travel is corrected to be within the specified range.

    Standard parking brake pedal travel:
    5 to 7 clicks at 294 N (30 kgf, 66 lbf)

    Text in Illustration 
    *1
    		Adjusting Nut

  1. Using a wrench or an equivalent tool, hold the adjusting nut and tighten the lock nut.

    Torque:
    5.4 N*m{ 55 kgf*cm , 48 in.*lbf }

  1. Operate the parking brake pedal 3 to 4 times and check the parking brake pedal travel.

  1. Check that there is no brake drag against the shoe.




  1. Additionally, excessive pedal travel can be adjusted on the No. 1 parking brake cable as follows.

    1. Loosen the No. 1 parking brake cable lock nut.

    2. Turn the No. 1 parking brake cable adjustment nut until the parking brake pedal travel is correct.

      Standard parking brake pedal travel:
      5 to 7 clicks at 294 N (30 kgf, 66 lbf)

    3. Tighten the No. 1 parking brake cable lock nut.

      Torque:
      13 N*m{ 127 kgf*cm , 9 ft.*lbf }

      NOTICE:
      Pay attention that the adjustment nut does not rotate together with the lock nut during tightening.

    4. Depress and release the parking brake pedal 3 or 4 times. Then check the number of clicks when depressing the pedal.

      Standard:
      5 to 7 clicks

      If the result is not as specified, adjust the parking brake pedal travel again.

  1. Remove the hub nuts from the hub bolts.

  1. Install the lower instrument panel finish panel sub-assembly (Click here).

  1. When operating the parking brake lever, check that the brake warning light comes on.

    OK:
    The brake warning light always illuminates at the first click.
