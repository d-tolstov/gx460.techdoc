REAR SPEED SENSOR > INSTALLATION

for Preparation Click here



1. INSTALL SKID CONTROL SENSOR WIRE

  1. Install the 2 sensor clamps with the 2 bolts.

    Torque:
    13 N*m{ 127 kgf*cm , 9 ft.*lbf }

    NOTICE:
    Make sure the clamp rotation stopper touches the installation position.

  1. Connect the connector.

    NOTICE:
    Securely connect the connector.

  1. Attach the connector.

  1. Attach the 8 clamps.

    NOTICE:
    When attaching the clamps, do not twist the wire harness.



2. INSTALL REAR SPEED SENSOR LH

  1. Install the speed sensor with the nut.

    Torque:
    7.8 N*m{ 80 kgf*cm , 69 in.*lbf }

    NOTICE:
    1. Make sure there are no pieces of iron or other foreign matter attached to the sensor tip.
    2. While inserting the speed sensor into the axle hole, do not strike or damage the sensor tip.
    3. After installing the speed sensor, make sure there is no clearance or foreign matter between the sensor stay part and the axle.
    4. Make sure there is no foreign matter attached to the speed sensor rotor.

  1. Connect the speed sensor connector.

    NOTICE:
    Securely connect the connector.



3. INSTALL REAR SPEED SENSOR RH

HINT:
Use the same procedure described for the LH side.



4. INSTALL REAR WHEEL

Torque:
112 N*m{ 1142 kgf*cm , 83 ft.*lbf }
