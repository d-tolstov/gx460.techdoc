GENERAL
This system has the following functions: manual slide open and close, auto slide open and close, manual tilt up and down, auto tilt up and down, jam protection, key off operation, key-linked operation and wireless transmitter-linked operation.

HINT:
If the driver side door is opened with the engine switch off and the sliding roof open, the buzzer sounds.


FUNCTION OF MAIN COMPONENT

Component
Outline
Sliding roof drive gear sub-assembly (Sliding roof ECU)
The sliding roof ECU controls the rotational direction of the motor, which tilts or slides the sliding roof glass.
Map light assembly (Sliding roof switch)
The sliding roof switch outputs operation signals to the sliding roof ECU.

SYSTEM OPERATION
The sliding roof has the following features.

Function
Outline
Manual slide open and close
  1. This function causes the sliding roof to open (or close) when the slide open switch (or slide close switch) is pressed for less than 0.3 seconds.
  2. The sliding roof stops as soon as the switch is released.
Manual tilt up and down
  1. This function causes the sliding roof to tilt up (or tilt down) when the tilt up switch (or tilt down switch) is pressed for less than 0.3 seconds.
  2. The sliding roof stops as soon as the switch is released.
Auto slide open and close
This function causes the sliding roof to be fully opened (or closed) when the slide open switch (or slide close switch) is pressed for 0.3 seconds or more.
Auto tilt up and down
This function causes the sliding roof to be fully tilted up (or tilted down) when the tilt up switch (or tilt down switch) is pressed for 0.3 seconds or more.
Jam protection function
The jam protection function automatically stops the sliding roof and moves it open halfway (or fully tilts it up) if a foreign object gets jammed in the sliding roof during close or tilt down operation.
Key-off operation
The key-off operation function makes it possible to operate the sliding roof for approximately 43 seconds after the engine switch is turned off if the driver side door is not opened.
Wireless transmitter-linked operation
  1. When the main body ECU (multiplex network body ECU) receives an unlock signal from the door control transmitter continuously for 3.0 seconds or more, the sliding roof ECU controls the sliding roof motor in accordance with the signal to slide open or tilt up the sliding roof.
  2. When the main body ECU (multiplex network body ECU) receives a lock signal from the door control transmitter continuously for 3.0 seconds or more, the sliding roof ECU controls the sliding roof motor in accordance with the signal to slide close or tilt down the sliding roof.

HINT:
This function is linked with the power window wireless transmitter-linked function. Make sure that both the power window wireless transmitter-linked function (the default setting is OFF) and sliding roof wireless transmitter-linked function customize settings are ON before performing an inspection.

Key-linked operation
  1. When the key is not in the vehicle, the driver side door is unlocked, and the key in the driver side door is turned and held in the unlock direction for 2.0 seconds or more, the sliding roof ECU activates the sliding roof motor to slide open or tilt up the sliding roof while the key is turned.
  2. When the key is not in the vehicle, the driver side door is locked, and the key in the driver side door is turned and held in the lock direction for 2.0 seconds or more, the sliding roof ECU activates the sliding roof motor to slide close or tilt down the sliding roof while the key is turned.

HINT:
This function is linked with the power window key-linked function. Make sure that both the power window key-linked function (the default setting is OFF) and sliding roof key-linked function customize settings are ON before performing an inspection.

Sliding roof open warning
When the engine switch is turned from on (IG) to off and the driver side door is opened with the sliding roof open, the buzzer in the combination meter sounds (a 0.9 second sound is repeated for approximately 8 seconds). Then, the warning message appears on the multi-information display.
