PERSONAL LIGHT (for Rear Door) > INSTALLATION

for Preparation Click here


HINT:
  1. Use the same procedure for the RH side and LH side.
  2. The procedure listed below is for the LH side.



1. INSTALL REAR DOOR INSIDE HANDLE ILLUMINATION LIGHT ASSEMBLY LH

  1. Connect the connector.

  1. Attach the claw to install the light.



2. INSTALL REAR DOOR INNER GLASS WEATHERSTRIP LH



  1. Attach the 3 claws to install the rear door inner glass weatherstrip as shown in the illustration.



3. INSTALL REAR DOOR TRIM BOARD SUB-ASSEMBLY LH



  1. Install the clamp to the door panel.

  1. Connect the rear door lock remote control cable assembly and rear door inside locking cable assembly.

  1. Connect each connector.




  1. Attach the rear door trim board sub-assembly by attaching the 4 claws of the rear door inner glass weatherstrip as shown in the illustration.

  1. Attach the 9 clips and install the rear door trim board sub-assembly.

  1. Install the 3 screws.



4. INSTALL REAR POWER WINDOW REGULATOR SWITCH ASSEMBLY

  1. Connect the connector.

  1. Attach the 2 clips, 4 claws and guide to install the rear power window regulator switch assembly with rear door armrest base panel.



5. INSTALL REAR ARMREST ASSEMBLY LH

  1. Attach the 7 claws and 7 guides to install the rear armrest assembly.



6. INSTALL REAR DOOR INSIDE HANDLE BEZEL PLUG LH

  1. Attach the 3 claws to install the rear door inside handle bezel plug.
