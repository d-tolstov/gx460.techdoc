RADIATOR GRILLE > REASSEMBLY

for Preparation Click here



1. INSTALL RADIATOR GRILLE BRACKET LH

  1. Install the radiator grille bracket with the bolt.



2. INSTALL RADIATOR GRILLE BRACKET RH

HINT:
Use the same procedure described for the LH side.



3. INSTALL FRONT TELEVISION CAMERA ASSEMBLY (w/ Wide View Front Monitor System)

  1. Install the front television camera with the 3 screws.



4. INSTALL RADIATOR GRILLE (OR FRONT PANEL) EMBLEM (w/ Dynamic Radar Cruise Control System)

  1. Attach the claw to install the radiator grille (or front panel) emblem.

  1. Install 2 new spring nuts.



5. INSTALL RADIATOR GRILLE EMBLEM BASE (w/o Dynamic Radar Cruise Control System)

  1. Attach the claw to install the radiator grille emblem base.

  1. Install 2 new spring nuts.



6. INSTALL RADIATOR GRILLE (OR FRONT PANEL) EMBLEM (w/o Dynamic Radar Cruise Control System)



HINT:
When installing the radiator grille (or front panel) emblem, heat the vehicle body and radiator grille (or front panel) emblem using a heat light.

Standard:
Item
Temperature
Vehicle Body
40 to 60°C (104 to 140°F)
Emblem
20 to 30°C (68 to 86°F)

NOTICE:
Do not heat the vehicle body or radiator grille (or front panel) emblem excessively.

  1. Clean the vehicle body surface.

    1. Using a heat light, heat the vehicle body surface.

    2. Remove the double-sided tape from the vehicle body.

    3. Wipe off any tape adhesive residue with cleaner.

  1. Install a new radiator grille (or front panel) emblem.

    1. Using a heat light, heat the vehicle body and a new radiator grille (or front panel) emblem.

    2. Remove the peeling paper from the face of the radiator grille (or front panel) emblem.

      HINT:
      After removing the peeling paper, keep the exposed adhesive free from foreign matter.

    3. Align the boss positions and install the radiator grille (or front panel) emblem.

      HINT:
      Press the radiator grille (or front panel) emblem firmly to install it.

      		Boss



7. INSTALL HOOD TO RADIATOR GRILLE SEAL

  1. Attach the 15 clips to install the hood to radiator grille seal.
