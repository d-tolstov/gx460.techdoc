for Preparation Click here



1. INSTALL DISPLAY AND NAVIGATION MODULE DISPLAY (w/ Navigation System)

  1. Connect the connectors.




  1. Insert the display and navigation module display and attach the 7 clips on the backside of the display and navigation module display.

    NOTICE:
    When inserting the display, do not press the knobs on it.

  1. Install the display and navigation module display with the 4 bolts.

    Torque:
    12 N*m{ 122 kgf*cm , 9 ft.*lbf }



2. INSTALL AIR CONDITIONING CONTROL ASSEMBLY (w/o Navigation System)

  1. Connect the connectors.

  1. Attach the 7 clips and install the air conditioning control assembly.

  1. Install the 2 bolts.

    Torque:
    12 N*m{ 122 kgf*cm , 9 ft.*lbf }



3. INSTALL RADIO RECEIVER ASSEMBLY WITH BRACKET


  1. w/o Navigation System:

    1. Connect the connectors.




    2. Insert the radio receiver to attach the 6 clips and rib on its backside.

      NOTICE:
      When inserting the radio receiver, do not press the knobs on it.

    3. Install the radio receiver with the 4 bolts.

      Torque:
      13 N*m{ 127 kgf*cm , 9 ft.*lbf }



  1. w/ Navigation System:

    1. Connect the connectors.




    2. Insert the radio receiver to attach the 6 clips and 3 ribs on its backside.

      NOTICE:
      When inserting the radio receiver, do not press the knobs on it.

    3. Install the radio receiver with the 4 bolts.

      Torque:
      13 N*m{ 127 kgf*cm , 9 ft.*lbf }



4. INSTALL FRONT UPPER CONSOLE PANEL GARNISH

  1. Attach the 2 clips to install the front upper console panel garnish.



5. INSTALL UPPER CONSOLE PANEL SUB-ASSEMBLY

  1. Connect each connector.

  1. Attach the 7 clips to install the upper console panel.



6. INSTALL SHIFT LEVER KNOB SUB-ASSEMBLY



  1. Install the shift lever knob and twist it in the direction indicated by the arrow.



7. INSTALL NO. 2 INSTRUMENT PANEL REGISTER ASSEMBLY

  1. Attach the 8 clips to install the No. 2 instrument panel register.



8. INSTALL CENTER INSTRUMENT CLUSTER FINISH PANEL GARNISH



  1. Attach the 13 clips to install the center instrument cluster finish panel garnish together with the instrument cluster finish panel garnish.

9. INSTALL NO. 1 INSTRUMENT PANEL FINISH CUSHION

  1. Connect the connector.

  1. Attach the 4 clips to install the No. 1 instrument panel finish cushion.
