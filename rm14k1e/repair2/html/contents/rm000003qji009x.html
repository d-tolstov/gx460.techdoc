for Preparation Click here



1. INSTALL HEADLIGHT DIMMER SWITCH ASSEMBLY



  1. Install the headlight dimmer switch to the steering column, making sure that the stopper protrusions and cutout parts align.




  1. While loosening the band clamp, attach the claw to install the headlight dimmer switch.

  1. Connect the connector.



2. INSTALL WINDSHIELD WIPER SWITCH ASSEMBLY

  1. Attach the claw to install the windshield wiper switch assembly.

  1. Connect the connector.
