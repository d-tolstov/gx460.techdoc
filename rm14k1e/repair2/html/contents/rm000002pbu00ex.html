MAYDAY BATTERY > INSTALLATION

for Preparation Click here



1. INSTALL MAYDAY BATTERY



  1. Install the bracket to the mayday battery.



2. INSTALL MAYDAY BATTERY WITH BRACKET



  1. Connect the connector.




  1. Attach the 2 claws and install the mayday battery with bracket with the bolt.

    Torque:
    13 N*m{ 127 kgf*cm , 9 ft.*lbf }

  1. Attach the 4 claws to install the cover.



3. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).



4. CHECK SRS WARNING LIGHT

  1. Check the SRS warning light (Click here).



5. PERFORM REGISTRATION

  1. Perform registration (Click here).

    NOTICE:
    When replacing the mayday battery or DCM (telematics transceiver) together with the mayday battery, perform registration.
