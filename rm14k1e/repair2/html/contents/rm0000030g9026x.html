HINT:
  1. If a DTC is output during the DTC check, check the parts listed in the table below and proceed to the "See page" given.
  2. *1: "Comes on" means the Malfunction Indicator Lamp (MIL) illuminates.
  3. *2: "DTC stored" means the ECM stores the trouble code if the ECM detects the DTC detection condition.
  4. *3: A message is displayed on the multi-information display.
  5. *4: w/ OBD
  6. *5: w/o OBD
  7. These DTCs may be output when the clutches, brakes, gear components, etc. inside the automatic transmission are damaged.
