for Preparation Click here



1. INSTALL WATER PUMP ASSEMBLY



  1. Install a new gasket and the water pump with the 8 bolts shown in the illustration.

    Torque:
    for bolt A:
    47 N*m{ 479 kgf*cm , 35 ft.*lbf }
    for bolt B:
    23 N*m{ 235 kgf*cm , 17 ft.*lbf }
    for bolt C:
    20 N*m{ 204 kgf*cm , 15 ft.*lbf }



2. INSTALL WATER PUMP PULLEY

  1. Temporarily install the water pump pulley with the 4 bolts.




  1. Using SST, hold the water pump pulley and tighten the 4 bolts.

    SST
    09960-10010   (09962-01000, 09963-01000)

    Torque:
    21 N*m{ 214 kgf*cm , 15 ft.*lbf }



3. INSTALL WATER INLET HOUSING

  1. Install a new gasket to the water pump.

  1. Install the water inlet housing with the 3 bolts.

    Torque:
    21 N*m{ 214 kgf*cm , 15 ft.*lbf }

7. INSTALL FAN SHROUD

  1. Install the fan pulley to the water pump.

  1. Place the fan shroud together with the fluid coupling between the radiator and engine.

    NOTICE:
    Be careful not to damage the radiator core.

  1. Temporarily install the fluid coupling to the water pump with the 4 nuts. Tighten the nuts as much as possible by hand.




  1. Set the fan shroud on the radiator as shown in the illustration.

  1. Install the shroud with the 2 bolts.

    Torque:
    5.0 N*m{ 51 kgf*cm , 44 in.*lbf }

  1. Install the fan and generator V-ribbed belt (Click here).

  1. Tighten the 4 nuts of the fluid coupling.

    Torque:
    21 N*m{ 214 kgf*cm , 15 ft.*lbf }
