FREEZE FRAME DATA

  1. Whenever a DTC is stored, the skid control ECU stores the current vehicle (sensor) state as freeze frame data.

  2. The skid control ECU stores the number of times (maximum: 31) the engine switch has been turned from off to on (IG) since the last time the ABS/VSC was activated.

    HINT:
    1. If the vehicle is stopped or at a low speed (7 km/h (4 mph) or less), or if a DTC is stored, the skid control ECU will stop counting the number of times the engine switch has been turned from off to on (IG).
    2. Freeze frame data at the time the ABS/VSC operates:
      The skid control ECU stores and updates data whenever the ABS/VSC operates.
      When the ECU stores data at the time a DTC is stored, the data stored during ABS/VSC operation is cleared.
    3. Freeze frame data at the time a DTC is stored:
      When the skid control ECU stores data at the time a DTC is stored, no updates will be performed until the data is cleared.


CHECK FREEZE FRAME DATA

  1. Connect the intelligent tester to the DLC3.

  2. Turn the engine switch on (IG).

  3. Turn the intelligent tester on.

  4. Enter the following menus: Chassis / ABS/VSC/TRC / DTC.

  5. According to the display on the intelligent tester, select the trouble code data display with freeze frame data.

CLEAR FREEZE FRAME DATA

NOTICE:
Clearing the DTCs will also clear the freeze frame data.

  1. Connect the intelligent tester to the DLC3.

  2. Turn the engine switch on (IG).

  3. Clear the DTCs and freeze frame data (Click here).
