DTC    B15DF    USB Communication Malfunction

for Preparation Click here


DESCRIPTION

This DTC is stored when a malfunction occurs in communication error with USB device or "iPod".

DTC Code
DTC Detection Condition
Trouble Area
B15DF
When one of the conditions below is met:
  1. Communication error with USB device
  2. "iPod" control software of system microprocessor malfunction
  3. Communication error with "iPod"
  1. Multi-media interface ECU
  2. USB device or "iPod"

INSPECTION PROCEDURE

HINT:
After the inspection is completed, clear the DTCs.

1.REPLACE "iPod" OR USB DEVICE

  1. Disconnect the USB device or "iPod" from the No. 1 stereo jack adapter assembly.

  1. Turn the engine switch off.

    HINT:
    When this DTC has been stored, it is necessary to turn off the engine switch to make it possible for the vehicle to recognize a new device when it is connected.

  1. Turn the engine switch on (ACC).

  1. Connect a known good USB device or "iPod" to the No. 1 stereo jack adapter assembly.

    HINT:
    1. If the malfunction occurred when a USB device was in use, use another USB device for the inspection. If the malfunction occurred when an "iPod" was in use, use another "iPod" for the inspection.
    2. Refer to System Description for compatible formats and versions (Click here).
