3.CHECK SYSTEM



  1. Check that the remote control operation of the rear system is not locked by the display and navigation module display.

    1. Check if "Rear System Locked" appears on the television display.

      OK:
      The message does not appear.

      HINT:
      If the rear system is locked and the remote control is used in the rear seat area, an operation not possible message (for example, "Rear System Locked") is displayed. If the message is displayed, enter the multi-display assembly settings screen and change "Rear System Lock" to Off.
