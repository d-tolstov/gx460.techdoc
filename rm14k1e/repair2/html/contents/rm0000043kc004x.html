DTC    C1A0A    Front Radar Sensor Region Code Mismatch

for Preparation Click here


DESCRIPTION

The driving support ECU uses the millimeter wave radar sensor assembly to detect obstacles in front of the vehicle. Based on this information, the driving support ECU sends pre-crash control operation signals.
When the millimeter wave radar sensor assembly is replaced with a new one, it acquires the country specification information from the driving support ECU. Also, when the driving support ECU is replaced with a new one, it receives country specification information from the main body ECU. When country specification information in the sensor and driving support ECU do not match, DTC C1A0A is output.

DTC Code
DTC Detection Condition
Trouble Area
C1A0A
While the initial check is running, the engine switch on (IG) and the pre-crash safety system is operating, the country specification information in the driving support ECU and millimeter wave radar sensor assembly do not match for 1 second.
  1. Millimeter wave radar sensor assembly
  2. Driving support ECU
