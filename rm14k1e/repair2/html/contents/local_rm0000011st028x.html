DTC    B2412    Headlight Swivel Motor LH Malfunction


DESCRIPTION

WIRING DIAGRAM

INSPECTION PROCEDURE

CHECK FOR DTC

PERFORM ACTIVE TEST USING INTELLIGENT TESTER (SWIVEL MOTOR)

CHECK HEADLIGHT UNIT (OPERATION)

CHECK HARNESS AND CONNECTOR (HEADLIGHT SWIVEL ECU ASSEMBLY - BATTERY)

CHECK HARNESS AND CONNECTOR (HEADLIGHT SWIVEL ECU ASSEMBLY - BODY GROUND)

CHECK HARNESS AND CONNECTOR (HEADLIGHT SWIVEL ECU ASSEMBLY - HEADLIGHT UNIT)

CHECK HEADLIGHT UNIT (OPERATION)
