DTC    P0500    Vehicle Speed Sensor Malfunction

for Preparation Click here


DESCRIPTION

The speed sensor detects the wheel speed and sends the appropriate signals to the skid control ECU. The skid control ECU converts these wheel speed signals into a pulse signal and outputs it to the ECM via the combination meter assembly. The ECM determines the vehicle speed based on the frequency of this pulse signal.

DTC Code
DTC Detection Condition
Trouble Area
P0500
The vehicle speed signal from the skid control ECU is cut for 0.14 seconds or more while cruise control is in operation.
  1. Vehicle speed sensor signal circuit
  2. Vehicle speed sensor
  3. Combination meter assembly
  4. ECM

INSPECTION PROCEDURE

This DTC may be stored under the SFI system at the same time. In that case, perform troubleshooting for the SFI system first.
1.CHECK FOR DTC

  1. Clear the DTCs (Click here).

  1. Perform the following to make sure the DTC detection conditions are met.

    HINT:
    If the detection conditions are not met, the system cannot detect the malfunction.

    1. Drive the vehicle for more than 10 seconds when the engine coolant temperature is more than 20°C (68°F).

  1. Check for DTCs (Click here).

    OK:
    DTC P0500 is not output.
