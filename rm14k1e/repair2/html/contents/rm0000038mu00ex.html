REAR FLOOR AIRBAG SENSOR > INSTALLATION

for Preparation Click here



1. INSTALL REAR FLOOR SIDE AIRBAG SENSOR

  1. Turn the engine switch off.

  1. Disconnect the cable from the negative (-) battery terminal.

    CAUTION:
    Wait at least 90 seconds after disconnecting the cable from the negative (-) battery terminal to disable the SRS system.

    NOTICE:
    1. w/ Navigation System (for HDD):
      After the engine switch is turned off, the display and navigation module display (HDD navigation system) records various types of memory and settings. As a result, after turning the engine switch off, be sure to wait for the time specified in the following table before disconnecting the cable from the negative (-) battery terminal.

      Waiting Time before Disconnecting Cable from Negative (-) Battery Terminal:

      Condition
      		Waiting Time
      Vehicle enrolled in G-BOOK system
      		6 minutes
      Vehicle not enrolled in G-BOOK system
      		1 minute

    2. When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here)

  1. Install the rear floor airbag sensor with the bolt.

    Torque:
    9.0 N*m{ 92 kgf*cm , 80 in.*lbf }

    NOTICE:
    1. If the rear floor airbag sensor has been dropped, or there are any cracks, dents or other defects in the case, bracket or connector, replace it with a new one.
    2. When installing the rear floor airbag sensor, be careful that the SRS wiring does not interfere with other parts and that it is not pinched between other parts.

  1. Check that is the rear floor airbag sensor not loose.

  1. Connect the connector.



2. INSTALL REAR FLOOR SIDE AIRBAG SENSOR COVER

  1. Attach the clip and claw to install the cover.

  1. Return the floor carpet to its original position.
