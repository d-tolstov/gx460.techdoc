DTC    B15C4    Airbag Signal Malfunction/Not Input

for Preparation Click here


DESCRIPTION

If the telematics transceiver detects an error in the communication between the telematics transceiver and center airbag sensor assembly during the telematics transceiver self-check, this DTC will be stored.

DTC Code
DTC Detection Condition
Trouble Area
B15C4
There is an airbag signal problem.
  1. SRS airbag system
  2. Harness or connector
  3. Center airbag sensor assembly
  4. Telematics transceiver
