HINT:

1. INSTALL INSTRUMENT PANEL SAFETY PAD SUB-ASSEMBLY

  1. Attach the 2 guides to install the instrument panel safety pad.

  1. Connect the connectors and attach the clamps and claws.

  1. Install the 2 passenger airbag bolts <F>.

    Torque:
    20 N*m{ 204 kgf*cm , 15 ft.*lbf }

  1. Install the 6 bolts <D> and nut <E>.



  1. Install the front floor carpet.




    Text in Illustration 
    *1
    		Bolt <D>
    		*2
    		Nut <E>
    *3
    		Bolt <F>
    		*4
    		Guide



2. INSTALL FRONT SEAT ASSEMBLY LH

  1. Install the front seat assembly (Click here).



3. INSTALL FRONT SEAT ASSEMBLY RH

HINT:
Use the same procedure described for the LH side.



4. INSTALL FRONT NO. 3 SPEAKER ASSEMBLY

  1. Attach the clip and claw to install the front No. 3 speaker.

  1. Connect the connector.



5. INSTALL FRONT NO. 2 SPEAKER ASSEMBLY

  1. Connect the connector.

  1. Align the speaker with the positioning pins of the instrument panel and temporarily install the speaker.

  1. Install the front No. 2 speaker with the 2 screws.

    NOTICE:
    1. Do not touch the cone of the speaker.
    2. When installing the speaker to the instrument panel, be careful that the wires do not get caught between parts.



6. INSTALL NO. 1 INSTRUMENT PANEL SPEAKER PANEL SUB-ASSEMBLY

  1. Attach the 2 clips, claw and 2 guides to install the No. 1 instrument panel speaker panel.



7. INSTALL NO. 2 INSTRUMENT PANEL SPEAKER PANEL SUB-ASSEMBLY

HINT:
Use the same procedure described for the No. 1 instrument panel speaker panel.



8. INSTALL FRONT NO. 4 SPEAKER ASSEMBLY

  1. Connect the speaker connector.

  1. Align the speaker with the positioning pins of the instrument panel and temporarily install the speaker.

  1. Install the front No. 4 speaker with the 2 screws.

    NOTICE:
    1. Do not touch the cone of the speaker.

9. INSTALL UPPER INSTRUMENT CLUSTER FINISH PANEL

  1. Attach the 11 clips to install the upper instrument cluster finish panel.



10. INSTALL DISPLAY AND NAVIGATION MODULE DISPLAY (w/ Navigation System)

  1. Connect the connectors.




  1. Insert the display and navigation module display and attach the 7 clips on the backside of the display and navigation module display.

    NOTICE:
    When inserting the display, do not press the knobs on it.

  1. Install the display and navigation module display with the 4 bolts.

    Torque:
    12 N*m{ 122 kgf*cm , 9 ft.*lbf }



11. INSTALL AIR CONDITIONING CONTROL ASSEMBLY (w/o Navigation System)

  1. Connect the connectors.

  1. Attach the 7 clips and install the air conditioning control assembly.

  1. Install the 2 bolts.

    Torque:
    12 N*m{ 122 kgf*cm , 9 ft.*lbf }

14. INSTALL TELEMATICS TRANSCEIVER

  1. Insert the telematics transceiver with bracket into the vehicle.




  1. Install the telematics transceiver with bracket with the bolt and nut.

    Torque:
    for bolt A:
    13 N*m{ 127 kgf*cm , 9 ft.*lbf }
    for nut B:
    8.0 N*m{ 82 kgf*cm , 71 in.*lbf }

  1. Connect the connectors.



15. INSTALL MULTI-MEDIA INTERFACE ECU

  1. Install the multi-media interface ECU with the 2 nuts.

    Torque:
    5.5 N*m{ 56 kgf*cm , 49 in.*lbf }

  1. Connect the 3 connectors.



16. INSTALL FOUR WHEEL DRIVE CONTROL ECU

  1. Install the four wheel drive control ECU with the bolt.

    Torque:
    13 N*m{ 127 kgf*cm , 9 ft.*lbf }

  1. Connect the 2 connectors.



17. INSTALL DRIVING SUPPORT SWITCH CONTROL ECU

    12 N*m{ 122 kgf*cm , 9 ft.*lbf }

  1. Connect the connector.
