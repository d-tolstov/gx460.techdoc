HEADLIGHT ASSEMBLY > REASSEMBLY

for Preparation Click here


HINT:
  1. Use the same procedure for the RH and LH sides.
  2. The procedure listed below is for the LH side.



1. INSTALL HEADLIGHT REFLECTOR LH



  1. Connect the connector.

  1. Install the headlight reflector LH with the 3 screws.

    NOTICE:
    Align the cutout parts of the headlight reflector LH shaft with the cutout parts of the reflector to install the headlight reflector LH.



2. INSTALL HEADLIGHT GASKET

  1. Install a new gasket.



3. INSTALL HEADLIGHT LIGHT CONTROL ECU SUB-ASSEMBLY LH

  1. Connect the connector.

  1. Install the headlight light control ECU sub-assembly with the 4 screws.



4. INSTALL HEADLIGHT LEVELING MOTOR BASE PACKING

  1. Install the packing.



5. INSTALL HEADLIGHT LEVELING MOTOR LH



  1. Attach the tip of the headlight leveling motor with the reflector as shown in the illustration.




  1. Turn the headlight leveling motor in the direction indicated by the arrow in the illustration to install it.

    HINT:
    For the headlight leveling motor RH, rotate it in the opposite direction of the arrow shown in the illustration before pulling it out.



6. INSTALL FRONT TURN SIGNAL LIGHT BULB

  1. Install the bulb.



7. INSTALL FRONT TURN SIGNAL LIGHT SOCKET



  1. Turn the socket in the direction indicated by the arrow to install it.



8. INSTALL CLEARANCE LIGHT BULB

  1. Install the bulb.



9. INSTALL CLEARANCE LIGHT SOCKET



  1. Turn the socket in the direction indicated by the arrow to install it.



10. INSTALL NO. 2 HEADLIGHT BULB



  1. Turn the bulb in the direction indicated by the arrow to install it.



11. INSTALL DISCHARGE HEADLIGHT BULB



  1. Lock the set spring as shown in the illustration to install the discharge headlight bulb.

    NOTICE:
    Do not touch the glass part of the bulb.




  1. Turn the headlight control ECU socket in the direction indicated by the arrow to connect the headlight control ECU socket.



12. INSTALL NO. 1 HEADLIGHT BACK COVER



  1. Turn the back cover in the direction indicated by the arrow to install it.
