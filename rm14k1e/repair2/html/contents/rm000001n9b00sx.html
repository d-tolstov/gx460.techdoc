DTC    C1781/81    Suspension Control ECU Malfunction

for Preparation Click here


DESCRIPTION

If a malfunction in the suspension control ECU is detected, DTC C1781/81 is stored.

DTC Code
DTC Detection Condition
Trouble Area
C1781/81
One of the following conditions is met:
  1. A nonvolatile memory malfunction
  2. A CAN communication resistor malfunction
  3. A main relay lock-up malfunction
Suspension control ECU
