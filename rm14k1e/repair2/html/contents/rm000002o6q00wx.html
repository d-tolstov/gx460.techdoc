TELEPHONE ANTENNA > INSTALLATION

for Preparation Click here



1. INSTALL TELEPHONE ANTENNA ASSEMBLY



  1. Align the positioning pin of the telephone antenna with the hole and set the antenna on the vehicle.

  1. Attach the 2 claws to install the telephone antenna.

  1. Connect the connector.

  1. Install the antenna nut.

    Torque:
    4.5 N*m{ 46 kgf*cm , 40 in.*lbf }



2. INSTALL TELEPHONE ANTENNA PROTECTOR

  1. Attach the 2 claws to install the telephone antenna protector.



3. INSTALL ROOF HEADLINING ASSEMBLY

  1. Install the roof headlining assembly (Click here).



4. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).



5. CHECK SRS WARNING LIGHT

  1. Check the SRS warning light (Click here).
