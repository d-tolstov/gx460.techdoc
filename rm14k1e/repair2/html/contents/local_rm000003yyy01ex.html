DTC    B156B    Multi-media Interface ECU Disconnected


DESCRIPTION

WIRING DIAGRAM

INSPECTION PROCEDURE

CHECK OPTIONAL COMPONENTS (INCLUDING ASSOCIATED WIRING)

REMOVE OPTIONAL COMPONENTS (INCLUDING ASSOCIATED WIRING)

CHECK FOR DTC

CHECK HARNESS AND CONNECTOR (MULTI-MEDIA INTERFACE ECU POWER SOURCE)

CHECK HARNESS AND CONNECTOR (MULTI-MEDIA INTERFACE ECU - DISPLAY AND NAVIGATION MODULE DISPLAY)

REPLACE MULTI-MEDIA INTERFACE ECU
