POWER WINDOW CONTROL SYSTEM > Jam Protection Function does not Operate

for Preparation Click here


DESCRIPTION

  1. This problem may occur in all door windows.
    The jam protection function operates within a specified range during the manual up or auto up operation.

INSPECTION PROCEDURE

1.PERFORM INITIALIZATION

  1. Initialize the power window regulator motor (Click here).

  1. Check if the power window auto up/down function operates (Click here).

    OK:
    Auto up/down function operates.



NG
END (PROBLEM DUE TO INITIALIZATION FAILURE)
OK


2.CHECK JAM PROTECTION FUNCTION OPERATING RANGE

  1. Check if the jam protection function operates for applicable power window.

    HINT:
    The jam protection function does not operate between the fully closed position of the door glass and the position 4 mm (0.157 in.) below that.

    OK:
    Jam protection function operates.
