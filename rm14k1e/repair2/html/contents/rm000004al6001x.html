REAR SEAT CUSHION HEATER (for 60/40 Split Slide Walk-in Seat Type LH Side) > INSTALLATION

for Preparation Click here


CAUTION:
Wear protective gloves. Sharp areas on the parts may injure your hands.



1. INSTALL REAR SEAT CUSHION HEATER ASSEMBLY



  1. Set the seat cushion heater in place with the name stamp side facing the seat cushion cover.
    Text in Illustration 
    *1
    		Tack Pin
    *2
    		Name Stamp

  1. Install the seat cushion heater with new tack pins.



2. INSTALL SEPARATE TYPE REAR SEAT CUSHION COVER


  1. Using hog ring pliers, install the seat cushion cover to the seat cushion pad with new hog rings.

    NOTICE:
    1. Be careful not to damage the cover.
    2. When installing the hog rings, avoid wrinkling the cover.

3. INSTALL REAR NO. 2 SEAT PROTECTOR

  1. Install the 2 protectors to the cushion pad.



4. INSTALL SEAT CUSHION COVER WITH PAD



  1. Attach the hook.

  1. w/ Seat Heater System:

    1. Attach the 2 clamps to connect the seat heater wire harness.

    2. Connect the 2 seat heater connectors.




  1. Attach the hooks to install the seat cushion cover with pad.



5. INSTALL REAR SEAT CUSHION MOULDING LH



  1. Move the moulding in the direction of the arrow to attach the guide.

  1. Attach the 3 claws and clip to install the moulding.



6. INSTALL CENTER SEAT HINGE COVER RH



  1. Attach the 2 guides and install the cover with the clip.



7. INSTALL RECLINING ADJUSTER RELEASE HANDLE LH

  1. Install the handle with the 3 screws.



8. INSTALL REAR SEAT UPPER RECLINING COVER LH



  1. Attach the 2 claws to install the cover.



9. INSTALL REAR NO. 1 SEAT ASSEMBLY LH

  1. Install the rear No. 1 seat assembly (Click here).
