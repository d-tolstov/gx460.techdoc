    Torque:

13. INSTALL NO. 2 FUEL DELIVERY PIPE SUB-ASSEMBLY

  1. Install the 2 delivery pipe spacers and 4 insulators to the cylinder head LH.

  1. Install the delivery pipe together with the injectors to the cylinder head LH.



  1. Install the 2 bolts.

    Torque:
    21 N*m{ 214 kgf*cm , 15 ft.*lbf }


    NOTICE:

    Make sure that the part of the injector labeled B is between the parts of the delivery pipe labeled A.

    Text in Illustration

    		Fuel Injector
    *a
    		CORRECT
    *b
    		INCORRECT
