DTC    P0575    Cruise Control Input Circuit

for Preparation Click here


DESCRIPTION

This DTC indicates the presence of internal abnormalities in the ECM.

DTC Code
DTC Detection Condition
Trouble Area
P0575
Either condition is met:
  1. The cruise control input signal is abnormal.
  2. The stop light switch input signal is abnormal.
ECM

INSPECTION PROCEDURE

NOTICE:
When the ECM is replaced with a new one, initialization must be performed (Click here).

HINT:
The ECM receives signals from each sensor to control all functions of the dynamic radar cruise control system. When a trouble code is stored, the fail-safe function remains on until the engine switch is turned off.

1.CHECK FOR DTC

  1. Clear the DTCs (Click here).

  1. Perform the following to make sure the DTC detection conditions are met.

    HINT:
    If the detection conditions are not met, the system cannot detect the malfunction.

    1. Drive the vehicle at a speed of 50 km/h (30 mph) or more.

    2. Turn the cruise control switch on.

    3. Push the cruise control switch to -SET to activate cruise control.

  1. Check for DTCs (Click here).

    OK:
    DTC P0575 is not output.
