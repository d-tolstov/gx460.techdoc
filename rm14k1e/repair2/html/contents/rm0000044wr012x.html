DESCRIPTION

These DTCs are stored when a CAN communication circuit malfunction is detected by the display and navigation module display self-check.

DTC Code
DTC Detection Condition
Trouble Area
U0073
A sending malfunction (navigation to APGS).
CAN communication line
U0100
An ECM communication malfunction.
U0126
A steering angle sensor communication malfunction.
U0155
A meter ECU communication malfunction.
U0164
An air conditioner ECU communication malfunction.
U1110
No communication with clearance warning ECU
