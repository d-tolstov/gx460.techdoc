RSE CHECK

HINT:
  1. Use the remote control for each inspection.
  2. This mode checks the color display on the television display assembly.
  3. Illustrations may differ from the actual vehicle depending on the device settings and options. Therefore, some detailed areas may not be exactly the same as on the actual vehicle.
  4. The television display assembly is not a touch panel type. Do not touch the screen.



  1. Color Bar Check




    1. Select "Color Bar Check" from the "RSE Check" screen.




    2. Select a color bar from the "Color Bar Check Mode" screen.

    3. Check the display color.

      HINT:
      1. The entire screen turns to the color or stripe which is selected.
      2. The screen returns to the "Color Bar Check" screen if the remote controller is operated again.



  2. Remote Commander Check




    1. Select "Remote Commander Check" from the "RSE Check" screen.




    2. Operate each switch and check that the switch name and condition are correctly displayed.
