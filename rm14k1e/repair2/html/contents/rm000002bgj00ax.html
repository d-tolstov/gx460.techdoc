HEIGHT CONTROL DRYER > INSTALLATION

for Preparation Click here



1. INSTALL HEIGHT CONTROL DRYER ASSEMBLY

  1. Install a new O-ring.

  1. Install the height control dryer with the bolt.

    Torque:
    6.4 N*m{ 65 kgf*cm , 57 in.*lbf }




  1. Connect the No. 2 height control tube (Click here).

    HINT:
    For the connection procedure of the tube (type 2), refer to Precaution of the Suspension Control System section.



2. INSTALL HEIGHT CONTROL COMPRESSOR ASSEMBLY

  1. Install the height control compressor to the bracket with the 3 bolts.

    Torque:
    5.9 N*m{ 60 kgf*cm , 52 in.*lbf }

  1. Connect the connector.

  1. Attach the clamp.

  1. Attach the 3 clamps.

  1. Connect the connector.

  1. Attach the No. 2 height control tube.



3. CONNECT NO. 2 HEIGHT CONTROL TUBE



  1. Connect the No. 2 height control tube (Click here).

    HINT:
    For the connection procedure of the tube (type 2), refer to Precaution of the Suspension Control System section.



4. INSTALL HEIGHT CONTROL FILTER

  1. Install the height control filter with the 2 bolts.

    Torque:
    6.4 N*m{ 65 kgf*cm , 57 in.*lbf }

  1. Connect the 2 hoses with 2 new clips.



5. CHECK CONNECTION OF TUBES FOR AIR LEAK

  1. Check the connections of the tubes for air leakage (Click here).



6. INSTALL REAR BUMPER PROTECTOR SUB-ASSEMBLY LH

  1. Attach the guide.

  1. Install the rear bumper protector with the screw.



7. INSTALL HEIGHT CONTROL COMPRESSOR COVER

  1. Install the height control compressor cover with 3 new clips.



8. INSTALL REAR BUMPER PROTECTOR LH

  1. Install the rear bumper protector with the 2 screws and clip.



9. INSTALL REAR QUARTER PANEL MUDGUARD SUB-ASSEMBLY LH

  1. Install the rear quarter panel mudguard with the 4 screws.



10. INSTALL REAR WHEEL LH

Torque:
112 N*m{ 1142 kgf*cm , 83 ft.*lbf }



11. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).
