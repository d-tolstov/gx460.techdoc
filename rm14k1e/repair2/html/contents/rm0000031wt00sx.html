1. INSTALL ACCELERATOR PEDAL POSITION SENSOR ASSEMBLY

NOTICE:
  1. Avoid physical shocks to the accelerator pedal position sensor.
  2. Do not disassemble the accelerator pedal position sensor.

  1. Install the accelerator pedal position sensor with the 2 nuts.

    Torque:
    5.4 N*m{ 55 kgf*cm , 48 in.*lbf }

  1. Connect the accelerator pedal position sensor connector.



2. INSTALL FRONT NO. 1 CONSOLE BOX INSERT

  1. Attach the 2 clips and guide to install the front No. 1 console box insert.
