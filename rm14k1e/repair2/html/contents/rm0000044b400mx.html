DTC    P0503    Vehicle Speed Sensor Circuit Malfunction

for Preparation Click here


DESCRIPTION

If a malfunction (a rapid change in vehicle speed) in the vehicle speed signal being output from the skid control ECU is detected while cruise control is in operation, the ECM determines there to be a momentary interruption or noise and stores a DTC.

DTC Code
DTC Detection Condition
Trouble Area
P0503
A malfunction in the vehicle speed signal is detected while cruise control is in operation.
  1. Vehicle stability control system
  2. Vehicle speed sensor signal circuit
  3. Vehicle speed sensor
  4. Combination meter assembly
  5. ECM
