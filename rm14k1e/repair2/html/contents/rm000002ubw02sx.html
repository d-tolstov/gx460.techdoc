DTC    B279A    Theft Deterrent System Communication Line High Fixation

for Preparation Click here


DESCRIPTION

When the communication line (IMI - EFIO) between the ECM and ID code box is stuck on HI output, the ECM stores this DTC.

DTC Code
DTC Detection Condition
Trouble Area
DTC Output Confirmation Operation
B279A
The communication line (IMI - EFIO) between the ECM and ID code box is stuck on HI output (1 trip detection logic*).
  1. Harness or connector (disconnection or break in wire harness is main cause)
  2. ID code box
  3. ECM
Turn the engine switch on (IG) and wait 6 seconds.
  1. *: Only output while a malfunction is present.

INSPECTION PROCEDURE

NOTICE:
  1. When replacing the ID code box, refer to the Service Bulletin.
  2. The fixed 12 V power source voltage in the ECM is output through a resistance from terminal IMI. The ID code box grounds or does not ground this power source voltage accordingly.
  3. After performing repair, perform the operation that fulfills the DTC output confirmation operation, and then confirm that no DTCs are output again.

HINT:
When DTC B279A and the certification ECU DTC are output simultaneously, first perform troubleshooting for the certification ECU DTC.
