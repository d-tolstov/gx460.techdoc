DTC    C1812/12    Pressure Sensor Malfunction / Upside


DESCRIPTION

WIRING DIAGRAM

INSPECTION PROCEDURE

READ VALUE USING INTELLIGENT TESTER (OIL PRESSURE SENSOR)

RECONFIRM DTC

INSPECT STABILIZER CONTROL ECU (SPB1 VOLTAGE)

CHECK HARNESS AND CONNECTOR (STABILIZER CONTROL ECU - ACCUMULATOR PRESSURE SENSOR)

INSPECT ACCUMULATOR PRESSURE SENSOR (OUTPUT VOLTAGE)

READ VALUE USING INTELLIGENT TESTER (OIL PRESSURE SENSOR)

CHECK HARNESS AND CONNECTOR (STABILIZER CONTROL ECU - ACCUMULATOR PRESSURE SENSOR)
