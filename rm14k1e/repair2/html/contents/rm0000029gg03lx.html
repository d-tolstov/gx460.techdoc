DTC    B2341    Sensor (Motor) Failure

DTC    B2344    Position Failure

for Preparation Click here


DESCRIPTION

  1. When the sliding roof drive gear sub-assembly detects a motor malfunction and the sliding roof operation is stopped, DTC B2341 is stored.
  1. When the sliding roof drive gear sub-assembly detects a gear position malfunction and the sliding roof operation is stopped, DTC B2344 is stored.

DTC Code
DTC Detection Condition
Trouble Area
B2341
There is a sensor (motor) failure.
Sliding roof drive gear sub-assembly (Sliding roof motor)
B2344
There is a position failure.
Sliding roof drive gear sub-assembly (Sliding roof ECU)
