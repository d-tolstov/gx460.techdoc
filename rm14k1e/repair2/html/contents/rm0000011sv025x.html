DTC    B2414    Steering Position Sensor Malfunction

for Preparation Click here


DESCRIPTION

The headlight swivel ECU receives signals indicating the steering angle from the spiral with sensor cable sub-assembly (steering angle sensor) using CAN communication.

DTC Code
DTC Detection Condition
Trouble Area
B2414
A malfunction in the steering angle sensor.
  1. Spiral with sensor cable sub-assembly (steering angle sensor)
  2. Headlight swivel ECU assembly
