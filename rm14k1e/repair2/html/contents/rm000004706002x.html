CLEARANCE WARNING BUZZER > INSTALLATION

for Preparation Click here



1. INSTALL NO. 1 CLEARANCE WARNING BUZZER

  1. Attach the clamp to install the No. 1 clearance warning buzzer.

  1. Connect the connector.



2. INSTALL DISPLAY AND NAVIGATION MODULE DISPLAY (w/ Navigation System)

  1. Connect the connectors.




  1. Insert the display and navigation module display and attach the 7 clips on the backside of the display and navigation module display.

    NOTICE:
    When inserting the display, do not press the knobs on it.

  1. Install the display and navigation module display with the 4 bolts.

    Torque:
    12 N*m{ 122 kgf*cm , 9 ft.*lbf }



3. INSTALL AIR CONDITIONING CONTROL ASSEMBLY (w/o Navigation System)

  1. Connect the connectors.

  1. Attach the 7 clips and install the air conditioning control assembly.

  1. Install the 2 bolts.

    Torque:
    12 N*m{ 122 kgf*cm , 9 ft.*lbf }



4. INSTALL RADIO RECEIVER ASSEMBLY

  1. w/o Navigation System:
    Install the radio receiver assembly (Click here).

  1. w/ Navigation System:
    Install the radio receiver assembly (Click here).



5. INSTALL LOWER INSTRUMENT PANEL FINISH PANEL ASSEMBLY

  1. Connect each connector and cooler thermistor.

  1. Attach the 4 clips to install the lower instrument panel finish panel.



6. INSTALL NO. 1 INSTRUMENT PANEL FINISH CUSHION

  1. Connect the connector.

  1. Attach the 4 clips to install the No. 1 instrument panel finish cushion.



7. INSTALL FRONT UPPER CONSOLE PANEL GARNISH

  1. Attach the 2 clips to install the front upper console panel garnish.



8. INSTALL UPPER CONSOLE PANEL SUB-ASSEMBLY

  1. Connect each connector.

  1. Attach the 7 clips to install the upper console panel.



9. INSTALL SHIFT LEVER KNOB SUB-ASSEMBLY



  1. Install the shift lever knob and twist it in the direction indicated by the arrow.



10. INSTALL INSTRUMENT PANEL FINISH PANEL END LH

  1. Attach the 11 clips to install the instrument panel finish panel end.



11. INSTALL INSTRUMENT PANEL FINISH PANEL END RH

  1. Attach the 11 clips to install the instrument panel finish panel end.



12. INSTALL NO. 2 INSTRUMENT PANEL REGISTER ASSEMBLY

  1. Attach the 8 clips to install the No. 2 instrument panel register.



13. INSTALL CENTER INSTRUMENT CLUSTER FINISH PANEL GARNISH



  1. Attach the 13 clips to install the center instrument cluster finish panel garnish together with the instrument cluster finish panel garnish.
    Text in Illustration 
    *1
    		Center Instrument Cluster Finish Panel Garnish
    *2
    		Instrument Cluster Finish Panel Garnish



14. INSTALL INSTRUMENT SIDE PANEL RH

  1. Attach the 3 clips and 2 guides to install the instrument side panel.



15. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).
