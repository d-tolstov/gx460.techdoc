POWER OUTLET SOCKET > INSTALLATION

for Preparation Click here



1. INSTALL POWER POINT SOCKET COVER

  1. Attach the 2 claws to install the power point socket cover.



2. INSTALL POWER POINT SOCKET ASSEMBLY

  1. Attach the claw to install the power point socket.

  1. Connect the connector.



3. INSTALL UPPER CONSOLE PANEL SUB-ASSEMBLY

  1. Connect each connector.

  1. Attach the 7 clips to install the upper console panel.



4. INSTALL SHIFT LEVER KNOB SUB-ASSEMBLY



  1. Install the shift lever knob and twist it in the direction indicated by the arrow.



5. INSTALL INSTRUMENT PANEL FINISH PANEL END RH

  1. Attach the 11 clips to install the instrument panel finish panel end.



6. INSTALL INSTRUMENT PANEL FINISH PANEL END LH

  1. Attach the 11 clips to install the instrument panel finish panel end.
