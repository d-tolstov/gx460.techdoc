REAR NO. 1 SEAT ASSEMBLY (for 60/40 Split Slide Walk-in Seat Type LH Side) > REASSEMBLY

for Preparation Click here


CAUTION:
Wear protective gloves. Sharp areas on the parts may injure your hands.



1. INSTALL REAR SEAT TRACK ADJUSTING HANDLE LH



  1. Install the handle.

    HINT:
    Make sure the handle is straight when installing it, and check that it is centered.



2. INSTALL REAR SEAT CUSHION FRAME SUB-ASSEMBLY LH

  1. Install the seat cushion frame with the 6 nuts.

    Torque:
    32 N*m{ 326 kgf*cm , 24 ft.*lbf }




  1. Temporarily install the seatback frame with the 2 bolts and 2 "TORX" bolts.

  1. Install the bolt to the bolt hole for the outer belt.
    Text in Illustration 
    *1
    		Outer Belt Installation Bolt

  1. Using a T45 "TORX" socket wrench, tighten the 2 "TORX" bolts.

    Torque:
    42 N*m{ 428 kgf*cm , 31 ft.*lbf }

  1. Tighten the 2 bolts.

    Torque:
    42 N*m{ 428 kgf*cm , 31 ft.*lbf }

  1. Remove the bolt from the bolt hole for the outer belt.

  1. Attach the 2 airbag wire harness clamps to connect the airbag wire harness to the seat cushion frame.

  1. Attach the claw to connect the side airbag connector to the seat track rail.

  1. Install a new cable tie.



3. INSTALL REAR SEAT WIRE LH (w/ Seat Heater System)

  1. Attach the clamp to install the seat wire to the seat cushion frame.

  1. Attach the claw to connect the seat wire connector to the seat track rail.



4. INSTALL SEAT HEATER CONTROL SUB-ASSEMBLY (w/ Seat Heater System)

  1. Attach the 2 clamps to install the seat heater control.

  1. Connect the connector.



5. INSTALL REAR BACK LOCK CONTROL LEVER SUB-ASSEMBLY LH

  1. Install the lever with the 2 nuts.

    Torque:
    18 N*m{ 184 kgf*cm , 13 ft.*lbf }



6. INSTALL REAR NO. 2 SEAT RECLINING CONTROL CABLE

  1. Install the cable to the seat frame.

  1. Attach the 3 cable clamps and connect the cable.



7. INSTALL REAR SEAT RECLINING CONTROL CABLE

  1. Install the cable to the seat frame.

  1. Attach the 3 cable clamps and connect the cable.

  1. Install 4 new cable ties.



8. INSTALL NO. 1 SEAT 3 POINT TYPE BELT ASSEMBLY LH



NOTICE:
When installing the retractor, make sure the claws of the vehicle (labeled "a" in the illustration) only contact the installation areas of the retractor.

  1. Align the seat belt positioning holes with the claws and install the retractor of the seat belt with the nut and bolt as shown in the illustration.

    HINT:
    First install the bolt, and then install the nut.

    Torque:
    for bolt:
    7.5 N*m{ 76 kgf*cm , 66 in.*lbf }
    for nut:
    42 N*m{ 428 kgf*cm , 31 ft.*lbf }

    *2
    		Nut
