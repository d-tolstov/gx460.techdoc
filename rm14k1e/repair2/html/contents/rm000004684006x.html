FRONT SEATBACK HEATER > INSTALLATION

for Preparation Click here


CAUTION:
Wear protective gloves. Sharp areas on the parts may injure your hands.

HINT:
  1. Use the same procedure for the RH and LH sides.
  2. The procedure listed below is for the LH side.



1. INSTALL FRONT SEATBACK HEATER ASSEMBLY LH



  1. Set the seatback heater in place with the name stamp side facing the seatback cover.
    Text in Illustration 
    *1
    		Tack Pin
    *2
    		Name Stamp

  1. Install the seatback heater with new tack pins.



2. INSTALL SEPARATE TYPE FRONT SEATBACK COVER



  1. Using hog ring pliers, install the seatback cover to the seatback pad with new hog rings.
    Text in Illustration 
    *1
    		Hog Ring Pliers

    NOTICE:
    1. Be careful not to damage the cover.
    2. When installing the hog rings, avoid wrinkling the cover.




  1. Connect the seatback cover bracket to the seatback pad.



3. INSTALL SEATBACK COVER WITH PAD

  1. Install the seatback cover with pad to the seatback spring.

  1. Attach the 4 claws to install the 2 headrest supports.

  1. Attach the 2 claws and connect the seatback cover bracket to the seatback spring with the nut.

    Torque:
    5.5 N*m{ 56 kgf*cm , 49 in.*lbf }

    NOTICE:
    After the seatback trim cover is assembled, make sure the side airbag strap is not twisted.




  1. Using hog ring pliers, install new hog rings.
    Text in Illustration 
    *1
    		Hog Ring Pliers

  1. Attach the hooks.

  1. Connect the seat heater connector and attach the wire harness clamp.



4. INSTALL FRONT SEATBACK BOARD SUB-ASSEMBLY LH



  1. Attach the 2 guides and 2 claws to install the seatback board.

  1. Connect the interior illumination light connector.

  1. Attach the 2 hooks to connect the 2 rubber bands.



5. INSTALL FRONT SEAT ASSEMBLY

  1. Install the front seat assembly (Click here).
