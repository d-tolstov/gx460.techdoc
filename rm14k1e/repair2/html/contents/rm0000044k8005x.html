DTC    P1572    Improper Aiming of Radar Sensor Beam Axis

for Preparation Click here


DESCRIPTION

When the millimeter wave radar sensor assembly beam axis has deviated, the ECM detects this and stores DTC P1572.

DTC Code
DTC Detection Condition
Trouble Area
P1572
While the vehicle speed is 50 km/h (30 mph) or more with the cruise control switch on and the millimeter wave radar sensor performing self diagnosis, axis deviation is detected for 0.15 seconds or more.
Millimeter wave radar sensor assembly

INSPECTION PROCEDURE

NOTICE:
When the millimeter wave radar sensor assembly is replaced with a new one, adjustment of the millimeter wave radar sensor assembly beam axis must be performed (Click here).

    2. Turn the cruise control switch on.
