HOOD SUPPORT > INSTALLATION

for Preparation Click here


HINT:
  1. Use the same procedure for the RH and LH sides.
  2. The procedure listed below is for the LH side.



1. INSTALL HOOD SUPPORT BRACKET LH



  1. Install the hood support bracket with the 2 bolts.

    Torque:
    18 N*m{ 184 kgf*cm , 13 in.*lbf }



2. INSTALL ENGINE ROOM SIDE COVER LH (for LH Side)

  1. Install the engine room side cover with the 4 clips.



3. INSTALL ENGINE ROOM COVER SIDE (for RH Side)

  1. Install the engine room side cover with the 4 clips.



4. INSTALL HOOD SUPPORT ASSEMBLY LH

NOTICE:
  1. Avoid touching the piston rod as much as possible to prevent foreign matter from attaching to it. Be sure to hold the cylinder while servicing.
  2. Do not wear cotton gloves or other similar materials when handling the piston rod. Fibers may attach to the rod and result in gas leaks.
  3. In order to prevent the piston rod from deforming, do not apply any horizontal load to the hood support.



  1. When reusing the hood support assembly:




    1. Install the 2 stop rings to the hood support assembly.




  1. Install the hood support assembly.

    NOTICE:
    Install the hood support assembly while supporting the hood by hand.

  1. Check that the hood support assembly is connected to the ball joint and cannot be pulled off.
