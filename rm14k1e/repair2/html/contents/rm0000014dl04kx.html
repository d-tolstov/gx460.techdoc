SEAT BELT WARNING SYSTEM > Front Passenger Side Seat Belt Warning Light Malfunction

for Preparation Click here


DESCRIPTION

  1. The combination meter detects the state of the front seat inner belt RH when the front passenger seat is occupied with the engine switch on (IG). If the front passenger seat belt is not fastened, the front passenger side seat belt warning light of the heater control switch board*1 or display and navigation module display*2 blinks. If the seat belt is fastened, the warning light goes off.
  2. *1: w/o Navigation System
  3. *2: w/ Navigation System
