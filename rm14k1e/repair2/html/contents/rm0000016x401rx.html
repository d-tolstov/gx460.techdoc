FRONT AXLE HUB > INSTALLATION

for Preparation Click here


HINT:
  1. Use the same procedure for the RH and LH sides.
  2. The procedure listed below is for the LH side.



1. INSTALL FRONT AXLE HUB SUB-ASSEMBLY LH

  1. Apply MP grease to a new O-ring.

  1. Install the O-ring to the axle hub.

  1. Install the dust cover and axle hub to the steering knuckle with the 4 bolts.

    Torque:
    80 N*m{ 816 kgf*cm , 59 ft.*lbf }



2. INSTALL FRONT AXLE SHAFT NUT

  1. Clean the threaded parts on the drive shaft and axle shaft nut using a non-residue solvent.

  1. Install the front axle shaft nut.

    Torque:
    235 N*m{ 2396 kgf*cm , 173 ft.*lbf }

  1. Install the adjusting cap and a new cotter pin.



3. INSTALL FRONT AXLE HUB GREASE CAP

  1. Install the axle hub grease cap.

    NOTICE:
    Make sure the grease cap is securely installed to the axle hub.



4. INSTALL FRONT DISC

  1. Align the matchmarks and install the front disc.

    HINT:
    When replacing the disc with a new one, select the installation position where the disc has the smallest runout.



5. INSTALL DISC BRAKE CYLINDER ASSEMBLY LH

  1. Install the disc brake cylinder assembly with the 2 bolts.

    Torque:
    123 N*m{ 1254 kgf*cm , 91 ft.*lbf }

  1. Using a union nut wrench, connect the brake tube to the disc brake cylinder assembly.

    Torque:
    15 N*m{ 155 kgf*cm , 11 ft.*lbf }

    NOTICE:
    Use the formula to calculate special torque values for situations where a union nut wrench is combined with a torque wrench (Click here).



6. INSTALL FRONT WHEEL

Torque:
112 N*m{ 1137 kgf*cm , 82 ft.*lbf }



7. INSPECT AND ADJUST FRONT WHEEL ALIGNMENT

  1. Adjust the front wheel alignment (Click here).



8. CHECK SPEED SENSOR SIGNAL

  1. Inspect the speed sensor signal (Click here).
