DTC    B2652    Front Vertical Sensor Malfunction


DESCRIPTION

WIRING DIAGRAM

INSPECTION PROCEDURE

PERFORM ACTIVE TEST USING INTELLIGENT TESTER (POWER SEAT MOTOR FUNCTION)

CHECK FRONT POWER SEAT SWITCH LH (FRONT VERTICAL MOTOR CIRCUIT)

CHECK SEPARATE TYPE FRONT SEAT CUSHION SPRING ASSEMBLY LH (FRONT VERTICAL MOTOR)

CHECK HARNESS AND CONNECTOR (FRONT POWER SEAT SWITCH - FRONT VERTICAL MOTOR)

INSPECT SEPARATE TYPE FRONT SEAT CUSHION SPRING ASSEMBLY LH (FRONT VERTICAL MOTOR)

CHECK HARNESS AND CONNECTOR (FRONT POWER SEAT SWITCH - FRONT VERTICAL MOTOR)
