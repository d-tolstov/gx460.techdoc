SLIDING ROOF SYSTEM > Remote Control System does not Operate

for Preparation Click here


DESCRIPTION

The main body ECU (multiplex network body ECU) receives remote control signals from the door key cylinder and wireless transmitter. When the sliding roof is operated by a remote control signal, sliding roof operation signals are input via LIN communication and the sliding roof motor starts moving.

INSPECTION PROCEDURE

NOTICE:
  1. The sliding roof key-linked function and sliding roof wireless transmitter-linked function are linked with the power window key-linked function and power window wireless transmitter-linked function. Make sure that the power window and sliding roof customize settings are ON.
  2. When the sliding roof drive gear sub-assembly is removed and reinstalled or replaced, the sliding roof drive gear sub-assembly must be initialized (Click here).
  3. Since the sliding roof system has functions that use LIN communication, first confirm that there is no malfunction in the LIN communication system with the How to Proceed with Troubleshooting procedure (Click here).

1.CHECK POWER WINDOW CONTROL FUNCTION

  1. Check the power window key-linked function and power window wireless transmitter-linked function (Click here).

    The power window control functions operate normally.
