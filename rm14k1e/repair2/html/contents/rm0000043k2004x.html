DTC    B2064    Vehicle Type Signal Malfunction

for Preparation Click here


DESCRIPTION

The seat belt control ECU is connected to the main body ECU through CAN communication.
The seat belt control ECU receives the steering wheel position information from the main body ECU via CAN communication and stores the vehicle model information. If the seat belt control ECU cannot receive the steering wheel position information from the main body ECU, DTC B2064 is stored.

DTC Code
DTC Detection Condition
Trouble Area
B2064
During the initial check, the steering wheel position information sent to the seat belt control ECU from the main body ECU stops for 3 seconds or more.
  1. CAN communication system
  2. Main body ECU (Multiplex network body ECU)
  3. Seat belt control ECU
