DTC    C1380    Stop Light Control Relay Malfunction


DESCRIPTION

WIRING DIAGRAM

INSPECTION PROCEDURE

CHECK STOP LIGHT OPERATION

READ VALUE USING INTELLIGENT TESTER (STOP LIGHT SW)

PERFORM ACTIVE TEST USING INTELLIGENT TESTER (STOP LIGHT RELAY)

CHECK TERMINAL VOLTAGE (STP2)

RECONFIRM DTC

CHECK TERMINAL VOLTAGE AND RESISTANCE (+B, GND)

CHECK TERMINAL VOLTAGE (STP)

INSPECT STOP LP RELAY

CHECK HARNESS AND CONNECTOR (STOP LIGHT CONTROL RELAY - STOP LP RELAY)

CHECK TERMINAL VOLTAGE (STOP LIGHT POWER SOURCE TERMINAL)

CHECK TERMINAL VOLTAGE (STP)

INSPECT STOP LP RELAY

CHECK TERMINAL VOLTAGE (STOP LP RELAY POWER SOURCE TERMINAL)

CHECK TERMINAL VOLTAGE (STPO)

RECONFIRM DTC
