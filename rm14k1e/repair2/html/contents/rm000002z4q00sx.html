INSPECT MULTI-INFORMATION DISPLAY



  1. Connect the intelligent tester to the DLC3.

  2. Perform the Active Test (Click here).

    HINT:
    If this Active Test is being performed and the engine switch is turned off and then on (IG), the ODO/TRIP display background may turn white. If this occurs, end the Active Test and turn the engine switch off.

    OK:
    All dots in the multi-information display can be turned on and off using the intelligent tester.
