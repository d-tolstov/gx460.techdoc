REAR NO. 1 SEAT INNER BELT ASSEMBLY (for 60/40 Split Slide Walk-in Seat Type LH Side) > INSTALLATION

for Preparation Click here


CAUTION:
Wear protective gloves. Sharp areas on the parts may injure your hands.

HINT:
A bolt without a torque specification is shown in the standard bolt chart (Click here).



1. INSTALL REAR NO. 1 SEAT INNER BELT ASSEMBLY LH



  1. Install the inner belt and connect the 3 point type belt anchor with the bolt.

    Torque:
    42 N*m{ 428 kgf*cm , 31 ft.*lbf }

    NOTICE:
    Do not allow the anchor part of the front seat inner belt assembly to overlap the protruding part of the front seat adjuster.

    Text in Illustration 
    *a
    		Protruding Part



2. INSTALL SEAT CUSHION COVER WITH PAD



  1. Attach the hook.

  1. w/ Seat Heater System:

    1. Attach the 2 clamps to connect the seat heater wire harness.

    2. Connect the 2 seat heater connectors.




  1. Attach the hooks to install the seat cushion cover with pad.



3. INSTALL REAR SEATBACK LOCK STRIKER COVER LH

  1. Attach the 3 clips to install the cover.



4. INSTALL REAR SEAT CUSHION MOULDING LH



  1. Move the moulding in the direction of the arrow to attach the guide.

  1. Attach the 3 claws and clip to install the moulding.



5. INSTALL CENTER SEAT HINGE COVER RH



  1. Attach the 2 guides and install the cover with the clip.



6. INSTALL RECLINING ADJUSTER RELEASE HANDLE LH

  1. Install the handle with the 3 screws.



7. INSTALL REAR SEAT UPPER RECLINING COVER LH



  1. Attach the 2 claws to install the cover.



8. INSTALL REAR SEAT ASSEMBLY LH

  1. Install the rear seat assembly LH (Click here).
