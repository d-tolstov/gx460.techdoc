ABSORBER CONTROL ACTUATOR > INSTALLATION

for Preparation Click here



1. INSTALL SUSPENSION CONTROL BRACKET LH



  1. Using SST, install the bracket with the nut.

    SST
    09961-00950  

    Torque:
    15 N*m{ 153 kgf*cm , 11 ft.*lbf }



2. INSTALL ABSORBER CONTROL ACTUATOR LH

  1. Using a hexagon wrench (4 mm), install the absorber control actuator with the 2 hexagon socket head cap bolts.

    Torque:
    7.8 N*m{ 80 kgf*cm , 69 in.*lbf }

    NOTICE:
    Replace the absorber control actuator with a new one if it is dropped.

  1. Connect the connector, and then attach the 3 clamps.

    NOTICE:
    Securely connect the connector.



3. INSTALL FRONT WHEEL

Torque:
112 N*m{ 1142 kgf*cm , 83 ft.*lbf }



4. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).
