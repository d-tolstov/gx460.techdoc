1.TROUBLESHOOTING PRECAUTIONS

  1. When there is a problem with the contact points of the terminals or installation problems with the parts, removal and installation of the suspected problem parts may completely or temporarily return the system to normal.

  2. In order to determine the malfunctioning area, be sure to check the conditions at the time the malfunction occurred, such as by stored DTC and freeze frame data output, and record them before disconnecting each connector or removing and installing parts.

  3. Since the vehicle stability control system may be influenced by a malfunction in the other systems, be sure to check for DTCs in the other systems.


2.HANDLING PRECAUTIONS

  1. Be sure to remove and install the master cylinder solenoid and each sensor with the engine switch off unless otherwise specified in the inspection procedure.

  2. When removing and installing the master cylinder solenoid and each sensor, be sure to check that the normal display is output in the test mode inspection and DTC output inspection after installing all the parts.

  3. After replacing the master cylinder solenoid and/or yaw rate and acceleration sensor, be sure to perform calibration (Click here).


3.CAN COMMUNICATION SYSTEM PRECAUTION

  1. The CAN communication system is used for data communication between the skid control ECU, the steering angle sensor and the yaw rate and acceleration sensor. If there is trouble in the CAN communication line, a DTC for the communication line is output.

  2. If a DTC for the CAN communication line is output, repair the malfunction in the communication line and troubleshoot the vehicle stability control system.

  3. Since the CAN communication line has its own length and route, it cannot be repaired temporarily with a bypass wire, etc.


4.PRECAUTIONS WHEN USING INTELLIGENT TESTER

  1. When CAN communication is impossible due to malfunctions of the CAN communication line, it is also impossible to check DTCs using the intelligent tester.


5.CHASSIS DYNAMOMETER PRECAUTION

  1. Disable the TRC and VSC controls when using a chassis dynamometer.

    1. When testing with a drum tester, such as a speedometer tester, combination tester for the speedometer and brake or chassis dynamometer, or when jacking up the front wheels and turning the wheels, perform the following procedure to enter inspection mode and stop the TRC and VSC systems. Enter inspection mode to disable TRC and VSC control when using a chassis dynamometer (Click here).

    2. When testing with a drum tester, such as a speedometer tester, combination tester for the speedometer and brake or chassis dynamometer, enter test mode using the intelligent tester, or switch to VSC OFF mode to turn TRC and VSC operation off using the VSC OFF switch.

      CAUTION:
      Secure the vehicle with a lock chain for safety.

      HINT:
      1. The vehicle may move unexpectedly out of the dynamometer because of TRC and VSC operations.
      2. Pressing the VSC OFF switch does not disable TRC and VSC operations completely.
