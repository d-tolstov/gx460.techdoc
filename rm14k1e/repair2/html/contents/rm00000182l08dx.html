DTC    B15A0    LAN Master Malfunction

DTC    B15AD    Navigation Processor Malfunction

for Preparation Click here


DESCRIPTION

These DTCs are stored when a malfunction occurs in the display and navigation module display.

DTC Code
DTC Detection Condition
Trouble Area
B15A0
Either condition is met:
  1. A bus malfunction.
  2. A sub-microprocessor malfunction.
Display and navigation module display
B15AD
Either condition is met:
  1. A short to ground, short to +B or open occurs in the gyro signal circuit.
  2. A gyro sensor malfunction.

  1. Clear the DTCs (Click here).

  1. Check for DTCs and check if the same trouble occurs again.
