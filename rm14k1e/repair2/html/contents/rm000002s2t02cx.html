DTC    B1500    Fuel Sender Open Detected

for Preparation Click here


DESCRIPTION

This DTC is stored when the combination meter detects a fuel sender gauge malfunction.

DTC Code
DTC Detection Condition
Trouble Area
B1500
The IG voltage is 9.5 V or higher and the following condition is met:
1. A fuel sender gauge operation malfunction.
  1. Harness or connector
  2. Combination meter assembly
  3. Fuel suction plate sub-assembly
  4. Fuel sender gauge assembly
