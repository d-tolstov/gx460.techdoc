2. INSTALL NO. 1 COOLER EVAPORATOR SUB-ASSEMBLY

  1. Sufficiently apply compressor oil to 2 new O-rings and the fitting surface of the hose joint.

    Compressor oil:
    ND-OIL 8 or equivalent

  1. Install the 2 O-rings to the evaporator.

  1. Install the evaporator.

  1. Attach the 4 claws to install the unit case.

  1. Install the 6 screws.



3. INSTALL DAMPER SERVO SUB-ASSEMBLY LH


  1. Align the grooves on the damper servo sub-assembly with the protrusions on the unit and install the damper servo sub-assembly.




    Text in Illustration 
    *a
    		Air Conditioning Unit Side
    		*b
    		Damper Servo Side

  1. Install the 4 screws.



4. INSTALL DAMPER SERVO SUB-ASSEMBLY RH


  1. Align the grooves on the damper servo sub-assembly with the protrusions on the unit and install the damper servo sub-assembly.




    Text in Illustration 
    *a
    		Air Conditioning Unit Side
    		*b
    		Damper Servo Side

  1. Install the 3 screws.



5. INSTALL COOLER EXPANSION VALVE

  1. Install the expansion valve.



6. INSTALL AIR CONDITIONING TUBE AND ACCESSORY ASSEMBLY

  1. Sufficiently apply compressor oil to 2 new O-rings and the fitting surface of the hose joint.

    Compressor oil:
    ND-OIL 8 or equivalent

  1. Install the 2 O-rings to the air conditioner tube and accessory assembly.

  1. Using a 4 mm hexagon wrench, install the 2 bolts.

    Torque:
    3.5 N*m{ 36 kgf*cm , 31 in.*lbf }

  1. Install new butyl tape.

  1. Install new packing.



7. INSTALL AIR CONDITIONING HARNESS

  1. Attach the clamps to install the harness.

  1. Connect the connectors.



8. INSTALL HEATER RADIATOR UNIT SUB-ASSEMBLY

  1. Install the radiator.

  1. Install the 2 brackets with the 2 screws.



9. INSTALL DRAIN COOLER HOSE



  1. Install the drain cooler hose.



10. INSTALL ASPIRATOR HOSE

  1. Attach the 2 claws to install the aspirator hose.



11. INSTALL DEFROSTER NOZZLE ASSEMBLY

  1. Attach the 6 claws to install the defroster nozzle assembly.



12. INSTALL BLOWER ASSEMBLY

  1. Attach the 2 claws to install the blower assembly.

  1. Install the screw.

    Torque:
    2.7 N*m{ 28 kgf*cm , 24 in.*lbf }
