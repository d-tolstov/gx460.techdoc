1. CUSTOMIZE ENTRY AND START SYSTEM (for Entry Function)

HINT:
The following items can be customized.

NOTICE:
  1. When the customer requests a change in a function, first make sure that the function can be customized.
  2. Record the current settings before customizing.

  1. Customizing with the intelligent tester

    1. Connect the intelligent tester to the DLC3.

    2. Turn the engine switch on (IG).

    3. Turn the intelligent tester on.

    4. Enter the following menus: Utility / Customize / Entry & Start or Warning.

    5. Select the setting by referring to the table below.

      Display
      		Default
      		Content
      		Setting
      		Relevant ECU
      Park Wait Time (Waiting time to permit unlocking door after locking)
      		2.5s
      		Function that sets the period of time (lock confirmation time) that the door is prevented from being unlocked by operating the front door outside handle assembly after an entry lock operation is performed.
      		0.5s, 1.5s, 2.5s or 5s
      		Certification ECU
      Ignition Available Area
      (Entry ignition available area)
      		All
      		Function that sets the area that the key must be in before the engine switch can be operated.
      		Front or All
      		Certification ECU
      Glass Hatch Open Operation
      		ON

      		ON or OFF

      Key Low Battery Warning (Warn when key battery becomes weak)

      		ON or OFF

  1. Customizing with Multi-information display

    1. Turn the engine switch on (IG).

    2. With the vehicle stationary or at a vehicle speed of less than 8 km/h (4.9 mph), press and hold the menu switch on the steering pad switch to display the meter customize mode screen.

    3. Select the setting by referring to the table below.

      Display
      		Default
      		Content
      		Setting
      		Relevant ECU
      SELECT DOORS TO UNLOCK
      		DRIVER
      		Function that sets which doors are unlocked by the entry unlock function.
      		ALL or DRIVER
      		Certification ECU
      ACCESS SYSTEM WITH ELEC. KEY
      		ON
      		Function that enables or disables the entry and start system.
      		ON or OFF
      		Certification ECU

  1. Entry unlock mode switching function

    1. The following entry unlock modes can be selected when using the entry unlock mode switching function. The settings can be customized through the multi-display, the multi-information display in the combination meter assembly or manually.

      1. Driver side door entry unlock mode: When the sensor of a front door outside handle assembly is touched, an entry unlock operation is performed simultaneously for all doors.
      2. Driver side door entry unlock mode: When the sensor of the front door outside handle assembly LH is touched, an entry unlock operation is performed for the driver side door only. If any other door handle is touched, an entry unlock operation is performed simultaneously for all doors.

    2. When using the multi-information display in the combination meter assembly:
      Customize the settings of the entry unlock mode switching function (see above).

    3. When customizing through manual operation:

      1. Turn the engine switch off.
      2. Check that the LED of the key is not illuminated, and then press and hold the lock and unlock buttons of the key for 5 seconds or more.
      3. Check the current setting.

        HINT:
        1. All door entry unlock mode is the default setting.
        2. The mode changes from all door unlock mode to driver side door unlock mode and then back to all door unlock mode.
        3. The answer-back (wireless buzzer, buzzer in the combination meter assembly) performed for each mode is indicated in the table below.

          Currently Selected Mode
          		Answer-back (Wireless Buzzer)
          		Answer-back (Buzzer in Combination Meter Assembly)
          All door unlock mode
          		Buzzer sounds twice (short beeps)
          		Buzzer sounds once ("pong" sound)
          Driver side door unlock mode
          		Buzzer sounds 3 times (short beeps)
          		Buzzer sounds once ("pong" sound)

      4. Release the lock and unlock buttons of the key.
      5. Check that the LED of the key is not illuminated, and then press and hold the lock and unlock buttons of the key for 5 seconds or more to change the mode.

        HINT:
        Repeat the procedure as necessary to select the desired mode.

      6. Unlock the doors with the wireless operation, and then open any door

  1. Entry cancel function

    HINT:
    While the entry and start system is canceled, it is possible to lock and unlock the doors with the wireless operation, and the start system can be operated by holding the key against the engine switch.


    1. The following functions are disabled when the entry and start system is canceled.

      1. Entry unlock/lock functions
      2. Entry ignition function
      3. Key lock-in prevention function
      4. Entry glass hatch open function
      5. Entry warning functions

    2. When using the multi-information display in the combination meter assembly:
      Cancel the entry and start system (see above).

    3. When canceling the system through manual operation:

      1. Make sure the following conditions are met:
        1. The engine switch is off.
        2. The driver side door is closed.
        3. The driver side door is unlocked.
      2. Press the unlock button of the key.
      3. Open the driver side door within 5 seconds of completing the step above (driver side door: closed at beginning of step → opened).
      4. With the driver side door open, press the unlock button of the key 2 times within 5 seconds of completing the step above.

        NOTICE:
        If the driver side door is closed before or while pressing the unlock button, the entry cancel setting mode will end.

      5. Close and open the driver side door twice within 30 seconds of completing the above step (driver side door: open at beginning of step → closed → opened → closed → opened).
      6. With the driver side door open, press the unlock button of the key 2 times within 30 seconds of completing the step above

        NOTICE:
        If the driver side door is closed before or while pressing the unlock button, the entry cancel setting mode will end.

      7. Close and open the driver side door within 30 seconds of completing the step above (driver side door: open at beginning of step → closed → opened)
      8. Close the driver side door within 5 seconds of completing the above step.
      9. Check that the wireless buzzer sounds twice (short beeps) to confirm that the entry and start system has been canceled.

    4. Perform the following procedure to restore the entry and start system to the active state from the canceled state.

      1. Perform the procedures to cancel the entry and start system again.
      2. Check that the wireless buzzer sounds once (short beep) to confirm that the entry and start system has been restored to the active state.

        HINT:
        1. The system changes between the canceled state and the active state each time the procedure to cancel the system through manual operation is performed.
        2. The buzzer sounds twice when the system changes from the active state to the canceled state, and sounds once when the system changes from the canceled state to the active state.
