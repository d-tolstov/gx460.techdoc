DTC    B2651    Reclining Sensor Malfunction


DESCRIPTION

WIRING DIAGRAM

INSPECTION PROCEDURE

PERFORM ACTIVE TEST USING INTELLIGENT TESTER (POWER SEAT MOTOR FUNCTION)

CHECK FRONT POWER SEAT SWITCH LH (RECLINING MOTOR CIRCUIT)

CHECK SEPARATE TYPE FRONT SEATBACK SPRING ASSEMBLY LH (RECLINING MOTOR)

CHECK HARNESS AND CONNECTOR (FRONT POWER SEAT SWITCH - RECLINING MOTOR)

INSPECT SEPARATE TYPE FRONT SEATBACK SPRING ASSEMBLY LH (RECLINING MOTOR)

CHECK HARNESS AND CONNECTOR (FRONT POWER SEAT SWITCH - RECLINING MOTOR)
