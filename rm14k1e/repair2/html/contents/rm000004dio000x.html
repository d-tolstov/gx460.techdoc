1. CUSTOMIZE AIR CONDITIONING SYSTEM

HINT:
The following items can be customized.

NOTICE:
  1. When the customer requests a change in a function, first make sure that the function can be customized.
  2. Record the current settings before customizing.

  1. Customizing with the intelligent tester

    1. Connect the intelligent tester to the DLC3.

    2. Turn the engine switch on (IG).

    3. Turn the intelligent tester on.

    4. Enter the following menus: Utility / Customize / Air Conditioner.

    5. Select the setting by referring to the table below.

      Display
      		Default
      		Content
      		Setting
      		Relevant ECU
      Set Temperature Shift
      		Normal
      		Function to set the temperature shift.
      		+2 C, +1 C, Normal, -1 C or -2 C
      		Air conditioning amplifier assembly
      Compressor Mode
      		Automatic
      		Function to turn the A/C on automatically by pressing the AUTO button when the blower is on and the A/C is off.
      		Automatic or Manual
      		Air conditioning amplifier assembly
      Air Inlet Mode
      		Automatic
      		Function to automatically change to recirculation mode when the A/C is turned on.
      		Automatic or Manual
      		Air conditioning amplifier assembly
      Foot/DEF Auto Mode
      		ON
      		Function to turn on the airflow from the front and rear footwell register ducts and front defroster automatically when auto mode is on.
      		ON or OFF
      		Air conditioning amplifier assembly
      Foot/DEF Automatic Blow Up Function
      		ON
      		Function to change the blower level automatically when the front defroster is on.
      		ON or OFF
      		Air conditioning amplifier assembly
      Button Press Buzzer*
      		ON
      		Function to sound a buzzer when a button is pressed.
      		ON or OFF
      		Air conditioning amplifier assembly
      Emission Gas Sensor Shift
      		Normal
      		Function to change the sensitivity of the exhaust gas sensor.
      		Much More, More, Little More, Normal, Little Less, Less or Much Less
      		Air conditioning amplifier assembly
      Add Foot Air in FACE Mode
      		ON
      		When the FACE mode is selected, an additional amount of air is blown to the foot area.
      		ON or OFF
      		Air conditioning amplifier assembly

      HINT:
      *: This item is displayed even if the vehicle is not equipped with a navigation assembly. Even if the setting is changed, nothing happens because the vehicle does not have the buzzer sounding function.
