NAVIGATION SYSTEM (for DVD) > Steering Pad Switch Circuit

for Preparation Click here


DESCRIPTION

This circuit sends an operation signal from the steering pad switch to the display and navigation module display.
If there is an open in the circuit, the audio system cannot be operated using the steering pad switch.
If there is a short in the circuit, the same condition as when switch is continuously depressed occurs.
Therefore, the display and navigation module display cannot be operated using the steering pad switch and the display and navigation module display itself cannot function.
