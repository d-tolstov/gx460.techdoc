STEERING WHEEL > INSTALLATION

for Preparation Click here



1. INSTALL STEERING WHEEL ASSEMBLY

  1. Align the matchmarks on the steering wheel assembly and steering main shaft assembly.

  1. Install the steering wheel assembly set nut.

    Torque:
    50 N*m{ 510 kgf*cm , 37 ft.*lbf }



2. INSTALL STEERING WHEEL HEATER CONTROL ASSEMBLY

  1. Install the controller with the 2 screws.

  1. Connect the 2 connectors.



3. INSTALL CRUISE CONTROL MAIN SWITCH

  1. Install the switch with the 2 screws.

  1. Connect the connector.



4. INSTALL STEERING PAD SWITCH ASSEMBLY

  1. Attach the 2 claws to install the steering pad switch assembly.

  1. Install the 4 screws.

  1. Connect the connector.



5. INSTALL STEERING PAD



  1. Support the steering pad with one hand.

  1. Connect the 2 connectors to the steering pad.

    NOTICE:
    When handling the airbag connector, take care not to damage the airbag wire harness.

  1. Connect the horn connector.

  1. Confirm that the groove along the circumference of the "TORX" screw is attached to the screw case and place the steering pad onto the steering wheel.

  1. Using a T30 "TORX" socket wrench, tighten the 2 screws.

    Torque:
    8.8 N*m{ 90 kgf*cm , 78 in.*lbf }



6. INSTALL NO. 2 STEERING WHEEL LOWER COVER

  1. Attach the 2 claws to install the cover.



7. INSTALL NO. 3 STEERING WHEEL LOWER COVER

  1. Attach the 2 claws to install the cover.



8. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).



9. INSPECT SRS WARNING LIGHT

  1. Inspect the SRS warning light (Click here).
