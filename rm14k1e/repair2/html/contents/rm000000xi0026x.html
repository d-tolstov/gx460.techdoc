POWER WINDOW CONTROL SYSTEM > Front Passenger Side Power Window does not Operate with Front Passenger Side Power Window Switch

for Preparation Click here


DESCRIPTION

  1. If the manual up/down function does not operate, there may be a malfunction in the power window regulator switch, front power window regulator motor, harness or connector.

WIRING DIAGRAM




INSPECTION PROCEDURE

NOTICE:
Inspect the fuses for circuits related to this system before performing the following inspection procedure.

HINT:
Since the power window control system has functions that use LIN communication, first confirm that there is no malfunction in the communication system by inspecting the LIN communication functions in accordance with the "How to Proceed with Troubleshooting" procedures. Then, conduct the following inspection procedure.

1.CHECK FOR DTC (B2312)

  1. Check if DTC B2312 is output (Click here).

    OK:
    DTC B2312 is not output.
