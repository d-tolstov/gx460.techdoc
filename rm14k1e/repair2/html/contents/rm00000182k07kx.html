DTC    B15CE    Seatback Display Disconnected

for Preparation Click here


DESCRIPTION

The television display assembly and multi-display controller sub-assembly are connected by an AVC-LAN communication line.
The display and navigation module display is connected to the multi-display controller sub-assembly by a MOST communication line.
This DTC will be stored when an AVC-LAN communication error occurs between the multi-display controller sub-assembly and television display assembly.

DTC Code
DTC Detection Condition
Trouble Area
B15CE
Either condition is met:
  1. The television display assembly is not connected while the engine switch is on (IG) or (ACC).
  2. Communication between the master unit and television display assembly is not possible when the engine is started.
  1. Television display assembly power source circuit
  2. AVC-LAN circuit between multi-display controller sub-assembly and television display assembly
  3. Multi-display controller sub-assembly
  4. Television display assembly LH
  5. Television display assembly RH

HINT:
  1. Even if no fault is present, this DTC may be stored depending on the battery condition or engine start voltage.
  2. The display and navigation module display is the master unit.
