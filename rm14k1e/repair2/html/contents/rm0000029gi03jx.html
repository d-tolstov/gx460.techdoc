DTC    B2343    Position Initialization Incomplete

for Preparation Click here


DESCRIPTION

This DTC is stored when the sliding roof drive gear sub-assembly has not been initialized.

DTC Code
DTC Detection Condition
Trouble Area
B2343
The sliding roof drive gear sub-assembly has not been initialized.
  1. Sliding roof drive gear sub-assembly (Sliding roof ECU)
  2. Map light assembly (Sliding roof switch)
  3. Harness or connector

WIRING DIAGRAM
