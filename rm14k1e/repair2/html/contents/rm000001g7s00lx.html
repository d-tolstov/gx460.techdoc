FAIL-SAFE FUNCTION

  1. When an air suspension system malfunction is detected under normal control operation, the suspension control ECU suspends switch control of the height control operation.

    Air Suspension System 
    DTC Code
    		Malfunction Item
    		Fail-safe Operation
    C1713/13
    C1714/14
    		Rear height control sensor circuit malfunction
    1. If there is an error on only 1 side, the system continues vehicle height control.
    2. If errors occur on both sides, the system interrupts vehicle height control.
    C1715/15
    C1716/16
    C1717/17
    		Acceleration sensor circuit malfunction
    		Interrupts damping force control after fixing the front to the 13th step and the rear to the 15th step.
    C1731/31
    C1732/32
    C1733/33
    C1734/34
    		Absorber control actuator circuit malfunction
    		Interrupts damping force control after fixing the front to the 13th step and the rear to the 15th step (except malfunctioning actuator).
    C1745/45
    		Exhaust solenoid valve circuit malfunction
    1. Auto mode:
      The system sets the target vehicle height to normal, performs vehicle height ascent control and interrupts control.
    2. Manual mode:
      The system accepts a switch operation for ascent, provided that the vehicle height is below normal.
    3. Continues damping force control.
    C1746/46
    		Gate solenoid valve circuit malfunction
    1. Interrupts vehicle height control.
    2. Continues damping force control.
    C1747/47
    		Leveling solenoid valve circuit malfunction
    1. Interrupts vehicle height control.
    2. Continues damping force control.
    C1751/51
    		Compressor relay coil malfunction
    1. Auto mode:
      The system sets the target vehicle height to normal, performs vehicle height descent control and interrupts control.
    2. Manual mode:
      The system accepts a switch operation for descent, provided that the vehicle height is above normal.
    3. Continues damping force control.
    C1752/52
    		Height control compressor motor circuit malfunction
    1. Auto mode:
      The system sets the target vehicle height to normal, performs vehicle height descent control and interrupts control.
    2. Manual mode:
      The system accepts a switch operation for descent, provided that the vehicle height is above normal.
    3. Continues damping force control.
    C1761/61
    		Continuous electric current to height control compressor motor
    1. Auto mode:
      Powered continuously the first time: When the target vehicle height is set to normal or HI, and the vehicle height is more than the target, the system performs vehicle height descent control and interrupts control.
      Powered continuously the second time: The system sets the target vehicle height to normal, effects vehicle height descent control and interrupts control.
    2. Manual mode:
      When the compressor motor is powered continuously both the first and second times, the system accepts a switch operation for descent, provided that the vehicle height is above normal.
    3. Continues damping force control.
    C1762/62
    		Continuous electric current to exhaust solenoid
    1. Auto mode:
      The system sets the target vehicle height to normal, performs vehicle height ascent control and interrupts control.
    2. Manual mode:
      The system accepts a switch operation for ascent, provided that the vehicle height is below normal.
    3. Continues damping force control.
    C1781/81
    		Suspension control ECU Malfunction
    		Interrupts vehicle height control and damping force control.
    C1782/82
    		Power source voltage malfunction
    1. The system interrupts vehicle height control only when there is an error in the +B power source.
    2. Interrupts the damping force control.
    C1783/83
    		Speed sensor signal malfunction
    1. Continues vehicle height control.
    2. When the front right side is malfunctioning, fixes the front to the 13th step and the rear to the 15th step.
    C1784/84
    		Steering angle sensor signal malfunction
    1. Continues vehicle height control.
    2. Interrupts damping force control after fixing the front to the 13th step and the rear to the 15th step.
    C1787/87
    		Forward/rearward/sideways acceleration sensor signal malfunction
    1. Continues height control.
    2. Interrupts the damping force of roll posture control.
    C1789/89
    		Vehicle identification malfunction
    1. Continues height control.
    2. Interrupts damping force control after fixing the front to the 13th step and the rear to the 15th step.
    U0100/71
    		Lost communication with ECM
    		Continues vehicle height control and damping force control.
    U0114/71
    		Lost communication with four wheel drive control ECU
    		Continues vehicle height control and damping force control.
    U0122/71
    		Lost communication with skid control ECU
    1. The target vehicle height is set to normal height.
    2. Interrupts damping force control after fixing the front to the 13th step and the rear to the 15th step.
    U0124/71
    		Lost communication with yaw rate and acceleration sensor
    1. Continues vehicle height control.
    2. Interrupts damping force control after fixing the front to the 13th step and the rear to the 15th step.
    U0126/71
    		Lost communication with steering angle sensor
    1. Continues vehicle height control.
    2. Interrupts damping force control after fixing the front to the 13th step and the rear to the 15th step.
    U0140/71
    		Lost communication with main body ECU
    		Continues vehicle height control and damping force control.
