DESCRIPTION

The signals generated by operating the tilt and telescopic switch are sent to the multiplex tilt and telescopic ECU through the serial line.

INSPECTION PROCEDURE

1.READ VALUE USING INTELLIGENT TESTER (TILT UP/DOWN SWITCH, TELESCO SHORT/LONG SWITCH)

  1. Use the Data List to check if the tilt and telescopic switch is functioning properly.
