DTC    C1747/47    Leveling Solenoid Valve Circuit

for Preparation Click here


DESCRIPTION

The suspension control ECU opens the leveling solenoid valve (No. 2 height control valve) to allow compressed air to flow into the height control cylinder (pneumatic cylinder) and raise the vehicle height.
The No. 2 height control valve consists of a leveling solenoid valve and a gate solenoid valve.
The leveling solenoid valve opens and closes the passage between the height control compressor and the height control cylinder (pneumatic cylinder).

DTC Code
DTC Detection Condition
Trouble Area
C1747/47
Either condition is met:
  1. With the height control valve inactivated, an open signal of the height control valve is detected for 1 second or more.
  2. With the height control valve activated, a short signal of the valve is detected 8 times successively.
  1. Harness or connector
  2. Leveling solenoid valve (No. 2 height control valve)
  3. Suspension control ECU

HINT:
  1. Once the suspension control ECU stores DTC C1747/47 in memory, vehicle height control is suspended until a normal signal is input to the suspension control ECU from the height control valves. However, control is resumed if the engine switch is turned off and then on (IG) again.
  2. Malfunction in the leveling solenoid valve:
    Height control operation is suspended and the height control indicator does not change by operating the height control switch.
