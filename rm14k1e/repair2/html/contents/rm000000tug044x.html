DTC    B2790    ID BOX EEPROM Malfunction

for Preparation Click here


DESCRIPTION

When an internal malfunction occurs in the ID code box, the certification ECU stores this DTC.

DTC Code
DTC Detection Condition
Trouble Area
DTC Output Confirmation Operation
B2790
An internal malfunction occurs in the ID code box (EEPROM access malfunction) (1 trip detection logic*).
  1. ID code box
  2. Certification ECU
An immobiliser set/unset operation is performed (the engine switch is turned off or on (IG) while carrying the key).
  1. *: Only output while a malfunction is present.

Vehicle Condition and Fail-safe Operation when Malfunction Detected 
Vehicle Condition when Malfunction Detected
Fail-safe Operation when Malfunction Detected
  1. Engine running: Engine cannot be started next time
  2. Engine stopped: Engine cannot be started
