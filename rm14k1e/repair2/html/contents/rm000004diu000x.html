1. CUSTOMIZE POWER DOOR LOCK CONTROL SYSTEM

HINT:
The following items can be customized.

NOTICE:
  1. When the customer requests a change in a function, first make sure that the function can be customized.
  2. Record the current settings before customizing.

  1. Customizing with the intelligent tester

    1. Connect the intelligent tester to the DLC3.

    2. Turn the engine switch on (IG).

    3. Turn the intelligent tester on.

    4. Enter the following menus: Utility / Customize / Door Lock.

    5. Select the setting by referring to the table below.

      Display
      		Default
      		Content
      		Setting
      		Relevant ECU
      Unlock Key Twice
      		ON
      		Function that unlocks only the driver door when the driver door key cylinder is turned to unlock once, and unlocks all doors when it is turned to unlock twice. For the OFF setting, turning it once unlocks all doors.
      		ON or OFF
      		Main body ECU (Multiplex network body ECU)
      Auto Lock
      		ON
      		Function that locks all doors when the vehicle speed reaches a certain level.
      		ON or OFF
      		Main body ECU (Multiplex network body ECU)
      Auto Lock/Shift
      		OFF
      		Function that locks all doors when the shift lever is moved from P to any position other than P while the engine is running and all doors are closed.
      		ON or OFF
      		Main body ECU (Multiplex network body ECU)
      Auto Unlock/Shift
      		OFF
      		Function that unlocks all doors when the shift lever is moved to P from any position other than P while the engine switch is on (IG).
      		ON or OFF
      		Main body ECU (Multiplex network body ECU)
      All Unlock/Open-Close
      		OFF
      		Function that unlocks the other doors when the driver door is opened within 10 seconds after the engine switch is turned off.
      		ON or OFF
      		Main body ECU (Multiplex network body ECU)

  1. Customizing with Multi-information display

    1. Turn the engine switch on (IG).

    2. With the vehicle stationary or at a vehicle speed of less than 8 km/h (4.9 mph), press and hold the menu switch on the steering pad switch to display the meter customize mode screen.

    3. Select the setting by referring to the table below.

      Display
      		Default
      		Content
      		Setting
      		Relevant ECU
      Auto Lock at Speed
      		ON
      		Function that locks all doors when the vehicle speed reaches a certain level.
      		ON or OFF
      		Main body ECU (Multiplex network body ECU)
      Auto Lock at Sift
      		OFF
      		Function that locks all doors when the shift lever is moved from P to any position other than P while the engine is running and all doors are closed.
      		ON or OFF
      		Main body ECU (Multiplex network body ECU)
      Unlock at Park
      		OFF
      		Function that unlocks all doors when the shift lever is moved to P from any position other than P while the engine switch is on (IG).
      		ON or OFF
      		Main body ECU (Multiplex network body ECU)
      Unlock All at Driver Door Open
      		OFF
      		Function that unlocks the other doors when the driver door is opened within 10 seconds after the engine switch is turned off.
      		ON or OFF
      		Main body ECU (Multiplex network body ECU)
