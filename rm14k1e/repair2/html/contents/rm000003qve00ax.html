TURN SIGNAL FLASHER ASSEMBLY > INSTALLATION

for Preparation Click here



1. INSTALL TURN SIGNAL FLASHER ASSEMBLY

  1. Install the turn signal flasher with the nut.

  1. Connect the connector.



2. INSTALL LOWER INSTRUMENT PANEL FINISH PANEL SUB-ASSEMBLY

  1. Connect each connector and cooler thermistor.

  1. Attach the 4 clips to install the lower instrument panel finish panel.



3. INSTALL INSTRUMENT PANEL FINISH PANEL END LH

  1. Attach the 11 clips to install the instrument panel finish panel end.



4. INSTALL NO. 1 INSTRUMENT PANEL UNDER COVER SUB-ASSEMBLY

  1. Connect the connector and attach the clamp.

  1. Attach the 2 clips and 2 guides to install the No. 1 instrument panel under cover.

  1. Install the screw.



5. INSTALL COWL SIDE TRIM BOARD LH

  1. Attach the clip and claw to install the cowl side trim board.

  1. Install the clip.



6. INSTALL DOOR SCUFF PLATE ASSEMBLY LH

  1. Connect the connector.

  1. Attach the 4 clips, 10 claws and 2 guides to install the door scuff plate.
