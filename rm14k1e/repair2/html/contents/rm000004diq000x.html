1. CUSTOMIZE METER / GAUGE SYSTEM

HINT:
The following items can be customized.

NOTICE:
  1. When the customer requests a change in a function, first make sure that the function can be customized.
  2. Record the current settings before customizing.

  1. Customizing with intelligent tester

    1. Connect the intelligent tester to the DLC3.

    2. Turn the engine switch on (IG).

    3. Turn the intelligent tester on.

    4. Enter the following menus: Utility / Customize / Warning or Display.

    5. Select the setting by referring to the table below.

      Tester Display
      		Default
      		Content
      		Setting
      		Relevant ECU
      Driver Side Seatbelt Warning Buzzer*
      		ON
      		Function to turn on/off the seat belt warning buzzer.
      		ON or OFF
      		Main body ECU
      Front Passenger Side Seatbelt Warning Buzzer*
      		ON
      		Function to turn on/off the seat belt warning buzzer.
      		ON or OFF
      		Main body ECU
      ECO Drive Indicator
      		ON
      		Function to enable or disable the eco driving indicator zone display in the multi-information display
      		ON or OFF
      		ECM

      1. *: This setting is only valid for the buzzer which sounds at 20 km/h (12 mph) or more.
