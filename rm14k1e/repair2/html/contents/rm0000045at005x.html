GLASS HATCH OPENER SWITCH > INSTALLATION

for Preparation Click here



1. INSTALL SWITCH COVER

  1. Install the switch cover.



2. INSTALL GLASS HATCH OPENER SWITCH ASSEMBLY

  1. Install the opener switch with the screw.



3. INSTALL LOWER BACK DOOR OUTSIDE GARNISH SUB-ASSEMBLY LH

  1. Clean the vehicle body surface.

    1. Using a heat light, heat the vehicle body surface.

    2. Wipe off any tape with cleaner.

  1. Install a new outside garnish.

    1. Using a heat light, heat the vehicle body and garnish.

    2. Remove the peeling paper from the face of the outside garnish.

  1. Attach the 5 claws to install the garnish.

  1. Install the screw.

  1. Attach the clamp and connect the connector.



4. INSTALL LOWER BACK DOOR OUTSIDE GARNISH MOULDING LH

  1. Attach the 4 claws to install the outside garnish moulding.
