FREEZE FRAME DATA

  1. Whenever a DTC is stored, the headlight swivel ECU stores the current vehicle (sensor) state as Freeze Frame Data.


CHECK FREEZE FRAME DATA

  1. Connect the intelligent tester to the DLC3.

  2. Turn the engine switch on (IG).

  3. Turn the intelligent tester on.

  4. Enter the following menus: Body Electrical / AFS / Trouble Codes.

  5. According to the display on the intelligent tester, select the trouble code data display with Freeze Frame Data.

CLEAR FREEZE FRAME DATA

NOTICE:
Clearing the DTCs will also clear the Freeze Frame Data.

  1. Connect the intelligent tester to the DLC3.

  2. Turn the engine switch on (IG).

  3. Clear the DTCs and Freeze Frame Data (Click here).
