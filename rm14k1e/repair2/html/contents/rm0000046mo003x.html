1. INSTALL NO. 2 COOLER EVAPORATOR SUB-ASSEMBLY

  1. Install the liquid tube, cooler magnet valve and No. 2 cooler evaporator sub-assembly and tighten the 2 nuts.

    Torque:
    13 N*m{ 136 kgf*cm , 10 ft.*lbf }

  1. Attach the 2 claws.

  1. Install the cooler evaporator assembly.

  1. Install the cooling unit case.




  1. Install the 4 bolts.

    Torque:
    for bolt A:
    5.4 N*m{ 55 kgf*cm , 48 in.*lbf }

  1. Install the 5 holding springs.

  1. Install the 3 bolts.



2. INSTALL COOLING UNIT DOOR

  1. Install the cooling unit door with the 8 screws and 2 bolts.

  1. Install new packing.



3. INSTALL COOLER CONTROL SWITCH SUB-ASSEMBLY

  1. Attach the 2 claws to install the cooler control switch sub-assembly.



4. INSTALL NO. 1 COOLER WIRING HARNESS SUB-ASSEMBLY

  1. Connect the 3 connectors and install the No. 1 cooler wiring harness sub-assembly.

  1. Install new packing.



5. INSTALL NO. 3 COOLER UNIT DRAIN HOSE

  1. Install the No. 3 cooler unit drain hose.



6. INSTALL AIR CONDITIONING AMPLIFIER ASSEMBLY

  1. Install the air conditioning amplifier assembly with the 2 screws.

  1. Connect the connector.



7. INSTALL COOLING BOX ASSEMBLY

  1. Attach the 12 claws to install the cooling box assembly.

  1. Install the 6 screws.



8. INSTALL CONSOLE COMPARTMENT DOOR HINGE SUB-ASSEMBLY

  1. Install the console compartment door hinge sub-assembly with the 4 screws.

  1. Attach the 5 claws.

  1. Install the 6 screws.



9. INSTALL CONSOLE COMPARTMENT DOOR SUB-ASSEMBLY

  1. Install the console compartment door sub-assembly with the 6 screws.
