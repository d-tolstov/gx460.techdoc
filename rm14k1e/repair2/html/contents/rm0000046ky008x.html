STEERING KNUCKLE > INSTALLATION

for Preparation Click here


HINT:
  1. Use the same procedure for the RH and LH sides.
  2. The procedure listed below is for the LH side.



1. INSTALL STEERING KNUCKLE LH



  1. Install the steering knuckle to the front suspension upper arm with the nut.

    Torque:
    110 N*m{ 1122 kgf*cm , 81 ft.*lbf }

  1. Install a new cotter pin.



2. INSTALL FRONT LOWER BALL JOINT ATTACHMENT LH

  1. Install the front lower ball joint attachment with the 2 bolts.

    Torque:
    160 N*m{ 1632 kgf*cm , 118 ft.*lbf }



3. CONNECT TIE ROD END SUB-ASSEMBLY LH

  1. Connect the tie rod end to the steering knuckle with the nut.

    Torque:
    91 N*m{ 928 kgf*cm , 67 ft.*lbf }

    NOTICE:
    Tighten the nut up to an additional 60° if the holes for the cotter pin are not aligned.

  1. Install a new cotter pin.



4. INSTALL FRONT AXLE HUB SUB-ASSEMBLY LH

  1. Install the front axle hub (Click here).



5. INSTALL FRONT SPEED SENSOR LH

  1. Install the speed sensor with the bolt.

    Torque:
    8.5 N*m{ 87 kgf*cm , 75 in.*lbf }

    NOTICE:
    1. Make sure there are no pieces of iron or other foreign matter attached to the sensor tip.
    2. While inserting the speed sensor into the knuckle hole, do not strike or damage the sensor tip.
    3. After installing the speed sensor, make sure there is no clearance or foreign matter between the sensor stay part and the knuckle.
    4. Make sure there is no foreign matter attached to the speed sensor rotor.



6. INSTALL FRONT WHEEL

Torque:
112 N*m{ 1137 kgf*cm , 82 ft.*lbf }



7. INSPECT AND ADJUST FRONT WHEEL ALIGNMENT

  1. Adjust the front wheel alignment (Click here).



8. CHECK FRONT SPEED SENSOR

  1. Check the front speed sensor (Click here).
