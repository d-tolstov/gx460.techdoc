OUTER REAR VIEW MIRROR > INSTALLATION

for Preparation Click here


HINT:
  1. Use the same procedure for the RH and LH sides.
  2. The procedure listed below is for the LH side.



1. INSTALL OUTER REAR VIEW MIRROR ASSEMBLY LH

  1. Attach the claw to install the outer rear view mirror, and then install the 3 nuts.

    Torque:
    8.0 N*m{ 82 kgf*cm , 71 in.*lbf }

  1. w/ Side Monitor System:
    Connect the 2 connectors and attach the clamp.

  1. w/o Side Monitor System:
    Connect the connector and attach the clamp.



2. INSTALL FRONT DOOR INNER GLASS WEATHERSTRIP LH



  1. Install the front door inner glass weatherstrip.



3. INSTALL FRONT DOOR TRIM BOARD SUB-ASSEMBLY LH



  1. Connect the front door lock remote control cable assembly and front door inside locking cable assembly.

  1. Connect 2 connectors.




  1. Attach the front door trim board sub-assembly by attaching the 4 claws of the front door inner glass weatherstrip as shown in the illustration.




  1. Attach the 12 clips and front door trim board retainer to install the front door trim board sub-assembly.

  1. Install the 3 screws.



4. INSTALL POWER WINDOW REGULATOR MASTER SWITCH ASSEMBLY WITH FRONT DOOR ARMREST BASE PANEL

  1. Connect the connector.

  1. Attach the 2 clips, 4 claws and guide to install the power window regulator master switch assembly with front door armrest base panel.



5. INSTALL POWER WINDOW REGULATOR SWITCH ASSEMBLY WITH FRONT DOOR ARMREST BASE PANEL (for Front Passenger Side)


6. INSTALL FRONT ARMREST ASSEMBLY LH

  1. Attach the 7 claws and 7 guides to install the front armrest assembly.



7. INSTALL FRONT DOOR INSIDE HANDLE BEZEL PLUG LH

  1. Attach the 3 claws to install the front door inside handle bezel plug.



8. INSTALL FRONT DOOR LOWER FRAME BRACKET GARNISH LH

  1. Attach the 2 claws to install the front door lower frame bracket garnish.
