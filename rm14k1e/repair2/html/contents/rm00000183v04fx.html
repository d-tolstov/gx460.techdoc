for Preparation Click here


DESCRIPTION


DTC Code
DTC Detection Condition
Trouble Area
01-DC
Transmission to the component shown by the sub-code failed (Detecting this DTC does not always mean actual failure).
If the same sub-code is recorded in other components, check harness for power supply and communication system of all components shown by code

HINT:
This code may be stored if the engine is started and idled for 60 seconds and then the engine switch is turned to the start position again.

NOTICE:
  1. Before starting troubleshooting, be sure to clear DTCs to clear codes stored due to the reasons described in Hint above. Then, check for DTCs and troubleshoot according to the output DTCs.
  2. The radio receiver assembly is the master unit.
  3. Be sure to clear and recheck DTCs after the inspection is completed to confirm that no DTCs are output.

    2. If "01-DC" is output for any other components, check if the same physical address is displayed.
