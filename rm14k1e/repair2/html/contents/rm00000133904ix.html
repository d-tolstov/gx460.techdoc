CHECK MULTIPLEX NETWORK MASTER SWITCH ASSEMBLY


  1. Disconnect the K3 multiplex network master switch connector.

  2. Measure the voltage and resistance according to the value(s) in the table below.

    HINT:
    Measure the values on the wire harness side with the connector disconnected.


    Terminal No. (Symbol)
    		Wiring Color
    		Terminal Description
    		Condition
    		Specified Condition
    K3-11 (B) - K3-12 (GND)
    		L - W-B
    		Power supply
    		Always
    		11 to 14 V
    K3-12 (GND) - Body ground
    		W-B - Body ground
    		Ground
    		Always
    		Below 1 Ω
    If the result is not as specified, there may be a malfunction in the wire harness side.

  3. Reconnect the K3 multiplex network master switch connector.

  4. Measure the voltage according to the value(s) in the table below.

    Terminal No. (Symbol)
    		Wiring Color
    		Terminal Description
    		Condition
    		Specified Condition
    K3-15 (DOWN) - K3-12 (GND)
    		B - W-B
    		Power window motor down input
    		Engine switch on (IG), driver side power window regulator switch off
    		11 to 14 V
    K3-15 (DOWN) - K3-12 (GND)
    		B - W-B
    		Power window motor down input
    		Engine switch on (IG), driver side power window switch down (Manual operation)
    		Below 1 V
    K3-18 (LED) - K3-12 (GND)
    		V - W-B
    		LED illumination signal
    		Engine switch on (IG)
    		11 to 14 V
    K3-18 (LED) - K3-12 (GND)
    		V - W-B
    		LED illumination signal
    		Approximately 43 seconds after Engine switch turned off
    		Below 1 V
    K3-20 (UP) - K3-12 (GND)
    		P - W-B
    		Power window motor up input
    		Engine switch on (IG), driver side power window switch off
    		11 to 14 V
    K3-20 (UP) - K3-12 (GND)
    		P - W-B
    		Power window motor up input
    		Engine switch on (IG), driver side power window switch up (Manual operation)
    		Below 1 V
    If the result is not as specified, the multiplex network master switch may have a malfunction.
