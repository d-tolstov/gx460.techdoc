5. INSTALL REAR DOOR FRONT WINDOW FRAME MOULDING LH

HINT:
When installing the window frame moulding, heat the vehicle body and window frame moulding using a heat light.

Standard:
Item
Temperature
Vehicle Body
40 to 60°C (104 to 140°F)
Window Frame Moulding
20 to 30°C (68 to 86°F)

NOTICE:
Do not heat the vehicle body or window frame moulding excessively.

  1. Clean the vehicle body surface.

    1. Using a heat light, heat the vehicle body surface.

    2. Remove the double-sided tape from the vehicle body surface.

    3. Wipe off any tape adhesive residue with cleaner.

  1. Install a new rear door front window frame moulding.

    1. Using a heat light, heat a new rear door front window frame moulding and the vehicle body surface.

    2. Remove the peeling paper from the face of the rear door front window frame moulding.

      HINT:
      After removing the peeling paper, keep the exposed adhesive free from foreign matter.

    3. Attach the clip and double-sided tape to install the rear door front window frame moulding.

      HINT:
      Press the rear door front window frame moulding firmly to install it.

  1. Install a new door window frame moulding clip.

11. INSTALL REAR DOOR CHECK ASSEMBLY LH

  1. Apply MP grease to the sliding areas of the rear door check assembly.




  1. Install the door check to the door panel with the 2 nuts.

    Torque:
    8.0 N*m{ 82 kgf*cm , 71 in.*lbf }

  1. Apply adhesive to the threads of the bolt.

    Adhesive:
    Toyota Genuine Adhesive 1324, Three Bond 1324 or equivalent

  1. Install the door check to the body panel with the bolt.

    Torque:
    27 N*m{ 275 kgf*cm , 20 ft.*lbf }

15. INSTALL REAR DOOR OUTSIDE HANDLE ASSEMBLY LH



  1. Insert the front end of the rear door outside handle assembly into the rear door outside handle frame.

    NOTICE:
    If the bellcrank lever is not pulled and held when installing the outside handle, the bellcrank lever will interfere with the outside handle and the release plate will be damaged.

  1. Insert the rear end of the rear door outside handle assembly into the rear door outside handle frame. Next, slide the rear door outside handle assembly toward the front of the vehicle to install it.




  1. Move the lever in the direction indicated by the arrow in the illustration to lock the door outside handle assembly.

19. INSTALL REAR DOOR LOCK ASSEMBLY LH

NOTICE:
  1. When reusing the removed rear door lock assembly, replace the door lock wiring harness seal on the connector with a new one.
  2. Do not allow grease or dust to adhere to the surface of the connector which contacts the door lock wiring harness seal.
  3. Reusing the door lock wiring harness seal or using a damaged door lock wiring harness seal may allow water into the connection. This may result in a malfunction of the rear door lock assembly.




  1. Apply MP grease to the sliding parts of the rear door lock assembly.
    Text in Illustration 

    		Slide

  1. Install a new door lock wiring harness seal to the rear door lock assembly.

  1. Insert the rear door lock assembly into the rear door outside handle release plate, and set it on the rear door panel.

  1. Check that the rear door outside handle frame release plate is securely connected to the rear door lock assembly.

  1. Using a T30 "TORX" wrench, install the rear door lock assembly with the 3 screws.

    Torque:
    5.0 N*m{ 51 kgf*cm , 44 in.*lbf }

24. INSTALL REAR DOOR GLASS SUB-ASSEMBLY LH

  1. Connect the cable to the negative (-) battery terminal and rear power window regulator motor connector.




  1. Connect the power window regulator switch assembly and move the arm of the regulator so that the roller on the arm can be seen through the service hole.

  1. Disconnect the power window regulator switch assembly and rear power window regulator motor connector.

  1. Disconnect the cable from the negative (-) battery terminal.

    NOTICE:
    1. w/ Navigation System (for HDD):
      After the engine switch is turned off, the display and navigation module display (HDD navigation system) records various types of memory and settings. As a result, after turning the engine switch off, be sure to wait for the time specified in the following table before disconnecting the cable from the negative (-) battery terminal.

      Waiting Time before Disconnecting Cable from Negative (-) Battery Terminal :

      Condition
      		Waiting Time
      Vehicle enrolled in G-BOOK system
      		6 minutes
      Vehicle not enrolled in G-BOOK system
      		1 minute

    2. When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).




  1. Insert the rear door glass sub-assembly into the rear door panel along the rear door glass run as indicated by the arrow in the illustration.
