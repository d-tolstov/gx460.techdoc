DTC    B2312    Power Window Switch Malfunction


DESCRIPTION

WIRING DIAGRAM

INSPECTION PROCEDURE

CHECK FOR DTC

CHECK DTC OUTPUT

READ VALUE USING INTELLIGENT TESTER (MULTIPLEX NETWORK MASTER SWITCH)

CHECK HARNESS AND CONNECTOR (MULTIPLEX NETWORK MASTER SWITCH - FRONT POWER WINDOW REGULATOR MOTOR ASSEMBLY LH)

READ VALUE USING INTELLIGENT TESTER (POWER WINDOW REGULATOR SWITCH RH)

INSPECT POWER WINDOW REGULATOR SWITCH ASSEMBLY RH

CHECK HARNESS AND CONNECTOR (POWER WINDOW REGULATOR SWITCH ASSEMBLY RH - FRONT POWER WINDOW REGULATOR MOTOR ASSEMBLY RH)

READ VALUE USING INTELLIGENT TESTER (REAR POWER WINDOW REGULATOR SWITCH LH)

INSPECT REAR POWER WINDOW REGULATOR SWITCH ASSEMBLY LH

CHECK HARNESS AND CONNECTOR (REAR POWER WINDOW REGULATOR SWITCH ASSEMBLY LH - REAR POWER WINDOW REGULATOR MOTOR ASSEMBLY LH)

READ VALUE USING INTELLIGENT TESTER (REAR POWER WINDOW REGULATOR SWITCH RH)

INSPECT REAR POWER WINDOW REGULATOR SWITCH ASSEMBLY RH

CHECK HARNESS AND CONNECTOR (REAR POWER WINDOW REGULATOR SWITCH ASSEMBLY RH - REAR POWER WINDOW REGULATOR MOTOR ASSEMBLY RH)
