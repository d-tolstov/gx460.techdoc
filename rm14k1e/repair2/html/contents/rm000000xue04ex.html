Turn signal flasher assembly (flasher relay)
Blinks the hazard warning lights when performing answer-back in accordance with commands from the certification ECU sent through the main body ECU (multiplex network body ECU) via CAN communication.
Wireless door lock buzzer
Sounds when performing answer-back*1 or when warning functions operate*2 in accordance with certification ECU control.
Electrical key

Sends request codes to each electrical key antenna.
Distinguishes and verifies the ID code from the key and sends signals to each ECU in response to operated functions (controls entire system).
Performs encryption code communication with the ID code box when ignition operations are performed.
