SEAT BELT WARNING SYSTEM > Front Passenger Side Seat Belt Warning Light Malfunction


DESCRIPTION

WIRING DIAGRAM

INSPECTION PROCEDURE

READ VALUE USING INTELLIGENT TESTER (FRONT PASSENGER SIDE BUCKLE SWITCH)

PERFORM ACTIVE TEST USING INTELLIGENT TESTER (FRONT PASSENGER SIDE SEAT BELT WARNING LIGHT)

INSPECT HEATER CONTROL SWITCH BOARD

CHECK HARNESS AND CONNECTOR (COMBINATION METER - HEATER CONTROL SWITCH BOARD, BATTERY AND BODY GROUND)

INSPECT DISPLAY AND NAVIGATION MODULE DISPLAY

CHECK HARNESS AND CONNECTOR (COMBINATION METER - DISPLAY AND NAVIGATION MODULE DISPLAY, BATTERY AND BODY GROUND)

INSPECT SEPARATE TYPE FRONT SEAT CUSHION PAD (OCCUPANT DETECTION SENSOR)

INSPECT FRONT SEAT INNER BELT ASSEMBLY RH

CHECK HARNESS AND CONNECTOR (COMBINATION METER - FRONT SEAT INNER BELT RH AND BODY GROUND)
