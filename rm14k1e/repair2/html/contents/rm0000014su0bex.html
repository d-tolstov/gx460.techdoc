for Preparation Click here


INSPECTION PROCEDURE

1.CHECK CELLULAR PHONE

  1. Check if the voice on the other side can be heard using cellular phone.

    OK:
    The voice can be heard.



NG
CELLULAR PHONE IS FAULTY
OK


2.CHECK NAVIGATION SYSTEM

  1. Check that voice guidance can be heard from the front speaker.

    OK:
    The voice guidance can be heard.



NG
GO TO "VOICE GUIDANCE DOES NOT FUNCTION" IN PROBLEM SYMPTOMS TABLE (Click here)
OK


3.CHECK SETTINGS



  1. Check if the volume level is low on the "Call" screen.

    OK:
    The level is not low.



NG
TURN VOLUME UP
OK


4.CHECK SETTINGS (OPERATION CHECK)



  1. Enter "Hands-free Volume Setting" (Check Handsfree Volume Setting) (Click here).

  1. Check if the "Receive Voice Level" is set to "0".

    HINT:
    The "Receive Voice Level" can be set to 11 different levels, -5 to +5, each with a 3 dB difference.

  1. Check if the "Receive Voice Level" is set to the minimum or maximum level.

    HINT:
    When the "Receive Voice Level" is set to the minimum or maximum level, the sound may be distorted.

    OK:
    The "Receive Voice Level" is set to "0".



NG
SET "Receive Voice Level" TO "0"
OK
