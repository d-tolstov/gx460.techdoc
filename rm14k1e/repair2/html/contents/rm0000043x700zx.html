    5.CHECK FOR DTC*

    1. Check for DTCs and note any codes that are output.

    2. Clear the DTCs.

    3. Check for DTCs. Based on the DTCs output in the first step, try to force output of the navigation system DTCs by simulating the conditions indicated by the DTCs.

      HINT:
      1. Even if the malfunction symptom is not confirmed, check for DTCs as the system stores past DTCs.
      2. Refer to the detailed description on the diagnostic screen as necessary (Click here).
      3. Check and clear past diagnostic trouble codes. Check the diagnostic trouble code and inspect the area the code indicates (Click here).
