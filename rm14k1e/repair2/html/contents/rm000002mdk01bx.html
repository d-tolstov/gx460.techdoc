STARTER > INSTALLATION

for Preparation Click here



1. INSTALL FLYWHEEL HOUSING SIDE COVER


2. INSTALL STARTER ASSEMBLY

  1. Install the starter with the 2 bolts.

    Torque:
    37 N*m{ 377 kgf*cm , 27 ft.*lbf }

  1. Connect the starter connector.

  1. Connect the starter wire with the nut.

    Torque:
    9.8 N*m{ 100 kgf*cm , 87 in.*lbf }

  1. Attach the starter wire clamp.



3. INSTALL STARTER COVER

  1. Install the starter cover with the 3 bolts.

    Torque:
    12 N*m{ 117 kgf*cm , 8 ft.*lbf }



4. INSTALL EXHAUST MANIFOLD SUB-ASSEMBLY RH

  1. Install the exhaust manifold sub-assembly RH (Click here).



5. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).
