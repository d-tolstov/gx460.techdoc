DTC    B15DE    Telematics Transceiver Location Data Blackout

for Preparation Click here


DESCRIPTION

The telematics transceiver checks the reception condition of the location data every minute after the engine switch is turned on (ACC). If the location data is not received from the display and navigation module display for 5 minutes, this DTC will be stored.

DTC Code
DTC Detection Condition
Trouble Area
B15DE
The location information from USB communication is interrupted.
  1. Harness or connector
  2. Telematics transceiver
  3. Display and navigation module display
