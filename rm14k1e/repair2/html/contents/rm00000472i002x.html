REAR VIEW MONITOR DIAGNOSIS SYSTEM

  1. For rear view monitor system diagnosis, signals received by the parking assist ECU can be checked and the rear view monitor system can be calibrated, adjusted and checked using the accessory meter assembly.

    NOTICE:
    Depending on the parts that are replaced or operations that are performed during vehicle inspection or maintenance, calibration of other systems as well as the rear view monitor system may be needed (Click here).

    HINT:
    The displayed screens and items may differ depending on vehicle specifications.


DIAGNOSIS SCREEN TRANSITION


  1. w/o Multi-terrain Monitor





  2. w/ Multi-terrain Monitor




START DIAGNOSTIC MODE

  1. Start the engine.

  2. Turn on the accessory meter.

  3. Retract the outer rear view mirrors.

  4. Operate the light control switch within 10 seconds:
    Off → Turn on → Turn off → Turn on → Turn off → Turn on → Turn off.


FINISH DIAGNOSTIC MODE

HINT:
There are 2 methods to finish diagnostic mode. Finish the mode by using either of them.

  1. Method 1

    1. Turn the engine switch off.

  2. Method 2

    1. Press the MODE switch of the accessory meter assembly


SIGNAL CHECK (PARKING ASSIST ECU INPUT SIGNAL)

  1. Start diagnostic mode.




  2. SIGNAL CHECK

    1. On the SIGNAL CHECK screen, it is possible to inspect the state of signals sent to the parking assist ECU and check the settings.

      Item
      		Inspection Detail
      		Note
      SPEED
      		Speed signal input
      		When "CHK" (red) is displayed, selecting "NEXT" will not change the display to the next screen.
      CAMERA SW
      		Steering pad switch assembly (wide view front and side monitor switch) signal input
      BCTY
      		State of CAN communication with main body ECU (multiplex network body ECU)
      MIRROR SW
      		Outer rear view mirror retract signal input
      SHIFT
      		Shift signal input
      STR SIG
      		Steering angle sensor signal input
      HANDLE
      		Steering wheel position signal input
      BODY
      		Body size signal
      DEST
      		Destination information signal input
      MTM
      		Multi-terrain monitor judgment signal
      T/M
      		Transmission judgment signal

      HINT:
      1. When "CHK" (red) is displayed, perform inspections based on the result of the following inspections.
      2. If performing the adjustment after proceeding to the next screen, confirm that all items display "OK" (blue) before selecting "NEXT".

      1. SPEED inspection

        HINT:
        If "CHK" (red) is displayed for "SPEED", check for DTCs and perform troubleshooting based on the output DTCs (Click here).

      2. CAMERA SW inspection
        1. Check that "OK" (blue) is displayed for "CAMERA SW" and select "OK".
        2. Check that "OK" (blue) changes to "CHK" (red). Press the wide view front and side monitor switch (steering pad switch assembly) and check that "CHK" (red) returns to "OK" (blue).

          HINT:
          If "CHK" (red) remains displayed or the "CAMERA SW" inspection result is not normal, perform troubleshooting according to Problem Symptoms Table ("CHK" message(s) are displayed on the SIGNAL CHECK screen) (Click here).

      3. BCTY inspection

        HINT:
        If "CHK" (red) is displayed for "BCTY", check for DTCs and perform troubleshooting based on the output DTCs (Click here).

      4. MIRROR SW inspection

        HINT:
        If "CHK" (red) is displayed for "MIRROR SW", check for DTCs and perform troubleshooting based on the output DTCs (Click here).

      5. SHIFT inspection

        HINT:
        If "CHK" (red) is displayed for "SHIFT", check for DTCs and perform troubleshooting based on the output DTCs (Click here).

      6. STR SIG inspection

        HINT:
        If "CHK" (red) is displayed for "STR SIG", check for DTCs and perform troubleshooting based on the output DTCs (Click here).

      7. HANDLE inspection

        HINT:
        If "CHK" (red) is displayed for "HANDLE", check for DTCs and perform troubleshooting based on the output DTCs (Click here).

      8. BODY inspection

        HINT:
        If "CHK" (red) is displayed for "BODY", check for DTCs and perform troubleshooting based on the output DTCs (Click here).

      9. DEST inspection

        HINT:
        If "CHK" (red) is displayed for "DEST", check for DTCs and perform troubleshooting based on the output DTCs (Click here).

      10. MTM inspection

        HINT:
        If "CHK" (red) is displayed for "MTM", check for DTCs and perform troubleshooting based on the output DTCs (Click here).

      11. T/M inspection

        HINT:
        If "CHK" (red) is displayed for "T/M", check for DTCs and perform troubleshooting based on the output DTCs (Click here).

  3. Finish diagnostic mode.
