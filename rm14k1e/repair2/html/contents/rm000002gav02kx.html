DESCRIPTION

These DTCs are stored when the self-diagnosis of the headlight swivel ECU determines that the CAN communication system is malfunctioning.

DTC Code
DTC Detection Condition
Trouble Area
U0073
Control module communication bus is off.
CAN communication system
U0100
Lost communication with the ECM/PCM "A".
U0126
Lost communication with the steering angle sensor.
U0129
Lost communication with the brake system control module.
U0142
Lost communication with the main body ECU (multiplex network body ECU).
U1000
CAN communication failure.
U1121
Lost communication with the driver seat module.
