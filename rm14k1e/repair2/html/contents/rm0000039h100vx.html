REAR DOOR LOCK > INSTALLATION

for Preparation Click here



1. INSTALL REAR DOOR INSIDE LOCKING CABLE ASSEMBLY LH

  1. Install the rear door inside locking cable assembly.

  1. Attach the 3 claws.



2. INSTALL REAR DOOR LOCK REMOTE CONTROL CABLE ASSEMBLY LH

  1. Install the rear door lock remote control cable assembly.

  1. Attach the claw.



3. INSTALL REAR DOOR LOCK ASSEMBLY LH

NOTICE:
  1. When reusing the removed rear door lock assembly, replace the door lock wiring harness seal on the connector with a new one.
  2. Do not allow grease or dust to adhere to the surface of the connector which contacts the door lock wiring harness seal.
  3. Reusing the door lock wiring harness seal or using a damaged door lock wiring harness seal may allow water into the connection. This may result in a malfunction of the rear door lock assembly.




  1. Apply MP grease to the sliding parts of the rear door lock assembly.
    Text in Illustration 

    		Slide

  1. Install a new door lock wiring harness seal to the rear door lock assembly.

  1. Insert the rear door lock assembly into the rear door outside handle release plate, and set it on the rear door panel.

  1. Check that the rear door outside handle frame release plate is securely connected to the rear door lock assembly.

  1. Using a T30 "TORX" wrench, install the rear door lock assembly with the 3 screws.

    Torque:
    5.0 N*m{ 51 kgf*cm , 44 in.*lbf }



4. INSTALL REAR DOOR GLASS SUB-ASSEMBLY LH

  1. Connect the cable to the negative (-) battery terminal and rear power window regulator motor connector.




  1. Connect the power window regulator switch assembly and move the arm of the regulator so that the roller on the arm can be seen through the service hole.

  1. Disconnect the power window regulator switch assembly and rear power window regulator motor connector.

  1. Disconnect the cable from the negative (-) battery terminal.

    NOTICE:
    1. w/ Navigation System (for HDD):
      After the engine switch is turned off, the display and navigation module display (HDD navigation system) records various types of memory and settings. As a result, after turning the engine switch off, be sure to wait for the time specified in the following table before disconnecting the cable from the negative (-) battery terminal.

      Waiting Time before Disconnecting Cable from Negative (-) Battery Terminal :

      Condition
      		Waiting Time
      Vehicle enrolled in G-BOOK system
      		6 minutes
      Vehicle not enrolled in G-BOOK system
      		1 minute

    2. When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).




  1. Insert the rear door glass sub-assembly into the rear door panel along the rear door glass run as indicated by the arrow in the illustration.



5. INSTALL REAR DOOR QUARTER WINDOW GLASS LH



  1. Install the rear door quarter window glass together with the rear door quarter window weatherstrip in the direction indicated by the arrow in the illustration.



6. INSTALL REAR DOOR WINDOW DIVISION BAR SUB-ASSEMBLY LH

  1. Insert the rear door window division bar sub-assembly from above.

  1. Install the rear door window division bar sub-assembly to the glass run.

  1. Install the temporary bolt to the rear door window division bar sub-assembly.

  1. Install the rear door window division bar sub-assembly.

  1. Install the 2 bolts and screw.

  1. Tighten the temporary bolt to install the rear door window division bar sub-assembly.



7. INSTALL REAR DOOR GLASS RUN LH

  1. Install the glass run.



8. INSTALL REAR DOOR SERVICE HOLE COVER LH

  1. Apply new butyl tape to the rear door panel.




  1. Pass the rear door lock remote control cable assembly and rear door inside locking cable assembly through a new rear door service hole cover.

  1. Attach the rear door service hole cover using to the reference points on the rear door panel.
    Text in Illustration 
    *1
    		Reference Point

    NOTICE:
    1. There should be no wrinkles or folds after attaching the service hole cover.
    2. After attaching the service hole cover, check the seal quality.

  1. Connect the connector.

  1. Attach the 2 clamps to the rear door wire.



9. INSTALL REAR DOOR INNER GLASS WEATHERSTRIP LH



  1. Attach the 3 claws to install the rear door inner glass weatherstrip as shown in the illustration.



10. INSTALL REAR DOOR TRIM BOARD SUB-ASSEMBLY LH



  1. Install the clamp to the door panel.

  1. Connect the rear door lock remote control cable assembly and rear door inside locking cable assembly.

  1. Connect each connector.




  1. Attach the rear door trim board sub-assembly by attaching the 4 claws of the rear door inner glass weatherstrip as shown in the illustration.

  1. Attach the 9 clips and install the rear door trim board sub-assembly.

  1. Install the 3 screws.



11. INSTALL POWER WINDOW REGULATOR SWITCH ASSEMBLY WITH REAR DOOR BASE PANEL

  1. Connect the connector.

  1. Attach the 2 clips, 4 claws and guide to install the rear power window regulator switch assembly with rear door armrest base panel.



12. INSTALL REAR ARMREST ASSEMBLY LH

  1. Attach the 7 claws and 7 guides to install the rear armrest assembly.



13. INSTALL REAR DOOR INSIDE HANDLE BEZEL PLUG LH

  1. Attach the 3 claws to install the rear door inside handle bezel plug.



14. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL
