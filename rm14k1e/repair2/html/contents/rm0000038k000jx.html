REAR BUMPER > INSTALLATION

for Preparation Click here


HINT:
A bolt without a torque specification is shown in the standard bolt chart (Click here).



1. INSTALL ELECTRICAL KEY ANTENNA

  1. Install the electrical key antenna with the bolt.

    Torque:
    8.0 N*m{ 82 kgf*cm , 71 in.*lbf }



2. INSTALL REAR BUMPER SIDE MOUNTING BRACKET LH

  1. Connect the connector.

  1. Install the rear bumper side mounting bracket with the bolt.

    Torque:
    8.0 N*m{ 82 kgf*cm , 71 in.*lbf }



3. INSTALL REAR BUMPER SIDE MOUNTING BRACKET RH

HINT:
Use the same procedure described for the LH side.



4. INSTALL NO. 2 REAR BUMPER SIDE SUPPORT LH

  1. Install the No. 2 rear bumper side support with the bolts.

    Torque:
    8.0 N*m{ 82 kgf*cm , 71 in.*lbf }



5. INSTALL UPPER REAR BUMPER RETAINER LH

  1. Attach the clip to install the upper rear bumper retainer.



6. INSTALL UPPER REAR BUMPER RETAINER RH

HINT:
Use the same procedure described for the LH side.



7. INSTALL REAR COMBINATION LIGHT LENS AND BODY LH

  1. Connect the connector.

  1. Attach the clip and 2 claws to install the light.

  1. Install the 2 bolts.



8. INSTALL REAR COMBINATION LIGHT LENS AND BODY RH

HINT:
Use the same procedure described for the LH side.



9. INSTALL REAR BUMPER BAR REINFORCEMENT LH

  1. Install the rear bumper bar reinforcement with the 3 bolts.

    Torque:
    25 N*m{ 255 kgf*cm , 18 ft.*lbf }



10. INSTALL REAR BUMPER BAR REINFORCEMENT RH

HINT:
Use the same procedure described for the LH side.



11. INSTALL REAR BUMPER SIDE SUPPORT LH

  1. Attach the 4 claws to install the rear bumper side support.

  1. Install the 3 screws.



12. INSTALL REAR BUMPER SIDE SUPPORT RH

HINT:
Use the same procedure described for the LH side.



13. INSTALL NO. 3 REAR BUMPER SIDE SUPPORT

  1. Install the No. 3 rear bumper side support with the screw.

    Torque:
    3.0 N*m{ 31 kgf*cm , 27 ft.*lbf }



14. INSTALL REAR BUMPER COVER

  1. Connect the 3 connectors.

  1. Attach the 12 claws to install the rear bumper cover.

  1. Install the 6 bolts, 2 screws and 2 clips.

    Torque:
    for bolt:
    8.0 N*m{ 82 kgf*cm , 71 in.*lbf }



15. INSTALL REAR BUMPER BAR PLATE

  1. Attach the 14 claws to install the rear bumper bar plate.



16. INSTALL JACK BOX HOLE COVER

  1. Attach the 2 claws to install the jack box hole cover.



17. INSTALL REAR BUMPER PROTECTOR LH

  1. Install the rear bumper protector with the 2 screws and clip.



18. INSTALL REAR BUMPER PROTECTOR RH

HINT:
Use the same procedure described for the LH side.



19. INSTALL REAR QUARTER PANEL MUDGUARD LH

  1. Install the rear quarter panel mudguard with the 4 screws.



20. INSTALL REAR QUARTER PANEL MUDGUARD RH

HINT:
Use the same procedure described for the LH side.
