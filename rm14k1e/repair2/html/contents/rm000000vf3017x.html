NAVIGATION ANTENNA > INSTALLATION

for Preparation Click here



1. INSTALL NAVIGATION ANTENNA ASSEMBLY

  1. Install the navigation antenna with the 2 screws.

  1. Attach the 3 clamps.



2. INSTALL DEFROSTER NOZZLE ASSEMBLY

  1. Install the defroster nozzle with the 4 screws <A>.



3. INSTALL NO. 2 SIDE DEFROSTER NOZZLE DUCT

HINT:
Use the same procedure described for the No. 1 side defroster nozzle duct.



4. INSTALL NO. 1 SIDE DEFROSTER NOZZLE DUCT

  1. Attach the 2 claws to install the No. 1 side defroster nozzle duct.

  1. Install the screw <A>.



5. INSTALL NO. 2 HEATER TO REGISTER DUCT

HINT:
Use the same procedure described for the No. 1 heater to register duct.



6. INSTALL NO. 1 HEATER TO REGISTER DUCT

  1. Install the No. 1 heater to register duct with the 4 screws <A>.



7. INSTALL INSTRUMENT PANEL SAFETY PAD SUB-ASSEMBLY

  1. Install the instrument panel safety pad sub-assembly (Click here).



8. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).



9. CHECK SRS WARNING LIGHT

  1. Check the SRS warning light (Click here).
