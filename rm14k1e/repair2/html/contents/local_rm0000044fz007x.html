DTC    B1569    DVD-ROM Deck Disconnected


DESCRIPTION

WIRING DIAGRAM

INSPECTION PROCEDURE

CHECK OPTIONAL COMPONENTS (INCLUDING ASSOCIATED WIRING)

REMOVE OPTIONAL COMPONENTS (INCLUDING ASSOCIATED WIRING)

CHECK FOR DTC

CHECK HARNESS AND CONNECTOR (NAVIGATION MODULE BOARD - BATTERY AND BODY GROUND)

CHECK IEEE CABLE CONNECTOR

REPLACE IEEE CABLE CONNECTOR

REPLACE NAVIGATION MODULE BOARD
