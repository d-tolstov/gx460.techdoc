DTC    B2452    Height Control Sensor Data Out of Range When Initializing

for Preparation Click here


DESCRIPTION

The headlight swivel ECU stores this DTC if the sensor value received from the height control sensor is out of range when performing initialization of the headlight swivel ECU; for example, the vehicle is not level or is being jacked up.

DTC Code
DTC Detection Condition
Trouble Area
B2452
The sensor value received from the height control sensor is out of range when performing initialization.
  1. Vehicle condition when initializing
  2. Headlight swivel ECU assembly

INSPECTION PROCEDURE

1.PERFORM INITIALIZATION

  1. Perform initialization of the headlight swivel ECU (Click here).

    HINT:
    1. Before initializing, make sure that the vehicle condition is proper for initialization.
    2. Be sure to perform the initialization as instructed.
