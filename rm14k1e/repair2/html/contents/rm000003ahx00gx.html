RADIO RECEIVER > INSTALLATION

for Preparation Click here



1. INSTALL NO. 2 RADIO BRACKET

  1. Install the bracket with the 3 screws.

    Torque:
    3.0 N*m{ 31 kgf*cm , 27 in.*lbf }



2. INSTALL NO. 1 RADIO BRACKET

  1. Install the bracket with the 3 screws.

    Torque:
    3.0 N*m{ 31 kgf*cm , 27 in.*lbf }



3. INSTALL LOWER CENTER INSTRUMENT CLUSTER FINISH PANEL SUB-ASSEMBLY (w/ Navigation System)

  1. Attach the 2 claws to install the lower center instrument cluster finish panel.

  1. Install the 4 screws.



4. INSTALL RADIO RECEIVER ASSEMBLY WITH BRACKET


  1. w/o Navigation System:

    1. Connect the connectors.




    2. Insert the radio receiver to attach the 6 clips and rib on its backside.

      NOTICE:
      When inserting the radio receiver, do not press the knobs on it.

    3. Install the radio receiver with the 4 bolts.

      Torque:
      13 N*m{ 127 kgf*cm , 9 ft.*lbf }



  1. w/ Navigation System:

    1. Connect the connectors.




    2. Insert the radio receiver to attach the 6 clips and 3 ribs on its backside.

      NOTICE:
      When inserting the radio receiver, do not press the knobs on it.

    3. Install the radio receiver with the 4 bolts.

      Torque:
      13 N*m{ 127 kgf*cm , 9 ft.*lbf }



5. INSTALL FRONT CONSOLE UPPER PANEL GARNISH

  1. Attach the 2 clips to install the front upper console panel garnish.



6. INSTALL UPPER CONSOLE PANEL SUB-ASSEMBLY

  1. Connect each connector.

  1. Attach the 7 clips to install the upper console panel.



7. INSTALL SHIFT LEVER KNOB SUB-ASSEMBLY



  1. Install the shift lever knob and twist it in the direction indicated by the arrow.



8. INSTALL INSTRUMENT PANEL FINISH PANEL END RH

  1. Attach the 11 clips to install the instrument panel finish panel end.



9. INSTALL INSTRUMENT PANEL FINISH PANEL END LH

  1. Attach the 11 clips to install the instrument panel finish panel end.



10. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).
