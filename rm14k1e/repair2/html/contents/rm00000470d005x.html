for Preparation Click here



1. INSTALL NO. 1 ULTRASONIC SENSOR RETAINER



HINT:
Use the same procedure to install the No. 1 ultrasonic sensor retainer on the other side.

  1. Align the keyhole and protrusion as shown in the illustration.

  1. Install the ultrasonic sensor retainer to the front bumper cover.

    NOTICE:
    Do not damage the bumper cover with the protrusion when installing the retainer.

    Text in Illustration 
    *1
    		Keyhole



2. INSTALL NO. 1 ULTRASONIC SENSOR

HINT:
Use the same procedure to install the No. 1 ultrasonic sensor on the other side.

  1. Attach the 2 claws to install the No. 1 ultrasonic sensor.

  1. Connect the connector.



3. INSTALL FRONT BUMPER COVER

  1. w/ LEXUS Parking Assist-sensor System:
    Connect the connector.

  1. Connect the headlight cleaner hose and 2 fog light connectors.

  1. Attach the 14 claws to install the front bumper cover.

  1. Install the 4 screws.




  1. Install the 2 bolts labeled A, 2 bolts labeled B, 2 screws and 4 clips.

    Torque:
    for bolt A:
    8.0 N*m{ 82 kgf*cm , 71 in.*lbf }

    Text in Illustration 
    *1
    		Protective Tape
    *2
    		Bolt A
    *3
    		Bolt B



4. INSTALL RADIATOR GRILLE

  1. Attach the 6 claws to install the radiator grille.

  1. Install the 2 bolts and 2 screws.



5. INSTALL UPPER RADIATOR SUPPORT SEAL

  1. Install the upper radiator support seal with the 13 clips.



6. ADJUST FOG LIGHT AIMING

  1. Adjust fog light aiming (Click here).
