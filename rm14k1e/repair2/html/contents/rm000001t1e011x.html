DTC    B2781    Open / Short in Steering Lock ECU

for Preparation Click here


DESCRIPTION

The steering lock ECU and steering lock motor are built into the steering lock actuator assembly.
The steering lock ECU detects whether the steering lock is in the lock or unlock position by using the lock position sensor of the steering lock motor.
The steering lock ECU receives lock and unlock command signals from the certification ECU and responds with lock and unlock operation signals.

HINT:
The steering lock ECU is not connected to the CAN communication system. However, the steering lock ECU is connected to the certification ECU via LIN communication and communicates with other components via CAN communication through the certification ECU.


DTC Code
Detection Condition
Trouble Area
DTC Output Confirmation Operation
B2781
One of the following conditions is met (1-trip detection logic*1):
  1. The steering lock/unlock position sensor is malfunctioning*2.
  2. The steering lock motor activation circuit has a short circuit.
  3. The steering lock motor activation circuit has an open circuit.
  1. Steering lock actuator assembly (steering lock ECU)
  2. LIN communication system
No confirmation operation is necessary (monitoring is continuous).
  1. *1: Only output while a malfunction is present and the engine switch is on (IG).
  2. *2: When both detection switches turn on, the system is determined to be malfunctioning.



Vehicle Condition and Fail-safe Function when Malfunction Detected 
Vehicle Condition when Malfunction Detected
Fail-safe Function when Malfunction Detected
The steering cannot be locked or unlocked. For this reason, the engine cannot be started.
Prohibits the engine from being started (the engine does not crank).

INSPECTION PROCEDURE

NOTICE:
  1. When using the intelligent tester with the engine switch off, perform either of the following: 1) Turn a courtesy light switch on and off at intervals of 1.5 seconds or less until communication between the intelligent tester and vehicle begins, or 2) connect the intelligent tester to the vehicle and select "MANUAL" from the initial screen on the intelligent tester, and then select "KEY REGIST" under Model Code.
  2. After performing repairs, confirm that no DTCs are output again.
  3. Before replacing the steering lock actuator assembly (steering lock ECU), refer to the Service Bulletin.

HINT:
When all the ECUs that are connected to the LIN communication cannot communicate with the certification ECU, DTC B2785 is stored.
