1.AIR SUSPENSION SYSTEM PRECAUTIONS

  1. Be sure to turn the height control OFF switch on to switch the height control system off and cancel the auto leveling function when:

    1. Jacking up the vehicle.
    2. A trailer etc. is attached to the vehicle.

  2. When replacing the suspension control ECU, be sure to replace it with a new one. If the suspension control ECU is replaced with a normally functioning one from another vehicle, the vehicle identification information will not match, which may cause the air suspension system to stop operating or have undesirable effects on other systems.
