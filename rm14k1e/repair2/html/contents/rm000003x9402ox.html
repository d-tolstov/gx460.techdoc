DTC    B1591    No Response from Audio HU

DTC    B15D1    Audio HU Disconnected

for Preparation Click here


DESCRIPTION

The display and navigation module display and the radio receiver assembly are connected by the MOST communication line.
When a MOST communication error occurs between the display and navigation module display and the radio receiver assembly, this DTC will be stored.

DTC Code
DTC Detection Condition
Trouble Area
B1591
Within the time-out period, there is no response from the radio receiver assembly to the following requests from the master unit:
  1. The system inspection request.
  2. The system inspection result request.
  3. The diagnostic memory request.
  1. Harness or connector
  2. Radio receiver assembly
  3. Stereo component amplifier assembly
  4. Multi-display controller sub-assembly*2
  5. Display and navigation module display*3
B15D1*1
Either condition is met:
  1. Radio receiver assembly is/was not connected while the engine switch is on (IG) or on (ACC).
  2. Communication between the master unit and the radio receiver assembly is not possible when the engine is started.
  1. *2: w/ Rear Seat Entertainment System
  2. *3: w/o Rear Seat Entertainment System

HINT:
  1. *1: Even if no fault is present, this DTC may be stored depending on the battery condition or engine start voltage.
  2. The display and navigation module display is the master unit.
