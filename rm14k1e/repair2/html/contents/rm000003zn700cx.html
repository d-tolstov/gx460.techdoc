G-BOOK SYSTEM > "NRES" is Displayed in "DCM" on the System Check Mode Screen

for Preparation Click here


DESCRIPTION

If the telematics transceiver does not receive any of the following from the display and navigation module display within a specified time, "NRES" will be displayed.
  1. System inspection request signal
  2. System inspection result request signal
  3. Diagnosis memory request signal

Inspect the fuse for circuits related to this system before performing the following inspection procedure.

2.CHECK HARNESS AND CONNECTOR (TELEMATICS TRANSCEIVER - BATTERY AND BODY GROUND)



  1. Disconnect the G56 telematics transceiver connector.

  1. Measure the resistance according to the value(s) in the table below.

    Standard Resistance:
    Tester Connection
    		Condition
    		Specified Condition
    G56-4 (E) - Body ground
    		Always
    		Below 1 Ω

  1. Measure the voltage according to the value(s) in the table below.

    Standard Voltage:
    Tester Connection
    		Switch Condition
    		Specified Condition
    G56-1 (+B) - Body ground
    		Engine switch off
    		11 to 14 V
    G56-7 (IG2) - Body ground
    		Engine switch on (IG)
    		11 to 14 V
    G56-8 (ACC) - Body ground
    		Engine switch on (ACC)
    		11 to 14 V
