POWER WINDOW CONTROL SYSTEM DESCRIPTION

  1. The power window control system controls the power window operation using the power window regulator motors. The main controls of this system are the multiplex network master switch, which is mounted on the driver side door, the power window regulator switch, which is mounted on the front passenger door, and the power window regulator switches, which are mounted on the rear doors. Operating a window switch results in electrical power being transmitted to the corresponding power window regulator motor.


FUNCTION OF MAIN COMPONENTS
The power window control system consists of the following components:

Component
Outline
Multiplex network master switch assembly
  1. Controls power window operation for all seats.
  2. When lock switch is set to lock position, multiplex network master switch prohibits operation of power windows from other regulator switches. However, operation of all power windows from multiplex network master switch is not affected.
Power window regulator switch assembly
  1. Located on front passenger door.
  2. Switch controls front passenger window operation.
Rear power window regulator switch assembly
  1. for LH Side
  2. for RH Side
  1. Located on rear doors.
  2. Each switch controls window operation for its respective window.
Power window regulator motor assembly
  1. Receives switch signals and changes signals into motor activation. As a result, window position changes.
  2. Uses Hall-IC in regulator motor with ECU to detect window position and jamming.

SYSTEM FUNCTION
The power window control system has the following functions:

Function
Outline
Manual up-and-down function
Function that causes the window to go up while the power window switch is being pulled halfway up and to go down while pushed halfway down. The window stops as soon as the switch is released.
All door auto up-and-down function
Function that enables the windows of all doors to be fully opened or closed by one full pull or push of the multiplex network master switch or power window regulator switch.
Window lock function
  1. Function that disables power window regulator switch operation when the window lock switch is pressed.
  2. The power window regulator switch assemblies can be operated when the window lock switch is pressed again.
Jam protection function
Function that automatically stops the power window and moves it downward if a foreign object gets jammed in the door window during auto up and manual up operation.
Remote control function
Function that allows the multiplex network master switch to control manual and auto up-and-down operations of the front passenger door window and rear door windows.
Key-off operation function
  1. Function that makes it possible to operate all power windows for approximately 43 sec. after the engine switch is turned off when the driver side door or front passenger side door has not been opened.
  2. When 45 sec. have elapsed, or the driver door or front passenger door is opened after the engine switch is turned off, the key-off operation function will be disabled.

HINT:
  1. If the window lock switch is off, the windows can be operated using any of the power window regulator switches. If the window lock switch is on, only the multiplex network master switch can be used to operate the windows (remote operation remains possible).
  2. Even if the driver door or front passenger door is opened during automatic window operation while the key-off operation function is on, the window will continue operating.

Key-linked open and close function*
  1. When the key is not inside the vehicle and the driver side door is locked, turning and maintaining the driver side door key cylinder in the lock position for 2 seconds or more causes the main body ECU to activate the power window regulator motors to close all the power windows while the key is turned.
  2. Similarly, when the driver side door is unlocked, turning and maintaining the driver side door key cylinder in the unlock position for 2 seconds or more will cause all power windows to open.
Wireless transmitter-linked open and close function*
  1. When the certification ECU receives an unlock signal from the electrical key transmitter for 3 seconds or more, the main body ECU (multiplex network body ECU) sends an operation request, signal to the power window ECU in each power window regulator motor assembly. As a result the power window regulator motors open all power windows.
  2. When the certification ECU receives a lock signal from the electrical key transmitter for 3 seconds or more, the main body ECU (multiplex network body ECU) sends an operation request signal to the power window ECU in each power window regulator motor assembly. As a result, the power window regulator motors close all power windows.
Diagnosis
Function that allows diagnosis of the power window control system if any of the power window ECUs (built into each power window regulator motor) detect a malfunction in the system. If a malfunction is detected, the power window ECU will illuminate the LEDs on the multiplex network master switch and the respective power window regulator switch assemblies to inform the driver and passenger of the malfunction.
Fail-safe
A fail-safe function to disable a part of power window functions if the pulse sensor in the power window motor has a malfunction.
Auto up-and-down function and remote control function are disabled.
  1. *: Default setting is OFF.
