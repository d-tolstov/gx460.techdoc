SEAT BELT WARNING SYSTEM > Driver Side Seat Belt Warning Light does not Operate

for Preparation Click here


DESCRIPTION

When the engine switch is turned on (IG), the center airbag sensor transmits front seat inner belt LH state signals to the main body ECU, and then the main body ECU sends signals to the combination meter through the CAN. If the driver side seat belt is not fastened, the combination meter blinks the driver side seat belt warning light. If the seat belt is fastened, the warning light goes off.
