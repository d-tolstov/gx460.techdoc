DTC    C1379    Downhill Assist Control Switch Malfunction (Test Mode DTC)

for Preparation Click here


DESCRIPTION

w/ Downhill Assist Control:
DTC C1379 is cleared when the downhill assist control switch sends a downhill assist control operation signal or when test mode ends.
w/ Crawl Control:
DTC C1379 is cleared when the crawl control switch sends a crawl control operation signal or when test mode ends.

DTC Code
DTC Detection Condition
Trouble Area
C1379
Stored only during test mode.
  1. Combination switch
  2. Harness or connector
  3. Master cylinder solenoid (Skid control ECU)
