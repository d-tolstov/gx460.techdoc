DTC    C1240    Yaw Rate Sensor Discrimination

for Preparation Click here


DESCRIPTION

The skid control ECU determines if the yaw rate and acceleration sensor identification information sent via CAN communication matches the yaw rate and acceleration sensor identification information memorized by the skid control ECU.

DTC Code
DTC Detection Condition
Trouble Area
C1240
The yaw rate and acceleration sensor identification information memorized by the skid control ECU does not match the yaw rate and acceleration sensor identification information sent via CAN communication for 1 second or more.
  1. Proper yaw rate and acceleration sensor not installed
  2. Master cylinder solenoid (Skid Control ECU)
