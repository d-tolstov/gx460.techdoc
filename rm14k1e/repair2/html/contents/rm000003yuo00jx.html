DTC    B15C5    Manual Button Malfunction

for Preparation Click here


DESCRIPTION

If the telematics transceiver detects an error in the communication between the telematics transceiver and emergency call switch during the telematics transceiver self-check, this DTC will be stored.

DTC Code
DTC Detection Condition
Trouble Area
B15C5
A problem with the emergency call switch is detected.
  1. Map light assembly
  2. Harness or connector
  3. Telematics transceiver
