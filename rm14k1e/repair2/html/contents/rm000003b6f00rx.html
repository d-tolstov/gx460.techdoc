DISPLAY AND NAVIGATION ASSEMBLY > INSTALLATION

for Preparation Click here



1. INSTALL HAZARD WARNING SIGNAL SWITCH ASSEMBLY

  1. Attach the 2 claws to install the switch.



2. INSTALL CENTER INSTRUMENT PANEL REGISTER ASSEMBLY



  1. Attach the 8 claws to install the center instrument panel register.



3. INSTALL NO. 2 RADIO BRACKET

  1. Install the No. 2 radio bracket with the 3 bolts.

    Torque:
    3.0 N*m{ 31 kgf*cm , 27 in.*lbf }



4. INSTALL NO. 1 RADIO BRACKET

  1. Install the No. 1 radio bracket with the 3 bolts.

    Torque:
    3.0 N*m{ 31 kgf*cm , 27 in.*lbf }

  1. Attach the claw and clamp to install the connector.



5. INSTALL DISPLAY AND NAVIGATION MODULE DISPLAY

  1. Connect the connectors.




  1. Insert the display and navigation module display and attach the 7 clips on the backside of the display and navigation module display.

    NOTICE:
    When inserting the display, do not press the knobs on it.

  1. Install the display and navigation module display with the 4 bolts.

    Torque:
    12 N*m{ 122 kgf*cm , 9 ft.*lbf }



6. INSTALL LOWER INSTRUMENT PANEL FINISH PANEL ASSEMBLY

  1. Connect each connector and cooler thermistor.

  1. Attach the 4 clips to install the lower instrument panel finish panel.



7. INSTALL CENTER INSTRUMENT CLUSTER FINISH PANEL GARNISH



  1. Attach the 13 clips to install the center instrument cluster finish panel garnish together with the instrument cluster finish panel garnish.
    Text in Illustration 
    *1
    		Center Instrument Cluster Finish Panel Garnish
    *2
    		Instrument Cluster Finish Panel Garnish



8. INSTALL NO. 2 INSTRUMENT PANEL REGISTER ASSEMBLY

  1. Attach the 8 clips to install the No. 2 instrument panel register.



9. INSTALL RADIO RECEIVER ASSEMBLY WITH BRACKET


  1. w/o Navigation System:

    1. Connect the connectors.




    2. Insert the radio receiver to attach the 6 clips and rib on its backside.

      NOTICE:
      When inserting the radio receiver, do not press the knobs on it.

    3. Install the radio receiver with the 4 bolts.

      Torque:
      13 N*m{ 127 kgf*cm , 9 ft.*lbf }



  1. w/ Navigation System:

    1. Connect the connectors.




    2. Insert the radio receiver to attach the 6 clips and 3 ribs on its backside.

      NOTICE:
      When inserting the radio receiver, do not press the knobs on it.

    3. Install the radio receiver with the 4 bolts.

      Torque:
      13 N*m{ 127 kgf*cm , 9 ft.*lbf }



10. INSTALL UPPER CONSOLE PANEL SUB-ASSEMBLY



  1. Put protective tape around the upper console panel.
    Text in Illustration 
    *1
    		Protective Tape




  1. Using a moulding remover, detach the 7 clips.

  1. Disconnect each connector and move the upper console panel in the direction of the arrow to remove it.



11. INSTALL SHIFT LEVER KNOB SUB-ASSEMBLY



  1. Install the shift lever knob and twist it in the direction indicated by the arrow.



12. INSTALL FRONT CONSOLE UPPER PANEL GARNISH

  1. Attach the 2 clips to install the front upper console panel garnish.



13. INSTALL INSTRUMENT PANEL FINISH PANEL END RH

  1. Attach the 11 clips to install the instrument panel finish panel end.



14. INSTALL INSTRUMENT PANEL FINISH PANEL END LH

  1. Attach the 11 clips to install the instrument panel finish panel end.



15. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).
