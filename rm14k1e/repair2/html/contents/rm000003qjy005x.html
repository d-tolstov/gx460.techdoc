1. INSTALL REAR HEIGHT CONTROL SENSOR SUB-ASSEMBLY RH



  1. Install the height control sensor with the 2 bolts.
    Text in Illustration 
    *1
    		Matchmarks

    Torque:
    13 N*m{ 133 kgf*cm , 10 ft.*lbf }

  1. Align the matchmarks on the link and bracket.

  1. Connect the sensor link with the nut.

    Torque:
    5.4 N*m{ 55 kgf*cm , 48 in.*lbf }

  1. Attach the sensor connector, and then connect the connector.

  1. Attach the clamp.
