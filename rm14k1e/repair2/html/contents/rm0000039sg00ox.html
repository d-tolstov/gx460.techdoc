STEERING COLUMN ASSEMBLY > INSTALLATION

for Preparation Click here



1. CONNECT NO. 2 STEERING INTERMEDIATE SHAFT SUB-ASSEMBLY

  1. Align the matchmarks on the No. 2 steering intermediate shaft and power steering gear.

  1. Install the bolt.

    Torque:
    36 N*m{ 367 kgf*cm , 27 ft.*lbf }



2. INSTALL NO. 1 STEERING COLUMN HOLE COVER SUB-ASSEMBLY

  1. Install the hole cover with the 4 bolts.

    Torque:
    5.0 N*m{ 51 kgf*cm , 44 in.*lbf }

    NOTICE:
    Do not fold back the boot part of the steering hole cover or extend it excessively. If it is extended excessively, return it to its original position.



3. CONNECT STEERING INTERMEDIATE SHAFT ASSEMBLY

  1. Align the matchmarks on the No. 2 steering intermediate shaft assembly and steering intermediate shaft assembly.

    HINT:
    Install the steering intermediate shaft to the inside of the vehicle.

  1. Install the bolt.

    Torque:
    36 N*m{ 367 kgf*cm , 27 ft.*lbf }



4. INSTALL FRONT NO. 1 FENDER APRON TO FRAME SEAL LH

  1. Install the front No. 1 fender apron to frame seal LH with the 5 clips.



5. INSTALL STEERING COLUMN ASSEMBLY

  1. Align the matchmarks on the steering intermediate shaft, steering shaft stopper and steering column.

  1. Install the steering column with the 4 nuts.

    Torque:
    26 N*m{ 265 kgf*cm , 19 ft.*lbf }

  1. Install the bolts.

    Torque:
    36 N*m{ 367 kgf*cm , 27 ft.*lbf }

  1. Attach the claws to install the wire harness protector and wire harness.



6. INSTALL NO. 1 INSTRUMENT LOWER PANEL AIRBAG ASSEMBLY

  1. Install the instrument lower panel airbag assembly (Click here).



7. INSTALL COMBINATION SWITCH ASSEMBLY WITH SPIRAL CABLE SUB-ASSEMBLY

  1. Using pliers, grip the claws of the clamp and install the combination switch assembly with spiral cable sub-assembly to the steering column assembly with the clamp.

  1. Connect the connectors to the combination switch with spiral cable.



8. INSTALL STEERING COLUMN UPPER COVER

  1. Attach the claw to install the steering column upper cover.

  1. Attach the 4 clips to install the steering column upper cover to the instrument cluster finish panel.



9. INSTALL STEERING COLUMN LOWER COVER

  1. Attach the 2 claws to install the steering column lower cover.

    NOTICE:
    Do not damage the tilt and telescopic switch.

  1. Install the 3 screws.



10. INSTALL STEERING WHEEL ASSEMBLY

  1. Install the steering wheel assembly (Click here).



11. INSTALL FRONT WHEEL LH


12. CHECK FRONT WHEELS FACING STRAIGHT AHEAD


13. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
  1. Reset the auto tilt away function setting to the previous condition by changing the customize parameter (Click here).
  2. When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).



14. INSPECT SRS WARNING LIGHT

  1. Inspect the SRS warning light (Click here).
