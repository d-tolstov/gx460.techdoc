1. VEHICLE CONTRACT SETTING


HINT:
  1. If the display and navigation module display or the telematics transceiver is replaced on vehicles that do not have a contract for the G-BOOK service, perform the vehicle contract setting.
  2. If the display and navigation module display or the telematics transceiver is replaced on vehicles that have a contract for the G-BOOK service, perform the vehicle contract setting and the procedure to resume the G-BOOK service and to start the emergency call service.






  1. Check the ID before replacement (when diagnosis can be activated).

    HINT:
    1. If the display and navigation module display is replaced, check the G-BOOK ID.
    2. If the telematics transceiver is replaced, check the DCM ID.


    1. Enter diagnostic mode (Click here).




    2. Select "Service Information" on the Service Menu screen.




    3. Select "G-BOOK Information for Contract" on the Service Information screen.




    4. Check the G-BOOK ID or DCM ID.

      HINT:
      The DCM ID is also specified on the telematics transceiver label.

  1. Replace the display and navigation module display or the telematics transceiver.

    1. Replace the display and navigation module display or the telematics transceiver at a dealership.



  1. Check the ID after replacement.

    HINT:
    1. If the display and navigation module display is replaced, check the G-BOOK ID.
    2. If the telematics transceiver is replaced, check the DCM ID.


    1. Enter diagnostic mode (Click here).




    2. Select "Service Information" on the Service Menu screen.




    3. Select "G-BOOK Information for Contract" on the Service Information screen.




    4. Check the G-BOOK ID or DCM ID.

      HINT:
      The DCM ID is also specified on the telematics transceiver label.



  1. Fill-in the Vehicle Contract Setting Request Form.




    1. After filling in the Vehicle Contract Setting Request Form, send it back to the G-BOOK center.

  1. Perform the procedure to resume the G-BOOK service.

    HINT:
    The screen illustrations provided are samples only and may be different from the actual screens.


    1. Press the "INFO/TEL" switch on the display and navigation module display.

    2. Perform the operations shown in the illustration and select G-BOOK.com.

    3. Check that G-BOOK.com is displayed.

    4. After the guidance screen is displayed, perform the procedure to resume the G-BOOK service by following the instructions on the screen.

  1. Perform the procedure to start the emergency call service.

    1. After completing the procedure to resume the G-BOOK service, perform the procedure to start the emergency call service by following the instructions on the screen.

      HINT:
      The emergency call service will be available when the procedure is completed successfully.
