FRONT DOOR BELT MOULDING > INSTALLATION

for Preparation Click here


HINT:
  1. Use the same procedure for the RH and LH sides.
  2. The procedure listed below is for the LH side.



1. INSTALL FRONT DOOR BELT MOULDING LH

  1. Attach the 6 claws to install the front door belt moulding.



2. INSTALL FRONT DOOR GLASS RUN LH



  1. Install the front door glass run.



3. INSTALL FRONT DOOR GLASS SUB-ASSEMBLY LH

  1. Connect the cable to the negative (-) battery terminal.

  1. Connect the power window regulator master switch assembly and move the front door glass sub-assembly so that the door glass bolt installation locations can be seen.

  1. Disconnect the cable from the negative (-) battery terminal and power window regulator master switch assembly.

    NOTICE:
    1. w/ Navigation System (for HDD):
      After the engine switch is turned off, the display and navigation module display (HDD navigation system) records various types of memory and settings. As a result, after turning the engine switch off, be sure to wait for the time specified in the following table before disconnecting the cable from the negative (-) battery terminal.

      Waiting Time before Disconnecting Cable from Negative (-) Battery Terminal :

      Condition
      		Waiting Time
      Vehicle enrolled in G-BOOK system
      		6 minutes
      Vehicle not enrolled in G-BOOK system
      		1 minute

    2. When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).




  1. Insert the front door glass sub-assembly into the front door panel along the front door glass run as indicated by the arrows in the order shown in the illustration.




  1. Install the front door glass sub-assembly with the 2 bolts.

    HINT:
    Tighten the bolts in the order shown in the illustration.

    Torque:
    5.5 N*m{ 56 kgf*cm , 49 in.*lbf }



4. INSTALL OUTER REAR VIEW MIRROR ASSEMBLY LH

  1. Attach the claw to install the outer rear view mirror, and then install the 3 nuts.

    Torque:
    8.0 N*m{ 82 kgf*cm , 71 in.*lbf }

  1. w/ Side Monitor System:
    Connect the 2 connectors and attach the clamp.

  1. w/o Side Monitor System:
    Connect the connector and attach the clamp.



5. INSTALL FRONT DOOR SERVICE HOLE COVER LH

  1. Apply new butyl tape to the front door panel.




  1. Pass the front door lock remote control cable assembly and front door inside locking cable assembly through a new front door service hole cover.

  1. Attach the front door service hole cover using the reference points on the front door panel.

    NOTICE:
    1. There should be no wrinkles or folds after attaching the service hole cover.
    2. After attaching the service hole cover, check the seal quality.
    3. Securely install the front door service hole cover preventing wrinkles and air bubbles.

    Text in Illustration 
    *1
    		Reference Point




  1. Attach the 2 clamps.

  1. Install the bolt to the front door wire.

    Torque:
    8.5 N*m{ 87 kgf*cm , 75 in.*lbf }



6. INSTALL FRONT DOOR INNER GLASS WEATHERSTRIP LH



  1. Install the front door inner glass weatherstrip.



7. INSTALL FRONT DOOR TRIM BOARD SUB-ASSEMBLY LH



  1. Connect the front door lock remote control cable assembly and front door inside locking cable assembly.

  1. Connect 2 connectors.




  1. Attach the front door trim board sub-assembly by attaching the 4 claws of the front door inner glass weatherstrip as shown in the illustration.




  1. Attach the 12 clips and front door trim board retainer to install the front door trim board sub-assembly.

  1. Install the 3 screws.



8. INSTALL POWER WINDOW REGULATOR MASTER SWITCH ASSEMBLY WITH FRONT DOOR ARMREST BASE PANEL

  1. Connect the connector.

  1. Attach the 2 clips, 4 claws and guide to install the power window regulator master switch assembly with front door armrest base panel.



9. INSTALL FRONT ARMREST ASSEMBLY LH

  1. Attach the 7 claws and 7 guides to install the front armrest assembly.



10. INSTALL FRONT DOOR INSIDE HANDLE BEZEL PLUG LH

  1. Attach the 3 claws to install the front door inside handle bezel plug.



11. INSTALL FRONT DOOR LOWER FRAME BRACKET GARNISH LH

  1. Attach the 2 claws to install the front door lower frame bracket garnish.



12. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).



13. INITIALIZE POWER WINDOW CONTROL SYSTEM

  1. Initialize the power window control system (Click here).



14. ADJUST SIDE TELEVISION CAMERA ASSEMBLY

HINT:
Adjust the side television camera for the passenger side only.

  1. w/ Rear View Monitor System:
    Adjust the side television camera (Click here).

  1. Parking Assist Monitor System:
    Adjust the side television camera (Click here).
