REAR POWER SEAT CONTROL SYSTEM DESCRIPTION
  1. The third-row seats are equipped with a power folding system.
  2. This system has the following functions: reclining, luggage room expansion, folding and return, and jam protection.
  3. Both the left and right seats are equipped with this system. Each system operates independently.

FUNCTIONS OF MAIN COMPONENTS

Component
Function
No. 3 Seatback Frame Sub-assembly (Reclining Motor)
(Seatback Position Sensor)
The fold seat control ECU provides the voltage that is used to operate the reclining motor. The reclining motor reclines and returns the seatback of the third-row seat (a Hall-type position sensor is installed in the reclining motor to detect the seatback position).
Seat Leg Assembly (Cushion Motor)
(Seat Cushion Position Sensor)
The fold seat control ECU provides the voltage that is used to operate the cushion motor. The cushion motor stows and returns the third-row seat (a Hall-type position sensor is installed in the cushion motor to detect the seat cushion position).
Rear Power Seat Switch
Sends a seatback angle adjustment signal to the fold seat control ECU when this switch is operated. The seatback angle is adjusted based on this signal.
Fold Seat Switch
Sends a third-row seat folding or return operation signal to the fold seat control ECU when this switch is operated. The third-row seat is folded or returned to the original position based on this signal.
Fold Seat Control ECU
This ECU receives signals from each operation switch and drives the motor to perform storage, return and reclining of the seat. When the conditions required for operation are not fulfilled or when the system detects that something is interfering with the movement of the seat, this ECU sounds an internal alarm.
