DTC    B2782    Power Source Control ECU Malfunction


DESCRIPTION

WIRING DIAGRAM

INSPECTION PROCEDURE

INSPECT STEERING LOCK ECU (IGE VOLTAGE)

CLEAR DTC AND DATA LIST MALFUNCTION RECORD

READ VALUE USING INTELLIGENT TESTER (POWER SUPPLY OPEN)

CHECK HARNESS AND CONNECTOR (POWER MANAGEMENT CONTROL ECU - STEERING LOCK ECU)
