DTC    C1379    Downhill Assist Control Switch Malfunction (Test Mode DTC)


DESCRIPTION

WIRING DIAGRAM

INSPECTION PROCEDURE

CONFIRM VEHICLE SPECIFICATIONS

READ VALUE USING INTELLIGENT TESTER (DOWNHILL ASSIST CONTROL SW)

CHECK TEST MODE DTC

INSPECT DOWNHILL ASSIST CONTROL SWITCH (COMBINATION SWITCH)

CHECK HARNESS AND CONNECTOR (SKID CONTROL ECU - DOWNHILL ASSIST CONTROL SWITCH)

CHECK CAN BUS

READ VALUE USING INTELLIGENT TESTER (CRAWL CONTROL SWITCH)

CHECK TEST MODE DTC

CHECK HARNESS AND CONNECTOR (CRAWL CONTROL SWITCH - DRIVING SUPPORT SWITCH CONTROL ECU)

INSPECT CRAWL CONTROL SWITCH (COMBINATION SWITCH)
