for Preparation Click here


HINT:
  1. Use the same procedure for the RH and LH sides.
  2. The procedure listed below is for the LH side.



1. INSTALL FRONT SEAT INNER BELT ASSEMBLY



  1. Install the front seat inner belt with the nut.

    Torque:
    42 N*m{ 428 kgf*cm , 31 ft.*lbf }

    NOTICE:
    Do not allow the anchor part of the front seat inner belt assembly to overlap the protruding parts of the front seat adjuster.

    Text in Illustration 
    *a
    		Protruding Part

  1. for Driver Side:
    Connect the 2 connectors and attach the 5 clamps.

  1. for Passenger Side:
    Connect the 2 connectors and attach the 4 clamps.



2. INSTALL FRONT SEAT INNER CUSHION SHIELD LH



  1. Attach the 3 claws to install the cushion shield.



3. INSTALL FRONT SEAT ASSEMBLY LH

  1. Install the front seat assembly LH (Click here).
