FRONT STABILIZER BAR > INSTALLATION

for Preparation Click here



1. TEMPORARILY INSTALL FRONT STABILIZER LINK ASSEMBLY RH



  1. Temporarily install the front stabilizer link assembly and spacer with the bolt.

    HINT:
    Make sure that the identification mark on the front stabilizer link assembly faces inward and toward the front of the vehicle.

    Text in Illustration 
    *a
    		Mark Position



2. TEMPORARILY INSTALL FRONT STABILIZER WITH TUBE CYLINDER ASSEMBLY

  1. Install the 2 bleeder plug caps to the front stabilizer cylinder with tube cylinder assembly.




  1. Temporarily install the front stabilizer with tube cylinder assembly with the nut and bolt.

    HINT:
    Pass the tube side of the front stabilizer with tube cylinder assembly under the return tube sub-assembly before installing it.

    NOTICE:
    1. Turn the bolt while holding the nut.
    2. Do not hold the front stabilizer with tube cylinder assembly by the cylinder boot.



3. CONNECT NO. 1 FRONT STABILIZER CONTROL TUBE ASSEMBLY

  1. Apply suspension fluid to the threads of the flare nuts.

  1. Using a union nut wrench, connect the No. 1 front stabilizer control tube to the front stabilizer with tube cylinder assembly and tighten the flare nuts.

    Torque:
    44 N*m{ 450 kgf*cm , 33 ft.*lbf }

    HINT:
    Use the formula to calculate special torque values for situations where a union nut wrench is combined with a torque wrench (Click here).

  1. Tighten the bolt.

    Torque:
    29 N*m{ 296 kgf*cm , 21 ft.*lbf }



4. INSTALL FRAME APRON SEAL BRACKET

  1. Install the bracket with the bolt.

    Torque:
    29 N*m{ 296 kgf*cm , 21 ft.*lbf }



5. INSTALL FRONT STABILIZER BAR



  1. Install the 2 front stabilizer lower bracket bushes to the front stabilizer bar.

    HINT:
    Align the protrusions on the bushes with the identification marks on the front stabilizer bar with the protrusions facing inward.

    Text in Illustration 
    *a
    		Protrusion
    *b
    		Mark Position




  1. With the identification marks of the front stabilizer bar facing downwards, support the front stabilizer bar with a jack.

    NOTICE:
    Place a wooden block between the jack and front stabilizer bar to prevent damage.

    HINT:
    Place the jack under the center of the vehicle.




  1. Install the front stabilizer bar and 2 front stabilizer lower brackets with the 4 bolts.

    Torque:
    40 N*m{ 408 kgf*cm , 30 ft.*lbf }

    HINT:
    Make sure that the protrusions on the front stabilizer lower bracket bushes are positioned within 180° of the stabilizer cylinder and stabilizer link.

    Text in Illustration 
    *1
    		Stabilizer Link
    *2
    		Stabilizer Cylinder



6. INSTALL FRONT STABILIZER END BRACKET



  1. Install the 2 front stabilizer brackets and 2 front stabilizer link bushes with the 4 bolts.

    Torque:
    75 N*m{ 765 kgf*cm , 55 ft.*lbf }

    HINT:
    There are stamps on the front stabilizer brackets to distinguish between the right and left brackets.



7. INSTALL FRONT SUSPENSION MEMBER BRACE SUB-ASSEMBLY

  1. Install the 2 member braces with the 6 bolts.

    Torque:
    30 N*m{ 306 kgf*cm , 22 ft.*lbf }



8. BLEED SUSPENSION FLUID

  1. Bleed the suspension fluid (Click here).



9. INSTALL FRONT WHEEL

Torque:
112 N*m{ 1142 kgf*cm , 83 ft.*lbf }



10. TIGHTEN FRONT STABILIZER LINK ASSEMBLY RH



  1. Tighten the bolt on the front stabilizer link assembly.

    Torque:
    140 N*m{ 1428 kgf*cm , 103 ft.*lbf }

    NOTICE:
    Tighten the bolt with the wheels on the ground.



11. TIGHTEN FRONT STABILIZER WITH TUBE CYLINDER ASSEMBLY



  1. Tighten the bolt on the front stabilizer with tube cylinder assembly.

    Torque:
    130 N*m{ 1326 kgf*cm , 96 ft.*lbf }

    NOTICE:
    Tighten the bolt with the wheels on the ground.



12. INSTALL FRONT FENDER APRON SEAL LH

  1. Install the front fender apron seal LH with the 7 clips.



13. INSTALL NO. 1 ENGINE UNDER COVER SUB-ASSEMBLY



  1. Hook the No. 1 engine under cover to the vehicle body as shown in the illustration.

  1. Install the 4 bolts.

    Torque:
    29 N*m{ 296 kgf*cm , 21 ft.*lbf }



14. INSTALL LOWER FRONT BUMPER COVER

  1. Install the lower front bumper cover with the 5 bolts and clip.

    Torque:
    8.0 N*m{ 82 kgf*cm , 71 in.*lbf }



15. INSTALL STABILIZER CONTROL VALVE PROTECTOR

  1. Install the stabilizer control valve protector with the 2 bolts.

    Torque:
    29 N*m{ 296 kgf*cm , 21 ft.*lbf }



16. INSTALL SIDE STEP ASSEMBLY LH

  1. Attach the 6 clips to install the side step.

  1. Install the 3 screws.

  1. Using a T30 "TORX" socket wrench, install the 2 screws.

    Torque:
    3.0 N*m{ 31 kgf*cm , 27 in.*lbf }

  1. Install the side step with the 6 bolts.

    Torque:
    18 N*m{ 184 kgf*cm , 13 ft.*lbf }

  1. Connect the connector.

  1. Attach the clamp.



17. INSPECT VEHICLE HEIGHT

  1. Inspect the vehicle height (Click here).
