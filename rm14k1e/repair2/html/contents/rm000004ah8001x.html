FRONT CONSOLE BOX > INSTALLATION

for Preparation Click here


HINT:
A bolt without a torque specification is shown in the standard bolt chart (Click here).



1. INSTALL REAR CONSOLE BOX ASSEMBLY

  1. Install the rear console box with the 2 screws and 4 bolts.

  1. Attach the clamp.

  1. Connect each connector.



2. INSTALL REAR CONSOLE END PANEL SUB-ASSEMBLY

  1. Connect the connector.

  1. Attach the 8 clips to install the rear console end panel.



3. INSTALL REAR UPPER CONSOLE PANEL SUB-ASSEMBLY

  1. Attach the 4 clips and 2 guides to install the rear upper console panel.



4. INSTALL CONSOLE COIN BOX

  1. Install the console coin box.



5. INSTALL UPPER CONSOLE PANEL SUB-ASSEMBLY

  1. Connect each connector.

  1. Attach the 7 clips to install the upper console panel.



6. INSTALL SHIFT LEVER KNOB SUB-ASSEMBLY



  1. Install the shift lever knob and twist it in the direction indicated by the arrow.



7. INSTALL INSTRUMENT PANEL FINISH PANEL END LH

  1. Attach the 11 clips to install the instrument panel finish panel end.



8. INSTALL INSTRUMENT PANEL FINISH PANEL END RH

  1. Attach the 11 clips to install the instrument panel finish panel end.



9. INSTALL FRONT NO. 1 CONSOLE BOX INSERT

  1. Attach the 2 clips and guide to install the front No. 1 console box insert.



10. INSTALL FRONT NO. 2 CONSOLE BOX INSERT

  1. Attach the 2 clips and guide to install the front No. 2 console box insert.



11. INSTALL NO. 1 INSTRUMENT PANEL UNDER COVER SUB-ASSEMBLY

  1. Connect the connector and attach the clamp.

  1. Attach the 2 clips and 2 guides to install the No. 1 instrument panel under cover.

  1. Install the screw.



12. INSTALL NO. 2 INSTRUMENT PANEL UNDER COVER SUB-ASSEMBLY

  1. Connect the foot light and attach the clamp.

  1. Attach the 3 clips and 2 guides to install the No. 2 instrument panel under cover.

  1. Install the screw.
