DRIVER MONITOR ECU > INSTALLATION

for Preparation Click here


HINT:
A bolt without a torque specification is shown in the standard bolt chart (Click here).



1. INSTALL DRIVER MONITOR ECU ASSEMBLY

  1. Install the driver monitor ECU assembly with the bolt.

  1. Connect the connector.



2. INSTALL LOWER NO. 1 INSTRUMENT PANEL AIRBAG ASSEMBLY



  1. Connect the connector.

    NOTICE:
    When handling the airbag connector, take care not to damage the airbag wire harness.

  1. Install the airbag assembly with the 4 bolts.

    Torque:
    10 N*m{ 102 kgf*cm , 7 ft.*lbf }



3. INSTALL LOWER INSTRUMENT PANEL FINISH PANEL SUB-ASSEMBLY

  1. Connect each connector and cable.

  1. Attach the 13 clips and 2 guides to install the lower instrument panel finish panel.

  1. Install the 2 bolts <C>.

  1. Attach the 2 claws to close the cover.



4. INSTALL LOWER INSTRUMENT PANEL FINISH PANEL ASSEMBLY

  1. Connect each connector and cooler thermistor.

  1. Attach the 4 clips to install the lower instrument panel finish panel.



5. INSTALL NO. 1 INSTRUMENT PANEL FINISH CUSHION

  1. Connect the connector.

  1. Attach the 4 clips to install the No. 1 instrument panel finish cushion.



6. INSTALL NO. 2 SWITCH HOLE BASE

  1. Connect each connector.

  1. Attach the 5 clips and claw to install the No. 2 switch hole base.



7. INSTALL INSTRUMENT SIDE PANEL LH

  1. Attach the 3 clips and 2 guides to install the instrument side panel.



8. INSTALL COWL SIDE TRIM BOARD LH

  1. Attach the clip and claw to install the cowl side trim board.

  1. Install the clip.



9. INSTALL DOOR SCUFF PLATE ASSEMBLY LH

  1. Connect the connector.

  1. Attach the 4 clips, 10 claws and 2 guides to install the door scuff plate.



10. INSTALL INSTRUMENT PANEL FINISH PANEL END LH

  1. Attach the 11 clips to install the instrument panel finish panel end.



11. INSTALL NO. 1 INSTRUMENT PANEL UNDER COVER SUB-ASSEMBLY

  1. Connect the connector and attach the clamp.

  1. Attach the 2 clips and 2 guides to install the No. 1 instrument panel under cover.

  1. Install the screw.



12. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).



13. CHECK SRS WARNING LIGHT

  1. Check the SRS warning light (Click here).
