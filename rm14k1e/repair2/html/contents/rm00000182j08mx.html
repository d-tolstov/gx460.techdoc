DTC    B1593    No Response from Stereo Component Amplifier

DTC    B15D3    Stereo Component Amplifier Disconnected

for Preparation Click here


DESCRIPTION

The display and navigation module display and the stereo component amplifier assembly are connected by the MOST communication line.
When a MOST communication error occurs between the display and navigation module display and the stereo component amplifier assembly, these DTCs are stored.

DTC Code
DTC Detection Condition
Trouble Area
B1593
No response to the diagnostic memory request from the master unit.
  1. Harness or connector
  2. Stereo component amplifier assembly
  3. Display and navigation module display
  4. Radio receiver assembly
B15D3
A device that is listed in the MOST network connected device record of the master unit is missing.

HINT:
For the MOST network, the display and navigation module display is the master unit.
