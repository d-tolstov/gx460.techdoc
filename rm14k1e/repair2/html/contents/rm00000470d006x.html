1. INSTALL NO. 1 ULTRASONIC SENSOR RETAINER



HINT:
Use the same procedure to install the No. 1 ultrasonic sensor retainer on the other side.

  1. Align the keyhole and protrusion as shown in the illustration.

  1. Install the ultrasonic sensor retainer to the front bumper cover.

    NOTICE:
    Do not damage the bumper cover with the protrusion when installing the retainer.

3. INSTALL NO. 1 ULTRASONIC SENSOR

HINT:
Use the same procedure to install the No. 1 ultrasonic sensor on the other side.

  1. Attach the 2 claws to install the No. 1 ultrasonic sensor.

  1. Connect the connector.



4. INSTALL ULTRASONIC SENSOR CLIP

HINT:
Use the same procedure to install the ultrasonic sensor clip on the other side.

  1. Attach the 4 claws to install the ultrasonic sensor clip.



5. INSTALL REAR BUMPER COVER

  1. Connect the 3 connectors.

  1. Attach the 12 claws to install the rear bumper cover.

  1. Install the 6 bolts, 2 screws and 2 clips.

    Torque:
    for bolt:
    8.0 N*m{ 82 kgf*cm , 71 in.*lbf }



6. INSTALL REAR BUMPER BAR PLATE

  1. Attach the 14 claws to install the rear bumper bar plate.



7. INSTALL JACK BOX HOLE COVER

  1. Attach the 2 claws to install the jack box hole cover.



8. INSTALL REAR BUMPER PROTECTOR LH

  1. Install the rear bumper protector with the 2 screws and clip.



9. INSTALL REAR BUMPER PROTECTOR RH

HINT:
Use the same procedure described for the LH side.



10. INSTALL REAR QUARTER PANEL MUDGUARD LH

  1. Install the rear quarter panel mudguard with the 4 screws.



11. INSTALL REAR QUARTER PANEL MUDGUARD RH

HINT:
Use the same procedure described for the LH side.
