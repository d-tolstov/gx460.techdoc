POWER MANAGEMENT CONTROL ECU > INSTALLATION

for Preparation Click here


HINT:
A bolt without a torque specification is shown in the standard bolt chart (Click here).



1. INSTALL POWER MANAGEMENT CONTROL ECU

  1. Attach the 2 claws to install the power management control ECU.



2. INSTALL ECU INTEGRATION BOX RH

  1. Install the ECU integration box RH with the 2 nuts and bolt.

  1. Attach the clamp.

  1. Connect all the connectors.

    NOTICE:
    1. Do not twist the connectors of ECU when connecting them.
    2. Do not apply more force than necessary to the connectors. If more than 98 N (10 kgf, 22 lbf) of force is applied to a connector, the connector or connector holder may be damaged.



3. INSTALL NO. 2 AIR DUCT SUB-ASSEMBLY

  1. Attach the 2 claws to install the No. 2 air duct sub-assembly.

  1. Install the screw.



4. INSTALL TELEMATICS TRANSCEIVER

  1. Insert the telematics transceiver with bracket into the vehicle.




  1. Install the telematics transceiver with bracket with the bolt and nut.

    Torque:
    for bolt A:
    13 N*m{ 127 kgf*cm , 9 ft.*lbf }
    for nut B:
    8.0 N*m{ 82 kgf*cm , 71 in.*lbf }

  1. Connect the connectors.



5. INSTALL INSTRUMENT PANEL BOX ASSEMBLY



  1. Connect each connector and attach the clamp.

  1. Attach the 4 clips to install the glove compartment door.

  1. Install the 3 screws <B> and 2 bolts <C>.
    Text in Illustration 
    *1
    		Screw <B>
    *2
    		Bolt <C>



6. INSTALL INSTRUMENT PANEL BOX DOOR COVER

  1. Attach the 2 claws and 2 guides to install the instrument panel box door cover.



7. INSTALL LOWER NO. 2 INSTRUMENT PANEL AIRBAG ASSEMBLY



  1. Connect the connector.

    NOTICE:
    When handling the airbag connector, take care not to damage the airbag wire harness.

  1. Attach the 6 claws to install the instrument panel airbag.

  1. Install the 3 bolts.

    Torque:
    10 N*m{ 102 kgf*cm , 7 ft.*lbf }



8. INSTALL LOWER INSTRUMENT COVER LH

  1. Attach the 4 claws to install the lower instrument cover.



9. INSTALL NO. 2 INSTRUMENT PANEL REGISTER ASSEMBLY

  1. Attach the 8 clips to install the No. 2 instrument panel register.



10. INSTALL CENTER INSTRUMENT CLUSTER FINISH PANEL GARNISH



  1. Attach the 13 clips to install the center instrument cluster finish panel garnish together with the instrument cluster finish panel garnish.
    Text in Illustration 
    *1
    		Center Instrument Cluster Finish Panel Garnish
    *2
    		Instrument Cluster Finish Panel Garnish



11. INSTALL INSTRUMENT SIDE PANEL RH

  1. Attach the 3 clips and 2 guides to install the instrument side panel.



12. INSTALL INSTRUMENT PANEL FINISH PANEL END RH

  1. Attach the 11 clips to install the instrument panel finish panel end.



13. INSTALL FRONT NO. 2 CONSOLE BOX INSERT

  1. Attach the 2 clips and guide to install the front No. 2 console box insert.



14. INSTALL NO. 2 INSTRUMENT PANEL UNDER COVER SUB-ASSEMBLY

  1. Connect the foot light and attach the clamp.

  1. Attach the 3 clips and 2 guides to install the No. 2 instrument panel under cover.

  1. Install the screw.



15. INSTALL COWL SIDE TRIM BOARD RH

  1. Attach the clip and claw to install the cowl side trim board.

  1. Install the clip.



16. INSTALL DOOR SCUFF PLATE ASSEMBLY RH

  1. w/ Illumination:
    Connect the connector.

  1. Attach the 4 clips, 10 claws and 2 guides to install the door scuff plate.



17. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).



18. CHECK SRS WARNING LIGHT

  1. Check the SRS warning light (Click here).
