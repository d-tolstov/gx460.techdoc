FRONT SPEED SENSOR > INSTALLATION

for Preparation Click here


HINT:
  1. The procedure listed below is for the LH side.
  2. Other than areas where instructions are provided, use the same procedures for the RH and LH sides.



1. INSTALL FRONT SPEED SENSOR

  1. Install the speed sensor with the bolt.

    Torque:
    8.5 N*m{ 87 kgf*cm , 75 in.*lbf }

    NOTICE:
    1. Make sure there are no pieces of iron or other foreign matter attached to the sensor tip.
    2. While inserting the speed sensor into the knuckle hole, do not strike or damage the sensor tip.
    3. After installing the speed sensor, make sure there is no clearance or foreign matter between the sensor stay part and the knuckle.
    4. Make sure there is no foreign matter attached to the speed sensor rotor.



2. INSTALL SKID CONTROL SENSOR CLAMP

  1. Install the skid control sensor clamp with the bolt.

    Torque:
    13 N*m{ 127 kgf*cm , 9 ft.*lbf }

    NOTICE:
    Install the clamp so that the rotation stopper touches the knuckle.



3. INSTALL FRONT SKID CONTROL SENSOR WIRE

  1. Connect the connector as follows.

    1. for LH:
      1. Attach the connector, and then connect the connector.

      NOTICE:
      Securely connect the connector.

    2. for RH:
      1. Install the skid control sensor clamp with the bolt.

      Torque:
      5.0 N*m{ 51 kgf*cm , 44 in.*lbf }

      NOTICE:
      Make sure the clamp rotation stopper touches the installation position.

      2. Attach the connector, and then connect the connector.

      NOTICE:
      1. Securely connect the connector.
      2. When connecting the connector, do not twist the wire harness.

      3. Attach the clip.

  1. Install the 2 harness clamps with the 2 bolts.

    Torque:
    13 N*m{ 127 kgf*cm , 9 ft.*lbf }

    NOTICE:
    1. When installing the clamps, do not twist the wire harness.
    2. Make sure the clamp rotation stopper touches the installation position.

  1. Install the 2 harness clamps with the 2 bolts.

    Torque:
    13 N*m{ 127 kgf*cm , 9 ft.*lbf }

    NOTICE:
    1. When installing the clamps, do not twist the wire harness.
    2. Make sure the clamp rotation stopper touches the installation position.

  1. Attach the clip.

  1. Connect the connector.

    NOTICE:
    Securely connect the connector.



4. INSTALL FRONT WHEEL

Torque:
112 N*m{ 1142 kgf*cm , 83 ft.*lbf }
