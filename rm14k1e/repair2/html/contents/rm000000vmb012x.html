HINT:
A bolt without a torque specification is shown in the standard bolt chart (Click here).



1. INSTALL DISC PLAYER BRACKET



  1. Align the protrusions of the disc player bracket, labeled A, with the indentations of the navigation module board.

  1. Install the disc player bracket with the 2 screws labeled B.



2. INSTALL NO. 2 DISC PLAYER BRACKET



  1. Align the protrusions of the No. 2 disc player bracket, labeled A, with the indentations of the navigation module board.

  1. Install the No. 2 disc player bracket with the 2 screws labeled B.



3. INSTALL DISC PLAYER COVER

  1. Install the disc player cover with the 2 screws.

  1. Install the screw.



4. INSTALL NAVIGATION MODULE BOARD WITH BRACKET



  1. Install the navigation module board with bracket with the 2 screws and bolt.

    Torque:
    for bolt A:
    8.0 N*m{ 82 kgf*cm , 71 in.*lbf }

  1. Connect the connectors and attach the clamp.




  1. Return the floor carpets to their original positions as shown in the illustration and attach the fastener.



5. INSTALL FRONT SEAT ASSEMBLY LH

  1. Install the front seat assembly LH (Click here).



6. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).



7. CHECK SRS WARNING LIGHT

  1. Check the SRS warning light (Click here).
