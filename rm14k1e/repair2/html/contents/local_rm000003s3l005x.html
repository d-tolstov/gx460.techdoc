DTC    B2650    Slide Sensor Malfunction


DESCRIPTION

WIRING DIAGRAM

INSPECTION PROCEDURE

PERFORM ACTIVE TEST USING INTELLIGENT TESTER (POWER SEAT MOTOR FUNCTION)

CHECK FRONT POWER SEAT SWITCH LH (SLIDE MOTOR CIRCUIT)

CHECK SEPARATE TYPE FRONT SEAT CUSHION SPRING ASSEMBLY LH (SLIDE MOTOR)

CHECK HARNESS AND CONNECTOR (FRONT POWER SEAT SWITCH - SLIDE MOTOR)

INSPECT SEPARATE TYPE FRONT SEAT CUSHION SPRING ASSEMBLY LH (SLIDE MOTOR)

CHECK HARNESS AND CONNECTOR (FRONT POWER SEAT SWITCH - SLIDE MOTOR)
