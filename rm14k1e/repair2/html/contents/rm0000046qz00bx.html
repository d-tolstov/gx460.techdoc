NAVIGATION SYSTEM (for DVD) > Microphone Circuit between Microphone and Navigation ECU

for Preparation Click here


DESCRIPTION

This circuit sends a microphone signal from the telephone microphone assembly to the display and navigation module display.
It also supplies power from the display and navigation module display to the telephone microphone assembly.
