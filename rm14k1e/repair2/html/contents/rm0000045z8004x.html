VEHICLE STABILITY CONTROL SYSTEM > Multi-terrain Select Indicator Light does not Come ON

for Preparation Click here


DESCRIPTION

When the multi-function switch of the steering pad switch is operated, multi-terrain select control is enabled and the multi-terrain select indicator light illuminates.
After selecting a road surface type displayed on the multi-information display using the multi-function switch, the system optimizes throttle control and brake fluid pressure control for wheels that slip when accelerating, which allows the power lost when wheels slip during acceleration to be distributed to all 4 wheels.
The slip indicator light blinks while the system is operating.
The system can operate under the following conditions:
  1. The vehicle stability control system is normal.
  2. The vehicle speed is 12 km/h (7 mph) or less.
  3. Crawl control is not operating.
  4. The selected road surface is "Mud & Sand" and the transfer is set to H4 or L4, or the selected road surface is "Loose Rock", "Mogul" or "Rock" and the transfer is set to L4.
When there is a malfunction in the multi-terrain select system, the multi-terrain select indicator light blinks.
