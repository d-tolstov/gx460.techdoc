LIGHTING SYSTEM > Headlight Dimmer Switch Circuit

for Preparation Click here


DESCRIPTION

The main body ECU receives the following signals:
  1. Headlight dimmer switch tail, head, AUTO, high or high flash signal
  2. Front fog light switch signal
  3. Rear fog light switch signal
