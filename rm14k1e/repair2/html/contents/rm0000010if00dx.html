FRONT AIRBAG SENSOR > INSTALLATION

for Preparation Click here


HINT:
  1. Use the same procedure for the RH and LH sides.
  2. The procedure listed below is for the LH side.



1. INSTALL FRONT AIRBAG SENSOR LH

  1. Install the sensor with the 2 bolts.

    Torque:
    9.0 N*m{ 92 kgf*cm , 80 in.*lbf }

    NOTICE:
    1. If the airbag sensor has been dropped, or there are any cracks, dents or other defects in the case or connector, replace it with a new one.
    2. When installing the airbag sensor, make sure that the SRS wiring does not interfere with other parts and is not pinched between other parts.

  1. Check that the front airbag sensor is not loose.

  1. Connect the sensor connector.



2. INSTALL FRONT BUMPER COVER

  1. Install the front bumper cover (Click here).



3. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).



4. CHECK SRS WARNING LIGHT

  1. Check the SRS warning light (Click here).
