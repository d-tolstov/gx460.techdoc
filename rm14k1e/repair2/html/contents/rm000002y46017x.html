POWER MIRROR CONTROL SYSTEM > Power Retractable Mirrors do not Operate with Power Retract Mirror Switch

for Preparation Click here


SYSTEM DESCRIPTION

The main body ECU receives the retract switch signal from the outer mirror switch assembly. The main body ECU then sends the mirror retract/return signal to each outer mirror control ECU via LIN communication. Based on the signal, each outer mirror control ECU retracts or returns the outer rear view mirror assembly.
