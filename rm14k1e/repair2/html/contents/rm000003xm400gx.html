HINT:
Be sure to check for DTCs before performing this procedure (Click here).

    1. Park the vehicle with the steering wheel centered.

      HINT:
      Target bars for side camera adjustment should be made with 2 pieces of 2.5 cm wide tape; one piece should be 100 cm (3.28 ft.) (C+D) and the other should be 70 cm (2.30 ft.) (E+F) long. Check the tape color on the display and navigation module display and choose a tape color which can be easily seen.



  2. Start the diagnostic mode.

    1. Navigation System for HDD: Click here
    2. Navigation System for DVD: Click here

    NOTICE:
    Mode setting must be carried out with the engine start. Apply the parking brake, depress the brake pedal, move the shift lever to P and make sure that the vehicle is not moving.





    1. Select "Function Check/Setting" on the Service Menu screen.




    2. Select "Camera Setting" on the Function Check/Setting screen.




    3. Select "SIDE CAMERA SETTING" on the MODE SETTING screen.




    4. Select "NEXT" on the SIGNAL CHECK screen.

      NOTICE:
      1. When "CHK" (red) is displayed for any items on the SIGNAL CHECK screen, selecting "NEXT" will not change the screen to the SIDE CAMERA POSITION SETTING screen.
      2. When "CHK" (red) is displayed for any items on the SIGNAL CHECK screen, perform inspections using the SIGNAL CHECK screen (Click here).

      HINT:
      1. When the outer mirrors are retracted, selecting "NEXT" will not change the screen to the SIDE CAMERA POSITION SETTING screen.
      2. If the screen does not change to the SIDE CAMERA POSITION SETTING screen even though the outer mirrors are extended, perform the MIRROR SW check on the SIGNAL CHECK screen (Click here).
      3. If the screen does not change to the SIDE CAMERA POSITION SETTING screen even after "OK" (blue) is displayed as a result of the MIRROR SW check on the SIGNAL CHECK screen, replace the parking assist ECU (Click here).




  3. SIDE CAMERA POSITION SETTING

    HINT:
    The illustration shows the colors used on the side camera position setting screen.

    Text in Illustration 
    *a
    		Target Adjustment Bar for Side Camera Position Setting
    *b
    		Red Frame
    *c
    		Yellow Frame




    1. Perform the roll angle adjustment.
      Use switches A and B to rotate C so that it is parallel to the target adjustment bar.

    2. Perform the vertical and horizontal position adjustment.
      Use the directional switches D, E, F and G to move C so that the target adjustment bar is centered in C.

    3. Select "NEXT" to display SIDE VERIFY MODE.




  4. SIDE VERIFY MODE for front passenger side

    1. Check that the red cross and the target adjustment bar are aligned.

      HINT:
      If they are not aligned, select "BACK" and perform SIDE CAMERA POSITION SETTING again.

    2. Select "NEXT" to store the side camera aiming adjustment value and change the screen to the STEERING ANGLE SETTING screen.
      Check that the screen for the driver side camera displays correctly.

      HINT:
      1. When "NEXT" is selected, a beep will sound to confirm that the side camera aiming adjustment values have been stored (w/o Multi-terrain Monitor).
      2. The adjustment value will not be stored unless "NEXT" is selected.




  5. SIDE VERIFY MODE for driver side (w/ Multi-terrain Monitor)

    1. Check the screen for the driver side camera according to the prompts on the screen.

      HINT:
      1. When "NEXT" is selected, a beep will sound to confirm that the side camera aiming adjustment values have been stored.
      2. The adjustment value will not be stored unless "NEXT" is selected.

  6. Finish the diagnostic mode.
