AIRBAG SYSTEM > DIAGNOSTIC TROUBLE CODE CHART

for Preparation Click here


  1. If a trouble code is output during the DTC check, inspect the trouble areas listed for that code. For details of the code, refer to the "See page" below.
  2. When the SRS warning light remains on and the DTC output is the normal system code, a voltage source drop is likely to occur. This malfunction is not stored in memory by the center airbag sensor.
  3. If the power source voltage returns to normal, the SRS warning light automatically goes off.
  4. When 2 or more codes are output, the code with the lower number is output first.
  5. If a code not listed in the chart is output, the center airbag sensor may have a malfunction.
  6. If a DTC is output indicating a short to ground, short to B+ or open in an airbag squib circuit, other DTCs may not be output. First replace or repair the malfunctioning area indicated by the current DTC, and then check for DTCs again to output the other DTCs.
  7. Mark in the check mode column:
    *1: DTC is not output in check mode.
    *2: DTC is output in check mode.
