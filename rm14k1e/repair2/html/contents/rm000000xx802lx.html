WIRELESS DOOR LOCK CONTROL SYSTEM DESCRIPTION

  1. This system locks/unlocks the doors remotely. The wireless door lock control system has the following features:

    1. The certification ECU performs the code identification process and the main body ECU controls the door locks. A serial data link is provided for communication between the certification ECU and main body ECU.
    2. A key-integrated type transmitter is used and it contains the following 4 switches: the lock switch, unlock switch, panic switch* and glass hatch open switch.
      1. *: w/ Panic Alarm Control
    3. An LED is mounted on the transmitter to show if the battery is depleted.


FUNCTION OF MAIN COMPONENTS

Component
Function
Electrical key transmitter
  1. Contains the lock, unlock, panic*1 and glass hatch open switches.
  2. Transmits weak electric waves (recognition codes and function codes) to the door control receiver.
  3. Illuminates the indicator light (LED) during transmission.
Door control receiver
Receives weak electric waves (recognition codes and function codes) from the electrical key transmitter via the electrical key antenna and changes the waves to code data.
Front door courtesy light switch
Rear door courtesy light switch
Back door courtesy light switch
Turns on when the door is opened and off when the door is closed. Outputs the door status (open or closed) to the main body ECU.
Glass hatch courtesy switch
Turns on when the glass hatch is opened and off when the glass hatch is closed. Outputs the glass hatch status (open or closed) to the main body ECU.
Door lock position switch
Transmits the door lock positions of each door to the main body ECU.
Wireless door lock buzzer
Sounds when the door ajar warning function is operating.*2
Certification ECU
Matches the code data with the registered code and outputs the code data to the main body ECU.
  1. *1: w/ Panic Alarm Control
  2. *2: w/ Door Ajar Warning Function

SYSTEM FUNCTION

  1. The wireless door lock control system has the following functions:

    Function
    		Outline
    All door lock function
    		Pressing the lock switch locks all the doors.
    All door unlock (Unlock 2 operation)
    		Pressing the unlock switch twice within 3 seconds unlocks all the doors after the driver door is unlocked.
    Glass hatch open function
    		Pressing the glass hatch open switch for approximately 1 second or more opens the glass hatch.
    Chattering prevention function
    		If a switch is pressed, the corresponding switch operation is performed once. If a switch is held down, the corresponding switch operation occurs only once and does not repeatedly activate. When pressing a switch at 1 second intervals, the corresponding operation activates once for every press of the switch.
    Answer-back function
    1. When the doors are locked by wireless operation, the hazard warning lights flash once and the wireless door lock buzzer*1 sounds once.
    2. When the doors are unlocked by wireless operation, the hazard warning lights flash twice and the wireless door lock buzzer*1 sounds twice.
    Automatic lock function
    		If no doors are opened within 30 seconds after they are unlocked by the electrical key transmitter, all the doors lock again automatically.
    Illuminated entry function
    1. When the map light door switch is on and all the doors are locked, pressing the unlock switch causes the interior light illumination to turn on and all the doors to unlock simultaneously.
    2. If the doors have not been opened, the interior light turns off after approximately 15 seconds.
    Door ajar warning function*2
    		If any door is open or ajar, pressing the lock switch causes the wireless door lock buzzer to sound for approximately 10 seconds.
    Panic alarm function*3
    		Pressing the panic switch of the transmitter for more than approximately 0.8 seconds causes the system to perform the following operations:
    1. Sounds the horn and security horn.
    2. Flashes the headlights, taillights and hazard warning lights.
    3. Illuminates the interior lights (when the map light door switch is on).
    While the panic alarm is operating, pressing any switch of the transmitter cancels the alarm operation.
    Memory call function
    		If the unlock switch of a memory call registered transmitter is pressed and the driver door is opened, the following parts move to the memory call position: the front seat, steering wheel and outer mirrors.

    1. *1: w/ Wireless Buzzer Answer-back
    2. *2: w/ Door Ajar Warning Function
    3. *3: w/ Panic Alarm Control
