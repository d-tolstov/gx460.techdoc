CHECK DTC

  1. Connect the intelligent tester to the DLC3.

  2. Turn the engine switch on (IG).

  3. Turn the intelligent tester on.

  4. Enter the following menus: Chassis / Parking Assist Monitor System / DTC.

  5. Check the details of the DTCs (Click here).

    HINT:
    The parking assist ECU may store the DTCs of the following system. If DTCs other than those specified in Diagnostic Trouble Code Chart of the parking assist monitor system are output, refer to Diagnostic Trouble Code Chart for the applicable system.

CLEAR DTC

  1. Connect the intelligent tester to the DLC3.

  2. Turn the engine switch on (IG).

  3. Turn the intelligent tester on.

  4. Enter the following menus: Chassis / Parking Assist Monitor System / DTC.

  5. According to the display on the intelligent tester, select the trouble code data display with the clear button.
