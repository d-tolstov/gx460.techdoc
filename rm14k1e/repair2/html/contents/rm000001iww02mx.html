FRONT AXLE HUB > ON-VEHICLE INSPECTION

for Preparation Click here



1. REMOVE FRONT WHEEL


2. REMOVE FRONT DISC BRAKE CYLINDER ASSEMBLY LH



  1. Using a union nut wrench, disconnect the brake tube from the disc brake cylinder assembly.

    HINT:
    Use a container to catch brake fluid as it drains out.




  1. Remove the 2 bolts and disc brake cylinder assembly.



3. REMOVE FRONT DISC



  1. Place matchmarks on the disc and axle hub if planning to reuse the disc.
    Text in Illustration 
    *1
    		Matchmark

  1. Remove the front disc.



4. REMOVE FRONT AXLE HUB GREASE CAP

  1. Using a screwdriver and hammer, remove the front axle hub grease cap.

    NOTICE:
    Do not damage the axle hub.



5. INSPECT FRONT AXLE HUB BEARING LOOSENESS



  1. Using a dial indicator, measure the looseness near the center of the axle hub.

    Maximum looseness:
    0.05 mm (0.00197 in.)

    NOTICE:
    Make sure that the dial indicator is set at a right angle to the measurement surface.

    If the looseness is more than the maximum, replace the axle hub.



6. INSPECT FRONT AXLE HUB RUNOUT



  1. Using a dial indicator, measure the runout on the surface of the axle hub outside the hub bolts.

    Maximum runout:
    0.08 mm (0.00315 in.)

    NOTICE:
    Make sure that the dial indicator is set at a right angle to the measurement surface.

    If the runout is more than the maximum, replace the axle hub.



7. INSTALL FRONT AXLE HUB GREASE CAP

  1. Install the axle hub grease cap.

    NOTICE:
    Make sure the grease cap is securely installed to the axle hub.



8. INSTALL FRONT DISC

  1. Align the matchmarks and install the front disc.

    HINT:
    When replacing the disc with a new one, select the installation position where the disc has the smallest runout.



9. INSTALL DISC BRAKE CYLINDER ASSEMBLY LH

  1. Install the disc brake cylinder assembly with the 2 bolts.

    Torque:
    123 N*m{ 1254 kgf*cm , 91 ft.*lbf }

  1. Using a union nut wrench, connect the brake tube to the disc brake cylinder assembly.

    Torque:
    15 N*m{ 155 kgf*cm , 11 ft.*lbf }

    NOTICE:
    Use the formula to calculate special torque values for situations where a union nut wrench is combined with a torque wrench (Click here).



10. INSTALL FRONT WHEEL

Torque:
112 N*m{ 1137 kgf*cm , 82 ft.*lbf }
