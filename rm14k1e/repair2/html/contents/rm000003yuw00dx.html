DTC    B15EF    Helpnet Vehicle Installation Device ID Malfunction/Unmatched

for Preparation Click here


DESCRIPTION

This DTC will be stored when either of the following conditions is met:
  1. The mayday ID is not detected in the nonvolatile memory for 5 minutes during the mayday ID confirmations that are performed every minute after the engine switch is turned on (ACC).
  2. The mayday ID stored in the display and navigation module display and the mayday ID stored in the telematics transceiver are different.

DTC Code
DTC Detection Condition
Trouble Area
B15EF
The mayday ID is invalid.
-

INSPECTION PROCEDURE

1.CHECK DEVICE

  1. Check the part replacement history.

    1. Check if the display and navigation module display or the telematics transceiver has been replaced in the past.




  1. Check the mayday ID.

    1. Enter diagnostic mode (Click here).

    2. Select "Service Information" on the Service Menu screen.




    3. Select "DCM Information" on the Service Information screen.




    4. Check the mayday ID using the DCM Information screen.

  1. Contact the G-BOOK support center.

    1. Contact the G-BOOK support center and inform them about the following:

      1. Replacement history of the display and navigation module display
      2. Replacement history of the telematics transceiver
      3. Mayday ID

      HINT:
      The G-BOOK support center will perform an investigation and provide the results.

    2. Perform the vehicle contract setting.
