DTC    B15C0    Short in GPS Antenna

DTC    B15C1    Open in GPS Antenna

for Preparation Click here


DESCRIPTION

These DTCs are stored when a malfunction occurs in the GPS antenna.

DTC Code
DTC Detection Condition
Trouble Area
B15C0
A GPS antenna error.
  1. Navigation antenna assembly
  2. Display and navigation module display
B15C1
A error in the power source to the GPS antenna.

INSPECTION PROCEDURE

1.CHECK NAVIGATION ANTENNA ASSEMBLY

  1. Check if the navigation antenna is securely connected to the display and navigation module display.

    OK:
    Navigation antenna is securely connected.



NG
SECURELY CONNECT NAVIGATION ANTENNA ASSEMBLY
OK


2.REPLACE NAVIGATION ANTENNA ASSEMBLY

  1. Replace the navigation antenna assembly with a new or normally functioning one (Click here).

  1. Check if the same problem occurs again.

    1. Clear the DTCs (Click here).

    2. Recheck for DTCs and check if the same trouble occurs again.

      OK:
      No DTCs are output.



NG
REPLACE DISPLAY AND NAVIGATION MODULE DISPLAY (Click here)
OK

END (NAVIGATION ANTENNA IS DEFECTIVE)
