DTC    B2650    Slide Sensor Malfunction

for Preparation Click here


DESCRIPTION

When the front power seat switch does not receive a sensor signal despite forward or backward movement of the seat by power seat motor operation, this DTC is stored.

DTC Code
DTC Detection Condition
Trouble Area
B2650
The forward and backward lock detection position of the sensor is the same.
  1. Front power seat switch LH
  2. Separate type front seat cushion spring assembly LH (Slide motor)
  3. Harness or connector
