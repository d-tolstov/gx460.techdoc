POWER WINDOW CONTROL SYSTEM > Front Passenger Side Power Window does not Operate with Front Passenger Side Power Window Switch


DESCRIPTION

WIRING DIAGRAM

INSPECTION PROCEDURE

CHECK FOR DTC (B2312)

READ VALUE USING INTELLIGENT TESTER (POWER WINDOW REGULATOR SWITCH RH)

PERFORM ACTIVE TEST USING INTELLIGENT TESTER (POWER WINDOW)

INSPECT POWER WINDOW REGULATOR SWITCH ASSEMBLY RH

CHECK HARNESS AND CONNECTOR (POWER WINDOW REGULATOR SWITCH ASSEMBLY RH - BODY GROUND)

CHECK HARNESS AND CONNECTOR (POWER WINDOW REGULATOR SWITCH ASSEMBLY RH - FRONT POWER WINDOW REGULATOR MOTOR ASSEMBLY RH)

CHECK HARNESS AND CONNECTOR (FRONT POWER WINDOW REGULATOR MOTOR ASSEMBLY RH - BATTERY AND BODY GROUND)
