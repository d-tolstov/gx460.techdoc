BACK WINDOW LOCK > INSTALLATION

for Preparation Click here



1. INSTALL BACK WINDOW LOCK ASSEMBLY

  1. Install the back window lock assembly with the 3 bolts.

    Torque:
    8.0 N*m{ 82 kgf*cm , 71 in.*lbf }

  1. Connect the connector.



2. INSTALL BACK DOOR TRIM PANEL ASSEMBLY

  1. Attach the 15 clips to install the back door trim panel assembly.

  1. Install the bolt.

  1. Install the screw.

  1. Attach the claw to install the cover.



3. INSTALL BACK DOOR LOCK COVER

  1. Attach the 6 claws to install the back door lock cover.



4. INSTALL TOOL CASE

  1. Attach the 2 claws to install the tool case.



5. INSTALL UPPER TOOL BOX PANEL SUB-ASSEMBLY



  1. Attach the 3 claws.

  1. Install the upper tool box panel sub-assembly as shown in the illustration.



6. INSTALL BACK DOOR TRIM COVER



  1. Attach the 4 claws.

  1. Install the back door trim cover as shown in the illustration.



7. INSTALL BACK DOOR SIDE GARNISH LH

  1. Attach the 2 clips and claw to install the back door side garnish.



8. INSTALL BACK DOOR SIDE GARNISH RH

  1. Attach the 2 clips and claw to install the back door side garnish.



9. INSTALL BACK DOOR CENTER GARNISH

  1. Attach the 4 clips to install the back door center garnish.
