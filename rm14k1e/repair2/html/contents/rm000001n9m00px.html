SIGNAL CHECK (WHEN USING INTELLIGENT TESTER)

HINT:
  1. When entering test mode, the suspension control ECU first stores all the test mode DTCs. After completing test mode for each check item, the test mode DTCs that are determined to be normal by the suspension control ECU are cleared. The test mode DTCs for other check items may not be cleared when only a certain signal is inspected.
  2. When the system returns to normal mode, all the test mode DTCs are cleared.

  1. Turn the engine switch off.

  2. Connect the intelligent tester to the DLC3.

  3. Start the engine.

  4. Turn the intelligent tester on.

  5. Enter the following menus: Chassis / Air suspension / Utility / Signal Check.

  6. Check the display of the multi-information display and confirm that the system is in test mode.

  7. Check each item in the table below by performing the corresponding operation.

    Test Mode (Input Signal Check) Table 
    Check Item
    		Operation
    Absorber control switch signal
    		Slowly move absorber control switch from COMF to SPORT and then COMF
    Height control switch signal
    		First press "up" on height control switch and then press "down"
    Height control OFF switch signal
    		Turn height control OFF switch from on to off

  8. Read the test mode DTCs by following the prompts on the intelligent tester screen.

  9. Check the malfunction using the Test Mode DTC Chart below. After completing test mode, disconnect the intelligent tester and turn the engine switch off.


SIGNAL CHECK (WHEN USING SST CHECK WIRE)

HINT:
  1. When entering test mode, the suspension control ECU first stores all the test mode DTCs. After completing test mode for each check item, the test mode DTCs that are determined to be normal by the suspension control ECU are cleared. The test mode DTCs for other check items may not be cleared when only a certain signal is inspected.
  2. When the system returns to normal mode, all the test mode DTCs are cleared.

  1. Turn the engine switch off.




  2. Using SST, connect terminals 12 (TS) and 4 (CG) of the DLC3.

    SST
    09843-18040  
    Text in Illustration 
    *a
    		Front view of DLC3

    NOTICE:
    Do not connect the incorrect terminals as this will cause damage.

  3. Turn the engine switch on (IG).

  4. Check the display of the multi-information display and confirm that the system is in test mode.




  5. Using SST, connect terminals 13 (TC) and 4 (CG) of the DLC3.

    SST
    09843-18040  
    Text in Illustration 
    *a
    		Front view of DLC3

    NOTICE:
    1. Do not disconnect terminals 12 (TS) and 4 (CG).
    2. Do not connect the incorrect terminals as this will cause damage.

  6. Check each item in the table below by performing the corresponding operation.

    Test Mode (Input Signal Check) Table 
    Check Item
    		Operation
    Absorber control switch signal
    		Slowly move absorber control switch from COMF to SPORT and then COMF
    Height control switch signal
    		First press "up" on height control switch and then press "down"
    Height control OFF switch signal
    		Turn height control OFF switch from on to off

  7. Read the output test mode DTCs that are displayed on the multi-information display.

  8. Check the malfunction using the Test Mode DTC Chart below.
    After completing test mode, disconnect SST and the engine switch off.
