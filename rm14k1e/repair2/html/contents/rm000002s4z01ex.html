1. INSTALL OIL PUMP RELIEF VALVE



  1. Coat the oil pump relief valve with engine oil.

  1. Install the relief valve and relief valve spring to the oil pump cover hole.

  1. Install the relief valve plug.

    Torque:
    53 N*m{ 540 kgf*cm , 39 ft.*lbf }



2. INSTALL OIL PUMP COVER



  1. Install the driven rotor, drive rotor and oil pump cover with the 13 bolts.

    Torque:
    10 N*m{ 105 kgf*cm , 8 ft.*lbf }
