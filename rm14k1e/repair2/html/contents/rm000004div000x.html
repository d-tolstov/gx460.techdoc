1. CUSTOMIZE SLIDING ROOF SYSTEM

HINT:
The following items can be customized.

NOTICE:
  1. When the customer requests a change in a function, first make sure that the function can be customized.
  2. Record the current settings before customizing.

  1. Customizing with the intelligent tester

    1. Connect the intelligent tester to the DLC3.

    2. Turn the engine switch on (IG).

    3. Turn the intelligent tester on.

    4. Enter the following menus: Utility / Customize / Slide Roof.

    5. Select the setting by referring to the table below.

      Display
      		Default
      		Content
      		Setting
      		Relevant ECU
      Door Key Related Open*1
      		ON
      		The function to manually open the sliding roof linked with the power windows by holding the driver side door key for 2.0 seconds or more in the unlock position when the engine switch is off.
      		ON or OFF
      		Sliding roof drive gear sub-assembly (Sliding roof ECU)
      Door Key Related Close*1
      		ON
      		The function to manually close the sliding roof linked with the power windows by holding the driver side door key for 2.0 seconds or more in the lock position when the engine switch is off.
      		ON or OFF
      		Sliding roof drive gear sub-assembly (Sliding roof ECU)
      Wireless Key Related Open*2
      		ON
      		The function to manually open the sliding roof linked with the power windows by pressing the transmitter UNLOCK switch for 3.0 seconds or more when the engine switch is off.
      		ON or OFF
      		Sliding roof drive gear sub-assembly (Sliding roof ECU)
      Wireless Key Related Close*2
      		ON
      		The function to manually close the sliding roof linked with the power windows by pressing the transmitter LOCK switch for 3.0 seconds or more when the engine switch is off.
      		ON or OFF
      		Sliding roof drive gear sub-assembly (Sliding roof ECU)
      Door Key Related Operation
      		Slide
      		The function to select tilt up or slide open for the manual sliding roof operation linked with the power windows by holding the driver side door key for 2.0 seconds or more in the unlock position when the engine switch is off.
      		Slide or Tilt
      		Sliding roof drive gear sub-assembly (Sliding roof ECU)
      Wireless Key Related Operation
      		Slide
      		The function to select tilt up or slide open for the manual sliding roof operation linked with the power windows by pressing the transmitter UNLOCK switch for 3.0 seconds or more when the engine switch is off.
      		Slide or Tilt
      		Sliding roof drive gear sub-assembly (Sliding roof ECU)

      HINT:
      1. *1: This function is linked with the power window key-linked function. If the power window key-linked function is set to OFF, the sliding roof key-linked function will not operate even when it is set to ON.
      2. *2: This function is linked with the power window wireless transmitter-linked function. If the power window wireless transmitter-linked function is set to OFF, the sliding roof wireless transmitter-linked function will not operate even when it is set to ON.
