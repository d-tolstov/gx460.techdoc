1. INSTALL BACK DOOR HINGE ASSEMBLY LH

HINT:
Perform the following procedure if replacing the back door hinge assembly LH.

  1. Using a T30 "TORX" socket wrench, install the back door hinge with the 2 screws.

    Torque:
    8.0 N*m{ 82 kgf*cm , 71 in.*lbf }



2. INSTALL BACK DOOR HINGE ASSEMBLY RH

HINT:
Use the same procedure described for the LH side.



3. INSTALL BACK WINDOW GLASS SPACER


HINT:
Perform the following procedure if replacing any back window glass spacer.





Text in Illustration 
*1
No. 2 Back Window Glass Spacer
*2
No. 1 Back Window Glass Spacer

  1. Apply Primer G to the window where the spacer will be installed.

    Standard:
    Area
    		Specified Condition
    a
    		88.5 mm (3.48 in.)
    b
    		100.8 mm (3.97 in.)
    c
    		78.5 mm (3.09 in.)
    d
    		40.2 mm (1.58 in.)

    NOTICE:
    1. Allow the primer to dry for 3 minutes or more.
    2. Throw away any leftover primer.
    3. Do not apply too much primer.

    HINT:
    If the primer is applied to an area that is not specified, apply non-residue solvent to a clean cloth and wipe off the excess primer before it dries.

  1. Remove the peeling paper from a new spacer and install the spacer to the back door glass at the location shown in the illustration.



4. INSTALL BACK WINDOW LOCK STRIKER ASSEMBLY

  1. Install the lock striker with the 2 nuts.



5. INSTALL BACK DOOR LOCK STRIKER COVER

  1. Attach the 3 claws to install the lock striker cover.



6. INSTALL BACK DOOR GLASS

  1. Install the back door glass with the 4 bolts.

    Torque:
    20 N*m{ 204 kgf*cm , 15 ft.*lbf }

  1. Attach the 2 clamps and connect the connector.



7. INSTALL BACK WINDOW STAY ASSEMBLY LH

  1. Install the 2 stop rings to the back door stay assembly.




  1. Install the back window stay assembly.

    CAUTION:
    Install the back window stay while supporting the back door glass by hand.

  1. Check that the back window stay is attached to the ball joint and cannot be pulled off.



8. INSTALL BACK WINDOW STAY ASSEMBLY RH

HINT:
Use the same procedure described for the LH side.



9. INSTALL REAR WIPER MOTOR ASSEMBLY

  1. Attach the 2 guides and temporarily install the rear wiper motor assembly with the 3 bolts.

  1. Tighten the 3 bolts.

    Torque:
    5.5 N*m{ 56 kgf*cm , 49 in.*lbf }

  1. Connect the connector.



10. INSTALL REAR WIPER ARM



  1. Clean the wiper arm serrations.
    Text in Illustration 
    *1
    		Wiper Arm Pivot Serrations




  1. Clean the wiper pivot serrations with a wire brush.
    Text in Illustration 
    *1
    		Wiper Pivot Serrations
    *2
    		Wire Brush

  1. Operate the rear wiper and stop the rear wiper motor at the automatic stop position.

    HINT:
    The wiper motor does not operate if the back door and back door glass are not closed.




  1. Align the blade tip with the mark on the glass as shown in the illustration.

    Standard:
    Area
    		Standard Condition
    a
    		10.0 mm (0.394 in.)
    b
    		10.0 mm (0.394 in.)

    Text in Illustration 
    *1

    *a

    *b

  1. Install the rear wiper arm with the nut.

    Torque:
    5.5 N*m{ 56 kgf*cm , 49 in.*lbf }

    HINT:
    Hold the wiper arm by hand while tightening the nut.



11. INSTALL REAR SPOILER SUB-ASSEMBLY

  1. Attach the 3 clips to install the rear spoiler.

  1. Install the 6 bolts.

  1. Connect the connector.



12. INSTALL REAR SPOILER COVER RH

  1. Attach the 2 clips to install the rear spoiler cover.



13. INSTALL REAR NO. 1 SPOILER COVER

  1. Attach the 5 clips and fastening tape to install the rear No. 1 spoiler cover.
