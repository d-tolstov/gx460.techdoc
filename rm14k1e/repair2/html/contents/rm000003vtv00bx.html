HEIGHT CONTROL SWITCH > INSTALLATION

for Preparation Click here



1. INSTALL COMBINATION SWITCH ASSEMBLY (HEIGHT CONTROL SWITCH)

  1. Install the combination switch (height control switch) with the 4 screws.



2. INSTALL UPPER CONSOLE PANEL SUB-ASSEMBLY

  1. Connect each connector.

  1. Attach the 7 clips to install the upper console panel.



3. INSTALL SHIFT LEVER KNOB SUB-ASSEMBLY



  1. Install the shift lever knob and twist it in the direction indicated by the arrow.



4. INSTALL INSTRUMENT PANEL FINISH PANEL END LH

  1. Attach the 11 clips to install the instrument panel finish panel end.



5. INSTALL INSTRUMENT PANEL FINISH PANEL END RH

  1. Attach the 11 clips to install the instrument panel finish panel end.



6. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).



7. INSPECT SUSPENSION CONTROL SYSTEM

  1. Inspect the suspension control system (Click here).
