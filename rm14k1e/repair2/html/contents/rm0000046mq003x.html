1. INSTALL CONSOLE BOX RETAINER

  1. Install the console box retainer with 4 screws.



2. INSTALL COOLING BOX ASSEMBLY

  1. Connect the connector and install the console box assembly.

  1. Sufficiently apply compressor oil to 2 new O-rings.

    Compressor oil:
    ND-OIL 8 or equivalent

  1. Install the 2 O-rings to the 2 cooler pipes.

  1. Connect the 2 cooler pipes with the 2 bolts.

    Torque:
    5.4 N*m{ 55 kgf*cm , 48 in.*lbf }

  1. Connect the drain hose.

  1. Install the 8 bolts.



3. INSTALL CENTER FLOOR CARPET COVER RH

  1. Attach the 3 claws to install the center floor carpet cover RH.



4. INSTALL REAR CONSOLE END PANEL SUB-ASSEMBLY

  1. Connect the connector.

  1. Attach the 8 clips to install the rear console end panel.



5. INSTALL COOLING BOX TRAY

  1. Install the cooling box tray.



6. INSTALL REAR UPPER CONSOLE PANEL SUB-ASSEMBLY

  1. Attach the 4 clips and 2 guides to install the rear upper console panel.



7. INSTALL UPPER CONSOLE PANEL SUB-ASSEMBLY

  1. Connect each connector.

  1. Attach the 7 clips to install the upper console panel.



8. INSTALL SHIFT LEVER KNOB SUB-ASSEMBLY



  1. Install the shift lever knob and twist it in the direction indicated by the arrow.



9. INSTALL INSTRUMENT PANEL FINISH PANEL END LH

  1. Attach the 11 clips to install the instrument panel finish panel end.



10. INSTALL INSTRUMENT PANEL FINISH PANEL END RH

  1. Attach the 11 clips to install the instrument panel finish panel end.



11. INSTALL FRONT NO. 1 CONSOLE BOX INSERT

  1. Attach the 2 clips and guide to install the front No. 1 console box insert.



12. INSTALL FRONT NO. 2 CONSOLE BOX INSERT

  1. Attach the 2 clips and guide to install the front No. 2 console box insert.



13. INSTALL NO. 1 INSTRUMENT PANEL UNDER COVER SUB-ASSEMBLY

  1. Connect the connector and attach the clamp.

  1. Attach the 2 clips and 2 guides to install the No. 1 instrument panel under cover.

  1. Install the screw.



14. INSTALL NO. 2 INSTRUMENT PANEL UNDER COVER SUB-ASSEMBLY

  1. Attach the 2 clips and guide to install the front No. 2 console box insert.



15. INSTALL FRONT SEAT ASSEMBLY RH

  1. Install the front seat assembly RH (Click here).



16. CHARGE REFRIGERANT

    NOTICE:
    1. Do not operate the cooler compressor before charging refrigerant as the cooler compressor will not work properly without any refrigerant, and will overheat.
    2. Approximately 100 g (3.53 oz.) of refrigerant may need to be charged after bubbles disappear. The refrigerant amount should be checked by measuring its quantity, and not with the sight glass.



17. WARM UP ENGINE

  1. Warm up the engine at less than 1850 rpm for 2 minutes or more after charging the refrigerant.

    NOTICE:
    Be sure to warm up the compressor by turning the A/C switch is on after removing and installing the cooler refrigerant lines (including the compressor) to prevent damage to the compressor.



18. CHECK FOR REFRIGERANT GAS LEAK

  1. After recharging the refrigerant gas, check for refrigerant gas leakage using a halogen leak detector.

  1. Perform the operation under these conditions:

    1. Stop the engine.
    2. Secure good ventilation (the halogen leak detector may react to volatile gases other than refrigerant, such as evaporated gasoline or exhaust gas).
    3. Repeat the test 2 or 3 times.
    4. Make sure that some refrigerant remains in the refrigeration system. When the compressor is off: approximately 392 to 588 kPa (4.0 to 6.0 kgf/cm2, 57 to 85 psi).




  1. Using a halogen leak detector, check the refrigerant line for leakage.
    Text in Illustration 
    *1
    		Halogen Leak Detector
    *a
    		Check for Leakage

  1. If a gas leak is not detected on the drain hose, remove the blower motor control (blower resistor) from the cooling unit. Insert the halogen leak detector sensor into the unit and perform the test.

  1. Disconnect the connector and wait for approximately 20 minutes. Bring the halogen leak detector close to the pressure switch and perform the test.
