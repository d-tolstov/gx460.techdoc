PCV VALVE > INSTALLATION

for Preparation Click here



1. INSTALL PCV VALVE SUB-ASSEMBLY



  1. Apply a light coat of engine oil to the O-ring.
    Text in Illustration 
    *1
    		O-ring

    HINT:
    1. When reusing the PCV valve, inspect the O-ring.
    2. If the O-ring has scratches or cuts, replace the PCV valve.

  1. Using a 22 mm ball joint lock nut wrench, install the PCV valve sub-assembly.

    Torque:
    3.5 N*m{ 36 kgf*cm , 31 in.*lbf }

    NOTICE:
    Use the formula to calculate special torque values for situations where ball joint lock nut wrench is combined with a torque wrench (Click here).



2. CONNECT PCV VALVE HOSE



  1. Connect the PCV valve hose to the valve.

  1. Secure the hose with the clamp.
    Text in Illustration 
    *a
    		Upper Side
    *b
    		5 to 10 mm (0.197 to 0.394 in.)

    HINT:
    Make sure the direction of the hose clamp is as shown in the illustration.



3. INSTALL NO. 3 ENGINE COVER



  1. Install the No. 3 engine cover.
    Text in Illustration 

    		Engine Front

    HINT:
    Position the No. 3 engine cover so that the arrow mark faces the front of the engine and install it.
