DTC    B2312    Power Window Switch Malfunction

for Preparation Click here


DESCRIPTION

The power window regulator motor is driven by operating the power window regulator switch. The power window regulator consists of the motor, regulator and ECU.
This DTC is stored when the ECU built into the regulator motor determines that the multiplex network master switch assembly or power window regulator switch assembly is stuck.

HINT:
This DTC may be stored for all the windows.

NOTICE:
  1. When a power window regulator is reinstalled or replaced, the power window control system must be initialized.
  2. After any door glass or a door glass run has been replaced, the jam protection function may operate unexpectedly when the auto up function is used. In such cases, the auto up function can be resumed by repeating the following operation at least 5 times:
    1. Close the power window by fully pulling up the power window switch and holding it in the auto up position.
    2. Open the power window by fully pushing down the power window switch.

  1. The multiplex network master switch (door control switch and power window switch) is stuck.
  2. The multiplex network master switch (door control switch and power window switch) is held in the same position for more than 20 sec.
