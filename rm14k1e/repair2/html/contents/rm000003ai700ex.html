STEREO COMPONENT AMPLIFIER > INSTALLATION

for Preparation Click here



1. INSTALL NO. 2 AMPLIFIER BRACKET



  1. Set the aligning protrusions of the bracket, labeled A, into the holes of the amplifier.

  1. Install the bracket with the 2 screws labeled B.

    Torque:
    3.0 N*m{ 31 kgf*cm , 27 in.*lbf }



2. INSTALL NO. 1 AMPLIFIER BRACKET



  1. Set the aligning protrusions of the bracket, labeled A, into the holes of the amplifier.

  1. Install the bracket with the 2 screws labeled B.

    Torque:
    3.0 N*m{ 31 kgf*cm , 27 in.*lbf }



3. INSTALL NO. 2 AMPLIFIER COVER

  1. Install the No. 2 amplifier cover with the bolt and screw.

    Torque:
    for bolt:
    8.0 N*m{ 82 kgf*cm , 71 in.*lbf }



4. INSTALL STEREO COMPONENT AMPLIFIER ASSEMBLY WITH BRACKET



  1. Install the stereo component amplifier with the bolt and 2 screws.

    Torque:
    for bolt A:
    8.0 N*m{ 82 kgf*cm , 71 in.*lbf }

  1. Connect the 5 connectors.



5. INSTALL AMPLIFIER COVER

  1. Install the amplifier cover with the 3 clips.




  1. Return the floor carpet to its original position as shown in the illustration.



6. INSTALL FRONT SEAT ASSEMBLY RH

  1. Install the front seat assembly RH (Click here).



7. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).



8. CHECK SRS WARNING LIGHT

  1. Check the SRS warning light (Click here).
