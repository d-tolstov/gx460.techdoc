BACK DOOR OUTSIDE GARNISH > INSTALLATION

for Preparation Click here



1. INSTALL BACK DOOR OUTSIDE GARNISH SUB-ASSEMBLY

  1. Attach the 9 clips to install the back door outside garnish.

  1. Install the nut.

  1. Connect the connector.



2. INSTALL REAR TELEVISION CAMERA ASSEMBLY

  1. Install the rear television camera with the 2 bolts.

  1. Connect the connector.



3. INSTALL BACK DOOR OUTSIDE HANDLE LH

  1. Install the back door outside handle with the 2 nuts.

    Torque:
    5.0 N*m{ 51 kgf*cm , 44 in.*lbf }



4. INSTALL BACK DOOR SERVICE HOLE COVER



  1. Apply new butyl tape to the back door panel.

  1. Attach the rear door service hole cover using to the reference points on the rear door panel.
    Text in Illustration 
    *1
    		Reference Point

    NOTICE:
    1. There should be no wrinkles or folds after attaching the service hole cover.
    2. After attaching the service hole cover, check the seal quality.



5. INSTALL BACK DOOR TRIM PANEL ASSEMBLY

  1. Attach the 15 clips to install the back door trim panel assembly.

  1. Install the bolt.

  1. Install the screw.

  1. Attach the claw to install the cover.



6. INSTALL BACK DOOR LOCK COVER

  1. Attach the 6 claws to install the back door lock cover.



7. INSTALL TOOL CASE

  1. Attach the 2 claws to install the tool case.



8. INSTALL UPPER TOOL BOX PANEL SUB-ASSEMBLY



  1. Attach the 3 claws.

  1. Install the upper tool box panel sub-assembly as shown in the illustration.



9. INSTALL BACK DOOR TRIM COVER



  1. Attach the 4 claws.

  1. Install the back door trim cover as shown in the illustration.



10. INSTALL BACK DOOR SIDE GARNISH LH

  1. Attach the 2 clips and claw to install the back door side garnish.



11. INSTALL BACK DOOR SIDE GARNISH RH

  1. Attach the 2 clips and claw to install the back door side garnish.



12. INSTALL BACK DOOR CENTER GARNISH

  1. Attach the 4 clips to install the back door center garnish.



13. ADJUST REAR TELEVISION CAMERA ASSEMBLY (w/ Side Monitor System)

  1. Adjust the rear television camera (Click here).
