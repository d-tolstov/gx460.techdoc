ENTRY AND START SYSTEM (for Start Function) > Power Source Mode does not Change to ON (IG and ACC)

for Preparation Click here


DESCRIPTION

When the key is in the vehicle and the engine switch is pressed, the power management control ECU receives a signal and changes the power source mode.

HINT:
  1. When the power management control ECU is replaced with a new one and the cable is connected to the negative (-) battery terminal, the power source mode changes to on (IG).
  2. When the battery cable is disconnected and reconnected, the power source mode returns to the mode it was in before the battery cable was disconnected.

INSPECTION PROCEDURE

NOTICE:
  1. When using the intelligent tester with the engine switch off, perform either of the following: 1) Turn a courtesy light switch on and off at intervals of 1.5 seconds or less until communication between the intelligent tester and vehicle begins, or 2) connect the intelligent tester to the vehicle and select "MANUAL" from the initial screen on the intelligent tester, and then select "KEY REGIST" under Model Code.
  2. Make sure that no DTCs are output. If any DTCs are output, proceed to the Diagnostic Trouble Code Chart (Click here).
  3. The entry and start system uses multiplex communication. First perform the inspections in "How to Proceed with Troubleshooting" to confirm that there are no communication malfunctions before proceeding with troubleshooting (Click here).
  4. If the entry and start system is disabled through the customize function, enable the system before performing troubleshooting (Click here).
  5. Inspect the fuses of circuits related to this system before performing the following inspection procedure.
  6. After completing repairs, confirm that the problem does not occur.

3.INSPECT ENTRY AND START SYSTEM



  1. Remove the battery of the electrical key transmitter.

  1. With the brake pedal depressed, touch the key to the engine switch.

  1. Press and hold the engine switch for a maximum of 15 seconds and check if the engine can be started.

    OK:
    The engine can be started.

    HINT:
    1. If the engine can be started, it is clear that the key is not being detected properly inside the vehicle. There may be a problem with the formation of the interior detection area, a problem with the indoor electrical key antenna (reception problem), or wave interference.
    2. If the engine cannot be started, there is a problem with the engine switch, power management control ECU or a related wire harness.
