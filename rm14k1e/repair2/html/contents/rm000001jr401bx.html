REAR AIR CONDITIONING UNIT > REASSEMBLY

for Preparation Click here



1. INSTALL AIR CONDITIONING HARNESS ASSEMBLY


2. INSTALL REAR EVAPORATOR TEMPERATURE SENSOR



  1. Attach the 2 claws to install the sensor.

  1. Move the tip of the sensor in the direction indicated by the arrow in the illustration to install it to the bracket plate.



3. INSTALL REAR EVAPORATOR SUB-ASSEMBLY

  1. Sufficiently apply compressor oil to 2 new O-rings and the fitting surface of the hose joint.

    Compressor oil:
    ND-OIL 8 or equivalent

  1. Install the 2 O-rings to the evaporator.

  1. Install the evaporator to the rear cooling unit case RH.




  1. Install the tube connector.

  1. Using a 4 mm hexagon wrench, install the 2 hexagon bolts.

    Torque:
    3.5 N*m{ 36 kgf*cm , 31 in.*lbf }

  1. Install the packing.

  1. Install the rear cooling unit case LH with the 12 screws.

  1. Attach the 3 clamps.

  1. Attach the 2 clamps and install the air conditioning wiring harness to the rear cooling unit case.



4. INSTALL BLOWER MOTOR CONTROLLER

  1. Install the blower motor controller with the 2 screws.



5. INSTALL REAR BLOWER WITH FAN MOTOR SUB-ASSEMBLY

  1. Install the rear blower with fan motor with the 3 screws.



6. INSTALL HEATER BRACKET

  1. Install the bracket with the 3 screws.



7. INSTALL HEATER RADIATOR UNIT SUB-ASSEMBLY

  1. Attach the 2 claws to install the heater clamp.




  1. Install the radiator unit to the rear cooling unit.

  1. Attach the claw to close the heater clamp.

  1. Install the screw.



8. INSTALL REAR COOLING UNIT EXPANSION VALVE

  1. Sufficiently apply compressor oil to 2 new O-rings and the fitting surface of the hose joint.

    Compressor oil:
    ND-OIL 8 or equivalent

  1. Install the 2 O-rings to the air conditioning tube and accessory assembly.




  1. Install the rear cooling unit expansion valve.

  1. Sufficiently apply compressor oil to 2 new O-rings and the fitting surface of the hose joint.

    Compressor oil:
    ND-OIL 8 or equivalent

  1. Install the 2 O-rings to the tube.

  1. Install the tube with the 2 bolts.

    Torque:
    3.4 N*m{ 35 kgf*cm , 30 in.*lbf }

  1. Install the bracket.

  1. Install the 3 screws.

  1. Attach the clamp and install the connector.

  1. Install new packing.



9. INSTALL AIR CONDITIONING TUBE AND ACCESSORY ASSEMBLY



  1. Sufficiently apply compressor oil to 2 new O-rings and the fitting surface of the hose joint.

    Compressor oil:
    ND-OIL 8 or equivalent

  1. Install the 2 O-rings to the air conditioning tube and accessory assembly.

  1. Install the air conditioning tube and accessory assembly.

  1. Align the white marks on the hoses with the black marks on the pipes and install the air conditioning tube and accessory assembly to the rear cooling unit.

  1. Using pliers, grip the claws of the clips and slide the 2 clips.

    NOTICE:
    When installing the clips, align the protruding parts of the clips with the white marks of the hoses.

  1. Attach the clamp.

  1. Install the bolt and 2 screws.

    Torque:
    5.4 N*m{ 55 kgf*cm , 48 in.*lbf }

  1. Attach the 3 claws to install the cover.

  1. Align the cutout of the hose with the protrusion of the unit and connect the drain hose.

  1. Install new packing.



10. INSTALL REAR MODE DAMPER SERVO SUB-ASSEMBLY



  1. Align the damper servo as shown in the illustration and install it with the 2 screws.

  1. Connect the connector.
    Text in Illustration 
    *a
    		Alignment Point



11. INSTALL COOLER BRACKET

  1. Install the bracket with the 4 screws.



12. INSTALL REAR AIR MIX DAMPER SERVO SUB-ASSEMBLY



  1. Position the air mix damper servo so that the large tooth of the gear engages with the cutout part as shown in the illustration and install the air mix damper servo with the 2 screws.

  1. Connect the connector.
    Text in Illustration 
    *a
    		Alignment Point



13. INSTALL REAR SIDE NO. 4 AIR DUCT (w/ Rear Air Duct)

  1. Install the duct with the 3 screws.
