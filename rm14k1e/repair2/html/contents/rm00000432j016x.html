DTC    B161A/8A    Lost Communication with Front Satellite Sensor Bus

for Preparation Click here


DESCRIPTION

The front collision sensor circuit (front airbag sensor RH circuit and front airbag sensor LH circuit) is composed of the center airbag sensor assembly, front airbag sensor RH and front airbag sensor LH.
The front airbag sensor RH or front airbag sensor LH detects impacts to the vehicle and sends signals to the center airbag sensor assembly to determine if the airbag should be deployed.
DTC B161A/8A is stored when a malfunction is detected in the front collision sensor circuit (front airbag sensor RH circuit and front airbag sensor LH circuit).

DTC Code
DTC Detection Condition
Trouble Area
B161A/8A
One of the following conditions is met:
  1. The center airbag sensor assembly detects a line short circuit signal, open circuit signal, short circuit to ground signal or short circuit to B+ signal in the front collision sensor circuit (front airbag sensor RH circuit and front airbag sensor LH circuit).
  2. A front airbag sensor RH malfunction.
  3. A front airbag sensor LH malfunction.
  4. A center airbag sensor assembly malfunction.
  1. Instrument panel wire
  2. Engine room main wire
  3. Front airbag sensor RH
  4. Front airbag sensor LH
  5. Center airbag sensor assembly
