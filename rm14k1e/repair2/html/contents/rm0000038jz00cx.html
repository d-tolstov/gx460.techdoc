FRONT BUMPER > REASSEMBLY

for Preparation Click here



1. INSTALL LOWER RADIATOR GRILLE LH

  1. Attach the 8 claws to install the lower radiator grille.



2. INSTALL LOWER RADIATOR GRILLE RH

HINT:
Use the same procedure described for the LH side.



3. INSTALL FRONT BUMPER HOLE COVER ASSEMBLY LH

  1. Attach the 2 guides and 3 claws to install the front bumper hole cover.



4. INSTALL FRONT BUMPER HOLE COVER ASSEMBLY RH

HINT:
Use the same procedure described for the LH side.



5. INSTALL FOG LIGHT ASSEMBLY LH

  1. Connect the connector.

  1. Install the fog light with the screw.



6. INSTALL FOG LIGHT ASSEMBLY RH

HINT:
Use the same procedure described for the LH side.



7. INSTALL NO. 1 ULTRASONIC SENSOR RETAINER (w/ LEXUS Parking Assist-sensor System)



HINT:
Use the same procedure to install the No. 1 ultrasonic sensor retainer on the other side.

  1. Align the keyhole and protrusion as shown in the illustration.

  1. Install the ultrasonic sensor retainer to the front bumper cover.

    NOTICE:
    Do not damage the bumper cover with the protrusion when installing the retainer.

    Text in Illustration 
    *1
    		Keyhole



8. INSTALL NO. 1 ULTRASONIC SENSOR (w/ LEXUS Parking Assist-sensor System)

HINT:
Use the same procedure to install the No. 1 ultrasonic sensor on the other side.

  1. Attach the 2 claws to install the No. 1 ultrasonic sensor.

  1. Connect the connector.



9. INSTALL NO. 2 ENGINE ROOM WIRE

  1. w/ LEXUS Parking Assist-sensor System

    1. Attach the 6 clamps to install the No. 2 engine room wire.

    2. Connect the 6 connectors.

  1. w/o LEXUS Parking Assist-sensor System

    1. Attach the 6 clamps to install the No. 2 engine room wire.

    2. Connect the 2 connectors.



10. INSTALL HEADLIGHT WASHER ACTUATOR SUB-ASSEMBLY LH

  1. Attach the 2 claws to install the headlight washer actuator sub-assembly.

  1. Install the bolt.



11. INSTALL HEADLIGHT WASHER ACTUATOR SUB-ASSEMBLY RH

HINT:
Use the same procedure described for the LH side.



12. INSTALL HEADLIGHT CLEANER HOSE

  1. Attach the 6 clamps to install the hose to the lower radiator grille.

  1. Connect the headlight cleaner hose to the headlight washer actuator sub-assembly.



13. INSTALL TYPE 1 HEADLIGHT WASHER NOZZLE SUB-ASSEMBLY LH

  1. Attach the 2 claws to install the headlight washer nozzle.



14. INSTALL TYPE 1 HEADLIGHT WASHER NOZZLE SUB-ASSEMBLY RH

HINT:
Use the same procedure described for the LH side.



15. INSTALL FRONT SPOILER LH

  1. Install the front bumper spoiler with the 2 lower front bumper retainers.



16. INSTALL FRONT SPOILER RH

HINT:
Use the same procedure described for the LH side.



17. INSTALL FRONT BUMPER EXTENSION MOUNTING BRACKET

  1. Install the front bumper extension moulding bracket with the 2 screws.
