FRONT SEAT OUTER BELT ASSEMBLY > INSTALLATION

for Preparation Click here


HINT:
  1. Use the same procedure for the RH and LH sides.
  2. The procedure listed below is for the LH side.
  3. A bolt without a torque specification is shown in the standard bolt chart (Click here).



1. INSTALL FRONT SHOULDER BELT ANCHOR ADJUSTER ASSEMBLY



  1. Align the claw with the anchor plate positioning hole, and install the anchor adjuster with the 2 bolts.

    Torque:
    42 N*m{ 428 kgf*cm , 31 ft.*lbf }

    HINT:
    First install the lower bolt, and then install the upper bolt.



2. INSTALL FRONT SEAT OUTER BELT ASSEMBLY LH



NOTICE:
When installing the retractor, make sure the claws of the vehicle (labeled "a" in the illustration) only contact the installation areas of the retractor.

  1. Align the seat belt retractor positioning holes with the claws of the vehicle.

  1. Install the seat belt with the bolt.

    Torque:
    8.5 N*m{ 87 kgf*cm , 75 in.*lbf }




  1. Connect the pretensioner connector labeled A as shown in the illustration.

  1. Connect the tension reducer connector labeled B.

  1. w/ Pre-crash Safety System:
    Connect the pre-crash safety system connector labeled C.

  1. Install the shoulder anchor to the shoulder belt anchor adjuster with the nut.

    Torque:
    42 N*m{ 428 kgf*cm , 31 ft.*lbf }



3. INSTALL CENTER PILLAR GARNISH LH

  1. Pass the front seat outer belt floor anchor through the center pillar garnish.

  1. Attach the 2 clips and 2 guides to install the center pillar garnish.

  1. Install the bolt.



4. INSTALL NO. 2 ASSIST GRIP

HINT:
Use the same procedure for the other No. 2 assist grip.

  1. Attach the 2 claws to install the No. 2 assist grip.

  1. Install the 2 bolts.



5. INSTALL NO. 2 ASSIST GRIP PLUG LH

HINT:
Use the same procedure for the other front No. 2 assist grip plug.

  1. Attach the 2 claws to install the front No. 2 assist grip plug.



6. INSTALL LOWER CENTER PILLAR GARNISH LH

  1. Attach the 4 claws and 2 clips to install the lower center pillar garnish.

  1. Install the front seat outer belt floor anchor with the bolt.

    Torque:
    42 N*m{ 428 kgf*cm , 31 ft.*lbf }



7. INSTALL OUTER LAP BELT ANCHOR COVER

HINT:
Use the same procedure for the other outer lap belt anchor cover.

  1. Attach the 3 claws to install the outer lap belt anchor cover



8. INSTALL REAR DOOR OPENING TRIM WEATHERSTRIP LH



  1. Align the paint mark on the rear door opening trim weatherstrip with the mark position on the vehicle and install the rear door opening trim weatherstrip as shown in the illustration.
    Text in Illustration 
    *1
    		Paint Mark
    *2
    		Mark Position



9. INSTALL REAR DOOR SCUFF PLATE LH

  1. Attach the 3 clips, 6 claws and 2 guides to install the rear door scuff plate.



10. INSTALL QUARTER SCUFF PLATE LH (w/ Rear No. 2 Seat)

  1. Install the quarter scuff plate with the 2 bolts.



11. INSTALL REAR NO. 1 FLOOR STEP COVER (w/ Rear No. 2 Seat)

HINT:
Use the same procedure for all rear No. 1 floor step covers.

  1. Attach the 2 claws to install the rear No. 1 floor step cover.



12. INSTALL FRONT DOOR OPENING TRIM WEATHERSTRIP LH



  1. Align the paint mark on the front door opening trim weatherstrip with the mark position on the vehicle and install the front door opening trim weatherstrip as shown in the illustration.
    Text in Illustration 
    *1
    		Paint Mark
    *2
    		Mark Position



13. INSTALL DOOR SCUFF PLATE ASSEMBLY LH

  1. Connect the connector.

  1. Attach the 4 clips, 10 claws and 2 guides to install the door scuff plate.



14. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).
