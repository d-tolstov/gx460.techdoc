DTC    C1210    Zero Point Calibration of Yaw Rate Sensor Undone


DESCRIPTION

WIRING DIAGRAM

INSPECTION PROCEDURE

CHECK DTC (CAN COMMUNICATION SYSTEM)

PERFORM ZERO POINT CALIBRATION OF YAW RATE AND ACCELERATION SENSOR

PERFORM TEST MODE (SIGNAL CHECK)

CHECK YAW RATE AND ACCELERATION SENSOR INSTALLATION

READ VALUE USING INTELLIGENT TESTER (YAW RATE AND ACCELERATION SENSOR)
