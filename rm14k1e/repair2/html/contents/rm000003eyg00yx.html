PARKING BRAKE SWITCH > INSTALLATION

for Preparation Click here



1. INSTALL PARKING BRAKE SWITCH ASSEMBLY

  1. Install the parking brake switch assembly to the parking brake control pedal assembly with the screw.

    Torque:
    3.2 N*m{ 33 kgf*cm , 28 in.*lbf }



2. INSTALL PARKING BRAKE CONTROL PEDAL ASSEMBLY

  1. Temporarily install the parking brake control pedal assembly with the 3 nuts.

  1. Tighten the 3 nuts.

    Torque:
    13 N*m{ 127 kgf*cm , 9 ft.*lbf }

  1. Connect the parking brake switch connector.

  1. Install the No. 1 parking brake cable assembly to the body with the 2 nuts.

    Torque:
    6.0 N*m{ 61 kgf*cm , 53 in.*lbf }



3. INSTALL TURN SIGNAL FLASHER ASSEMBLY

  1. Install the turn signal flasher with the nut.

  1. Connect the connector.



4. INSTALL LOWER NO. 1 INSTRUMENT PANEL AIRBAG ASSEMBLY

  1. Install the lower No. 1 instrument panel airbag assembly (Click here).



5. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).
