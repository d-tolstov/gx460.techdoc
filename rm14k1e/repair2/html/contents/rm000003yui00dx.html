G-BOOK SYSTEM OUTLINE

  1. The G-BOOK system is a telematics service that links the vehicle and G-BOOK network (provides information from the G-BOOK center or content providers).

  2. To use the G-BOOK system, it is necessary to connect the telematics transceiver (data communication module) to the display and navigation module display.



  3. The user needs to follow the instructions on the display and navigation module display to start using the G-BOOK system service after applying for the service at a dealership.




FUNCTION OF MAIN COMPONENTS

Component
Function
Emergency call switch
When the emergency call switch is pressed, a switch signal is sent to the telematics transceiver (when a manual maintenance check or manual emergency call is performed).
LED Indicator Light
Green
  1. Turns on for 5 seconds after the engine switch is turned on (ACC).
  2. Turns on when the system is under a service contract and operating normally.
  3. Blinks during a call.
  4. Blinks during a manual maintenance check.
  5. Turns off when the system is malfunctioning.
Red
  1. Turns on for 5 seconds after the engine switch is turned on (ACC).
  2. Turns on when the telematics transceiver is not within the service area.
  3. Blinks when a call is made while the system is still malfunctioning.
  4. Turns off when the system is operating normally.
  5. Blinks when the mayday battery needs to be replaced.
  6. Blinks when in diagnostic mode to indicate a DTC.
Telephone microphone assembly
  1. When the operator service is used: Sends the microphone voice signal to the display and navigation module display.
  2. When an emergency call is made: Sends the microphone voice signal to the telematics transceiver.
Telephone antenna assembly
Sends and receives the data and voice signals used for the G-BOOK service through a communication network.
Telematics transceiver
  1. Uses the telephone antenna assembly to send and receive the data and voice signals used for the G-BOOK service through a communication network.
    1. When the operator service is used: Sends a received voice signal to the display and navigation module display.
    2. When the operator service is used: Sends a sent voice signal from the display and navigation module display to the telephone antenna assembly.
    3. When an emergency call is made: Sends a received voice signal to the vehicle speakers.
    4. When an emergency call is made: Receives a sent voice signal from the telephone microphone assembly.
    5. When an emergency call is made: Sends a mute signal to the stereo component amplifier assembly (mute function becomes active during the call).
    6. When an emergency call is made: Sends the location information to the G-BOOK center.
Mayday battery (Back-up battery)
  1. The mayday battery is a lithium manganese dioxide non-rechargeable battery.
  2. Checks the battery condition and sends the self-diagnosis result to the display and navigation module display via the telematics transceiver.
  3. The mayday battery conducts a self-check every time the engine switch is turned on (IG), and after an automatic emergency call or manual emergency call is made, and the result is sent to the DCM.
GPS antenna
Receives GPS radio waves and sends them to the display and navigation module display.
Main body ECU (Multiplex network body ECU)
Sends a security horn sounding signal to the display and navigation module display when the theft deterrent system is activated.
Display and navigation module display
  1. Sends and receives the data used for the G-BOOK service to and from the telematics transceiver using a telematics data signal.
    1. When the operator service is used: Sends the sent voice signal from the telephone microphone assembly to the telematics transceiver.
    2. When the operator service is used: Sends the received voice signal from the telematics transceiver to the stereo component amplifier assembly using a MOST signal.
    3. Sends the vehicle location information to the telematics transceiver.
    4. When a warning comes on, the display and navigation module display receives a warning ON signal from the No. 1 meter ECU sub-assembly via CAN communication and the telematics transceiver sends it to the G-BOOK center.
    5. When the theft deterrent system is activated, the telematics transceiver sends the information that the security horn has sounded to the G-BOOK center.
Stereo component amplifier assembly
Receives the received voice signal sent from the display and navigation module display as a MOST signal and outputs it as sound from the vehicle speakers (when the operator service is used).
Center airbag sensor assembly
Sends an activation signal to the telematics transceiver when airbags deploy (an automatic emergency call is made).

DEVICE USED FOR G-BOOK SYSTEM COMMUNICATION

  1. The telematics transceiver is used for data communication or emergency call service telephone calls and operator service telephone calls.

  2. The handsfree function cannot use the telematics transceiver. A user's "Bluetooth" compatible cellular phone needs to be used.

  3. Perform the following if the device used for G-BOOK system communication is replaced.

    HINT:
    1. If the display and navigation module display or the telematics transceiver is replaced on vehicles that do not have a contract for the G-BOOK service, perform the vehicle contract setting.
    2. If the display and navigation module display or the telematics transceiver is replaced on vehicles that have a contract for the G-BOOK service, perform the vehicle contract setting and the procedure to resume the G-BOOK service and to start the emergency call service.


G-BOOK ID OUTLINE

  1. The display and navigation module display functions as the G-BOOK device.

  2. The display and navigation module display has a serial number (G-BOOK ID) which can be used to determine the individual device when servicing.


MAYDAY ID OUTLINE

  1. The telematics transceiver functions as the emergency call service device to perform emergency calls.

  2. The telematics transceiver has a serial number (mayday ID) which can be used to identify the individual device when servicing.


DATA COMMUNICATION MODULE (DCM) OUTLINE

  1. The G-BOOK device uses the DCM (telematics transceiver) to provide access to the network service.

  2. The DCM has a DCM ID (printed on the label attached to the DCM).

  3. As is the case with cellular phones, the DCM has a phone number.

WARNING NOTIFICATION FUNCTION

NOTICE:
  1. Before performing inspections or repairs which may cause warning messages to occur or DTCs to be stored, such as a simulation test or road test, it is necessary to activate warning notification restraint mode so that the G-BOOK center does not mistake the warnings for indications of real problems.
  2. Warning notification restraint mode can be entered by using the intelligent tester or operating the display and navigation module display.

HINT:
The warning notification function sends a warning message signal, received from the combination meter assembly via CAN communication, to the G-BOOK center using the G-BOOK device if a warning is displayed due to a vehicle malfunction.

  1. Warning notification restraint mode

    HINT:
    1. Performing the following procedure will cause the vehicle to enter warning notification restraint mode.
    2. After warning notification restraint mode is entered, driving the vehicle 10 km (6 miles) will cancel the mode.


    1. Enter warning notification restraint mode (using the intelligent tester).

      1. Turn the engine switch off.
      2. Connect the intelligent tester to the DLC3.
      3. Start the engine.
      4. Enter the following menus: Powertrain / Engine / DTC.

    2. Enter warning notification restraint mode (using the display and navigation module display).

      1. Enter diagnostic mode of the navigation system (for HDD) (Click here).
