DTC    C1201    Engine Control System Malfunction

for Preparation Click here


DESCRIPTION

If a malfunction is detected in the engine control system, the ECM sends a malfunction signal to the skid control ECU. The skid control ECU stores this DTC and prohibits the following controls:
  1. VSC
  2. TRC
  3. Downhill assist control
  4. Crawl control
  5. Multi-terrain select control
