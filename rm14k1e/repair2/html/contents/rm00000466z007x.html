FRONT SEAT ASSEMBLY > INSTALLATION

for Preparation Click here


CAUTION:
Wear protective gloves. Sharp areas on the parts may injure your hands.

HINT:
  1. Use the same procedure for the RH and LH sides.
  2. The procedure listed below is for the LH side.



1. INSTALL FRONT SEAT ASSEMBLY

  1. Place the front seat assembly in the cabin.

    NOTICE:
    Be careful not to damage the vehicle body.

  1. Connect the seat wire connectors under the seat.

  1. Temporarily install the front seat assembly with the 4 bolts.

  1. Connect the cable to the negative (-) battery terminal.

    NOTICE:
    When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).

  1. Operate the power seat switch to move the seat to the rearmost position.




  1. Tighten the bolts in the order indicated in the illustration.

    Torque:
    37 N*m{ 377 kgf*cm , 27 ft.*lbf }

  1. Operate the power seat switch to move the seat to the foremost position.




  1. Tighten the bolts in the order indicated in the illustration.

    Torque:
    37 N*m{ 377 kgf*cm , 27 ft.*lbf }



2. INSTALL SEAT TRACK COVER LH



  1. Move the cover in the direction of the arrow to attach the 4 guides.

  1. Attach the 2 claws to install the cover.



3. INSTALL INNER SEAT TRACK BRACKET COVER LH



  1. Move the cover in the direction of the arrow to attach the 2 guides.

  1. Attach the 2 claws to install the cover.



4. INSTALL FRONT SEAT OUTER TRACK BRACKET COVER LH

  1. Operate the power seat switch to move the seat to the rearmost position.




  1. Move the cover in the direction of the arrow to attach the 2 claws and guide.

  1. Attach the 2 claws to install the cover.



5. INSTALL FRONT SEAT INNER TRACK BRACKET COVER LH



  1. Move the cover in the direction of the arrow to attach the 2 guides.

  1. Attach the 2 claws to install the cover.



6. INSTALL TELEVISION DISPLAY ASSEMBLY (w/ Rear Seat Entertainment System)

  1. Install the television display assembly with the 3 nuts.

    Torque:
    5.5 N*m{ 56 kgf*cm , 49 in.*lbf }

  1. Connect the connector.



7. INSTALL TELEVISION BASE (w/ Rear Seat Entertainment System)

  1. Attach the 3 guides and 3 claws to install the television base.



8. INSTALL FRONT SEAT HEADREST ASSEMBLY

  1. Install the front seat headrest.



9. CHECK SRS WARNING LIGHT

  1. Check the SRS warning light (Click here).



10. INSPECT FRONT SEAT ASSEMBLY

  1. Check the power seat operation (Click here).

  1. Check the climate control seat operation (Click here).
