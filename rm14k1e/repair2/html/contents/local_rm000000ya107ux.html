ENTRY AND START SYSTEM (for Start Function) > Engine does not Start


DESCRIPTION

WIRING DIAGRAM

INSPECTION PROCEDURE

CHECK BATTERY

READ VALUE USING INTELLIGENT TESTER (POWER SUPPLY OPEN) AND CHECK FOR DTC

CHECK ENGINE SWITCH CONDITION

CHECK CRANKING FUNCTION

CHECK SECURITY INDICATOR LIGHT (ENGINE IMMOBILISER SYSTEM UNSET)

CHECK ENGINE SWITCH INDICATOR LIGHT

READ VALUE USING INTELLIGENT TESTER (STARTER SW)

INSPECT ST RELAY

CHECK HARNESS AND CONNECTOR (POWER MANAGEMENT CONTROL ECU - BODY GROUND)

INSPECT STARTER ASSEMBLY

CHECK HARNESS AND CONNECTOR (BATTERY - STARTER AND ENGINE ROOM RELAY BLOCK)

READ VALUE USING INTELLIGENT TESTER (S CODE CHECK)

CHECK STEERING LOCK

READ VALUE USING INTELLIGENT TESTER (IMMOBILISER)

READ VALUE USING INTELLIGENT TESTER (STARTER REQUEST SIGNAL)

REPLACE ID CODE BOX

READ VALUE USING INTELLIGENT TESTER (SHIFT P SIGNAL, NEUTRAL SW/ CLUTCH SW, SHIFT POSITION P OR N)

READ VALUE USING INTELLIGENT TESTER (STOP LIGHT SWITCH1)

INSPECT PARK/NEUTRAL POSITION SWITCH

CHECK HARNESS AND CONNECTOR (POWER MANAGEMENT CONTROL ECU - PARK/NEUTRAL POSITION SWITCH)

INSPECT STOP LIGHT SWITCH ASSEMBLY

CHECK HARNESS AND CONNECTOR (POWER MANAGEMENT CONTROL ECU - STOP LIGHT SWITCH ASSEMBLY)

REPLACE CERTIFICATION ECU

REPLACE ID CODE BOX

READ VALUE USING INTELLIGENT TESTER (STEERING UNLOCK SWITCH)

READ VALUE USING INTELLIGENT TESTER (L CODE CHECK)

REPLACE STEERING LOCK ACTUATOR ASSEMBLY (STEERING LOCK ECU)

REPLACE STEERING LOCK ACTUATOR ASSEMBLY (STEERING LOCK ECU)

REPLACE CERTIFICATION ECU

REPLACE ID CODE BOX
