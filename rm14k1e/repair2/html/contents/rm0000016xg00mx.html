REAR AXLE SHAFT > REASSEMBLY

for Preparation Click here


NOTICE:
Do not allow foreign matter, etc. to contact the rear axle hub and bearing assembly.



1. INSTALL BRAKE DRUM OIL DEFLECTOR LH



  1. Install a new deflector gasket and deflector to the rear axle shaft.

    HINT:
    Align the 2 oil drain holes.

    Text in Illustration 
    *1
    		Oil Drain Holes

  1. Insert 6 new hub bolts.




  1. Temporarily install a washer and nut to each hub bolt as shown in the illustration.
    Text in Illustration 
    *1
    		Washer
    *2
    		Nut
    *a
    		Turn

  1. Install the hub bolts by tightening each nut.

  1. Remove the washer and nut from each hub bolt.



2. INSTALL REAR AXLE HUB AND BEARING ASSEMBLY LH

  1. Install a new rear axle hub and bearing to the parking brake plate.

    NOTICE:
    Make sure the bearing is securely installed to the parking brake plate.




  1. Using 2 socket wrenches and a press, press in the 4 housing bolts.



3. INSTALL REAR AXLE SHAFT LH



  1. Install a new washer and a new retainer to the axle hub as shown in the illustration.
    Text in Illustration 
    *a
    		Tapered Surface
    *b
    		Chamfered Surface

    NOTICE:
    1. Install the washer with its tapered surface facing downward.
    2. Install the retainer with its chamfered surface facing downward.

  1. Using SST and a press, press in the rear axle shaft.

    SST
    09631-12090  
    09726-40010  
    09951-01100  

    NOTICE:
    Make sure the bearing is securely installed to the rear axle shaft.



4. INSTALL REAR AXLE SHAFT SNAP RING LH



  1. Using snap ring expander, install a new rear axle shaft snap ring.
