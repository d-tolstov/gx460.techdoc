REAR NO. 1 SEAT ASSEMBLY (for 60/40 Split Slide Walk-in Seat Type RH Side) > REASSEMBLY

for Preparation Click here


CAUTION:
Wear protective gloves. Sharp areas on the parts may injure your hands.



1. INSTALL REAR SEAT TRACK ADJUSTING HANDLE RH



  1. Install the handle.

    HINT:
    Make sure the handle is straight when installing it, and check that it is centered.




  1. Install the seat cushion side frame.



2. INSTALL REAR SEAT CUSHION FRAME SUB-ASSEMBLY RH

  1. Install the seat cushion frame and seat leg plate with the 6 nuts.

    Torque:
    32 N*m{ 326 kgf*cm , 24 ft.*lbf }

  1. Temporarily install the seatback frame with the 4 "TORX" bolts.

  1. Using a T45 "TORX" socket wrench, tighten the 4 "TORX" bolts.

    Torque:
    29 N*m{ 296 kgf*cm , 21 ft.*lbf }

  1. Attach the 2 airbag wire harness clamps to connect the airbag wire harness to the seat cushion frame.

  1. Attach the claw to connect the side airbag connector to the seat track rail.

  1. Install a new cable tie.



3. INSTALL REAR SEAT CUSHION SPRING

  1. Attach the 2 hooks to install the spring.



4. INSTALL REAR SEAT COVER

  1. Install the cover with the screw.



5. INSTALL REAR SEAT WIRE RH (w/ Seat Heater System)

  1. Attach the clamp to install the seat wire to the seat cushion frame.

  1. Attach the claw to connect the seat wire connector to the seat track rail.



6. INSTALL SEAT HEATER CONTROL SUB-ASSEMBLY (w/ Seat Heater System)

  1. Attach the 2 clamps to install the seat heater control.

  1. Connect the connector.



7. INSTALL REAR SEATBACK LOCK PLATE LH

  1. Install the lock plate with the nut.

    Torque:
    18 N*m{ 184 kgf*cm , 13 ft.*lbf }

  1. Attach the 2 cable clamps and connect the cable.



8. INSTALL REAR SEAT CUSHION SET PLATE

  1. Install the set plate with the 2 nuts.

    Torque:
    18 N*m{ 184 kgf*cm , 13 ft.*lbf }



9. INSTALL REAR NO. 2 SEAT RECLINING CONTROL CABLE

  1. Install the cable.

  1. Attach the cable clamp and connect the cable.



10. INSTALL REAR SEAT RECLINING CONTROL CABLE

  1. Install the cable.

  1. Attach the 2 cable clamps and connect the cable.

  1. Install 3 new cable ties.



11. INSTALL RECLINING ADJUSTER INSIDE COVER RH

  1. Attach the claw and guide to install the cover.



12. INSTALL RECLINING ADJUSTER INSIDE COVER RH



  1. Move the cover in the direction of the arrow to attach the 2 claws.

  1. Attach the clip to install the cover.



13. INSTALL RECLINING ADJUSTER INSIDE COVER RH

  1. Attach the 2 claws to install the cover.

  1. Install the 2 screws.



14. INSTALL REAR SEAT INNER RECLINING COVER RH

  1. Attach the guide and install the cover with the 2 screws.



15. INSTALL REAR SEATBACK EDGE PROTECTOR

  1. Install the protector to the seatback frame.



16. INSTALL REAR SEATBACK HEATER ASSEMBLY (w/ Seat Heater System)



  1. Set the seatback heater in place with the name stamp side facing the seatback cover.
    Text in Illustration 
    *1
    		Tack Pin
    *2
    		Name Stamp

  1. Install the seatback heater with new tack pins.



17. INSTALL SEPARATE TYPE REAR SEATBACK COVER



  1. Using hog ring pliers, install the seatback cover to the seatback pad with new hog rings.
    Text in Illustration 
    *1
    		Hog Ring Pliers

    NOTICE:
    1. Be careful not to damage the cover.
    2. When installing the hog rings, avoid wrinkling the cover.




  1. Connect the seatback cover bracket to the seatback pad.




  1. w/ Seat Heater System:
    Connect the seatback heater wire harness to the seatback pad.



18. INSTALL SEATBACK COVER WITH PAD



  1. Attach the hooks.
    Text in Illustration 
    *1
    		Hog Ring Pliers
    *2
    		Hook
    *3
    		Hog Ring

  1. Using hog ring pliers, install the seatback cover with pad with new hog rings.

  1. Attach the 2 guides and connect the seatback cover bracket to the seatback frame with the nut.

    Torque:
    5.5 N*m{ 56 kgf*cm , 49 in.*lbf }

    NOTICE:
    After the seatback trim cover is assembled, make sure the side airbag strap is not twisted.



19. INSTALL REAR NO. 1 SEAT HEADREST SUPPORT ASSEMBLY

  1. Attach the 4 claws to install the 2 supports.



20. INSTALL REAR SEATBACK BOARD CARPET ASSEMBLY RH

  1. Attach the 7 clips to install the carpet.



21. INSTALL REAR SEATBACK COVER

  1. Attach the 4 claws to install the cover.



22. INSTALL REAR CENTER BACK LOCK CONTROL LEVER



  1. Attach the cable clamp to connect the cable to the lever.

  1. Move the lever in the direction of the arrow to attach the guide.

  1. Install the lever with the 2 screws.



23. INSTALL REAR SEAT COVER CAP

  1. Attach the guide and claw to install the cap.



24. INSTALL REAR SEAT LOCK CONTROL LEVER SUB-ASSEMBLY RH



  1. Connect the cable in the order shown in the illustration to install the lever.

  1. Install the 2 screws.



25. INSTALL RECLINING ADJUSTER KNOB CAP RH



  1. Attach the guide and claw to install the cap.



26. INSTALL REAR SEAT CUSHION EDGE PROTECTOR

  1. Install the protector to the cushion spring.



27. INSTALL REAR SEAT CUSHION HEATER ASSEMBLY (w/ Seat Heater System)



  1. Set the seat cushion heater in place with the name stamp side facing the seat cushion cover.
    Text in Illustration 
    *1
    		Tack Pin
    *2
    		Name Stamp

  1. Install the seat cushion heater with new tack pins.



28. INSTALL SEPARATE TYPE REAR SEAT CUSHION COVER


  1. Using hog ring pliers, install the seat cushion cover to the seat cushion pad with new hog rings.

    NOTICE:
    1. Be careful not to damage the cover.
    2. When installing the hog rings, avoid wrinkling the cover.





    Text in Illustration

29. INSTALL REAR NO. 2 SEAT PROTECTOR

  1. Install the 2 protectors to the cushion pad.



30. INSTALL SEAT CUSHION COVER WITH PAD

  1. Install the seat cushion cover with pad to the seat cushion frame.

  1. w/ Seat Heater System:

    1. Attach the 2 clamps to connect the seat heater wire harness.

    2. Connect the 2 seat heater connectors.




  1. Attach the hooks to install the seat cushion cover with pad.



31. INSTALL REAR SEATBACK LOCK STRIKER COVER RH

  1. Attach the 3 clips to install the cover.



32. INSTALL REAR SEAT INNER BELT ASSEMBLY

  1. Install the inner belt with the nut.

    Torque:
    42 N*m{ 428 kgf*cm , 31 ft.*lbf }



33. INSTALL REAR NO. 1 SEAT INNER BELT ASSEMBLY RH



  1. Install the inner belt with the bolt.

    Torque:
    42 N*m{ 428 kgf*cm , 31 ft.*lbf }

    NOTICE:
    Do not allow the anchor part of the front seat inner belt assembly to overlap the protruding part of the front seat adjuster.

    Text in Illustration 
    *a
    		Protruding Part



34. INSTALL REAR SEAT REAR CUSHION MOULDING LH

  1. Attach the guide and 3 claws to install the moulding.

  1. Install the screw.



35. INSTALL REAR SEAT CUSHION MOULDING RH



  1. Move the moulding in the direction of the arrow to attach the guide and install the moulding.

  1. Attach the 3 claws and clip to install the moulding.



36. INSTALL RECLINING ADJUSTER RELEASE HANDLE RH

  1. Install the handle with the 3 screws.



37. INSTALL REAR SEAT UPPER RECLINING COVER RH



  1. Attach the 2 claws to install the cover.
