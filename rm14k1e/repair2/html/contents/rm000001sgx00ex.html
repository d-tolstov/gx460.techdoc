DTC    C1794/94    Height Control OFF Switch Circuit (Test Mode DTC)

for Preparation Click here


DESCRIPTION

If the height control OFF switch is pressed when the vehicle is stopped with the engine running or the engine switch on (IG), vehicle height control is disabled and remains disabled even after the engine switch is turned off. Pressing the height control OFF switch again with the engine running or the engine switch on (IG) resumes vehicle height control. If the vehicle speed reaches approximately 30 km/h (19 mph) when vehicle height control is disabled, vehicle height control automatically resumes.
