DTC    B2416    Height Control Sensor Malfunction

DTC    B241A    Rear Height Control Sensor

for Preparation Click here


DESCRIPTION

The DTC is stored when the headlight swivel ECU detects malfunctions in the suspension control ECU.

DTC Code
DTC Detection Condition
Trouble Area
B2416
  1. Malfunction in the suspension control ECU.
  2. Open or short in the suspension control ECU circuit.
  1. Air suspension system
  2. Suspension control ECU
  3. Headlight swivel ECU assembly
  4. Harness or connector
B241A
  1. Malfunction in the suspension control ECU.
  2. Open or short in the suspension control ECU circuit.
  1. Air suspension system
  2. Suspension control ECU
  3. Headlight swivel ECU assembly
  4. Harness or connector
