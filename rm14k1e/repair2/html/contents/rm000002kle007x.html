STABILIZER CONTROL ECU > INSTALLATION

for Preparation Click here



1. INSTALL STABILIZER CONTROL ECU

  1. Install the stabilizer control ECU with the bolt.

    Torque:
    14 N*m{ 138 kgf*cm , 10 ft.*lbf }

    NOTICE:
    1. Replace the stabilizer control ECU with a new one if it is dropped.
    2. Make sure the bracket rotation stopper touches the installation position.

  1. Connect the connector.

    NOTICE:
    Securely connect the connector.



2. INSTALL LOWER NO. 1 INSTRUMENT PANEL AIRBAG ASSEMBLY

  1. Install the lower No. 1 instrument panel airbag assembly (Click here).



3. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).
