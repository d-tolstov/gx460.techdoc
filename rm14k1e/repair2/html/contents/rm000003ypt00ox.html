FUEL LID OPENER SWITCH > INSTALLATION

for Preparation Click here



1. INSTALL FUEL LID OPENER SWITCH

  1. Attach the 2 claws and install the fuel lid opener switch.



2. INSTALL LOWER INSTRUMENT PANEL FINISH PANEL SUB-ASSEMBLY

  1. Connect each connector and cable.

  1. Attach the 13 clips and 2 guides to install the lower instrument panel finish panel.

  1. Install the 2 bolts <C>.

  1. Attach the 2 claws to close the cover.



3. INSTALL INSTRUMENT PANEL FINISH PANEL END LH

  1. Attach the 11 clips to install the instrument panel finish panel end.



4. INSTALL NO. 1 INSTRUMENT PANEL UNDER COVER SUB-ASSEMBLY

  1. Connect the connector and attach the clamp.

  1. Attach the 2 clips and 2 guides to install the No. 1 instrument panel under cover.

  1. Install the screw.



5. INSTALL COWL SIDE TRIM BOARD LH

  1. Attach the clip and claw to install the cowl side trim board.

  1. Install the clip.



6. INSTALL DOOR SCUFF PLATE ASSEMBLY LH

  1. Connect the connector.

  1. Attach the 4 clips, 10 claws and 2 guides to install the door scuff plate.



7. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).



8. CHECK SRS WARNING LIGHT

  1. Check the SRS warning light (Click here).
