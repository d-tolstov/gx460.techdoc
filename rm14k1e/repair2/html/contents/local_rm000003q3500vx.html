INSPECTION PROCEDURE

CHECK FOR ANY OTHER DTCS OUTPUT (IN ADDITION TO P0401)

PERFORM ACTIVE TEST USING INTELLIGENT TESTER (OPERATE EGR VALVE)

INSPECT EGR VALVE ASSEMBLY

REPLACE EGR VALVE ASSEMBLY

CHECK FOR DEPOSIT (EGR VALVE ASSEMBLY TO INTAKE PORT)

CHECK FOR DEPOSIT (EXHAUST MANIFOLD CONVERTER TO EGR VALVE ASSEMBLY)

INSPECT EGR COOLER ASSEMBLY

READ VALUE USING INTELLIGENT TESTER (MANIFOLD ABSOLUTE PRESSURE SENSOR)

CHECK INTAKE SYSTEM

REPLACE MANIFOLD ABSOLUTE PRESSURE SENSOR

CONFIRM WHETHER MALFUNCTION HAS BEEN SUCCESSFULLY REPAIRED
