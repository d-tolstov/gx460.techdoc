DTC    U1002    Lost Communication with Gateway Module (Sensor Bus)

for Preparation Click here


DESCRIPTION

  1. The driving support ECU assembly stores this DTC when no signals can be received from the ECUs that have been memorized as those connected to the sensor bus.
  2. When the driving support ECU assembly receives a response signal from the ECUs connected to the sensor bus, the driving support ECU assembly recognizes and memorizes that the ECU is connected to the sensor bus. Based on this memorized data, the driving support ECU assembly monitors for malfunctions in the ECUs connected to the sensor bus when communicating with those ECUs. If the driving support ECU assembly cannot receive response signals from the ECUs that have been memorized as those connected to the sensor bus, the driving support ECU assembly determines that a malfunction exists.
  3. If 2 or more DTCs are output during the DTC check, one side of the CAN branch wire may be open (one side of the CANH [branch wire]/CANL [branch wire] of the ECU and/or sensor is open).

DTC Code
DTC Detection Condition
Trouble Area
U1002
Lost communication with the gateway module (sensor bus).
  1. Sensor bus circuit
  2. Millimeter wave radar sensor assembly main wire or connector
  3. Driving support ECU assembly main wire or connector
  4. Millimeter wave radar sensor assembly
  5. Driving support ECU assembly
  6. No. 3 junction connector
  1. For vehicles with pre-crash safety system only.

WIRING DIAGRAM




INSPECTION PROCEDURE

NOTICE:
Inspect the fuses for circuits related to this system before performing the following inspection procedure.

HINT:
Operating the engine switch, any switches or any doors triggers related ECU and sensor communication with the CAN, which causes resistance variation.

1.DISCONNECT CABLE FROM NEGATIVE BATTERY TERMINAL

  1. Disconnect the cable from the negative (-) battery terminal before measuring the resistances of the main wire and the branch wire.

    CAUTION:
    Wait at least 90 seconds after disconnecting the cable from the negative (-) battery terminal to disable the SRS system.

    NOTICE:
    1. w/ Navigation System (for HDD):
      After the engine switch is turned off, the display and navigation module display (HDD navigation system) records various types of memory and settings. As a result, after turning the engine switch off, be sure to wait for the time specified in the following table before disconnecting the cable from the negative (-) battery terminal.

      Waiting Time before Disconnecting Cable from Negative (-) Battery Terminal:

      Condition
      		Waiting Time
      Vehicle enrolled in G-BOOK system
      		6 minutes
      Vehicle not enrolled in G-BOOK system
      		1 minute

    2. When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).
