1.PRECAUTIONS WHEN CHECKING FOR DTCS

  1. When the cable is disconnected from the negative (-) battery terminal, the DTCs stored in the steering lock actuator assembly (steering lock ECU) are cleared. Be sure to record any output DTCs immediately.

  2. Normally, with the engine switch off, even if the intelligent tester is connected to the vehicle, the information stored in entry and start system related ECUs (the certification ECU, power management control ECU, main body ECU, etc.) cannot be read. However, entry and start system DTCs and Data List information can be confirmed even if the engine switch is off using either of the two methods below:

    1. Connect the intelligent tester to the vehicle, and turn a courtesy light switch on and off at intervals of 1.5 seconds or less until communication between the intelligent tester and vehicle begins (the intelligent tester screen changes when communication begins).
    2. Connect the intelligent tester to the vehicle and select "MANUAL" from the initial screen on the intelligent tester. Select "KEY REGIST" under Model Code.


2.PRECAUTIONS WHEN REPLACING STEERING LOCK ACTUATOR ASSEMBLY (STEERING LOCK ECU)

  1. If the steering lock actuator assembly (steering lock ECU) is replaced, be sure to confirm that the steering is unlocked by turning the steering wheel to the left and right before starting the engine. If the steering is locked for any reason, open and close a door with the engine switch off, and then unlock the steering by pressing the engine switch. This prevents the engine from starting while the steering is locked.


3.PRECAUTION FOR REGISTRATION

  1. If replacing any of the following parts, refer to the Service Bulletin.

    NOTICE:
    Any keys or ECUs for which registration has been performed cannot be reused.


    1. ID code box

    2. Certification ECU

    3. Steering lock actuator assembly (steering lock ECU)

    4. ECM

    5. Electrical key transmitter (key)
