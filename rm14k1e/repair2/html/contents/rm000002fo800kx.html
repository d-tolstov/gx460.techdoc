CONTROL VALVE ASSEMBLY > INSTALLATION

for Preparation Click here



1. INSTALL NO. 2 HEIGHT CONTROL VALVE SUB-ASSEMBLY

  1. Install the No. 2 height control valve with the 2 bolts.

    Torque:
    29 N*m{ 296 kgf*cm , 21 ft.*lbf }



2. CONNECT HEIGHT CONTROL TUBE

  1. Connect the 3 height control tubes (Click here).

    HINT:
    For the connection procedure of the tubes (type 1 and type 2), refer to Precaution of the Suspension Control System section.

  1. Connect the connector.



3. INSTALL FUEL PUMP CONTROL ECU ASSEMBLY

  1. Install the fuel pump ECU with the 2 bolts.

    Torque:
    31 N*m{ 316 kgf*cm , 23 ft.*lbf }

    NOTICE:
    Do not use the fuel pump ECU if it has been subjected to physical shocks, such as being dropped, etc.

  1. Connect the 2 connectors.



4. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).



5. CHECK CONNECTION OF TUBE FOR AIR LEAK

  1. Check the connections of the tubes for air leakage (Click here).
