ENTRY AND START SYSTEM (for Start Function) > DTC CHECK / CLEAR

for Preparation Click here


NOTICE:
If the intelligent tester is connected to the vehicle with the engine switch off, the information stored in entry and start system related ECUs (the certification ECU, power management control ECU and main body ECU) cannot be read. However, entry and start system DTCs and Data List information can be confirmed even if the engine switch is off using the method below:
  1. Connect the intelligent tester to the vehicle, and turn a courtesy light switch on and off at intervals of 1.5 seconds or less until communication between the intelligent tester and vehicle begins (the intelligent tester screen changes when communication begins).
  2. Connect the intelligent tester to the vehicle and select "MANUAL" from the initial screen on the intelligent tester. Select "KEY REGIST" under Model Code.


CHECK FOR DTC (POWER MANAGEMENT CONTROL ECU [POWER SOURCE CONTROL])

  1. Connect the intelligent tester to the DLC3.

  2. Turn the engine switch on (IG).

  3. Turn the intelligent tester on.

  4. Enter the following menus: Body / Power Source Control / DTC.

  5. Check for DTCs.

    HINT:
    Refer to the intelligent tester operator's manual for further details.

CLEAR DTC (POWER MANAGEMENT CONTROL ECU [POWER SOURCE CONTROL])

  1. Connect the intelligent tester to the DLC3.

  2. Turn the engine switch on (IG).

  3. Turn the intelligent tester on.

  4. Enter the following menus: Body / Power Source Control / DTC.

  5. Clear the DTCs by following the directions on the intelligent tester display.

    HINT:
    Refer to the intelligent tester operator's manual for further details.
