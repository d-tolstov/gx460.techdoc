NAVIGATION SYSTEM (for HDD) > Microphone Circuit between Microphone and Navigation ECU

for Preparation Click here


DESCRIPTION

w/ G-BOOK System
Using this circuit, the telematics transceiver sends power to the map light assembly (telephone microphone assembly), and the map light assembly (telephone microphone assembly) sends microphone signals to the display and navigation module display via the telematics transceiver. The display and navigation module display and map light assembly (telephone microphone assembly) are connected to each other using the microphone connection detection signal lines.
w/o G-BOOK System
Using this circuit, the display and navigation module display sends power to the map light assembly (telephone microphone assembly), and the map light assembly (telephone microphone assembly) sends microphone signals to the display and navigation module display. The display and navigation module display and map light assembly (telephone microphone assembly) are connected to each other using the microphone connection detection signal lines.
