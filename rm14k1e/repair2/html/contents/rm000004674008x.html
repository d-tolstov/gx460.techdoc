FRONT SEAT ASSEMBLY > REASSEMBLY

for Preparation Click here


CAUTION:
Wear protective gloves. Sharp areas on the parts may injure your hands.

HINT:
  1. Use the same procedure for the RH and LH sides.
  2. The procedure listed below is for the LH side.



1. INSTALL REAR SEAT WIRE LH (w/ Rear Seat Entertainment System)

  1. Attach the 3 clamps to install the seat audio wire to the seatback spring.



2. INSTALL NO. 2 SEAT WIRE

  1. Attach the 3 clamps to install the seat wire.

  1. Connect the 2 connectors.



3. INSTALL FRONT SEAT AIRBAG ASSEMBLY LH

  1. Install the side airbag to the seat frame with the 2 nuts.

    Torque:
    5.5 N*m{ 56 kgf*cm , 49 in.*lbf }

  1. Attach the wire harness 2 clamps.



4. INSTALL SEATBACK CLIMATE CONTROL BLOWER LH

  1. Install the blower with the 3 screws.

  1. Connect the connector.



5. INSTALL NO. 1 TELEVISION BASE

  1. Attach the seat audio wire clamp and 2 claws to install the television base.

  1. Install the 2 screws.

  1. Install the 9 clips.



6. INSTALL FRONT SEATBACK HEATER ASSEMBLY LH



  1. Set the seatback heater in place with the name stamp side facing the seatback cover.
    Text in Illustration 
    *1
    		Tack Pin
    *2
    		Name Stamp

  1. Install the seatback heater with new tack pins.



7. INSTALL SEPARATE TYPE FRONT SEATBACK COVER



  1. Using hog ring pliers, install the seatback cover to the seatback pad with new hog rings.
    Text in Illustration 
    *1
    		Hog Ring Pliers

    NOTICE:
    1. Be careful not to damage the cover.
    2. When installing the hog rings, avoid wrinkling the cover.




  1. Connect the seatback cover bracket to the seatback pad.



8. INSTALL SEATBACK COVER WITH PAD

  1. Install the seatback cover with pad to the seatback spring.

  1. Attach the 4 claws to install the 2 headrest supports.

  1. Attach the 2 claws and connect the seatback cover bracket to the seatback spring with the nut.

    Torque:
    5.5 N*m{ 56 kgf*cm , 49 in.*lbf }

    NOTICE:
    After the seatback trim cover is assembled, make sure the side airbag strap is not twisted.




  1. Using hog ring pliers, install new hog rings.
    Text in Illustration 
    *1
    		Hog Ring Pliers

  1. Attach the hooks.



9. INSTALL RECLINING ADJUSTER INSIDE COVER LH

  1. Attach the guide to install the cover.



10. INSTALL RECLINING ADJUSTER INSIDE COVER RH

HINT:
Use the same procedure described for the LH side.



11. INSTALL FRONT SEAT WIRE LH

  1. Install the seat wire to the seat cushion spring.



12. INSTALL SEAT HEATER CONTROL SUB-ASSEMBLY LH

  1. Attach the 2 clamps to install the seat heater control.

  1. Connect the connector.



13. INSTALL SEAT CUSHION CLIMATE CONTROL BLOWER ASSEMBLY LH

  1. Attach the 2 guides and install the blower by pushing it forward.

  1. Attach the 3 guides and install the bracket.

  1. Using a T20 "TORX" socket wrench, install the "TORX" screw.



14. INSTALL SEAT CUSHION CLIMATE CONTROL DUCT

  1. Attach the 2 claws and 3 clamps to install the duct.



15. INSTALL FRONT SEAT CUSHION EDGE PROTECTOR LH



  1. Move the protector in the direction of the arrow to attach the hook and install the protector.

  1. Install the screw.



16. INSTALL FRONT SEAT CUSHION EDGE PROTECTOR RH

HINT:
Use the same procedure described for the LH side.



17. INSTALL FRONT SEAT LOWER CUSHION SHIELD LH

  1. Attach the hook and clip to install the cushion shield.

  1. Install the screw.



18. INSTALL FRONT SEAT LOWER CUSHION SHIELD RH

  1. Attach the hook and clip to install the cushion shield.

  1. Install the screw.



19. INSTALL RECLINING ADJUSTER INSIDE COVER LH

  1. Attach the claw and guide to install the cover.



20. INSTALL FRONT SEAT CUSHION HEATER ASSEMBLY LH



  1. Set the seat cushion heater in place with the name stamp side facing the seat cushion cover.
    Text in Illustration 
    *1
    		Tack Pin
    *2
    		Name Stamp

  1. Install the seat cushion heater with new tack pins.




  1. Connect the wire harness to the seat cushion cover.



21. INSTALL SEPARATE TYPE FRONT SEAT CUSHION COVER



  1. Using hog ring pliers, install the seat cushion cover to the seat cushion pad with new hog rings.
    Text in Illustration 
    *1
    		Hog Ring Pliers

    NOTICE:
    1. Be careful not to damage the cover.
    2. When installing the hog rings, avoid wrinkling the cover.



22. INSTALL SEAT CUSHION COVER WITH PAD

  1. Install the seat cushion cover with pad to the seat cushion spring.

  1. for Front Passenger Side:
    Attach the claw to connect the occupant detection sensor connector to the seat cushion spring.

  1. Connect the seat heater connector and attach the 3 wire harness clamps.

  1. Attach the hooks.



23. INSTALL FRONT SEAT LEG COVER LH



  1. Move the cover in the direction of the arrow to install the cover.

  1. Attach the 2 claws.



24. INSTALL SEAT ADJUSTER COVER CAP

  1. Install the 2 cover caps.



25. INSTALL SEPARATE TYPE FRONT SEATBACK ASSEMBLY

  1. Install the seatback assembly with the 4 bolts.

    Torque:
    42 N*m{ 428 kgf*cm , 31 ft.*lbf }

  1. Connect the airbag wire harness.

  1. Attach the fastening tape to close the cover.

  1. Attach the 3 airbag wire harness clamps.

  1. Attach the claw to connect the airbag connector.

  1. w/ Rear Seat Entertainment System:
    Attach the seat audio wire harness clamps and connect the seat audio connector.

  1. Attach the 2 hooks.

  1. Attach the clamp to connect the No. 2 seat wire.

  1. Connect the seat heater connector and attach the wire harness clamp.



26. INSTALL FRONT SEAT INNER BELT ASSEMBLY LH



  1. Install the front seat inner belt with the nut.

    Torque:
    42 N*m{ 428 kgf*cm , 31 ft.*lbf }

    NOTICE:
    Do not allow the anchor part of the front seat inner belt assembly to overlap the protruding parts of the front seat adjuster.

    Text in Illustration 
    *a
    		Protruding Part

  1. for Driver Side:
    Connect the 2 connectors and attach the 5 clamps.

  1. for Passenger Side:
    Connect the 2 connectors and attach the 4 clamps.



27. INSTALL FRONT SEAT INNER CUSHION SHIELD LH



  1. Attach the 3 claws to install the cushion shield.



28. INSTALL FRONT SEAT NO. 1 INNER CUSHION SHIELD LH



  1. Attach the 2 claws to install the cushion shield.



29. INSTALL LUMBAR SWITCH ASSEMBLY



  1. Connect the lumbar switch connector.

  1. Attach the 4 claws to install the lumbar switch.



30. INSTALL FRONT POWER SEAT SWITCH LH

  1. for Driver Side:

    1. Connect the 2 connectors.

    2. Install the power seat switch with the 3 screws.

  1. for Front Passenger Side:

    1. Connect the connector.

    2. Install the power seat switch with the 3 screws.



31. INSTALL FRONT SEAT CUSHION SHIELD LH

  1. Attach the 3 wire harness clamps and connect the seat wire connector.

  1. Attach the 3 claws to close the wire harness cover.

  1. Connect the lifter motor connector.



  1. Remove the clip attached to the seat cushion spring and install it to the cushion shield.



  1. Attach the 2 clips, 3 claws and guide.

  1. Attach the guide of the front seat No. 1 inner cushion shield and install the screw.



32. INSTALL RECLINING POWER SEAT SWITCH KNOB



  1. Attach the 2 claws to install the knob.



33. INSTALL SLIDE AND VERTICAL POWER SEAT SWITCH KNOB



  1. Attach the 2 claws to install the knob.



34. INSTALL NO. 1 INTERIOR ILLUMINATION LIGHT ASSEMBLY

  1. Attach the 2 claws to install the light.

  1. Connect the connector.

  1. Install the seatback cover.



35. INSTALL FRONT SEATBACK BOARD SUB-ASSEMBLY LH



  1. Attach the 2 guides and 2 claws to install the seatback board.

  1. Connect the interior illumination light connector.

  1. Attach the 2 hooks to connect the 2 rubber bands.
