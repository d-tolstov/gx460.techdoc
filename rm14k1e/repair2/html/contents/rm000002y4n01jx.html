HOOD > REASSEMBLY

for Preparation Click here



1. INSTALL HOOD SUPPORT ASSEMBLY LH

NOTICE:
  1. Avoid touching the piston rod as much as possible to prevent foreign matter from attaching to it. Be sure to hold the cylinder while servicing.
  2. Do not wear cotton gloves or other similar materials when handling the piston rod. Fibers may attach to the rod and result in gas leaks.
  3. In order to prevent the piston rod from deforming, do not apply any horizontal load to the hood support.



  1. When reusing the hood support assembly:




    1. Install the 2 stop rings to the hood support assembly.




  1. Install the hood support assembly.

    NOTICE:
    Install the hood support assembly while supporting the hood by hand.

  1. Check that the hood support assembly is connected to the ball joint and cannot be pulled off.



2. INSTALL HOOD SUPPORT ASSEMBLY RH

HINT:
Use the same procedure described for the LH side.



3. CONNECT WASHER HOSE ASSEMBLY



  1. Attach the 4 clips and connect the washer hose assembly.



4. INSTALL WASHER NOZZLE SUB-ASSEMBLY

  1. Connect a new washer nozzle sub-assembly to the washer hose.

  1. Attach the 2 claws to install the washer nozzle sub-assembly.



5. INSPECT WASHER NOZZLE SUB-ASSEMBLY


  1. With the engine running, check the position that the washer fluid hits the windshield.

6. INSTALL HOOD INSULATOR



  1. Attach the 2 claws.

  1. Install the hood insulator with the 11 clips.



7. INSTALL HOOD TO RADIATOR SUPPORT SEAL

  1. Attach the 7 clips to install the hood to radiator support seal.
