REAR UPPER ARM > INSTALLATION

for Preparation Click here


HINT:
  1. Use the same procedure for the RH and LH sides.
  2. The procedure listed below is for the LH side.
  3. A bolt without a torque specification is shown in the standard bolt chart (Click here).



1. TEMPORARILY INSTALL REAR UPPER CONTROL ARM ASSEMBLY LH



  1. Temporarily install the upper control arm assembly and washer with the nut and bolt.
    Text in Illustration 
    *a
    		Front
    *b
    		Outer Side

    HINT:
    If the vehicle is equipped with an air suspension, install the rear upper control arm in the direction shown in the illustration.

  1. Temporarily install the upper control arm assembly to the rear axle housing with the nut and washer and bolt.



2. CONNECT REAR HEIGHT CONTROL SENSOR LINK

  1. Connect the height control sensor link sub-assembly (Click here).



3. INSTALL REAR WHEEL

Torque:
112 N*m{ 1142 kgf*cm , 83 ft.*lbf }



4. STABILIZE SUSPENSION

  1. Lower the vehicle.

  1. Bounce the vehicle up and down several times to stabilize the suspension.



5. TIGHTEN REAR UPPER CONTROL ARM ASSEMBLY LH



  1. Tighten the 2 nuts.

    Torque:
    80 N*m{ 816 kgf*cm , 59 ft.*lbf }

    HINT:
    Turn the nut while holding the bolt.
