DESCRIPTION


DTC Code
DTC Detection Condition
Trouble Area
62-7D
Either condition is met:
  1. An incompatible MP3/WMA file is used.
  2. Although the file has an extension of ".mp3" or ".wma", the header information cannot be read.
  1. Disc
  2. Radio receiver assembly
62-7E
Either condition is met:
  1. A disc with no music data is used.
  2. Playable files are not on the disc (MP3/WMA).
62-7F
A copy-protected file, which cannot be played, is used.
63-7D
Either condition is met:
  1. An incompatible MP3/WMA file is used.
  2. Although the file has an extension of ".mp3" or ".wma", the header information cannot be read.
63-7E
Either condition is met:
  1. A disc with no music data is used.
  2. Playable files are not on the disc (MP3/WMA).
63-7F
A copy-protected file, which cannot be played, is used.
