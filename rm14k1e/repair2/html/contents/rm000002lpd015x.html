1. INSTALL SLIDING ROOF DRIVE CABLE SUB-ASSEMBLY



  1. Hold down the window deflector.

    NOTICE:
    Make sure that the spring indicated by the arrow in the illustration is securely installed.




  1. Using a screwdriver, slide the drive cable in the direction indicated by the arrow in the illustration to install it.

    HINT:
    Tape the screwdriver tip before use.

    Text in Illustration 
    *1
    		Protective Tape

    		Slide




  1. Slide the rear roof drip channel forward and attach the 2 claws to install it.
    Text in Illustration 
    *a
    		Forward



  1. Install the 2 stoppers with the 2 screws.





2. ADJUST SLIDING ROOF DRIVE CABLE SUB-ASSEMBLY



  1. Using a screwdriver, align the matchmarks by sliding the drive cable in the directions indicated by the arrow in the illustration.

    HINT:
    Tape the screwdriver tip before use.

    Text in Illustration 
    *1
    		Matchmark A
    *2
    		Matchmark B
    *3
    		Protective Tape



3. INSTALL SLIDING ROOF SUNSHADE STOPPER LH



  1. Install the stopper.



4. INSTALL SLIDING ROOF SUNSHADE STOPPER RH

HINT:
Use the same procedure described for the LH side.



5. INSTALL SUNSHADE TRIM SUB-ASSEMBLY


  1. Insert the sunshade trim into the sliding roof housing.

6. INSTALL SLIDING ROOF DRIVE GEAR SUB-ASSEMBLY

  1. Install the drive gear with the 2 bolts.
