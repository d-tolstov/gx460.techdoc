1. INSTALL HIGH PITCHED HORN ASSEMBLY

  1. Install the high pitched horn with the bolt.

    Torque:
    19 N*m{ 194 kgf*cm , 14 ft.*lbf }

  1. Connect the connector.



2. INSTALL LOW PITCHED HORN ASSEMBLY

  1. Install the low pitched horn with the bolt.

    Torque:
    19 N*m{ 194 kgf*cm , 14 ft.*lbf }

  1. Connect the connector.



3. INSTALL UPPER RADIATOR SUPPORT SEAL
