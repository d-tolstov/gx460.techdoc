DESCRIPTION


DTC Code
DTC Detection Condition
Trouble Area
U0100
While the dynamic radar cruise control is either preparing for operation or operating, a communication stop between the ECM and skid control ECU or driving support ECU continues for 1 second or more.
  1. Master cylinder solenoid (skid control ECU)
  2. CAN communication system
  3. Driving support ECU
  4. ECM
U0122
While the dynamic radar cruise control is either preparing for operation or operating, a communication stop between the skid control ECU and driving support ECU continues for 1 second or more.
  1. ECM
  2. CAN communication system
  3. Master cylinder solenoid (skid control ECU)
U0123
While the dynamic radar cruise control is either preparing for operation or operating, a communication stop between the yaw rate sensor and driving support ECU continues for 1 second or more.
  1. Yaw rate sensor
  2. CAN communication system
  3. ECM
U0126
While the dynamic radar cruise control is either preparing for operation or operating, a communication stop between the spiral with sensor cable sub-assembly and driving support ECU continues for 1 second or more.
  1. Spiral with sensor cable sub-assembly
  2. CAN communication system
  3. ECM
U0235
While the dynamic radar cruise control is either preparing for operation or operating, a communication stop between the millimeter wave radar sensor assembly and driving support ECU continues for 1 second or more.
  1. Millimeter wave radar sensor assembly
  2. CAN communication system
  3. Driving support ECU
U1102
While the dynamic radar cruise control is either preparing for operation or operating, a communication stop between the driving support ECU and millimeter wave radar sensor assembly continues for 1 second or more.
  1. Millimeter wave radar sensor assembly
  2. CAN communication system
  3. Driving support ECU
U1104
While the dynamic radar cruise control is either preparing for operation or operating, a communication stop between the driving support ECU and ECM continues for 1 second or more.
  1. ECM
  2. CAN communication system
  3. Driving support ECU

INSPECTION PROCEDURE

HINT:
This circuit uses CAN communication. Therefore, if there are any malfunctions in the communication circuit, one or more DTCs in the CAN communication system are output.

1.CHECK CAN COMMUNICATION SYSTEM

  1. Select "Bus Check" from the "System Select" screen on the intelligent tester.

  1. Select "Communication Malfunction DTC" from the "Bus Check" screen, and then select "Enter".

    OK:
    CAN communication system DTCs are not output.
