DTC    B15E9    Telematics Communication Malfunction

for Preparation Click here


DESCRIPTION

This DTC will be stored when any of the following conditions is met:
  1. A manual maintenance check does not complete normally (for a reason such as no response from the G-BOOK center within a specified time).
  2. An automatic maintenance check does not complete normally (for a reason such as no response from the G-BOOK center within a specified time) 30 times in a row.
  3. A manual emergency call or automatic emergency call does not complete normally.

DTC Code
DTC Detection Condition
Trouble Area
B15E9
There is a communication processing error.
G-BOOK center

INSPECTION PROCEDURE

1.CHECK CONNECTION



  1. Check the connection to the G-BOOK center.

    1. Press the "INFO/TEL" switch on the display and navigation module display.

    2. Turn the engine switch on (ACC) and wait 1 minute.

    3. Perform the operations shown in the illustration and check that G-BOOK.com is displayed.

      HINT:
      1. If the G-BOOK.com screen is displayed, select "Renew" to check if it is possible to access G-BOOK.com.
      2. The screen illustrations provided are samples only and may be different from the actual screens.
