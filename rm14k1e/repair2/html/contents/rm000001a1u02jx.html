VANE PUMP > REASSEMBLY

for Preparation Click here


NOTICE:
When installing parts, coat the parts indicated by arrows with power steering fluid (Click here).



1. INSTALL VANE PUMP HOUSING OIL SEAL

  1. Coat the lip of a new oil seal with MP grease.




  1. Using SST and a press, press in the oil seal.

    SST
    09950-60010   (09951-00280)
    09950-70010   (09951-07100)

    NOTICE:
    Make sure that the oil seal is installed facing in the correct direction as shown in the illustration.



2. INSTALL VANE PUMP SHAFT WITH VANE PUMP PULLEY

  1. Coat the inside surface of the bushing in the vane pump front housing with power steering fluid.




  1. Gradually insert the vane pump shaft with vane pump pulley.
    Text in Illustration 
    *1
    		Protective Tape

    NOTICE:
    1. Do not damage the lip of the oil seal in the vane pump front housing.
    2. Wrap protective tape around the spline of the vane pump shaft with vane pump pulley in order to prevent damage to the oil seal.



3. INSTALL VANE PUMP FRONT SIDE PLATE



  1. Coat a new O-ring with power steering fluid and install it to the vane pump front housing.




  1. Coat a new O-ring with power steering fluid and install it to the front side plate.




  1. Align the notch of the front side plate with the notch of the vane pump front housing and install the front side plate.
    Text in Illustration 
    *a
    		Align

    NOTICE:
    Make sure that the front side plate is installed facing in the correct direction.



4. INSTALL VANE PUMP CAM RING



  1. Align the notch of the cam ring with the notch of the front side plate and install the cam ring with the inscribed mark facing upward.
    Text in Illustration 
    *a
    		Align
    *b
    		Inscribed Mark

    NOTICE:
    Make sure that the cam ring is installed facing in the correct direction.



5. INSTALL VANE PUMP ROTOR



  1. Install the vane pump rotor.

    HINT:
    The vane pump rotor can be installed with either side facing up.

  1. Coat the 10 vane pump plates with power steering fluid.




  1. Install the vane pump plates with the round end facing outward.
    Text in Illustration 
    *a
    		Round End
    *b
    		Outward

    NOTICE:
    Make sure that the vane pump plates are installed facing in the correct direction.



6. INSTALL VANE PUMP SHAFT SNAP RING

  1. Using a screwdriver and snap ring expander, install a new shaft snap ring to the vane pump shaft.

    NOTICE:
    1. Do not expand the shaft snap ring any further than needed.
    2. Make sure that the shaft snap ring fits completely into the groove.
    3. Do not damage the vane pump rotor and shaft.



7. INSTALL VANE PUMP REAR HOUSING



  1. Coat a new O-ring with power steering fluid and install it to the rear housing.




  1. Align the straight pin of the rear housing with the notches of the cam ring, front side plate and vane pump front housing.
    Text in Illustration 
    *a
    		Align

    NOTICE:
    Make sure that the O-ring is not protruding from anywhere when installing the vane pump rear housing.

  1. Install the rear housing with the 4 bolts.

    Torque:
    22 N*m{ 224 kgf*cm , 16 ft.*lbf }



8. SECURE VANE PUMP ASSEMBLY



  1. Using SST, secure the vane pump in a vise.

    SST
    09630-00014   (09631-00132)



9. INSTALL PUMP REAR BRACKET

  1. Install the bracket with the bolt.

    Torque:
    43 N*m{ 438 kgf*cm , 32 ft.*lbf }



10. INSTALL PRESSURE PORT UNION

  1. Coat a new O-ring with power steering fluid and install it to the pressure port union.

  1. Install the pressure port union.

    Torque:
    69 N*m{ 704 kgf*cm , 51 ft.*lbf }



11. INSTALL PRESSURE FEED TUBE

  1. Install 2 new gaskets to the pressure feed tube.

  1. Install the pressure feed tube.

    Torque:
    69 N*m{ 704 kgf*cm , 51 ft.*lbf }



12. INSPECT TOTAL PRELOAD

  1. Check that the pump rotates smoothly without abnormal noise.

  1. Temporarily install a service bolt.

    Recommended Service Bolt:
    Item
    		Value
    Thread diameter
    		10 mm (0.394 in.)
    Thread pitch
    		1.25 mm (0.0492 in.)
    Bolt length
    		50 mm (1.97 in.)




  1. Using a torque wrench, measure the pump rotating torque.

    Standard rotating torque:
    0.3 N*m (3 kgf*cm, 2 in.*lbf) or less

    Text in Illustration 
    *a
    		Service Bolt
    If the rotating torque is not as specified, check the vane pump housing oil seal.



13. INSTALL SUCTION PORT UNION

  1. Coat a new O-ring with power steering fluid and install it to the suction port union.

  1. Install the suction port union to the vane pump with the bolt.

    Torque:
    12 N*m{ 122 kgf*cm , 9 ft.*lbf }
