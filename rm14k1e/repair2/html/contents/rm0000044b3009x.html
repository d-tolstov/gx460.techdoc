HINT:
If the following conditions are detected while cruise control is in operation, the system clears the stored vehicle speed in the ECM and cancels cruise control.

Cruise control indicator light blinks
  1. There is an open or short in the stop light switch circuit
  2. There is a problem with the vehicle speed signal
  3. There is a problem with the throttle body assembly
Turn the cruise control switch on again
Cruise control indicator light blinks
  1. There is a problem with the input circuit of the stop light switch circuit
  2. There is a problem with the cancel circuit
  1. Turn the cruise control switch on again
  2. Turn the engine switch off and then on (IG) again
Cruise control indicator light remains on
(Cruise control is canceled)
Under either of the following conditions, cruise control operation is canceled but the stored vehicle speed is maintained:
  1. The vehicle speed is less than the low speed limit (approximately 40 km/h [25 mph]) while driving with the cruise control system in operation
  2. The vehicle stability control system is operated
  3. When the ECM detects a cruise control cancel signal from the four wheel control ECU
Push the cruise control switch to +RES
