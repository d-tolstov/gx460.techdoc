DTC    B2658    Short in Sensor with Motor Power Supply Circuit

for Preparation Click here


DESCRIPTION

This DTC is stored when a power seat motor operates (a position control sensor is being supplied with power) and the power supply voltage does not rise to the specified value.

DTC Code
DTC Detection Condition
Trouble Area
B2658
A problem with the voltage supplied to the position control sensor.
  1. Front power seat switch LH
  2. Separate type front seat cushion spring assembly LH
  3. Separate type front seatback spring assembly LH
  4. Harness or connector
