REAR CENTER SEAT OUTER BELT ASSEMBLY (for 60/40 Split Slide Walk-in Seat Type LH Side) > INSTALLATION

for Preparation Click here


CAUTION:
Wear protective gloves. Sharp areas on the parts may injure your hands.

HINT:
A bolt without a torque specification is shown in the standard bolt chart (Click here).



1. INSTALL NO. 1 SEAT 3 POINT TYPE BELT ASSEMBLY LH



NOTICE:
When installing the retractor, make sure the claws of the vehicle (labeled "a" in the illustration) only contact the installation areas of the retractor.

  1. Align the seat belt positioning holes with the claws and install the retractor of the seat belt with the nut and bolt as shown in the illustration.

    HINT:
    First install the bolt, and then install the nut.

    Torque:
    for bolt:
    7.5 N*m{ 76 kgf*cm , 66 in.*lbf }
    for nut:
    42 N*m{ 428 kgf*cm , 31 ft.*lbf }

    Text in Illustration 
    *1
    		Bolt
    *2
    		Nut



2. INSTALL REAR SEATBACK EDGE PROTECTOR

  1. Install the protector to the seatback frame.



3. INSTALL REAR SEAT SHOULDER BELT GUIDE

  1. Attach the guide and claw to install the belt guide.



4. INSTALL SEATBACK COVER WITH PAD



  1. Connect the outer belt to the seatback cover with pad.




  1. Attach the hooks.
    Text in Illustration 
    *1
    		Hog Ring Pliers
    *2
    		Hook
    *3
    		Hog Ring

  1. Using hog ring pliers, install the seatback cover with pad with new hog rings.

  1. Attach the 2 guides and connect the seatback cover bracket to the seatback frame with the nut.

    Torque:
    5.5 N*m{ 56 kgf*cm , 49 in.*lbf }

    NOTICE:
    After the seatback trim cover is assembled, make sure the side airbag strap is not twisted.



5. INSTALL REAR NO. 1 SEAT HEADREST SUPPORT ASSEMBLY

  1. Attach the 4 claws to install the 2 supports.



6. INSTALL REAR SEATBACK BOARD CARPET ASSEMBLY LH

  1. Attach the 7 clips to install the carpet.



7. INSTALL REAR SEATBACK COVER

  1. Attach the 4 claws to install the cover.



8. INSTALL REAR SEAT SHOULDER BELT COVER

  1. Connect the outer belt to the cover.




  1. Attach the 2 guides and 2 claws to install the cover.



9. INSTALL SEAT BELT ANCHOR COVER CAP



  1. Attach the 2 claws to install the cap.



10. INSTALL REAR SEAT LOCK CONTROL LEVER SUB-ASSEMBLY LH



  1. Connect the cable in the order shown in the illustration to install the lever.

  1. Install the 2 screws.



11. INSTALL RECLINING ADJUSTER KNOB CAP LH



  1. Attach the guide and claw to install the cap.



12. INSTALL REAR NO. 1 SEAT INNER BELT ASSEMBLY LH



  1. Install the inner belt and connect the 3 point type belt anchor with the bolt.

    Torque:
    42 N*m{ 428 kgf*cm , 31 ft.*lbf }

    NOTICE:
    Do not allow the anchor part of the anchor plate sub-assembly to overlap the protruding parts of the seatback frame.

    Text in Illustration 
    *a
    		Protruding Part



13. INSTALL CENTER SEATBACK ASSEMBLY

  1. Install the center seatback with the 2 bolts.

    Torque:
    21 N*m{ 214 kgf*cm , 15 ft.*lbf }



14. INSTALL SEAT CUSHION COVER WITH PAD



  1. Attach the hook.

  1. w/ Seat Heater System:

    1. Attach the 2 clamps to connect the seat heater wire harness.

    2. Connect the 2 seat heater connectors.




  1. Attach the hooks to install the seat cushion cover with pad.



15. INSTALL REAR SEATBACK LOCK STRIKER COVER LH

  1. Attach the 3 clips to install the cover.



16. INSTALL REAR SEAT CUSHION MOULDING LH



  1. Move the moulding in the direction of the arrow to attach the guide.

  1. Attach the 3 claws and clip to install the moulding.



17. INSTALL CENTER SEAT HINGE COVER RH



  1. Attach the 2 guides and install the cover with the clip.



18. INSTALL RECLINING ADJUSTER RELEASE HANDLE LH

  1. Install the handle with the 3 screws.



19. INSTALL REAR SEAT UPPER RECLINING COVER LH



  1. Attach the 2 claws to install the cover.



20. INSTALL REAR NO. 1 SEAT ASSEMBLY LH

  1. Install the rear No. 1 seat assembly LH (Click here).
