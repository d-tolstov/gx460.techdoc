1. INSTALL MULTIPLEX TILT AND TELESCOPIC ECU

  1. Install the ECU with the bolt.

    Torque:
    2.8 N*m{ 28 kgf*cm , 24 in.*lbf }

  1. Connect the connector.

    NOTICE:
    The connectors on both ends of the wire harness between the telescopic motor and ECU are the same shape. When installing them, connect the connectors of the same color.



2. INSTALL STEERING LOCK ACTUATOR ASSEMBLY

  1. Temporarily install the steering lock actuator assembly with 2 new tapered-head bolts.




  1. Tighten the 2 tapered-head bolt until the bolt heads breaks off.
