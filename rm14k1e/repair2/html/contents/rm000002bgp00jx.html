for Preparation Click here



1. INSTALL FRONT ACCELERATION SENSOR



  1. Install the front acceleration sensor with the bolt as shown in the illustration.

    Torque:
    8.5 N*m{ 87 kgf*cm , 75 in.*lbf }

    NOTICE:
    1. Replace the front acceleration sensor with a new one if it is dropped.
    2. Install the front acceleration sensor is oriented correctly.

  1. Connect the connector to the front acceleration sensor.

    NOTICE:
    Securely connect the connector.



2. INSTALL LOWER NO. 1 INSTRUMENT PANEL AIRBAG ASSEMBLY

  1. Install the lower No. 1 instrument panel airbag assembly (Click here).



3. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).
