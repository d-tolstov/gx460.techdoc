COMBINATION METER > INSTALLATION

for Preparation Click here



1. INSTALL COMBINATION METER ASSEMBLY

  1. Connect the connector.

  1. Attach the 2 clips.

  1. Install the combination meter assembly with the 2 screws.



2. INSTALL NO. 1 INSTRUMENT PANEL REGISTER ASSEMBLY

  1. Attach the 5 clips to install the No. 1 instrument panel register.



3. INSTALL INSTRUMENT CLUSTER FINISH PANEL SUB-ASSEMBLY

  1. Attach the guide.

  1. Attach the 4 claws.

  1. Attach the 2 clips to install the instrument cluster finish panel sub-assembly.

  1. Attach the 4 claws.



4. INSTALL NO. 1 INSTRUMENT PANEL FINISH CUSHION

  1. Connect the connector.

  1. Attach the 4 clips to install the No. 1 instrument panel finish cushion.



5. INSTALL NO. 2 SWITCH HOLE BASE

  1. Connect each connector.

  1. Attach the 5 clips and claw to install the No. 2 switch hole base.



6. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).



7. RESTORE AUTOAWAY/RETURN FUNCTION

  1. Restore the Autoaway/Return function setting to the previous condition by changing the customize parameter (Click here).
