WIDE VIEW FRONT MONITOR SYSTEM DESCRIPTION

  1. The wide view front monitor system is controlled by the parking assist ECU. This system is equipped with a front television camera assembly in the front radiator grille. Images from the camera are displayed on the display screen to inform the driver of conditions ahead, shorten the amount the vehicle has to enter an intersection, etc.


FUNCTION OF COMPONENTS

  1. The wide view front monitor system operates by controlling the components listed below.

    Components
    		Function
    Front television camera assembly
    		Located in the front radiator grille and sends images to the parking assist ECU.
    Parking assist ECU
    		Receives image signals and sends these signals to the display and navigation module display to display the view in front of the vehicle on the screen.
    1. Steering pad switch assembly
      1. Wide view front and side monitor switch
      2. Multi-function switch
    		When auto-display mode is not set and the vehicle speed is approximately 12 km/h (7 mph) or less, by pressing the wide view front and side monitor switch, the display changes from the displayed screen to the wide view front and side monitor simultaneous display screen.

    HINT:
    1. When the vehicle speed is approximately 12 km/h (7 mph) or less, auto-display mode is not set, the wide view front and side monitor switch is on and the shift lever is in R, only the wide view front and side monitor simultaneous display screen will be displayed. The display will not change to the wide view front monitor display screen or side monitor display screen.
    2. While the wide view front and side monitor display screen is displayed, if the DISP switch is pressed, the vehicle speed exceeds the display permission vehicle speed range (approximately 12 km/h (7 mph) or less), or the shift lever is moved to R and the screen changes to the rear view monitor, the display changes in the following order the next time the wide view front and side monitor simultaneous display screen is displayed: wide view front and side monitor simultaneous display screen → wide view front monitor display screen → side monitor display screen.
    3. While the side monitor display screen is displayed, if the DISP switch is pressed, the vehicle speed exceeds the display permission vehicle speed range (approximately 12 km/h (7 mph) or less), or the shift lever is moved to R and the screen changes to the rear view monitor, the display changes in the following order the next time the wide view front and side monitor simultaneous display screen is displayed: wide view front and side monitor simultaneous display screen → side monitor display screen → wide view front monitor display screen.

    Skid control ECU
    		Receives vehicle speed signals from the vehicle speed sensor and transmits these signals to the parking assist ECU using the CAN communication line.
    1. Display and navigation module display*1
    2. Accessory meter assembly*2
    		Receives front television camera assembly display signals from the parking assist ECU and displays the camera image on the display screen.

    1. *1: w/ Navigation system
    2. *2: w/o Navigation system


WIDE VIEW FRONT MONITOR SYSTEM CONTROL DESCRIPTION

  1. Wide view front monitor operation conditions (when auto-display mode is set) (w/ Navigation System).

    1. When either of the following conditions is met, the wide view front monitor display screen automatically changes to the wide view front and side monitor simultaneous display screen.

      Condition
      		Engine Switch
      		AUTO Switch
      		Vehicle Speed
      		Shift Lever
      1
      		On (IG)
      		On
      		Decreases to 10 km/h (6 mph) or less from a speed of 12 km/h (7 mph) or more
      		Moved from P to D or N
      2
      		On (IG)
      		On
      		12 km/h (7 mph) or less
      		Moved from R to D or N

      HINT:
      The AUTO switch is displayed in the bottom left when the wide view front monitor is displayed. Auto-display mode can be set/unset by turning the AUTO switch on/off.

  2. Wide view front monitor operation conditions (when auto-display mode is not set).

    1. When the following conditions are met and the front and side monitor main switch is pushed, the wide view front monitor display screen changes to the wide view front and side monitor simultaneous display screen.

      Engine Switch
      		AUTO Switch
      		Vehicle Speed
      On (IG)
      		Off
      		12 km/h (7 mph) or less
