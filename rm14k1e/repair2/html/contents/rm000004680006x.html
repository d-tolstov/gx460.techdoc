FRONT SEAT CUSHION HEATER > INSTALLATION

for Preparation Click here


CAUTION:
Wear protective gloves. Sharp areas on the parts may injure your hands.

HINT:
  1. Use the same procedure for the RH and LH sides.
  2. The procedure listed below is for the LH side.



1. INSTALL FRONT SEAT CUSHION HEATER ASSEMBLY LH



  1. Set the seat cushion heater in place with the name stamp side facing the seat cushion cover.
    Text in Illustration 
    *1
    		Tack Pin
    *2
    		Name Stamp

  1. Install the seat cushion heater with new tack pins.




  1. Connect the wire harness to the seat cushion cover.



2. INSTALL SEPARATE TYPE FRONT SEAT CUSHION COVER



  1. Using hog ring pliers, install the seat cushion cover to the seat cushion pad with new hog rings.
    Text in Illustration 
    *1
    		Hog Ring Pliers

    NOTICE:
    1. Be careful not to damage the cover.
    2. When installing the hog rings, avoid wrinkling the cover.



3. INSTALL SEAT CUSHION COVER WITH PAD

  1. Install the seat cushion cover with pad to the seat cushion spring.

  1. for Front Passenger Side:
    Attach the claw to connect the occupant detection sensor connector to the seat cushion spring.

  1. Connect the seat heater connector and attach the 3 wire harness clamps.

  1. Attach the hooks.

  1. Connect the airbag wire harness.

  1. Attach the fastening tape to close the cover.

  1. Connect the seat heater connector and attach the wire harness clamp.



4. INSTALL FRONT SEAT INNER BELT ASSEMBLY LH



  1. Install the front seat inner belt with the nut.

    Torque:
    42 N*m{ 428 kgf*cm , 31 ft.*lbf }

    NOTICE:
    Do not allow the anchor part of the front seat inner belt assembly to overlap the protruding parts of the front seat adjuster.

    Text in Illustration 
    *a
    		Protruding Part

  1. for Driver Side:
    Connect the 2 connectors and attach the 5 clamps.

  1. for Passenger Side:
    Connect the 2 connectors and attach the 4 clamps.



5. INSTALL FRONT SEAT INNER CUSHION SHIELD LH



  1. Attach the 3 claws to install the cushion shield.



6. INSTALL LUMBAR SWITCH ASSEMBLY



  1. Connect the lumbar switch connector.

  1. Attach the 4 claws to install the lumbar switch.



7. INSTALL FRONT POWER SEAT SWITCH LH

  1. for Driver Side:

    1. Connect the 2 connectors.

    2. Install the power seat switch with the 3 screws.

  1. for Front Passenger Side:

    1. Connect the connector.

    2. Install the power seat switch with the 3 screws.



8. INSTALL FRONT SEAT CUSHION SHIELD LH

  1. Attach the 3 wire harness clamps and connect the seat wire connector.

  1. Attach the 3 claws to close the wire harness cover.

  1. Connect the lifter motor connector.



  1. Remove the clip attached to the seat cushion spring and install it to the cushion shield.



  1. Attach the 2 clips, 3 claws and guide.

  1. Attach the guide of the front seat No. 1 inner cushion shield and install the screw.



9. INSTALL RECLINING POWER SEAT SWITCH KNOB



  1. Attach the 2 claws to install the knob.



10. INSTALL SLIDE AND VERTICAL POWER SEAT SWITCH KNOB



  1. Attach the 2 claws to install the knob.



11. INSTALL FRONT SEAT ASSEMBLY

  1. Install the front seat assembly (Click here).
