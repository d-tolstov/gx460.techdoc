FUEL PUMP > REASSEMBLY

for Preparation Click here


NOTICE:
Do not disconnect the hoses indicated in the illustration.




1. INSTALL FUEL MAIN VALVE ASSEMBLY



  1. Apply a light coat of gasoline to 2 new O-rings and install the O-rings to the fuel main valve.
    Text in Illustration 
    *1
    		New O-ring

  1. Install the fuel main valve to the fuel filter case.

    NOTICE:
    Make sure the O-rings are not cut or pinched during installation.



2. INSTALL FUEL PUMP

  1. Connect the fuel pump wire harness connector to the fuel pump.




  1. Apply a light coat of gasoline to a new O-ring. Then install the fuel pump spacer and O-ring to the fuel pump.
    Text in Illustration 
    *1
    		New O-ring
    *2
    		Fuel Pump Spacer




  1. Attach the 5 claws to the claw holes to install the fuel pump to the fuel filter case.
    Text in Illustration 
    *1
    		Fuel Filter Case



3. INSTALL NO. 1 FUEL SUB-TANK



  1. Attach the 3 claws to the claw holes to install the No. 1 fuel sub-tank.




  1. Attach the 3 claws to the claw holes.



  1. Install the tube part to the No. 1 fuel sub-tank.




    1. Attach the claw on the end of the tube to the claw hole. Connect the tube to the 2 clamps on the No. 1 fuel sub-tank.

  1. Connect the jet pump to the No. 1 fuel sub-tank.
    Text in Illustration 
    *1
    		Jet Pump

  1. Connect the fuel pump connector.



4. INSTALL FUEL SENDER GAUGE ASSEMBLY



  1. Set the fuel sender gauge on the No. 1 fuel sub-tank. Then slide the fuel sender gauge downward to install it.
    Text in Illustration 
    *1
    		No. 1 Fuel Sub-tank

    		Slide

  1. Connect the fuel sender gauge connector.
