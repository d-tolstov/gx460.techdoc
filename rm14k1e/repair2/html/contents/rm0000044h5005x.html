INITIALIZATION RELATED TO DYNAMIC RADAR CRUISE CONTROL SYSTEM

  1. Turn the engine switch on (IG).

  2. Turn the cruise control switch on.

  3. With the brake pedal depressed, push the cruise control switch to +RES 3 times within 3 seconds. Check that the skid control buzzer sounds at this time.

    NOTICE:
    1. Do not turn the light control switch on at this time as doing so causes the beam axis to deviate. If the light control switch is turned on by mistake, readjust the beam axis (Click here).
    2. If initialization cannot be performed, first inspect the stop light switch circuit, cruise control switch circuit and skid control buzzer circuit.

  4. Turn the engine switch off.
