REAR BRAKE > REASSEMBLY

for Preparation Click here



1. INSTALL PISTON SEAL



  1. Apply a light coat of lithium soap base glycol grease to a new piston seal.
    Text in Illustration 

    		Lithium soap base glycol grease

  1. Install the piston seal to the rear disc brake cylinder.



2. INSTALL CYLINDER BOOT



  1. Apply a light coat of lithium soap base glycol grease to the entire inner circumference of a new cylinder boot.
    Text in Illustration 

    		Lithium soap base glycol grease

  1. Apply a light coat of lithium soap base glycol grease to the entire outer circumference of the rear disc brake piston where it contacts the rear disc brake cylinder.

  1. Install the cylinder boot to the rear disc brake piston.

  1. Install the rear disc brake piston to the rear disc brake cylinder.

    NOTICE:
    Do not forcibly install the piston into the disc brake cylinder.



3. INSTALL REAR DISC BRAKE PISTON

  1. Install the cylinder boot to the rear disc brake cylinder.

    NOTICE:
    Install the boot securely to the groove of the disc brake cylinder.




  1. Push in the rear disc brake piston by hand and install the cylinder boot into the groove of the rear disc brake piston.

    NOTICE:
    Install the boot securely to the groove of the piston.



4. TEMPORARILY INSTALL REAR DISC BRAKE BLEEDER PLUG

  1. Temporarily install the bleeder plug to the rear disc brake cylinder.

    HINT:
    The bleeder plug will be tightened to the torque specification in the "Bleed Brake Line" procedure.
