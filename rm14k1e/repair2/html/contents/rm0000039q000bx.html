REAR BLOWER MOTOR > INSTALLATION

for Preparation Click here



1. INSTALL REAR BLOWER WITH FAN MOTOR SUB-ASSEMBLY

  1. Install the rear blower with fan motor with the 3 screws.



2. INSTALL HEATER BRACKET

  1. Install the bracket with the 3 screws and attach the clamp.

  1. Install the bolt.

  1. Connect the 2 connectors.



3. INSTALL REAR QUARTER TRIM PANEL ASSEMBLY RH

HINT:
Use the same procedure described for the LH side.



4. INSTALL DECK TRIM SIDE PANEL ASSEMBLY RH

  1. Connect each connector.

  1. Attach the 3 clips and 8 claws to install the deck trim side panel.

  1. Install the 2 bolts and screw.

  1. Install the rear No. 2 seat outer belt floor anchor with the bolt.

    Torque:
    42 N*m{ 428 kgf*cm , 31 ft.*lbf }

  1. Install the rear No. 1 seat outer belt floor anchor with the bolt.

    Torque:
    42 N*m{ 428 kgf*cm , 31 ft.*lbf }



5. INSTALL ASSIST GRIP SUB-ASSEMBLY (w/ Rear No. 2 Seat)

HINT:
Use the same procedure for the other assist grip.

  1. Install the assist grip with the 2 bolts.



6. INSTALL ASSIST GRIP PLUG (w/ Rear No. 2 Seat)

HINT:
Use the same procedure for all assist grip plugs.

  1. Attach the 2 claws to install the assist grip plug.



7. INSTALL FRONT DECK SIDE TRIM COVER (w/ Tonneau Cover)

HINT:
Use the same procedure for the other front deck side trim cover.

  1. Attach the 2 claws to install the front deck side trim cover



8. INSTALL NO. 1 TONNEAU COVER HOLDER CAP (w/o Tonneau Cover)

HINT:
Use the same procedure for the other No. 1 tonneau cover holder cap.

  1. Attach the 2 claws to install the No. 1 tonneau cover holder cap.



9. INSTALL NO. 1 LUGGAGE COMPARTMENT TRIM HOOK

HINT:
Use the same procedure for the other No. 1 luggage compartment trim hook.

  1. Insert the No. 1 luggage compartment trim hook with the hook oriented horizontally to install it.



10. INSTALL REAR NO. 1 SEAT LAP BELT OUTER ANCHOR COVER

HINT:
Use the same procedure for the other rear No. 1 seat outer lap belt anchor cover.

  1. Attach the 3 claws to install the rear No. 1 seat outer lap belt anchor cover.



11. INSTALL TONNEAU COVER ASSEMBLY (w/ Tonneau Cover)

  1. Install the tonneau cover.



12. INSTALL OUTER LAP BELT ANCHOR COVER

HINT:
Use the same procedure for the other outer lap belt anchor cover.

  1. Attach the 3 claws to install the outer lap belt anchor cover



13. INSTALL REAR DOOR OPENING TRIM WEATHERSTRIP RH

HINT:
Use the same procedure described for the LH side.



14. INSTALL REAR DOOR SCUFF PLATE RH

HINT:
Use the same procedure described for the LH side.



15. INSTALL QUARTER SCUFF PLATE RH (w/ Rear No. 2 Seat)

HINT:
Use the same procedure described for the LH side.



16. INSTALL REAR NO. 1 FLOOR STEP COVER (w/ Rear No. 2 Seat)

HINT:
Use the same procedure for all rear No. 1 floor step covers.

  1. Attach the 2 claws to install the rear No. 1 floor step cover.
