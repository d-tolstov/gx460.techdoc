DTC    P1613    Secondary Air Injection Driver Malfunction


DESCRIPTION

MONITOR DESCRIPTION

MONITOR STRATEGY

CONFIRMATION DRIVING PATTERN

WIRING DIAGRAM

INSPECTION PROCEDURE

CHECK AIR INJECTION CONTROL DRIVER (POWER SOURCE CIRCUIT)

CHECK HARNESS AND CONNECTOR (AIR INJECTION CONTROL DRIVER - ECM, BODY GROUND)

INSPECT AIR INJECTION CONTROL DRIVER (DI TERMINAL VOLTAGE)

PERFORM ACTIVE TEST USING INTELLIGENT TESTER

REPLACE AIR INJECTION CONTROL DRIVER

CHECK WHETHER DTC OUTPUT RECURS
