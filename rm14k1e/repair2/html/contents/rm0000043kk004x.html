DESCRIPTION

These DTCs are stored when a communication malfunction occurs between the sensors and ECUs that detect collisions, or between ECUs that perform pre-crash control.

DTC Code
DTC Detection Condition
Trouble Area
U0122
While the engine switch on (IG), communication stops between the skid control ECU and driving support ECU or seat belt control ECU for 1 second or more.
  1. Master cylinder solenoid (Skid control ECU)
  2. CAN communication system
  3. Seat belt control ECU
  4. Driving support ECU
U0123
While the engine switch on (IG), communication stops between the yaw rate sensor and driving support ECU for 1 second or more.
  1. Yaw rate sensor
  2. CAN communication system
  3. Driving support ECU
U0126
While the engine switch on (IG), communication stops between the steering angle sensor and driving support ECU for 1 second or more.
  1. Steering angle sensor
  2. CAN communication system
  3. Driving support ECU
U0140
While the engine switch on (IG), communication stops between the main body ECU and seat belt control ECU for 1 second or more.
  1. Main body ECU (Multiplex network body ECU)
  2. CAN communication system
  3. Seat belt control ECU
U0151
While the engine switch on (IG), communication stops between the center airbag sensor assembly and seat belt control ECU for 1 second or more.
  1. Center airbag sensor assembly
  2. CAN communication system
  3. Seat belt control ECU
U0235
While the engine switch on (IG), communication stops between the millimeter wave radar sensor assembly and driving support ECU for 1 second or more.
  1. Millimeter wave radar sensor assembly
  2. CAN communication system
  3. Driving support ECU
U1002
While the engine switch is on (IG), communication stops between the driving support ECU and millimeter wave radar sensor assembly for 1 second or more.
  1. Driving support ECU
  2. CAN communication system
  3. Millimeter wave radar sensor assembly
U1100
While the engine switch on (IG), communication stops between the seat belt control ECU and driving support ECU for 1 second or more.
  1. Seat belt control ECU
  2. CAN communication system
  3. Driving support ECU
U1104
While the engine switch on (IG), communication stops between the seat belt control ECU or millimeter wave radar sensor assembly and driving support ECU for 1 second or more.
  1. Millimeter wave radar sensor assembly
  2. Seat belt control ECU
  3. CAN communication system
  4. Driving support ECU
