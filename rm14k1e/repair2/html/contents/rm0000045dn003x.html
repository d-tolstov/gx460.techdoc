POWER SEAT CONTROL ECU (for Rear No. 2 Seat) > INSTALLATION

for Preparation Click here


NOTICE:
  1. If the fold seat control ECU is replaced with a new one, perform initialization (initial position memorization) of the fold seat control ECU (Click here).
  2. If the fold seat control ECU is exchanged with one from another vehicle, perform initialization (initial position reset and initial position memorization) of the fold seat control ECU (Click here).

HINT:
  1. Use the same procedure for the RH and LH sides.
  2. The procedure listed below is for the LH side.



1. INSTALL FOLD SEAT CONTROL ECU

  1. Install the ECU with the 2 nuts.

    Torque:
    8.0 N*m{ 82 kgf*cm , 71 in.*lbf }

  1. Connect the 2 connectors.

  1. Install the 3 clips.



2. INSTALL DECK BOARD ASSEMBLY

  1. Attach the 4 clips.

  1. Install the deck board with the 2 bolts.




  1. Attach the 2 claws to close the cover.

    HINT:
    Use the same procedure to close the cover on the other side.



3. PERFORM INITIALIZATION

NOTICE:
  1. If the fold seat control ECU is replaced with a new one, perform initialization (initial position memorization) of the fold seat control ECU (Click here).
  2. If the fold seat control ECU is exchanged with one from another vehicle, perform initialization (initial position reset and initial position memorization) of the fold seat control ECU (Click here).



4. CHECK REAR NO. 2 SEAT

  1. Check the power seat operation (Click here).
