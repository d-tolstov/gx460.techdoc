REAR SEATBACK HEATER (for 60/40 Split Slide Walk-in Seat Type RH Side) > INSTALLATION

for Preparation Click here


CAUTION:
Wear protective gloves. Sharp areas on the parts may injure your hands.



1. INSTALL REAR SEATBACK HEATER ASSEMBLY



  1. Set the seatback heater in place with the name stamp side facing the seatback cover.
    Text in Illustration 
    *1
    		Tack Pin
    *2
    		Name Stamp

  1. Install the seatback heater with new tack pins.



2. INSTALL SEPARATE TYPE REAR SEATBACK COVER



  1. Using hog ring pliers, install the seatback cover to the seatback pad with new hog rings.
    Text in Illustration 
    *1
    		Hog Ring Pliers

    NOTICE:
    1. Be careful not to damage the cover.
    2. When installing the hog rings, avoid wrinkling the cover.




  1. Connect the seatback cover bracket to the seatback pad.




  1. w/ Seat Heater System:
    Connect the seatback heater wire harness to the seatback pad.



3. INSTALL SEATBACK COVER WITH PAD



  1. Attach the hooks.
    Text in Illustration 
    *1
    		Hog Ring Pliers
    *2
    		Hook
    *3
    		Hog Ring

  1. Using hog ring pliers, install the seatback cover with pad with new hog rings.

  1. Attach the 2 guides and connect the seatback cover bracket to the seatback frame with the nut.

    Torque:
    5.5 N*m{ 56 kgf*cm , 49 in.*lbf }

    NOTICE:
    After the seatback trim cover is assembled, make sure the side airbag strap is not twisted.



4. INSTALL REAR NO. 1 SEAT HEADREST SUPPORT ASSEMBLY

  1. Attach the 4 claws to install the 2 supports.



5. INSTALL REAR SEATBACK BOARD CARPET ASSEMBLY RH

  1. Attach the 7 clips to install the carpet.



6. INSTALL REAR SEATBACK COVER

  1. Attach the 4 claws to install the cover.



7. INSTALL REAR CENTER BACK LOCK CONTROL LEVER



  1. Attach the cable clamp to connect the cable to the lever.

  1. Move the lever in the direction of the arrow to attach the guide.

  1. Install the lever with the 2 screws.



8. INSTALL REAR SEAT COVER CAP

  1. Attach the guide and claw to install the cap.



9. INSTALL REAR SEAT LOCK CONTROL LEVER SUB-ASSEMBLY RH



  1. Connect the cable in the order shown in the illustration to install the lever.

  1. Install the 2 screws.



10. INSTALL RECLINING ADJUSTER KNOB CAP RH



  1. Attach the guide and claw to install the cap.



11. INSTALL SEAT CUSHION COVER WITH PAD

  1. Install the seat cushion cover with pad to the seat cushion frame.

  1. w/ Seat Heater System:

    1. Attach the 2 clamps to connect the seat heater wire harness.

    2. Connect the 2 seat heater connectors.




  1. Attach the hooks to install the seat cushion cover with pad.



12. INSTALL REAR SEAT REAR CUSHION MOULDING LH

  1. Attach the guide and 3 claws to install the moulding.

  1. Install the screw.



13. INSTALL REAR SEAT CUSHION MOULDING RH



  1. Move the moulding in the direction of the arrow to attach the guide and install the moulding.

  1. Attach the 3 claws and clip to install the moulding.



14. INSTALL RECLINING ADJUSTER RELEASE HANDLE RH

  1. Install the handle with the 3 screws.



15. INSTALL REAR SEAT UPPER RECLINING COVER RH



  1. Attach the 2 claws to install the cover.



16. INSTALL REAR NO. 1 SEAT ASSEMBLY RH

  1. Install the rear No. 1 seat assembly (Click here).
