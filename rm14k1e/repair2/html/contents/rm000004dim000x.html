1. CUSTOMIZE FRONT POWER SEAT CONTROL SYSTEM

HINT:
The following items can be customized.

NOTICE:
  1. When the customer requests a change in a function, first make sure that the function can be customized.
  2. Record the current settings before customizing.

  1. Customizing with the intelligent tester

    1. Connect the intelligent tester to the DLC3.

    2. Turn the engine switch on (IG).

    3. Turn the intelligent tester on.

    4. Enter the following menus: Utility / Customize / Seat or Smart & Entry.

    5. Select the setting by referring to the table below.

      Tester Display
      		Default
      		Content
      		Setting
      		Relevant ECU
      Seat Move Distance
      		Standard
      		Allows customization of the seat movement distance for the power easy access system.
      		OFF, Little or Standard
      		Front power seat switch LH (seat ECU)
      Seat Position / Tilt & Telesco memory recall setting
      		D Door
      		This customize function switches the condition of the seat position, mirror position and tilt and telescopic memory recall. Possible recall activation parameters are "driver door unlocked with smart entry and start system and opened" or "any door unlocked with smart entry and start system and opened".
      		D Door or ANY Door
      		Main body ECU

  1. Customizing with Multi-information Display

    1. Turn the engine switch on (IG).

    2. With the vehicle stationary or at a vehicle speed of less than 8 km/h (4.9 mph), press and hold the menu switch on the steering pad switch to display the meter customize mode screen.

    3. Select the setting by referring to the table below.

      Tester Display
      		Default
      		Content
      		Setting
      		Relevant ECU
      DRIVER SEAT EASY-EXIT
      		FULL
      		Allows customization of the seat movement distance for the power easy access system.
      		OFF, PARTIAL or FULL
      		Front power seat switch LH (seat ECU)
