DTC    B15B0    Display Screen Malfunction

for Preparation Click here


DESCRIPTION

This DTC is stored when a malfunction occurs in the display and navigation module display or multi-display controller sub-assembly.

HINT:
  1. The trouble area cannot be determined by checking DTCs with the intelligent tester. It is necessary to display the "System Check Mode" screen to determine the trouble area.
  2. If the picture circuit is faulty, the display and navigation module display stores the DTC.
  3. If the picture circuit between the multi-display controller sub-assembly and the television display assembly LH or television display assembly RH is faulty, the multi-display controller sub-assembly stores this DTC.


DTC Code
DTC Detection Condition
Trouble Area
B15B0
Picture circuit (TFT unit) malfunction.
  1. Display and navigation module display
  2. Television display assembly LH
  3. Television display assembly RH
  4. Multi-display controller sub-assembly
