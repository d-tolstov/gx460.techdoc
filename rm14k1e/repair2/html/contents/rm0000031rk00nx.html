for Preparation Click here



1. INSTALL FRONT CRANKSHAFT OIL SEAL

  1. Apply MP grease to the lip of a new front crankshaft oil seal.

    NOTICE:
    1. Do not allow foreign matter to contact the lip of the oil seal.
    2. Do not allow MP grease to contact the dust seal.




  1. Using SST and a hammer, tap in the front crankshaft oil seal to a depth between 0 and 1.0 mm (0 to 0.0394 in.) from the timing chain cover edge.

    SST
    09223-22010  
    09506-35010  

    NOTICE:
    1. Keep the lip free from foreign matter.
    2. Do not tap the front crankshaft oil seal at an angle.



2. INSTALL CRANKSHAFT TIMING GEAR KEY

  1. Install the crankshaft timing gear key to the crankshaft.



3. INSTALL CRANKSHAFT PULLEY



  1. Align the key groove of the pulley with the pulley set key, and slide on the pulley to install it.

  1. Using SST, install the pulley set bolt.

    SST
    09213-70011  
    09330-00021  

    Torque:
    300 N*m{ 3059 kgf*cm , 221 ft.*lbf }

    Text in Illustration 
    *a
    		Hold

    		Turn
