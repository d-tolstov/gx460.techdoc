GLASS HATCH OPENER SYSTEM DESCRIPTION

  1. An unlock request signal from the glass hatch opener switch is sent to the main body ECU (multiplex network body ECU). The main body ECU (multiplex network body ECU) receives the signal and unlocks the back window lock.


FUNCTION OF MAIN COMPONENTS

Component
Function
Glass hatch opener switch assembly
Sends an unlock request signal to the main body ECU (multiplex network body ECU) when operated.
Back window lock assembly
Receives the unlock request signal from the main body ECU (multiplex network body ECU) and unlocks the back door window lock.
Main body ECU (Multiplex network body ECU)
Receives the unlock request signal from the glass hatch opener switch and sends the signal to the back window lock.
