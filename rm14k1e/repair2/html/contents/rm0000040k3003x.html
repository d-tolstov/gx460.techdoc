1. INSTALL NO. 2 EGR PIPE SUB-ASSEMBLY



  1. Install a new gasket and the No. 2 EGR pipe with the 2 bolts.

    Torque:
    10 N*m{ 102 kgf*cm , 7 ft.*lbf }

    Text in Illustration 
    *1
    		No. 2 EGR Pipe
    *2
    		Claw
    *3
    		EGR Cooler

    NOTICE:
    Make sure the claws of the gasket are not caught between the No. 2 EGR pipe and EGR cooler.



2. INSTALL EGR COOLER ASSEMBLY



  1. Install the EGR cooler with the 2 bolts.

    Torque:
    21 N*m{ 214 kgf*cm , 15 ft.*lbf }

    Text in Illustration 
    *1
    		No. 2 EGR Pipe Sub-assembly
    *2
    		Pin

    HINT:
    Tighten the bolts with the pin of the No. 2 EGR pipe sub-assembly inserted into the positioning hole of the cylinder block.



3. INSTALL NO. 1 EGR PIPE SUB-ASSEMBLY

  1. Install 2 new gaskets.




  1. Install the EGR pipe with 4 new nuts and tighten the nuts in the sequence shown in the illustration.

    Torque:
    21 N*m{ 214 kgf*cm , 15 ft.*lbf }
