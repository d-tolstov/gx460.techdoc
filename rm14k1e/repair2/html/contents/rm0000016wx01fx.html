FRONT DRIVE SHAFT ASSEMBLY > REASSEMBLY

for Preparation Click here



1. INSTALL FRONT AXLE HUB DUST SEAL



  1. Using a brass bar and hammer, install a new dust seal.

    NOTICE:
    Do not damage the dust seal.



2. INSTALL FRONT DRIVE SHAFT DUST COVER



  1. Using SST and a press, install a new dust cover.

    SST
    09527-10011  



3. INSTALL SHAFT SNAP RING



  1. Install a new shaft snap ring.



4. INSTALL OUTBOARD JOINT BOOT



HINT:
Before installing the boot, wrap vinyl tape around the spline of the shaft to prevent damage to the boot.

Text in Illustration 
*1
Vinyl Tape

  1. Temporarily install a new outboard joint boot with 2 new clamps to the outboard joint.

  1. Pack the outboard joint and boot with grease from the boot kit.

    Standard grease capacity:
    266 to 276 g (9.4 to 9.7 oz.)



5. INSTALL FRONT NO. 2 AXLE OUTBOARD JOINT BOOT CLAMP

  1. Hold the drive shaft lightly in a vise between aluminum plates.

    NOTICE:
    Do not overtighten the vise.




  1. Place SST onto the front No. 2 axle outboard joint boot clamp.

    SST
    09521-24010  

  1. Tighten SST so that the No. 2 front axle outboard joint boot clamp is pinched.
    Text in Illustration 
    *a
    		Hold
    *b
    		Turn

    NOTICE:
    Do not overtighten SST.




  1. Using SST, measure the clearance of the No. 2 front axle out board joint boot clamp.

    SST
    09240-00020  

    Standard clearance:
    2.1 mm (0.0827 in.) or less

    NOTICE:
    If the measured value is more than the specified value, retighten the clamp.



6. INSTALL FRONT NO. 1 AXLE OUTBOARD JOINT BOOT CLAMP

  1. Hold the drive shaft lightly in a vise between aluminum plates.

    NOTICE:
    Do not overtighten the vise.

  1. Secure the outboard joint boot clamp to the boot.




  1. Place SST onto the outboard joint boot clamp.

    SST
    09521-24010  

  1. Tighten SST so that the front axle outboard joint boot clamp is pinched.

7. INSTALL INBOARD JOINT BOOT

  1. Temporarily install a new inboard joint boot to the outboard joint.



8. INSTALL FRONT AXLE INBOARD JOINT ASSEMBLY

  1. Install new parts to the outboard joint in the following order.
    1.
    		Front axle inboard joint boot clamp
    2.
    		Front axle inboard joint boot
    3.
    		Front No. 2 axle inboard joint boot clamp

  1. Secure the outboard joint shaft in a vise between aluminum plates.

    NOTICE:
    Do not overtighten the vise.




  1. Align the matchmarks and install the tripod joint to the outboard joint.
    Text in Illustration 
    *1
    		Matchmark
    *a
    		Beveled Side

    NOTICE:
    Position tripod so that the beveled side of the tripod faces toward the inboard joint.

  1. Align the matchmarks and install the tripod to the inboard joint.

  1. Using a brass bar and hammer, tap the tripod onto the outboard joint.

    NOTICE:
    1. Do not hit the roller portion.
    2. Keep the tripod joint free of foreign matter.




  1. Using a snap ring expander, install a new snap ring.

  1. Pack the inboard joint assembly and boot with grease from the boot kit.

    Standard grease capacity:
    239 to 249 g (8.4 to 8.8 oz.)




  1. Align the matchmarks and install the inboard joint onto the outboard joint.
    Text in Illustration 
    *1
    		Matchmark
