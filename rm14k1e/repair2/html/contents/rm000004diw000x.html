1. CUSTOMIZE LIGHTING SYSTEM (INT)

HINT:
The following items can be customized.

NOTICE:
  1. When the customer requests a change in a function, first make sure that the function can be customized.
  2. Record the current settings before customizing.

  1. Customizing with the intelligent tester

    1. Connect the intelligent tester to the DLC3.

    2. Turn the engine switch on (IG).

    3. Turn the intelligent tester on.

    4. Enter the following menus: Utility / Customize / Illuminated Entry.

    5. Select the setting by referring to the table below.

      Display
      		Default
      		Content
      		Setting
      		Relevant ECU
      Lighting Time
      		15 s
      		Changes the lighting time after closing all of the doors (the lights will fade out immediately when turning the engine switch from off to on (ACC) or (IG)).
      		7.5 s, 15 s or 30 s
      		Main body ECU (Multiplex network body ECU)
      I/L when ACC OFF
      		ON
      		This function turns on the lights when the engine switch is turned from on (IG) or (ACC) to off (the room light turns on when the interior light switch is set to DOOR).
      		ON or OFF
      		Main body ECU (Multiplex network body ECU)
      I/L ON W/Door Key Unlock
      		ON
      		This function turns on the room light when the door is unlocked with a transmitter (the room light turns on when the map light switch is set to DOOR).
      		ON or OFF
      		Main body ECU (Multiplex network body ECU)
      Room Light when Aprchd
      		ON
      		This function turns on the room light when the key is brought within the vehicle exterior detection area (the room light turns on when the map light switch is set to DOOR).
      		ON or OFF
      		Main body ECU (Multiplex network body ECU)
      Inside Foot Light
      		ON
      		This function controls the interior foot lights.
      		ON or OFF
      		Main body ECU (Multiplex network body ECU)
      Center Console Spot Light
      		ON
      		Lights up the center console spot light when the engine switch is turned to on (ACC) or (IG).
      		ON or OFF
      		Main body ECU (Multiplex network body ECU)
      Mirror Foot Light Time
      		15 s
      		Side step light illumination duration.
      		7.5 s, 15 s or 30 s
      		Main body ECU (Multiplex network body ECU)
      Light Control
      		ON
      		This function controls the interior foot lights and inside handle illumination.
      		ON or OFF
      		Main body ECU (Multiplex network body ECU)
      Interior Light Control
      		ON
      		This function controls the interior lights.
      		ON or OFF
      		Main body ECU (Multiplex network body ECU)
      Exterior Light Control
      		ON
      		This function controls the side step lights.
      		ON or OFF
      		Main body ECU (Multiplex network body ECU)
      Scuff Plate Lamp linked to Smart
      		ON
      		This function turns on the side step light when the key is brought within the vehicle exterior detection area.
      		ON or OFF
      		Main body ECU (Multiplex network body ECU)
      Scuff Plate Lamp linked to Wireless Key
      		ON
      		This function turns on the side step light when the door is unlocked with the wireless key.
      		ON or OFF
      		Main body ECU (Multiplex network body ECU)
      Scuff Plate Lamp linked to D-Door Unlock SW
      		ON
      		This function turns on the side step light when the door is unlocked with the manual unlock switch.
      		ON or OFF
      		Main body ECU (Multiplex network body ECU)
      Scuff Plate Lamp linked to Door open
      		ON
      		This function turns on the side step light when the driver door is opened.
      		ON or OFF
      		Main body ECU (Multiplex network body ECU)

  1. Customizing with Multi-information display

    1. Turn the engine switch on (IG).

    2. With the vehicle stationary or at a vehicle speed of less than 8 km/h (4.9 mph), press and hold the menu switch on the steering pad switch to display the meter customize mode screen.

    3. Select the setting by referring to the table below.

      Display
      		Default
      		Content
      		Setting
      		Relevant ECU
      INTERIOR LIGHTS OFF TIME ADJUSTMENT
      		15 S
      		Changes the lighting time after closing all of the doors (the lights will fade out immediately when turning the engine switch from off to on (ACC) or (IG)).
      		7.5 S, 15 S or 30 S
      		Main body ECU (Multiplex network body ECU)
      EXT. LIGHTS OFF TIME ADJUSTMENT
      		15 S
      		Side step light illumination duration
      		OFF, 7.5 S, 15 S or 30 S
      		Main body ECU (Multiplex network body ECU)
