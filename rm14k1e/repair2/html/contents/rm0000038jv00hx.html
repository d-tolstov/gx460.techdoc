FRONT BUMPER > INSTALLATION

for Preparation Click here


HINT:
A bolt without a torque specification is shown in the standard bolt chart (Click here).



1. INSTALL FRONT BUMPER BRACKET SUB-ASSEMBLY LH

  1. Install the front bumper bracket with the 2 bolts.



2. INSTALL FRONT BUMPER BRACKET SUB-ASSEMBLY RH

HINT:
Use the same procedure described for the LH side.



3. INSTALL UPPER CENTER FRONT BUMPER RETAINER

  1. Install the upper center front bumper retainer with the 3 bolts.

    Torque:
    8.0 N*m{ 82 kgf*cm , 71 in.*lbf }



4. INSTALL FRONT BUMPER SIDE SUPPORT LH

  1. Attach the 2 claws to install the front bumper side support.

  1. Install the screw.



5. INSTALL FRONT BUMPER SIDE SUPPORT RH

HINT:
Use the same procedure described for the LH side.



6. INSTALL HEADLIGHT ASSEMBLY LH

  1. Connect the connectors.

  1. Attach the clamp.

  1. Install the headlight with the bolt, 2 screws and clip.



7. INSTALL HEADLIGHT ASSEMBLY RH

HINT:
Use the same procedure described for the LH side.



8. INSTALL FRONT BUMPER BAR REINFORCEMENT LH

  1. Install the front bumper bar reinforcement with the 2 screws.



9. INSTALL FRONT BUMPER BAR REINFORCEMENT RH

HINT:
Use the same procedure described for the LH side.



10. INSTALL NO. 2 FRONT BUMPER EXTENSION SUB-ASSEMBLY LH

  1. Install the No. 2 front bumper extension with the 4 bolts.

    Torque:
    65 N*m{ 663 kgf*cm , 48 ft.*lbf }



11. INSTALL NO. 2 FRONT BUMPER EXTENSION SUB-ASSEMBLY RH

HINT:
Use the same procedure described for the LH side.



12. INSTALL FRONT BUMPER REINFORCEMENT SUB-ASSEMBLY

  1. Install the front bumper reinforcement with the 6 nuts.

    Torque:
    30 N*m{ 306 kgf*cm , 22 ft.*lbf }



13. INSTALL LOWER FRONT BUMPER COVER

  1. Install the lower front bumper cover with the 5 bolts and clip.



14. INSTALL FRONT BUMPER ENERGY ABSORBER

  1. Install the front bumper energy absorber.



15. INSTALL FRONT BUMPER COVER

  1. w/ LEXUS Parking Assist-sensor System:
    Connect the connector.

  1. Connect the headlight cleaner hose and 2 fog light connectors.

  1. Attach the 14 claws to install the front bumper cover.

  1. Install the 4 screws.




  1. Install the 2 bolts labeled A, 2 bolts labeled B, 2 screws and 4 clips.

    Torque:
    for bolt A:
    8.0 N*m{ 82 kgf*cm , 71 in.*lbf }

    Text in Illustration 
    *1
    		Protective Tape
    *2
    		Bolt A
    *3
    		Bolt B



16. INSTALL RADIATOR GRILLE

  1. Attach the 6 claws to install the radiator grille.

  1. Install the 2 bolts and 2 screws.



17. INSTALL UPPER RADIATOR SUPPORT SEAL

  1. Install the upper radiator support seal with the 13 clips.



18. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (Click here).



19. ADJUST HEADLIGHT AIMING

  1. Adjust the headlight aiming (Click here).



20. ADJUST FOG LIGHT AIMING

  1. Adjust the fog light aiming (Click here).
