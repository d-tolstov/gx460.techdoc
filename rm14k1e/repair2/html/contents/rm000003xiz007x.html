1. INSPECT HEIGHT CONTROL INDICATOR



  1. Turn the engine switch on (IG) and check the height control indicator on the combination meter.

    OK:
    After the height control indicator (shown in the illustration) is displayed for approximately 2 seconds, the actual vehicle height is indicated by the height control mode indicator.



2. INSPECT VEHICLE HEIGHT SELECTION

HINT:
Perform this inspection with the vehicle unloaded.

  1. Start the engine.

  1. Using the height control switch, check the vehicle height adjustment amount and vehicle height adjustment time for each vehicle height mode.

    HINT:
    The select mode light of the height control indicator blinks during vehicle height adjustment and remains on after the vehicle height adjustment is completed.


    Standard:

    Mode Transition
    		Operation Details
    		Vehicle Height Adjustment Time
    N mode to HI mode
    		From when "up" on height control switch pressed and control begins to when vehicle height is adjusted to HI mode
    		Approximately 20 seconds
    HI mode to N mode
    		From when "down" on height control switch pressed and control begins to when vehicle height is adjusted to N mode
    		Approximately 20 seconds
    N mode to LO mode
    		From when "down" on height control switch pressed and control begins to when vehicle height is adjusted to LO mode
    		Approximately 20 seconds
    LO mode to N mode
    		From when "up" on height control switch pressed and control begins to when vehicle height is adjusted to N mode
    		Approximately 20 seconds



3. INSPECT AUTO-LEVELING FUNCTION

  1. Unload the vehicle.

  1. Start the engine.

  1. Select N mode.

  1. Check that the N mode light blinks and then remains on.

    HINT:
    The selected mode light on the height control indicator blinks while the vehicle height is changing and remains on when the target height is obtained.

  1. Turn the engine switch off and wait for 60 seconds or more.

    HINT:
    The auto-leveling function is only able to lower the vehicle for 60 seconds after the engine switch is turned off.

  1. Load the vehicle with 4 people (2 in the front seats and 2 in the rear seats).

    HINT:
    Each person should weigh approximately 55 kg (121.3 lbs).

  1. Start the engine and inspect the vehicle height adjustment time.

    Standard time:
    Approximately 20 seconds

  1. With the engine running, wait for 60 seconds to discharge compressed air from the pneumatic cylinder assembly.

  1. Turn the engine switch off and wait for 60 seconds or more.

    HINT:
    The auto-leveling function is only able to lower the vehicle for 60 seconds after the engine switch is turned off.

  1. Unload the vehicle.

  1. Start the engine and inspect the vehicle height adjustment time.

    Standard time:
    Approximately 20 seconds
