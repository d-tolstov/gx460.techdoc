FRONT UPPER SUSPENSION ARM > INSTALLATION

for Preparation Click here


HINT:
  1. Use the same procedure for the RH and LH sides.
  2. The procedure listed below is for the LH side.
  3. A bolt without a torque specification is shown in the standard bolt chart (Click here).



1. TEMPORARILY INSTALL FRONT SUSPENSION UPPER ARM ASSEMBLY LH



  1. Temporarily install the suspension upper arm and 2 washers with the bolt and nut.

    HINT:
    After stabilizing the suspension, tighten the nut.




  1. Connect the bracket with the bolt.

    Torque:
    8.0 N*m{ 82 kgf*cm , 71 in.*lbf }




  1. Install a new nut and clip.

    Torque:
    110 N*m{ 1122 kgf*cm , 81 ft.*lbf }



2. INSTALL SKID CONTROL SENSOR WIRE

  1. Connect the connector as follows.

    1. for LH:
      1. Attach the connector, and then connect the connector.

      NOTICE:
      Securely connect the connector.

    2. for RH:
      1. Install the skid control sensor clamp with the bolt.

      Torque:
      5.0 N*m{ 51 kgf*cm , 44 in.*lbf }

      NOTICE:
      Make sure the clamp rotation stopper touches the installation position.

      2. Attach the connector, and then connect the connector.

      NOTICE:
      1. Securely connect the connector.
      2. When connecting the connector, do not twist the wire harness.

      3. Attach the clip.

  1. Install the 2 harness clamps with the 2 bolts.

    Torque:
    13 N*m{ 127 kgf*cm , 9 ft.*lbf }

    NOTICE:
    1. When installing the clamps, do not twist the wire harness.
    2. Make sure the clamp rotation stopper touches the installation position.

  1. Install the 2 harness clamps with the 2 bolts.

    Torque:
    13 N*m{ 127 kgf*cm , 9 ft.*lbf }

    NOTICE:
    1. When installing the clamps, do not twist the wire harness.
    2. Make sure the clamp rotation stopper touches the installation position.

  1. Attach the clip.

  1. Connect the connector.

    NOTICE:
    Securely connect the connector.



3. INSTALL FRONT WHEEL

Torque:
112 N*m{ 1142 kgf*cm , 83 ft.*lbf }



4. STABILIZE SUSPENSION

  1. Lower the vehicle.

  1. Bounce the vehicle up and down several times to stabilize the suspension.



5. TIGHTEN FRONT SUSPENSION UPPER ARM ASSEMBLY LH

  1. Tighten the nut.

    Torque:
    115 N*m{ 1173 kgf*cm , 85 ft.*lbf }



6. INSPECT AND ADJUST FRONT WHEEL ALIGNMENT

  1. Inspect and adjust the front wheel alignment (Click here).
