DTC    B15EE    Telematics Contract Invalid

for Preparation Click here


DESCRIPTION

If an unauthorized contract flag is stored by the G-BOOK device, this DTC will be stored.

DTC Code
DTC Detection Condition
Trouble Area
B15EE
The contract is invalid.
-

INSPECTION PROCEDURE

1.CHECK CONTRACT FLAG



  1. Check the contract flag.

    1. Enter diagnostic mode (Click here).

    2. Select "Service Information" on the Service Menu screen.




    3. Select "G-BOOK Information for Contract" on the Service Information screen.




    4. Check if the flag information shown in the illustration indicates "1".
      Result 
      Result
      		Proceed to
      "1" not indicated
      		A
      "1" indicated
      		B

2.PERFORM OPERATION



  1. Perform "Download necessary data".

    1. Press the "INFO/TEL" switch on the display and navigation module display.

    2. Perform the operations shown in the illustration and select "Download necessary data".

      HINT:
      The screen illustrations provided are samples only and may be different from the actual screens.

  1. Perform a manual maintenance check

4.CONTACT G-BOOK SUPPORT CENTER

  1. Contact the G-BOOK support center.

      2. The manual maintenance check did not complete normally.
