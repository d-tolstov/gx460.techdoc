1. CUSTOMIZE GLASS HATCH OPENER SYSTEM

HINT:
The following items can be customized.

NOTICE:
  1. When the customer requests a change in a function, first make sure that the function can be customized.
  2. Record the current settings before customizing.

  1. Customizing with the intelligent tester

    1. Connect the intelligent tester to the DLC3.

    2. Turn the engine switch on (IG).

    3. Turn the intelligent tester on.

    4. Enter the following menus: Utility / Customize / Wireless Door Lock or Entry & Start .

    5. Select the setting by referring to the table below.

      Display
      		Default
      		Content
      		Setting
      		Relevant ECU
      Wireless Glass Hatch Open
      		Long1
      		This function changes the operation method of opening the glass hatch by the transmitter.
      		1tim ON, 2tim ON, Long1, Long2 or Prohibit
      		Main body ECU (Multiplex network body ECU)
      Glass Hatch Open Operation
      		ON
      		Function to permit opening the glass hatch with the electronic key.
      		ON or OFF
      		Certification ECU
