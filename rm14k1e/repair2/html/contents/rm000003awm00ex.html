1. INSTALL HOLDER ANTENNA ASSEMBLY (for 5 Door)


  1. Install the holder antenna with the bolt as shown in the part of the illustration labeled A.



    Torque:
    7.0 N*m{ 71 kgf*cm , 62 in.*lbf }

  1. Attach the clamp and install the holder antenna with the bolt as shown in the part of the illustration labeled B.

    Torque:
    7.5 N*m{ 76 kgf*cm , 66 in.*lbf }

  1. Connect the 3 connectors.



2. INSTALL ROOF HEADLINING ASSEMBLY

  1. Install the roof headlining assembly (Click here).



3. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

NOTICE:
