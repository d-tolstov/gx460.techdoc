NOTICE:
  1. After replacing the master cylinder solenoid and/or yaw rate and acceleration sensor, perform calibration.
  2. VSC is prohibited during test mode.

HINT:
  1. By switching the skid control ECU from normal mode to test mode, abnormality detection sensitivity is enhanced and troubleshooting can be conducted efficiently.
  2. Perform a sensor check in test mode after the speed sensor or sensor rotor has been repaired or replaced.
  3. If the engine switch is turned from on (IG) to on (ACC) or off during test mode, DTCs related to the signal check function are not cleared. During test mode, do not change to normal mode.
  4. During test mode, the skid control ECU stores all DTCs related to the signal check function, and the DTCs are cleared if normality is confirmed. Any remaining DTCs indicate abnormalities that were found.


TEST MODE (SIGNAL CHECK) PROCEDURE

  1. When using the intelligent tester:

    1. Place the vehicle on a level surface with an inclination of less than 1%.

    2. Turn the engine switch off.

    3. Check that the shift lever is in P.

      CAUTION:
      Apply the parking brake for safety.

    4. Connect the intelligent tester to the DLC3.

    5. Turn the engine switch on (IG).

    6. Turn the intelligent tester on.

    7. Enter the following menus: Chassis / ABS/VSC/TRC / Utility / Signal Check.

    8. Check that the ABS warning light and slip indicator light blink at 0.125 second intervals (0.125 seconds on and 0.125 seconds off).

    9. Start the engine.



  2. When using SST check wire:

    1. Place the vehicle on a level surface with an inclination of less than 1%.

    2. Turn the engine switch off.

    3. Check that the shift lever is in P.

      CAUTION:
      Apply the parking brake for safety.




    4. Using SST, connect terminals 12 (TS) and 4 (CG) of the DLC3.

      SST
      09843-18040

      *a
      		Front view of DLC3

    5. Turn the engine switch on (IG).

    6. Check that the ABS warning light and slip indicator light blink at 0.125 second intervals (0.125 seconds on and 0.125 seconds off).

    7. Start the engine.

  3. ABS Signal Check

    HINT:
    Check that the ABS warning light is blinking in the test mode blinking pattern (0.125 seconds on and 0.125 seconds off) before performing the ABS signal check.

  4. Master Cylinder Pressure Sensor and Acceleration Sensor Check

    1. Keep the vehicle stationary on a level surface for 1 second or more.

    2. With the vehicle stationary, release the brake pedal for 1 second or more, and then quickly depress the brake pedal with a force of 98 N (10 kgf, 22 lbf) or more for 1 second.

    3. Check that the ABS warning light comes on for 3 seconds.

      HINT:
      During test mode, the ABS warning light turns on for 3 seconds every time the above operation is performed.

  5. Center Differential Lock Detection Switch Check

    1. Press the center differential lock switch to lock the center differential.

      HINT:
      Move the vehicle either forward or backward a little to engage the center differential lock.

    2. Press the center differential lock switch to unlock the center differential.

  6. Speed Sensor Check

    NOTICE:
    1. Before performing the speed sensor signal check, complete the master cylinder pressure sensor check.
    2. The signal check may not be completed if the wheels of the vehicle spin or the steering wheel is turned during this check.


    1. Drive the vehicle straight forward at a speed of 45 km/h (28 mph) or more for several seconds (forward signal check).

    2. Check that the ABS warning light goes off.

    3. Drive the vehicle in reverse for more than 1 second at 3 km/h (2 mph) or more (reverse signal check).

    4. Check that the ABS warning light goes off.

      HINT:
      Drive the vehicle in reverse and check the speed sensor signal. Note that the signal check cannot be completed if the vehicle speed is 45 km/h (28 mph) or more.

      NOTICE:
      1. The sensor check may not be completed if wheelspin occurs.
      2. The ABS warning light blinks when the sensor check is completed and the brake pedal is depressed.
      3. The ABS warning light comes on immediately after a malfunction has been detected during the speed sensor check.

    5. Stop the vehicle.

      NOTICE:
      1. The speed sensor signal check may not be completed if the speed sensor signal check is started while turning the steering wheel or spinning the wheels.
      2. If the signal check is not completed, the ABS warning light blinks while driving and the ABS does not operate.

      HINT:
      When the signal check is completed, the ABS warning light goes off while driving and blinks in the test mode pattern while stationary.

  7. VSC Signal Check

    HINT:
    Check that the slip indicator light is blinking in the test mode blinking pattern (0.125 seconds on and 0.125 seconds off) before performing the VSC signal check.

  8. Downhill Assist Control Switch Check (w/ Downhill Assist Control)

    1. Push the downhill assist control switch and check that the downhill assist control indicator light is blinking.

    2. Turn the downhill assist control switch off.




  9. Crawl Control Switch Check (w/ Crawl Control)
    Text in Illustration 
    *1
    		Crawl Control Switch (Combination Switch)
    *2
    		ON/OFF Switch
    *3
    		Speed Selector Switch

    1. Push the ON/OFF switch and check that the crawl indicator light is on while the switch is being pushed.

    2. Turn the ON/OFF switch off.

    3. Set the target vehicle speed to L (low) with the speed selector switch.

    4. Set the target vehicle speed to medium-low with the speed selector switch.

    5. Set the target vehicle speed to M (medium) with the speed selector switch.

    6. Set the target vehicle speed to medium-high with the speed selector switch.

    7. Set the target vehicle speed to H (high) with the speed selector switch.

    8. Set the target vehicle speed to L (low) with the speed selector switch.

      HINT:
      The speed selector status is displayed on the multi-information display.

  10. VSC OFF Switch Check

    1. Press the VSC OFF switch.

    2. Check that the VSC OFF indicator light comes on.

    3. Press the VSC OFF switch again to turn the VSC OFF indicator light off.

  11. End of Signal Check

    1. If the signal check is completed, the ABS warning light and slip indicator light blink (0.125 seconds on and 0.125 seconds off) when the vehicle is stopped and goes off while driving the vehicle.

      NOTICE:
      If the signal check is not completed, the ABS warning light blinks while driving and the ABS does not operate.

  12. Read test mode (signal check) DTCs (when using the intelligent tester).

    1. Read the DTC(s) by following the intelligent tester screen.

      NOTICE:
      1. If only DTCs other than test mode signal check DTCs are output, repair the malfunctions and clear the DTCs.
      2. If test mode signal check DTCs and other DTCs are output or if only test mode sensor check DTCs are output, repair the malfunctions, clear the DTCs, and perform the test mode inspection again.

      HINT:
      Refer to Signal Check DTCs.

    2. Turn the engine switch off and disconnect the intelligent tester.



  13. Read test mode (signal check) DTCs (when using SST check wire).




    1. Using SST, connect terminals 13 (TC), 12 (TS) and 4 (CG) of the DLC3.

      SST
      09843-18040  
      Text in Illustration 
      *a
      		Front view of DLC3

    2. Read the number of blinks of the ABS warning light and slip indicator light.

      HINT:
      1. If every sensor is normal, the normal code is output (a cycle of 0.25 seconds on and 0.25 seconds off is repeated).
      2. If 2 or more malfunctions are detected at the same time, the lowest numbered DTC is output first.

    3. Disconnect SST from terminals 13 (TC), 12 (TS) and 4 (CG) of the DLC3.
