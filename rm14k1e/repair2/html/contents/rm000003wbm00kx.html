1. INSTALL MULTIPLEX NETWORK BODY ECU

NOTICE:
  1. Make sure that no foreign objects contact the connecting surfaces.
  2. Do not touch the ECU connector.




  1. Insert the multiplex network body ECU until it contacts the housing sidewall of the guide part as shown in the illustration.
    Text in Illustration 
    *1
    		Housing Sidewall
    *2
    		Guide Part

    HINT:
    Be sure to maintain an angle of 20° or more as shown in the illustration.




  1. Slide the guide of the multiplex network body ECU along the housing sidewall toward the junction block fuses as shown in the illustration.
    Text in Illustration 
    *1
    		Housing Sidewall
    *2
    		Guide Part
    *3
    		Junction Block Fuse




  1. Slide the multiplex network body ECU so that it contacts side A as shown in the illustration.
    Text in Illustration 
    *1
    		Side A

    NOTICE:
    Do not apply excessive force to the multiplex network body ECU.




  1. With the multiplex network body ECU to side A of the junction block (point of rotation), rotate it downward as shown in the illustration.
    Text in Illustration 
    *1
    		Side A Contact Portion




  1. Press the pushing area until the claw attaches to install the multiplex network body ECU.
    Text in Illustration 
    *1
    		Pushing Area

    NOTICE:
    1. Make sure to press only the pushing area.
    2. Confirm the engagement of the multiplex network body ECU and junction block by listening for a lock sound.
    3. Do not apply excessive force to the multiplex network body ECU.

    HINT:
    If a lock sound cannot be heard, visually check the engagement of the lock part. The engagement can also be confirmed if the multiplex network body ECU and junction block are flush.



2. INSTALL DRIVER SIDE JUNCTION BLOCK ASSEMBLY



  1. Attach the claw to install the connector as shown in the illustration.

    NOTICE:
    Be sure to connect each connector securely.




  1. Attach the 2 claws to lock the connector lock as shown in the illustration.




  1. Attach the claw to connect the connector as shown in the illustration.

    NOTICE:
    Be sure to connect the connector securely.

  1. Install the driver side junction block assembly with the bolt and 2 nuts.

    Torque:
    8.0 N*m{ 82 kgf*cm , 71 in.*lbf }




  1. Connect the connector labeled (1).

    NOTICE:
    Be sure to connect each connector securely.

  1. Attach the 2 claws to connect the 2 connectors labeled (2) as shown in the illustration.

    NOTICE:
    Be sure to connect each connector securely.

  1. Connect the 3 connectors.

    NOTICE:
    Be sure to connect each connector securely.



3. INSTALL LOWER NO. 1 INSTRUMENT PANEL AIRBAG ASSEMBLY



  1. Connect the connector.

    NOTICE:
    When handling the airbag connector, take care not to damage the airbag wire harness.

  1. Install the airbag assembly with the 4 bolts.

    Torque:
    10 N*m{ 102 kgf*cm , 7 ft.*lbf }



4. INSTALL LOWER INSTRUMENT PANEL FINISH PANEL SUB-ASSEMBLY



  1. Using a screwdriver, detach the 2 claws and open the cover.

    HINT:
    Tape the screwdriver tip before use.

    Text in Illustration 
    *1
    		Protective Tape




  1. Remove the 2 bolts <C>.

  1. Detach the 13 clips and 2 guides.

  1. Disconnect each connector and cable and remove the lower instrument panel finish panel.



5. INSTALL NO. 1 INSTRUMENT PANEL REGISTER ASSEMBLY

  1. Attach the 5 clips to install the No. 1 instrument panel register.



6. INSTALL LOWER INSTRUMENT PANEL FINISH PANEL ASSEMBLY

  1. Connect each connector and cooler thermistor.

  1. Attach the 4 clips to install the lower instrument panel finish panel.



7. INSTALL NO. 1 INSTRUMENT PANEL FINISH CUSHION

  1. Connect the connector.

  1. Attach the 4 clips to install the No. 1 instrument panel finish cushion.



8. INSTALL NO. 2 SWITCH HOLE BASE

  1. Connect each connector.

  1. Attach the 5 clips and claw to install the No. 2 switch hole base.



9. INSTALL INSTRUMENT SIDE PANEL LH

  1. Attach the 3 clips and 2 guides to install the instrument side panel.



10. INSTALL INSTRUMENT PANEL FINISH PANEL END LH

  1. Attach the 11 clips to install the instrument panel finish panel end.



11. INSTALL FRONT NO. 1 CONSOLE BOX INSERT

  1. Attach the 2 clips and guide to install the front No. 1 console box insert.



12. INSTALL NO. 1 INSTRUMENT PANEL UNDER COVER SUB-ASSEMBLY

  1. Connect the connector and attach the clamp.

  1. Attach the 2 clips and 2 guides to install the No. 1 instrument panel under cover.

  1. Install the screw.



13. INSTALL COWL SIDE TRIM BOARD LH

  1. Attach the clip and claw to install the cowl side trim board.

  1. Install the clip.



14. INSTALL DOOR SCUFF PLATE ASSEMBLY LH

  1. Connect the connector.

  1. Attach the 4 clips, 10 claws and 2 guides to install the door scuff plate.
