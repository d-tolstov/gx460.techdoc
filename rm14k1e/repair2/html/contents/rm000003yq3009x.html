CURTAIN SHIELD AIRBAG ASSEMBLY > INSTALLATION

for Preparation Click here


HINT:
  1. Use the same procedure for the RH and LH sides.
  2. The procedure listed below is for the LH side.



1. INSTALL CURTAIN SHIELD AIRBAG ASSEMBLY LH

  1. Install the curtain shield airbag with the 17 new bolts.

    Torque:
    9.8 N*m{ 100 kgf*cm , 87 in.*lbf }

    NOTICE:
    Do not twist the curtain shield airbag when installing it.

  1. Connect the connector.

    NOTICE:
    When handling the airbag connector, take care not to damage the airbag wire harness.



2. INSTALL NO. 1 AIR DUCT REAR SIDE

  1. Insert the upper part of the duct into the roof side duct and, using that point as a pivot point, swing the duct into place and install it to the rear cooling unit.

  1. Attach the 2 claws to install the cooler plate.



3. INSTALL NO. 2 AIR DUCT REAR SIDE

  1. Install the duct with the 2 clips and attach the clamp.



4. INSTALL ROOF HEADLINING ASSEMBLY

  1. Install the roof headlining (Click here).
