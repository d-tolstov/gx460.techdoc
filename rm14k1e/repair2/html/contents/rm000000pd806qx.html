CAUTION:
When using the intelligent tester, observe the following items for safety reasons:
  1. Read the intelligent tester instruction books before using the tester.
  2. Make sure that the tester cable is not caught on the pedals, shift lever and steering wheel when driving with the tester connected to the vehicle.
  3. When driving the vehicle for testing purposes while using the tester, 2 persons are required. One person drives the vehicle and the other operates the tester.

NOTICE:
  1. Perform initialization (throttle position) after replacing the throttle body assembly or cleaning any throttle body components (Click here).
  2. Perform initialization (throttle position) after reconnecting the battery cable or replacing the ECM (Click here).
  3. Perform Dynamic Laser Cruise Control System initialization after replacing the ECM (Click here).
  4. Perform Reset Memory (AT initialization) when replacing the automatic transmission assembly, engine assembly or ECM (Click here).

HINT:
Reset Memory cannot be completed by only disconnecting the battery.
