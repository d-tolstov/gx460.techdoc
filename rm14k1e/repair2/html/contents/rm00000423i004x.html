MANIFOLD ABSOLUTE PRESSURE SENSOR > INSTALLATION

for Preparation Click here



1. INSTALL MANIFOLD ABSOLUTE PRESSURE SENSOR

  1. Install a new O-ring to the manifold absolute pressure sensor.

  1. Install the manifold absolute pressure sensor with the bolt.

    Torque:
    9.0 N*m{ 92 kgf*cm , 80 in.*lbf }

  1. Connect the manifold absolute pressure sensor connector.



2. INSTALL V-BANK COVER SUB-ASSEMBLY



  1. Attach the 2 V-bank cover hooks to the No. 1 V-bank cover bracket. Then align the 2 V-bank cover grommets with the 2 pins and press down on the V-bank cover to attach the pins.
